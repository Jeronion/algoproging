
**Динамическое программирование (DP)** — это метод оптимизации вычислений, который позволяет эффективно решать задачи, разбивая их на подзадачи и используя результаты этих подзадач многократно.

---

## Основная идея

1. **Разбиение на подзадачи**: Сложная задача разбивается на более простые, часто повторяющиеся подзадачи.
2. **Запоминание результатов (Memoization / Таблица)**: Решения подзадач сохраняются, чтобы не вычислять их заново.
3. **Оптимальная структура**: DP работает для задач с:
	-  **Оптимальной подструктурой**: решение задачи можно получить из решений подзадач.
	-  **Перекрывающимися подзадачами**: подзадачи повторяются многократно.

---

## Подходы

### 1. Top-Down (С мемоизацией)

* Реализация через рекурсию.
* Используется словарь или массив для хранения уже вычисленных значений.
* Пример: вычисление чисел Фибоначчи.

```python
def fib(n, memo={}):
    if n in memo:
        return memo[n]
    if n <= 1:
        return n
    memo[n] = fib(n-1, memo) + fib(n-2, memo)
    return memo[n]

print(fib(10))  # 55
```

---

### 2. Bottom-Up (Табличный метод)

* Построение решения от наименьших подзадач к исходной задаче.
* Обычно используется массив для хранения промежуточных результатов.
* Пример: числа Фибоначчи через DP:

```python
def fib_bottom_up(n):
    if n <= 1:
        return n
    dp = [0] * (n + 1)
    dp[1] = 1
    for i in range(2, n+1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]

print(fib_bottom_up(10))  # 55
```

## Примеры задач, решаемых DP

1. **Фибоначчи**
2. **Задача о рюкзаке** (0/1 Knapsack)
3. **Минимальное количество монет для суммы**
4. **Редакционное расстояние** (Levenshtein distance)
5. **Путь с максимальной суммой в треугольнике**

## Советы по применению DP

1. **Определите подзадачи**: что нужно посчитать для меньших значений.
2. **Сформулируйте рекуррентное соотношение**.
3. **Выберите подход**: Top-Down или Bottom-Up.
4. **Проверяйте повторяющиеся подзадачи**: если они есть, DP подходит.
5. **Оптимизация памяти**: иногда можно хранить только несколько последних значений вместо всей таблицы.
