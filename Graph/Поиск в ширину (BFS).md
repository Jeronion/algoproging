# BFS (Поиск в ширину) — Полное руководство

BFS (Breadth-First Search, поиск в ширину) — это алгоритм обхода графа, который исследует вершины **уровень за уровнем**, начиная с выбранной стартовой вершины.

Главная идея: сначала посещаем все соседние вершины текущей вершины, затем их соседей и так далее.

BFS используется для:

- Поиска кратчайшего пути в невзвешенном графе
- Проверки связности графа
- Поиска минимального числа шагов (например, в лабиринтах)
- Построения дерева обхода

Граф можно представлять как:

- **Список смежности** — массив списков соседей
- **Матрица смежности** — `n x n` матрица ребер

---

## 1. Итеративная реализация BFS

BFS обычно реализуют через **очередь**, чтобы обработать вершины в порядке их открытия.

```python
from collections import deque

def bfs(graph, start):
    visited = set()
    queue = deque([start])
    
    while queue:
        vertex = queue.popleft()
        if vertex not in visited:
            visited.add(vertex)
            print(vertex, end=' ')
            # Добавляем соседей в очередь
            for neighbor in graph[vertex]:
                if neighbor not in visited:
                    queue.append(neighbor)
    
    return visited

# Пример графа
graph = {
    'A': ['B', 'C'],
    'B': ['A', 'D', 'E'],
    'C': ['A', 'F'],
    'D': ['B'],
    'E': ['B', 'F'],
    'F': ['C', 'E']
}

bfs(graph, 'A')
```

**Особенности BFS:**

- Обходит граф "по слоям"
- Использует очередь, чтобы хранить вершины для посещения
- Находит кратчайший путь в невзвешенном графе

## 2. Рекурсивная реализация BFS

Рекурсивный BFS встречается редко, потому что естественная структура BFS — очередь, а рекурсия лучше подходит для DFS.  
Тем не менее, можно реализовать рекурсивно с передачей очереди:

```python
from collections import deque

def bfs_recursive(graph, queue, visited=None):
    if visited is None:
        visited = set()
    
    if not queue:
        return visited
    
    vertex = queue.popleft()
    if vertex not in visited:
        visited.add(vertex)
        print(vertex, end=' ')
        for neighbor in graph[vertex]:
            if neighbor not in visited:
                queue.append(neighbor)
    
    return bfs_recursive(graph, queue, visited)

# Использование:
queue = deque(['A'])
bfs_recursive(graph, queue)
```

**Особенности рекурсивного BFS:**

- Редко используется на практике
- Ограничен глубиной рекурсии (stack limit)
- Можно использовать для учебных целей

---

## 3. Сравнение BFS и DFS

|Характеристика|DFS|BFS|
|---|---|---|
|Порядок обхода|Вглубь (по пути до конца)|Вширь (по уровням)|
|Используемая структура|Рекурсия или стек|Очередь|
|Применение|Поиск пути, топологическая сортировка|Кратчайший путь, поиск по слоям|
|Контроль стека/очереди|Стек вызовов или явный стек|Очередь|
|Риск переполнения|Рекурсивный DFS: да|Нет (использует очередь)|
