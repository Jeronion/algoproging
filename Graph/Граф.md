# Структура данных «Граф»

Граф — это одна из ключевых структур данных в информатике и математике, используемая для представления связей между объектами. Графы применяются в самых разных областях: от навигации и социальных сетей до компиляторов, биоинформатики и анализа данных.

## Основные понятия

Граф состоит из двух основных компонентов:

- **Вершины (узлы)** — объекты, которые представляют сущности системы
- **Рёбра (дуги)** — связи между вершинами

Формально граф обозначается как **G = (V, E)**, где:

- `V` — множество вершин
- `E` — множество рёбер

## Виды графов

### По направленности рёбер

- **Неориентированный граф** — рёбра не имеют направления (A — B)
- **Ориентированный граф (орграф)** — рёбра имеют направление (A → B)

### По наличию весов

- **Невзвешенный граф** — все рёбра равнозначны
- **Взвешенный граф** — каждому ребру сопоставлен вес (стоимость, расстояние и т.д.)

### По структуре

- **Связный граф** — между любой парой вершин существует путь
- **Несвязный граф** — граф состоит из нескольких компонент связности
- **Циклический граф** — содержит хотя бы один цикл
- **Ациклический граф** — не содержит циклов (например, дерево)
- **Полный граф** — каждая вершина соединена со всеми остальными

## Способы представления графов

### Матрица смежности

Это двумерный массив $N × N$, где $N$ — количество вершин.  
Элемент `A[i][j]` показывает наличие ребра между вершинами $i$ и $j$.

#### Реализация

```python
graph = [
    [0, 1, 1, 0],
    [0, 0, 1, 0],
    [1, 0, 0, 1],
    [0, 0, 0, 0]
]
```

**Плюсы:**

- Быстрый доступ к информации о наличии ребра
- Простота реализации

**Минусы:**

- Большой расход памяти (`O(N²)`)

### Список смежности

Каждой вершине сопоставляется список вершин, с которыми она соединена.

#### Реализация

```python
graph = {
    0: [1, 2],
    1: [2],
    2: [0, 3],
    3: []
}
```

**Плюсы:**

- Экономия памяти
- Удобен для разреженных графов

**Минусы:**

- Проверка наличия ребра может занимать больше времени

## Основные операции над графами

- Добавление и удаление вершин
- Добавление и удаление рёбер
- Обход графа
	- Поиск в глубину (DFS)
	- Поиск в ширину (BFS)
- Поиск путей и компонент связности
	- Алгоритм Дейкстра

## Области применения графов

- Социальные сети (пользователи и их связи)
- Карты и навигационные системы
- Компьютерные сети
- Анализ зависимостей в программировании
- Искусственный интеллект и машинное обучение

## Заключение

Графы — универсальная и мощная структура данных, позволяющая моделировать сложные взаимосвязи между объектами. Понимание их устройства и способов обработки является важной частью подготовки программиста и специалиста в области компьютерных наук.
