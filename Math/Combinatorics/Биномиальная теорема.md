
## Определение

**Биномиальная теорема** — это формула, позволяющая разложить выражение вида $(a + b)^n$ в сумму степеней $a$ и $b$ для любого неотрицательного целого $n$. Она играет ключевую роль в алгебре, комбинаторике, теории вероятностей и анализе алгоритмов.

---

## Формула

Для натурального $n$:

$$
(a + b)^n = \sum_{k=0}^{n} \binom{n}{k} a^{n-k} b^k
$$

где $\binom{n}{k} = \frac{n!}{k!(n-k)!}$ — **биномиальные коэффициенты**.

### Пример

$(x + 2)^3 = \binom{3}{0}x^3 + \binom{3}{1}x^2\cdot 2 + \binom{3}{2}x\cdot 2^2 + \binom{3}{3}2^3 = x^3 + 6x^2 + 12x + 8$.

---

## Свойства

1. **Число членов** — $n+1$.
2. **Симметрия коэффициентов**: $\binom{n}{k} = \binom{n}{n-k}$.
3. **Сумма коэффициентов**: $\sum_{k=0}^{n} \binom{n}{k} = 2^n$.
4. **Альтернирующая сумма**: $\sum_{k=0}^{n} (-1)^k \binom{n}{k} = 0$.
5. **Связь с треугольником Паскаля**: каждая строка треугольника даёт коэффициенты для $(a+b)^n$.

---

## Треугольник Паскаля

Первые строки:

```
n=0: 1
n=1: 1 1
n=2: 1 2 1
n=3: 1 3 3 1
n=4: 1 4 6 4 1
```

Каждое число равно сумме двух чисел над ним.

---

## Общий член разложения

$T_{k+1} = \binom{n}{k} a^{n-k} b^k$, где $k$ начинается с 0.

### Поиск конкретного члена

Если требуется найти член, содержащий $a^r b^{n-r}$, то это $T_{r+1}$.

#### Пример: найти коэффициент при $x^3$ в $(2x - 1)^5$.

Общий член: $T_{k+1} = \binom{5}{k} (2x)^{5-k}(-1)^k$.  
Нужно $5-k = 3 \Rightarrow k=2$.  
Коэффициент: $\binom{5}{2} 2^3 (-1)^2 = 10 \cdot 8 = 80$.

---

## Применения

- **Вычисление приближений** (например, $(1+x)^n$ при малых $x$).
- **Задачи на остатки** (например, найти остаток от деления $2^{100}$ на 7).
- **Подсчёт комбинаций** (число способов выбрать $k$ элементов из $n$).
- **Вероятностные распределения** (биномиальное распределение).

---

## Связь с комбинаторикой

Биномиальный коэффициент $\binom{n}{k}$ — это число способов выбрать $k$ объектов из $n$ без учёта порядка. Эта связь используется в задачах на сочетания.

---

## Вычисление на Python

```python
import math

def binomial_coeff(n, k):
    return math.comb(n, k)

def expand_binomial(a, b, n):
    terms = []
    for k in range(n+1):
        coeff = math.comb(n, k) * (a ** (n-k)) * (b ** k)
        terms.append(coeff)
    return terms

# Пример: коэффициенты (x + 2)^3
print(expand_binomial(1, 2, 3))  # [1, 6, 12, 8]
```
