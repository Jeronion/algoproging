
## Определение

**Биномиальное разложение** — это способ представления степени бинома (двучлена) в виде суммы членов, содержащих степени обоих слагаемых. Формула биномиального разложения позволяет раскрывать выражения вида $(x + a)^n$ для любых натуральных (и даже рациональных) показателей.

---

## Основные формулы

### 1. Для натурального показателя $n \in \mathbb{N}$

$$
(x + a)^n = \sum_{k=0}^{n} \binom{n}{k} x^{n-k} a^k
$$

где $\binom{n}{k} = \frac{n!}{k!(n-k)!}$ — биномиальный коэффициент.

В развёрнутом виде:

$$
(x + a)^n = \binom{n}{0}x^n a^0 + \binom{n}{1}x^{n-1}a^1 + \dots + \binom{n}{n}x^0 a^n
$$

**Пример:**  
$(x + 2)^3 = x^3 + 6x^2 + 12x + 8$.

### 2. Для рационального показателя (обобщение)

При $|x| < 1$ и произвольном $n \in \mathbb{R}$:

$$
(1 + x)^n = 1 + n x + \frac{n(n-1)}{2!}x^2 + \frac{n(n-1)(n-2)}{3!}x^3 + \dots
$$

Этот ряд сходится при $|x| < 1$ (и при $x=1$ для некоторых $n$).

---

## Характеристики биномиального разложения

- **Количество членов:** $n+1$ (для натурального $n$).
- **Сумма показателей:** в каждом члене сумма степеней $x$ и $a$ равна $n$.
- **Симметрия коэффициентов:** $\binom{n}{k} = \binom{n}{n-k}$.
- **Знакочередование:** $(x - a)^n = \sum_{k=0}^{n} (-1)^k \binom{n}{k} x^{n-k} a^k$.
- **Связь с треугольником Паскаля:** коэффициенты в строке $n$ треугольника Паскаля дают $\binom{n}{k}$.
- **Общий член:** $T_{k+1} = \binom{n}{k} x^{n-k} a^k$ (нумерация с 0).
- **Средний член:** при чётном $n$ — один средний член; при нечётном — два.

---

## Примеры

### Пример 1: Разложение $(2x - 3)^4$

$$
(2x - 3)^4 = \sum_{k=0}^{4} \binom{4}{k} (2x)^{4-k}(-3)^k
= 16x^4 - 96x^3 + 216x^2 - 216x + 81
$$

### Пример 2: Поиск конкретного члена

Найти коэффициент при $x^3$ в $(3x + 2)^5$.

Общий член: $T_{k+1} = \binom{5}{k}(3x)^{5-k}2^k$.  
Нужно $5-k = 3 \Rightarrow k = 2$.  
Коэффициент: $\binom{5}{2} \cdot 3^3 \cdot 2^2 = 10 \cdot 27 \cdot 4 = 1080$.

### Пример 3: Применение для приближений

Вычислить $(1.01)^{10}$ с помощью разложения (до 3 членов):

$$
(1 + 0.01)^{10} \approx 1 + 10\cdot 0.01 + \frac{10\cdot 9}{2} \cdot 0.0001 = 1 + 0.1 + 0.0045 = 1.1045
$$

---

## Связь с комбинаторикой

Биномиальный коэффициент $\binom{n}{k}$ — это число способов выбрать $k$ элементов из $n$ (порядок не важен). Это делает биномиальное разложение основой для многих комбинаторных задач.
