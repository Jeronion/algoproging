
## Определение

**Функция Эйлера** $\varphi(n)$ (или $\phi(n)$) для целого положительного числа $n$ — это количество целых чисел $k$ в диапазоне $1 \le k \le n$, которые **взаимно просты** с $n$ (т.е. $\gcd(k, n) = 1$). Другими словами, это количество чисел, не превосходящих $n$ и не имеющих общих делителей с $n$, кроме 1.

---

## Формула (произведение Эйлера)

Если $n$ имеет разложение на п ростые множители:
$$n = p_1^{e_1} \cdot p_2^{e_2} \cdot \ldots \cdot p_k^{e_k}$$
где $p_i$ — различные простые числа, то
$$\varphi(n) = n \cdot \left(1 - \frac{1}{p_1}\right) \cdot \left(1 - \frac{1}{p_2}\right) \cdot \ldots \cdot \left(1 - \frac{1}{p_k}\right)$$

Эквивалентная форма:
$$\varphi(n) = n \prod_{p \mid n} \left(1 - \frac{1}{p}\right)$$

**Пример:**
$n = 12 = 2^2 \cdot 3^1$
$$\varphi(12) = 12 \cdot \left(1 - \frac{1}{2}\right) \cdot \left(1 - \frac{1}{3}\right) = 12 \cdot \frac{1}{2} \cdot \frac{2}{3} = 4$$
(Числа, взаимно простые с 12: 1, 5, 7, 11 → их 4.)

---

## Идея решения (алгоритм)

Для вычисления $\varphi(n)$ достаточно найти все простые делители $n$ и применить формулу. Эффективный способ:

1. Инициализировать `result = n`.
2. Найти все простые делители $p$ числа $n$ (проверяя делимость на числа до $\sqrt{n}$).
3. Для каждого найденного простого делителя $p$:
   - Пока $n$ делится на $p$, делить $n$ на $p$ (убираем все вхождения).
   - Обновить `result -= result // p` (что эквивалентно умножению на $(1 - 1/p)$).
4. Если после цикла $n > 1$ (остался простой делитель больше $\sqrt{n}$), повторить шаг 3 для этого $n$.
5. Вернуть `result`.

Этот алгоритм работает за $O(\sqrt{n})$ и не требует предварительного нахождения всех простых чисел.

---

## Реализация на Python

```python
def phi(n):
    result = n
    p = 2
    # Ищем простые делители до sqrt(n)
    while p * p <= n:
        if n % p == 0:
            # Убираем все вхождения p
            while n % p == 0:
                n //= p
            # Применяем формулу
            result -= result // p
        p += 1
    # Если остался простой делитель > sqrt(original n)
    if n > 1:
        result -= result // n
    return result

# Пример
print(phi(11))  # 10
print(phi(16))  # 8
```

## Сложность

- **Время:** $O(\sqrt{n})$ — перебор до $\sqrt{n}$.
- **Память:** $O(1)$ — только несколько переменных.

---

## Свойства функции Эйлера

1. **Для простого $p$:** $\varphi(p) = p - 1$.
2. **Для степени простого $p^k$:** $\varphi(p^k) = p^k - p^{k-1}$.
3. **Мультипликативность:** если $\gcd(a, b) = 1$, то $\varphi(ab) = \varphi(a) \cdot \varphi(b)$.
4. **Сумма по делителям:** $\sum_{d \mid n} \varphi(d) = n$.
5. **Теорема Эйлера:** если $\gcd(a, n) = 1$, то $a^{\varphi(n)} \equiv 1 \pmod{n}$ (обобщение малой теоремы Ферма).

## Применение

- **Криптография:** RSA, алгоритм шифрования (основан на трудности факторизации и свойствах $\varphi$).
- **Теория чисел:** подсчёт взаимно простых чисел, решение сравнений.
- **Комбинаторика:** число примитивных корней, структура циклических групп.
