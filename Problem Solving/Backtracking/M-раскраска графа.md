## Определение

Дан неориентированный граф с `V` вершинами и списком рёбер `edges`, а также целое число `m`. Требуется определить, можно ли раскрасить вершины графа в **не более чем m цветов** так, чтобы любые две **смежные вершины** имели разные цвета.

**Пример:**
```
V = 4, edges = [[0,1],[1,3],[2,3],[3,0],[0,2]], m = 3 → true
V = 3, edges = [[0,1],[1,2],[0,2]], m = 2 → false (треугольник нельзя раскрасить в 2 цвета)
```

---

## Идея решения (backtracking)

Используется рекурсивный перебор с возвратом: последовательно назначаем цвет каждой вершине (от 0 до V-1). Для текущей вершины `node` пробуем все цвета от 0 до m-1. Если цвет не конфликтует с уже окрашенными соседями, назначаем его и переходим к следующей вершине. Если в какой-то момент ни один цвет не подходит, возвращаемся и пробуем другой цвет для предыдущей вершины.

### Проверка безопасности

Цвет безопасен для вершины, если ни один из её соседей в графе ещё не имеет этого цвета.

### Структуры данных

- `graph` – список смежности (словарь: вершина → список соседей).
- `colors` – массив длины V, где `colors[v] = -1` означает, что вершина ещё не раскрашена, иначе – номер цвета (0..m-1).

---

## Алгоритм

1. Построить список смежности `graph` по списку рёбер.
2. Создать массив `colors` длины V, заполненный `-1`.
3. Запустить рекурсивную функцию `backtrack(node)`:
   - Если `node == V`, то все вершины раскрашены → вернуть `True`.
   - Для цвета `c` от 0 до m-1:
     - Если `isSafe(node, c)` (ни один сосед не имеет цвета `c`):
       - Назначить `colors[node] = c`.
       - Вызвать `backtrack(node + 1)`. Если она вернула `True`, то возвращаем `True`.
       - Иначе сбросить цвет обратно на `-1` (откат).
   - Вернуть `False` (ни один цвет не подошёл).
4. Вернуть результат `backtrack(0)`.

---

## Реализация

```python
from collections import defaultdict

class Solution:
    def graphColoring(self, v, edges, m):
        graph = defaultdict(list)
        for a, b in edges:
            graph[a].append(b)
            graph[b].append(a)

        colors = [-1] * v

        def isSafe(node, color):
            for nei in graph[node]:
                if colors[nei] == color:
                    return False
            return True

        def backtrack(node=0):
            if node == v:
                return True
            for color in range(m):
                if isSafe(node, color):
                    colors[node] = color
                    if backtrack(node + 1):
                        return True
                    colors[node] = -1
            return False

        return backtrack()
```

## Сложность

- **Время:** O(m^V) в худшем случае (для полного графа). Однако благодаря проверкам отсечения и малому V (≤10) решение работает быстро.
- **Память:** O(V + E) для хранения графа и стека рекурсии.

---

## Примечания

- Задача эквивалентна проверке, является ли граф **m-раскрашиваемым**.
- Если `m ≥ V`, то ответ всегда `True` (каждой вершине можно дать уникальный цвет).
- Для `m = 2` задача сводится к проверке двудольности графа (можно решить BFS, но backtracking тоже подходит).
- Порядок перебора цветов влияет на скорость, но не на результат.
