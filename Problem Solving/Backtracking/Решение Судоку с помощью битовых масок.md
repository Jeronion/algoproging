## Определение

Дана частично заполненная доска судоку размером 9×9, где `0` обозначает пустую клетку. Требуется заполнить её цифрами от 1 до 9 так, чтобы в каждой строке, каждом столбце и каждом подквадрате 3×3 все цифры были различны. Гарантируется, что решение существует и единственно.

---

## Идея решения (backtracking + битовые маски)

Используется рекурсивный перебор с возвратом (backtracking). Для ускорения проверок и экономии памяти применяются **битовые маски** для строк, столбцов и подквадратов:

- `rows[i]` — битовая маска, где `bit x` = 1, если цифра `x+1` уже встречалась в строке `i`.
- `cols[j]` — аналогично для столбца `j`.
- `submat[idx]` — для подквадрата с индексом `idx = (i//3)*3 + (j//3)`.

### Процесс

1. **Инициализация** – проходим по всей доске и для каждой заполненной клетки (не 0) устанавливаем соответствующие биты в масках.
2. **Рекурсивный поиск** – функция `backtrack(i, j)`:
   - Если `i == 8` и `j == 9` → доска полностью заполнена → возвращаем `True`.
   - Если `j > 8` → переходим на следующую строку: `i += 1, j = 0`.
   - Если текущая клетка уже заполнена (`mat[i][j] != 0`) → переходим к следующей клетке.
   - Иначе пробуем все цифры `x` от 0 до 8 (соответствуют числам `x+1`):
     - Если биты в `rows[i]`, `cols[j]`, `submat[idx]` для `x` уже установлены → пропускаем.
     - Устанавливаем `mat[i][j] = x+1`, обновляем маски (устанавливаем бит).
     - Рекурсивно вызываем `backtrack(i, j+1)`. Если успешно → возвращаем `True`.
     - Иначе откатываем: обнуляем клетку и сбрасываем биты (XOR).
   - Если ни одна цифра не подошла → возвращаем `False`.

## Реализация

```python
class Solution:
    def solveSudoku(self, mat):
        rows = [0] * 9
        cols = [0] * 9
        submat = [0] * 9

        # Инициализация масок
        for i in range(9):
            for j in range(9):
                if mat[i][j] == 0:
                    continue
                x = mat[i][j] - 1
                rows[i] |= (1 << x)
                cols[j] |= (1 << x)
                submat[i // 3 * 3 + j // 3] |= (1 << x)

        def backtrack(i=0, j=0):
            if i == 8 and j == 9:
                return True
            if j == 9:
                i += 1
                j = 0
            if mat[i][j] != 0:
                return backtrack(i, j + 1)

            idx = i // 3 * 3 + j // 3
            for x in range(9):
                if rows[i] & (1 << x):
                    continue
                if cols[j] & (1 << x):
                    continue
                if submat[idx] & (1 << x):
                    continue
                # Ставим цифру
                mat[i][j] = x + 1
                rows[i] |= (1 << x)
                cols[j] |= (1 << x)
                submat[idx] |= (1 << x)

                if backtrack(i, j + 1):
                    return True

                # Откат
                mat[i][j] = 0
                rows[i] ^= (1 << x)
                cols[j] ^= (1 << x)
                submat[idx] ^= (1 << x)

            return False

        backtrack()
        return mat
```

## Сложность

- **Время:** В худшем случае экспоненциальное, но благодаря использованию битовых масок и отсечению работает быстро для 9×9.
- **Память:** O(1) дополнительно (маски и стек рекурсии глубиной до 81).

---

## Примечания

- Битовые маски вместо массивов `bool` ускоряют проверки и уменьшают объём копий.
- XOR используется для сброса бита, что эквивалентно `&= ~(1<<x)`, но проще в написании и откате.
- Подквадрат индексируется как `(i//3)*3 + j//3`.
- Алгоритм изменяет исходную матрицу на месте.
