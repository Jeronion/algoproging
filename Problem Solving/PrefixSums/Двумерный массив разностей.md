## Определение

**Двумерный массив разностей** — это техника для эффективного выполнения множества операций **прибавления значения ко всем элементам прямоугольной подматрицы** (range update). Вместо обновления каждой ячейки вложенными циклами (O(n·m) на операцию), используются маркеры в четырёх углах вспомогательной матрицы `diff`. После всех обновлений префиксные суммы восстанавливают конечную матрицу за O(n·m).

---

## Принцип работы

Для обновления подматрицы с верхним левым углом `(r1, c1)` и нижним правым `(r2, c2)` на значение `v` выполняются четыре операции в `diff`:

```
diff[r1][c1]     += v
diff[r1][c2+1]   -= v   (если c2+1 < m)
diff[r2+1][c1]   -= v   (если r2+1 < n)
diff[r2+1][c2+1] += v   (если обе границы в пределах)
```

После обработки всех запросов:
1. Вычисляются **префиксные суммы по строкам** (горизонтально).
2. Затем **по столбцам** (вертикально). Массив `diff` превращается в матрицу накопленных изменений.
3. Исходная матрица складывается с `diff` (если требуется модифицировать исходник).

## Реализация на Python

```python
def apply_range_updates(mat, updates):
    n = len(mat)
    m = len(mat[0])
    diff = [[0] * m for _ in range(n)]

    # 1. Применяем все обновления через 4 маркера
    for v, r1, c1, r2, c2 in updates:
        diff[r1][c1] += v
        if c2 + 1 < m:
            diff[r1][c2 + 1] -= v
        if r2 + 1 < n:
            diff[r2 + 1][c1] -= v
        if r2 + 1 < n and c2 + 1 < m:
            diff[r2 + 1][c2 + 1] += v

    # 2. Горизонтальные префиксные суммы
    for i in range(n):
        for j in range(1, m):
            diff[i][j] += diff[i][j - 1]

    # 3. Вертикальные префиксные суммы
    for j in range(m):
        for i in range(1, n):
            diff[i][j] += diff[i - 1][j]

    # 4. Применяем изменения к исходной матрице
    for i in range(n):
        for j in range(m):
            mat[i][j] += diff[i][j]
    return mat
```

## Пример

```python
mat = [
    [1, 2, 3],
    [1, 1, 0],
    [4, -2, 2]
]
updates = [
    [2, 0, 0, 1, 1],   # прибавить 2 к подматрице [(0,0)..(1,1)]
    [-1, 1, 0, 2, 2]   # прибавить -1 к подматрице [(1,0)..(2,2)]
]
result = apply_range_updates(mat, updates)
for row in result:
    print(" ".join(map(str, row)))
# Вывод:
# 3 4 3
# 2 2 -1
# 3 -3 1
```

---

## Сложность

- **Каждое обновление:** O(1)
- **Построение итоговой матрицы:** O(n·m)
- **Память:** O(n·m) для `diff`

---

## Применение

- Кратное изменение прямоугольных областей в матрицах (игры, редакторы изображений).
- Обработка большого количества запросов на прибавление константы к подматрице.
- Вычисление префиксных сумм после массовых обновлений.
