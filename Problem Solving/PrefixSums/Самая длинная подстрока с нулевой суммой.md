## Определение

Дан массив целых чисел (положительных и отрицательных). Требуется найти длину **самого длинного непрерывного подмассива**, сумма элементов которого равна нулю.

**Пример:**
```
arr = [15, -2, 2, -8, 1, 7, 10, 23] → ответ 5 (подмассив [-2, 2, -8, 1, 7])
arr = [1, 0, -4, 3, 1, 0] → ответ 5 ([0, -4, 3, 1, 0])
arr = [2, 10, 4] → ответ 0
```

---

## Идея решения (хеш-таблица + префиксные суммы)

Используем тех же наблюдение, что и для задачи проверки существования подмассива с нулевой суммой: если префиксные суммы `prefix[i]` и `prefix[j]` равны (при `i < j`), то сумма элементов между `i+1` и `j` равна нулю. Длина такого подмассива = `j - i`.

Чтобы найти максимальную длину, нужно для каждого значения префиксной суммы запоминать **самый ранний индекс**, где оно встретилось. При повторном появлении вычисляем длину и обновляем максимум.

Дополнительно учитываем случай, когда вся префиксная сумма становится нулевой (подмассив от начала). Для этого инициализируем словарь с ключом `0: -1`.

### Алгоритм

1. Создать словарь `first_seen`, инициализировать `{0: -1}`.
2. Инициализировать `prefix_sum = 0`, `max_len = 0`.
3. Пройти по массиву с индексом `i` от `0` до `n-1`:
   - `prefix_sum += arr[i]`
   - Если `prefix_sum` уже есть в `first_seen`:
     - `length = i - first_seen[prefix_sum]`
     - `max_len = max(max_len, length)`
   - Иначе:
     - `first_seen[prefix_sum] = i`
4. Вернуть `max_len`.

---

## Реализация

```python
class Solution:
    def maxLength(self, arr):
        first_seen = {0: -1}
        pref = 0
        max_len = 0
        for i, val in enumerate(arr):
            pref += val
            if pref in first_seen:
                max_len = max(max_len, i - first_seen[pref])
            else:
                first_seen[pref] = i
        return max_len
```

---

## Сложность

- **Время:** O(n) — один проход по массиву.
- **Память:** O(n) — в худшем случае словарь хранит все различные префиксные суммы.

---

## Примечания

- Если массив состоит только из ненулевых чисел и ни одна префиксная сумма не повторяется, ответ 0.
- Метод также работает для поиска подмассива с суммой, равной любому заданному `k` (тогда проверяем `pref - k` в словаре).
- Важно инициализировать словарь с `0: -1`, чтобы учитывать подмассивы, начинающиеся с индекса 0.
