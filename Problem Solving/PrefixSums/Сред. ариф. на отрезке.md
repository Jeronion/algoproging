## Определение

Задача состоит в том, чтобы для заданного массива `arr` и множества запросов `(l, r)` (индексы начала и конца отрезка, включительно) быстро вычислить среднее арифметическое элементов на этом отрезке.

Среднее арифметическое на отрезке `[l, r]` определяется как:

$$
\text{mean}(l, r) = \frac{1}{r-l+1} \sum_{i=l}^{r} arr[i]
$$

---

## Постановка задачи

Дан массив `arr` длины `N` и список `queries` из `Q` запросов, каждый из которых представляет собой пару индексов `(l, r)` (в 0-индексации). Требуется для каждого запроса вычислить среднее значение на отрезке. Обратите внимание, что индексы в запросах могут быть заданы по-разному (например, в условии примера кода используется 0-индексация, но в коде `pref` строится так, что `pref[i]` хранит сумму первых i элементов, поэтому для отрезка `[l, r]` сумма равна `pref[r+1] - pref[l]`).

---

## Решение с префиксными суммами

### Идея

Чтобы отвечать на множество запросов суммы на отрезке быстро, предварительно вычисляется массив префиксных сумм `pref`, где `pref[i]` — сумма первых `i` элементов (т.е. элементов с индексами `0` до `i-1`). Тогда сумма на отрезке `[l, r]` вычисляется за O(1) как `pref[r+1] - pref[l]`. После этого среднее получается делением на длину отрезка `(r - l + 1)`.

### Реализация на Python

```python
def findMean(arr, queries):
    N = len(arr)
    Q = len(queries)
    result = []
    
    # построение массива префиксных сумм (длина N+1)
    pref = [0] * (N + 1)
    for i in range(1, N + 1):
        pref[i] = pref[i - 1] + arr[i - 1]
    
    for i in range(Q):
        l = queries[i][0]
        r = queries[i][1]
        n = r - l + 1  # количество элементов на отрезке
        # сумма на отрезке [l, r] = pref[r+1] - pref[l]
        total = pref[r + 1] - pref[l]
        result.append(total / n)  # среднее
    
    return result
```

### Пример

Пусть `arr = [1, 2, 3, 4, 5]`, запросы `queries = [(0, 2), (1, 4), (2, 2)]`.

- Префиксные суммы: `pref = [0, 1, 3, 6, 10, 15]`.
- Для запроса (0,2): сумма = `pref[3] - pref[0] = 6 - 0 = 6`, длина 3 → среднее 2.0.
- Для (1,4): сумма = `pref[5] - pref[1] = 15 - 1 = 14`, длина 4 → среднее 3.5.
- Для (2,2): сумма = `pref[3] - pref[2] = 6 - 3 = 3`, длина 1 → среднее 3.0.

Результат: `[2.0, 3.5, 3.0]`.

## Сложность

- **Предподсчёт:** O(N) времени и O(N) памяти для префиксного массива.
- **Обработка одного запроса:** O(1) времени.
- **Итоговая сложность:** O(N + Q) времени и O(N) дополнительной памяти.

---

## Примечания

- Индексация в запросах должна быть согласована с построением префиксного массива. В приведённом коде используется 0-индексация элементов массива, а `pref` имеет длину `N+1`, где `pref[i]` — сумма первых i элементов.
- Если запросы используют 1-индексацию, формула суммы изменится на `pref[r] - pref[l-1]`.
- Среднее может быть нецелым, поэтому результат хранится как число с плавающей точкой (в Python это float).
- Задача является классическим примером применения префиксных сумм для обработки множественных запросов на отрезке.
