## Условие

Задача известна как [LeetCode 238. Product of Array Except Self](https://leetcode.com/problems/product-of-array-except-self/).

Дан массив целых чисел `nums`. Требуется вернуть массив `output` такой, что `output[i]` равен произведению всех элементов исходного массива, **кроме** `nums[i]`. Нельзя использовать операцию деления, и алгоритм должен работать за O(n) времени.

**Вход:** массив `nums` (длиной n)
**Выход:** массив `output` той же длины, где `output[i] = ∏(nums[j])` для всех j ≠ i.

**Пример:**
```
nums = [1, 2, 3, 4]
output = [24, 12, 8, 6]
```

---

## Примечания

- Запрет на использование деления мотивирован тем, что в массиве могут быть нули, что привело бы к ошибкам или дополнительным проверкам.
- Решение работает для любых целых чисел, включая отрицательные.
- В оптимизированной версии выходной массив не считается дополнительной памятью, так как он требуется по условию.

---

## Идея решения

Можно вычислить для каждого индекса произведение всех элементов слева от него и произведение всех элементов справа, а затем перемножить эти два значения. Это делается с помощью двух вспомогательных массивов:

- `left[i]` — произведение всех элементов `nums[0..i-1]` (для i=0 считаем равным 1)
- `right[i]` — произведение всех элементов `nums[i+1..n-1]` (для i=n-1 считаем равным 1)

Тогда `output[i] = left[i] * right[i]`.

---

### Алгоритм

1. Создаём массив `left` длины n, заполняем: `left[0] = 1`, для i от 1 до n-1: `left[i] = left[i-1] * nums[i-1]`.
2. Создаём массив `right` длины n, заполняем: `right[n-1] = 1`, для i от n-2 до 0: `right[i] = right[i+1] * nums[i+1]`.
3. Для каждого i: `output[i] = left[i] * right[i]`.

Этот подход использует O(n) дополнительной памяти (два массива). Можно улучшить до O(1), если вычислять `right` на лету и сразу заполнять `output`.

### Оптимизация памяти

Сначала заполняем `output` как массив левых произведений. Затем проходим справа налево, накапливая правое произведение и умножая его на текущий `output[i]`.

## Реализация

### Классический вариант (с двумя массивами)

```python
def productExceptSelf(nums):
    n = len(nums)
    left = [1] * n
    right = [1] * n
    output = [0] * n

    # левые произведения
    for i in range(1, n):
        left[i] = left[i-1] * nums[i-1]

    # правые произведения
    for i in range(n-2, -1, -1):
        right[i] = right[i+1] * nums[i+1]

    # итоговый результат
    for i in range(n):
        output[i] = left[i] * right[i]

    return output
```

### Оптимизированный вариант (без дополнительных массивов)

```python
def productExceptSelf(nums):
    n = len(nums)
    output = [1] * n

    # левые произведения сразу в output
    for i in range(1, n):
        output[i] = output[i-1] * nums[i-1]

    # правые произведения накапливаем в переменной
    right = 1
    for i in range(n-1, -1, -1):
        output[i] *= right
        right *= nums[i]

    return output
```

## Сложность

- **Время:** O(n) — три (или два) прохода по массиву.
- **Память:** O(1) дополнительно (кроме выходного массива) в оптимизированной версии; в классической — O(n).
