## Определение

Дана двумерная матрица `mat` размера `n × m`. Поступает несколько запросов, каждый из которых задаёт координаты прямоугольного подмассива: `(r1, c1)` — верхний левый угол, `(r2, c2)` — нижний правый угол (включительно). Нужно для каждого запроса вычислить сумму элементов внутри этого прямоугольника.

**Пример:**
```
mat = [[1, 2, 3],
       [1, 1, 0],
       [4, 2, 2]]
Запрос [0,0,1,1] → сумма элементов с (0,0) по (1,1): 1+2+1+1 = 5.
```

---

## Идея решения (2D префиксные суммы)

Строим вспомогательную матрицу `prefix`, где `prefix[i][j]` хранит сумму всех элементов исходной матрицы в прямоугольнике от `(0,0)` до `(i,j)` включительно.

Формула для построения:
```
prefix[i][j] = mat[i][j]
             + (prefix[i-1][j] если i > 0)
             + (prefix[i][j-1] если j > 0)
             - (prefix[i-1][j-1] если i>0 и j>0)
```
Вычитание необходимо, так как верхняя левая область учтена дважды.

Для ответа на запрос `(r1, c1, r2, c2)` сумма подматрицы равна:
```
total = prefix[r2][c2]
      - (prefix[r1-1][c2] если r1 > 0)
      - (prefix[r2][c1-1] если c1 > 0)
      + (prefix[r1-1][c1-1] если r1>0 и c1>0)
```

---

## Алгоритм

1. Построить массив `prefix` размером `n × m`:
   - Пройти по всем `i, j` и вычислить `prefix[i][j]` по формуле.
2. Для каждого запроса `(r1, c1, r2, c2)`:
   - Начать с `total = prefix[r2][c2]`.
   - Если `r1 > 0`, вычесть `prefix[r1-1][c2]`.
   - Если `c1 > 0`, вычесть `prefix[r2][c1-1]`.
   - Если `r1 > 0` и `c1 > 0`, добавить `prefix[r1-1][c1-1]` (компенсация двойного вычитания).
   - Добавить результат в ответ.
3. Вернуть список ответов.

---

## Реализация

```python
class Solution:
    def prefixSum2D(self, mat, queries):
        n, m = len(mat), len(mat[0])
        # строим 2D префиксные суммы
        prefix = [[0] * m for _ in range(n)]
        for i in range(n):
            for j in range(m):
                prefix[i][j] = mat[i][j]
                if i > 0:
                    prefix[i][j] += prefix[i-1][j]
                if j > 0:
                    prefix[i][j] += prefix[i][j-1]
                if i > 0 and j > 0:
                    prefix[i][j] -= prefix[i-1][j-1]
        
        res = []
        for r1, c1, r2, c2 in queries:
            total = prefix[r2][c2]
            if r1 > 0:
                total -= prefix[r1-1][c2]
            if c1 > 0:
                total -= prefix[r2][c1-1]
            if r1 > 0 and c1 > 0:
                total += prefix[r1-1][c1-1]
            res.append(total)
        return res
```

---

## Сложность

- **Построение префикса:** O(n·m)
- **Каждый запрос:** O(1)
- **Память:** O(n·m) для хранения `prefix` (можно использовать ту же матрицу, если разрешено изменять входные данные, но обычно создают отдельную).

---

## Примечания

- Этот метод эффективен при большом количестве запросов, так как предобработка выполняется один раз, а ответ на каждый запрос занимает O(1).
- Если элементы матрицы могут быть отрицательными, алгоритм работает корректно.
- При `n·m` до 10⁵ и `q` до 10⁵ метод вполне применим.
