## Определение

Дан массив целых чисел (положительных и отрицательных). Требуется определить, существует ли **непрерывный подмассив** (непустой), сумма элементов которого равна нулю.

**Примеры:**
```
arr = [4, 2, -3, 1, 6] → true (подмассив [2, -3, 1])
arr = [4, 2, 0, 1, 6] → true (отдельный ноль)
arr = [-3, 2, 3, 1, 6] → false
```

---

## Идея решения (хеш-таблица + префиксные суммы)

Наивный подход O(n²) перебирает все подмассивы — не подходит для больших массивов.

Оптимальное решение O(n) использует **префиксные суммы** и **хеш-таблицу** (множество) для хранения всех встреченных сумм.

**Ключевое наблюдение:** если для двух индексов `i` и `j` (`i < j`) префиксные суммы равны (`prefix[i] == prefix[j]`), то сумма элементов между `i+1` и `j` равна нулю. Также если какая-либо префиксная сумма равна 0, то подмассив от начала до этого индекса имеет нулевую сумму.

### Алгоритм

1. Инициализировать `prefix_sum = 0` и  множество `seen` c 0 внутри.
2. Пройти по массиву `arr`:
   - `prefix_sum += arr[i]`
   - Если `prefix_sum` уже есть в `seen`, вернуть `True`.
   - Иначе добавить `prefix_sum` в `seen`.
3. Вернуть `False`.

---

## Реализация на Python

```python
def subarray_zero_sum(arr):
    prefix_sum = 0
    seen = {0}
    for val in arr:
        prefix_sum += val
        if prefix_sum in seen:
            return True
        seen.add(prefix_sum)
    return False
```

---

## Сложность

- **Время:** O(n) — один проход по массиву.
- **Память:** O(n) — в худшем случае множество хранит все различные префиксные суммы.
