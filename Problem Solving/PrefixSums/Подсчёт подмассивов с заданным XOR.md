## Определение

Дан массив целых чисел `arr` и целое число `k`. Требуется подсчитать количество **непрерывных подмассивов**, XOR элементов которых равен `k`.

**Пример:**
```
arr = [4, 2, 2, 6, 4], k = 6 → ответ 4
Подмассивы: [4,2], [4,2,2,6,4], [2,2,6], [6]
```

---

## Идея решения (хеш-таблица + префиксный XOR)

Наивный подход O(n²) перебирает все подмассивы. Оптимальное решение O(n) использует **префиксный XOR** и **хеш-таблицу**.

**Ключевое наблюдение:** если для индексов `i` и `j` (`i < j`) префиксные XOR равны `px[i]` и `px[j]`, то XOR подмассива `[i+1, j]` равен `px[j] ^ px[i]`. Нам нужно, чтобы `px[j] ^ px[i] = k`, откуда `px[i] = px[j] ^ k`.

Для каждого `j` мы ищем количество предыдущих индексов `i`, у которых `px[i]` равно `px[j] ^ k`. Также учитываем случай, когда `px[j] == k` (подмассив от начала).

### Алгоритм

1. Инициализировать `pref_xor = 0`, `count = 0`, словарь `freq = {}`.
2. Для каждого элемента `val` в массиве:
   - `pref_xor ^= val`
   - Если `pref_xor == k`, увеличить `count` на 1.
   - Если `(pref_xor ^ k)` есть в `freq`, добавить `freq[pref_xor ^ k]` к `count`.
   - Увеличить `freq[pref_xor]` на 1.
3. Вернуть `count`.

---

## Реализация

```python
def subarray_xor_count(arr, k):
    freq = {}
    pref_xor = 0
    count = 0
    for val in arr:
        pref_xor ^= val
        if pref_xor == k:
            count += 1
        if (pref_xor ^ k) in freq:
            count += freq[pref_xor ^ k]
        freq[pref_xor] = freq.get(pref_xor, 0) + 1
    return count

# Пример
arr = [4, 2, 2, 6, 4]
k = 6
print(subarray_xor_count(arr, k))  # 4
```

---

## Сложность

- **Время:** O(n) — один проход по массиву.
- **Память:** O(n) — словарь для частот префиксных XOR.

---

## Примечания

- Используется свойство: `a ^ a = 0`, `a ^ 0 = a`, ассоциативность и коммутативность.
- Ключевая формула: если `pref[j] ^ pref[i] = k`, то `pref[i] = pref[j] ^ k`.
- Метод аналогичен подсчёту подмассивов с заданной суммой, но вместо сложения — XOR.
