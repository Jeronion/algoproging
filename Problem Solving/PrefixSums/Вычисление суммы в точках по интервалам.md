## Определение

Даны `n` точек, пронумерованных от `1` до `n`. Дан список интервалов, каждый из которых имеет вид `[l, r, count]`. Для каждого интервала к каждой точке с номером от `l` до `r` (включительно) прибавляется значение `count`. Требуется вычислить итоговую сумму для каждой точки после обработки всех интервалов.

**Пример:**
```
n = 5
intervals = [[1,2,9], [2,4,5], [3,5,7]]
Результат: [9, 14, 12, 12, 7]
```

---

## Идея решения (разностный массив)

Наивный подход — для каждого интервала пробегать по всем точкам от `l` до `r` и прибавлять `count`. Это O(m·n), где `m` — количество интервалов.

Оптимизированный подход использует **массив разностей (difference array)**:

1. Создаём массив `diff` длины `n+1`, инициализированный нулями.
2. Для каждого интервала `[l, r, count]`:
   - `diff[l-1] += count`   (так как точки нумеруются с 1, а индексы массива с 0)
   - если `r < n`, то `diff[r] -= count`
3. Затем проходим по массиву, накапливая префиксную сумму:
   - `result[0] = diff[0]`
   - `result[i] = result[i-1] + diff[i]` для `i = 1..n-1`
4. Массив `result` содержит итоговые суммы для каждой точки.

**Почему это работает?**  
Операция `diff[l-1] += count` означает, что с точки `l` начинается прибавление `count`, а `diff[r] -= count` — что после точки `r` это прибавление прекращается. При вычислении префиксной суммы накопленное значение распространяется на все последующие позиции, пока не встретится вычитание.

---

## Реализация на Python

```python
def interval_contribution(n, intervals):
    diff = [0] * (n + 1)   # дополнительная ячейка для удобства
    for l, r, cnt in intervals:
        diff[l - 1] += cnt
        if r < n:
            diff[r] -= cnt
    result = [0] * n
    result[0] = diff[0]
    for i in range(1, n):
        result[i] = result[i - 1] + diff[i]
    return result

# Пример
n = 5
intervals = [[1,2,9], [2,4,5], [3,5,7]]
print(interval_contribution(n, intervals))  # [9, 14, 12, 12, 7]
```

---

## Сложность

- **Время:** O(n + m), где `n` — количество точек, `m` — количество интервалов.
- **Память:** O(n) для массивов `diff` и `result`.

---

## Примечания

- Массив `diff` размера `n+1` позволяет обрабатывать случай `r = n`, не выходя за границы.
- Метод особенно эффективен при большом количестве интервалов (до 10⁶) и большом `n`.
- Это классическое применение разностного массива (дифференциального подхода) для задач с диапазонными прибавлениями.
