## Определение

Дан двумерный массив (матрица) `mat` размера `n × m` (целые числа, могут быть отрицательными). Требуется найти **прямоугольную подматрицу** (непрерывный блок строк и столбцов) с **максимальной суммой** элементов.

**Пример:**
```
mat = [[1, 2, -1, -4, -20],
       [-8, -3, 4, 2, 1],
       [3, 8, 10, 1, 3],
       [-4, -1, 1, 7, -6]]
Ответ: 29 (подматрица с верхним левым углом (1,2) и нижним правым (2,4)? на самом деле в примере координаты могут быть (1,2) to (2,3)? неважно, ответ 29).
```

---

## Идея решения (Kadane's Algorithm + сжатие столбцов)

Прямой перебор всех возможных подматриц потребовал бы O(n²·m²) времени. Эффективное решение использует **алгоритм Кадане для одномерного случая** и технику **фиксации левой и правой границы столбцов**.

1. Для каждой пары столбцов `(left, right)` (где `left` от 0 до `m-1`, `right` от `left` до `m-1`) мы строим одномерный массив `temp` длины `n`, где `temp[i]` = сумма элементов в строке `i` между столбцами `left` и `right` (включительно).
2. Затем применяем алгоритм Кадане к массиву `temp`, чтобы найти максимальную сумму подмассива. Эта сумма соответствует максимальной сумме подматрицы, ограниченной по горизонтали столбцами `left` и `right`, а по вертикали — некоторым интервалом строк (который находит Кадане).
3. Перебираем все `(left, right)` и оставляем глобальный максимум.

### Обоснование
Так как мы перебираем все возможные левые и правые границы столбцов, а для каждой такой пары находим наилучшие строки (вертикальный интервал) с помощью Кадане, то в итоге будет рассмотрена любая прямоугольная подматрица.

---

## Алгоритм

1. Инициализировать `res = -∞`.
2. Для `left` от 0 до `m-1`:
   - Создать массив `temp` длины `n`, заполненный нулями.
   - Для `right` от `left` до `m-1`:
     - Для каждой строки `i` от 0 до `n-1`: `temp[i] += mat[i][right]`.
     - Вызвать функцию `maxSubarraySum(temp)`, которая возвращает максимальную сумму подмассива (алгоритм Кадане).
     - Обновить `res = max(res, best)`.
3. Вернуть `res`.

---

## Реализация на Python

```python
class Solution:
    # Одномерный алгоритм Кадане
    def maxSubarraySum(self, arr):
        max_ending = arr[0]
        best = arr[0]
        for x in arr[1:]:
            max_ending = max(x, max_ending + x)
            best = max(best, max_ending)
        return best
    
    def maxRectSum(self, mat):
        n = len(mat)
        m = len(mat[0])
        res = float('-inf')
        
        for left in range(m):
            temp = [0] * n
            for right in range(left, m):
                for i in range(n):
                    temp[i] += mat[i][right]
                best = self.maxSubarraySum(temp)
                res = max(res, best)
        return res
```

## Сложность

- **Время:** O(n·m²) — для каждой пары столбцов (O(m²)) мы проходим по всем строкам (O(n)) для обновления `temp`, а также вызываем Кадане за O(n). Итого O(m²·n).
  - Если `n` и `m` до 300, то ~27·10⁶ операций, что приемлемо.
- **Память:** O(n) — для временного массива `temp`.

## Примечания

- Алгоритм работает для матриц с отрицательными числами.
- Если все элементы отрицательные, ответом будет максимальный (наименьший по модулю отрицательный) элемент (Кадане это корректно обрабатывает).
- Это стандартное решение задачи "Maximum Sum Rectangle" (аналогично LeetCode 363, но там есть ограничение по сумме, а здесь ищем просто максимум).
