## Определение

Даны два массива `L[]` и `R[]` одинаковой длины `n`, определяющие `n` интервалов `[L[i], R[i]]` (включительно). Требуется найти целое число, которое **встречается в наибольшем количестве интервалов**. Если таких чисел несколько, выбрать **наименьшее**.

**Пример:**
```
L = [1, 4, 3, 1], R = [15, 8, 5, 4]
Интервалы: [1,15], [4,8], [3,5], [1,4].
Число 4 встречается во всех четырёх интервалах → ответ 4.
```

---

## Идея решения (массив разностей)

Прямой перебор каждого числа (до 10⁶) и подсчёт вхождений перебором всех интервалов был бы слишком медленным (O(n·maxVal)). Используем **массив разностей (difference array)** для эффективного подсчёта частоты покрытия каждой позиции.

Создаём массив `diff` размера `max_val + 2`, где `max_val` — максимальное возможное значение (в задаче ограничено 10⁶). Для каждого интервала `[L[i], R[i]]`:
- Увеличиваем `diff[L[i]]` на 1.
- Уменьшаем `diff[R[i] + 1]` на 1 (если `R[i]+1` не выходит за границы).

После обработки всех интервалов проходим по массиву, накапливая префиксную сумму `cur`. На каждом шаге `cur` — это количество интервалов, покрывающих текущее число `i`. Отслеживаем максимальное значение и соответствующий индекс.

---

## Алгоритм

1. Определить максимальную возможную координату. В задаче значения до 10⁶, поэтому можно зафиксировать `MAX = 10⁶`.
2. Создать массив `diff` размера `MAX + 2`, заполненный нулями.
3. Для каждого `i` от `0` до `n-1`:
   - `diff[L[i]] += 1`
   - `diff[R[i] + 1] -= 1`   (если `R[i] + 1 ≤ MAX`)
4. Инициализировать `cur = 0`, `max_count = 0`, `answer = 0`.
5. Для `i` от `0` до `MAX`:
   - `cur += diff[i]`
   - Если `cur > max_count`:
     - `max_count = cur`
     - `answer = i`
   - (если `cur == max_count` — ничего не делаем, так как нужно наименьшее число, а мы идём по возрастанию, и первое встреченное с максимальным `cur` уже является наименьшим)
6. Вывести `answer`.

---

## Реализация на Python

```python
def max_occured_integer(L, R):
    MAX = 10**6
    diff = [0] * (MAX + 2)
    for l, r in zip(L, R):
        diff[l] += 1
        diff[r + 1] -= 1
    cur = 0
    max_count = 0
    ans = 0
    for i in range(MAX + 1):
        cur += diff[i]
        if cur > max_count:
            max_count = cur
            ans = i
    return ans
```

---

## Сложность

- **Время:** O(n + MAX), где n — количество интервалов, MAX = 1 000 000 (константа). Для n ≤ 10⁶ это допустимо.
- **Память:** O(MAX) — массив размера около 1 000 002 элементов.

---

## Примечания

- Если максимальное значение `R[i]` меньше глобального максимума, можно вычислить точный `maxR = max(R)` и ограничить массив до `maxR+2`, что немного сократит память.
- Массив разностей гарантирует, что после накопления префиксной суммы `cur` даст точное количество покрытий для каждого целого числа в диапазоне.
- Метод эффективен и широко применяется для задач массового наложения интервалов.
