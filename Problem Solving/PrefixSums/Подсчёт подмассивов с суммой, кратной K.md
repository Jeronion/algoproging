## Определение

Дан массив целых чисел `arr` и целое число `k`. Требуется подсчитать количество **непрерывных подмассивов**, сумма элементов которых **делится на `k`** (т.е. `sum % k == 0`).

**Пример:**
```
arr = [4, 5, 0, -2, -3, 1], k = 5 → ответ 7
```

---

## Идея решения (префиксные суммы по модулю + хеш-таблица)

Наивный подход O(n²) перебирает все подмассивы. Оптимальное решение O(n) использует **префиксные суммы** и их **остатки от деления на k**.

**Ключевое наблюдение:** если для индексов `i` и `j` (`i < j`) остатки от деления префиксных сумм на `k` равны, то сумма подмассива между `i+1` и `j` делится на `k`. А именно, если `prefix[j] % k == prefix[i] % k`, то `(prefix[j] - prefix[i]) % k == 0`.

Также подмассив, начинающийся с индекса 0 и заканчивающийся в `i`, подходит, если `prefix[i] % k == 0`.

### Алгоритм

1. Инициализировать `prefix_mod = 0`, словарь `count` для хранения количества вхождений каждого остатка, `result = 0`.
2. Пройти по массиву с индексом `i` от `0` до `n-1`:
   - `prefix_mod = (prefix_mod + arr[i]) % k` (в Python `%` даёт неотрицательный остаток).
   - Если `prefix_mod == 0`, увеличить `result` на 1 (подмассив от 0 до `i`).
   - Добавить к `result` количество предыдущих вхождений этого остатка (из `count`).
   - Увеличить `count[prefix_mod]` на 1.
3. Вернуть `result`.

---

## Реализация

```python
from collections import defaultdict

def subarrays_divisible_by_k(arr, k):
    count = defaultdict(int)
    prefix_mod = 0
    result = 0
    for val in arr:
        prefix_mod = (prefix_mod + val) % k
        if prefix_mod == 0:
            result += 1
        result += count[prefix_mod]
        count[prefix_mod] += 1
    return result

# Пример
arr = [4, 5, 0, -2, -3, 1]
k = 5
print(subarrays_divisible_by_k(arr, k))  # 7
```

---

## Сложность

- **Время:** O(n) — один проход по массиву.
- **Память:** O(min(n, k)) — словарь хранит остатки (их не более k).

---

## Примечания

- В Python оператор `%` для отрицательных чисел возвращает неотрицательный остаток, что удобно для этой задачи (в отличие от некоторых других языков, где требуется дополнительная корректировка).
- Количество подмассивов, оканчивающихся в текущей позиции, равно числу ранее встреченных одинаковых остатков.
- Если требуется подсчитывать только ненулевые длины подмассивов, алгоритм уже это делает (подмассив длины 1 учитывается, только если его элемент кратен k).
