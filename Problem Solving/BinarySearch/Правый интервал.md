## Определение

Для каждого интервала `[start, end]` найти интервал с минимальным `start'` ≥ `end`. Вернуть массив индексов.

---

## Реализация (LeetCode 436)

```python
def findRightInterval(intervals: List[List[int]]) -> List[int]:
    sorted_intervals = sorted(enumerate(intervals), key=lambda x: x[1][0])
    result = [-1] * len(intervals)
    for i, (_, (start, end)) in enumerate(sorted_intervals):
        left, right = 0, len(intervals) - 1
        while left <= right:
            mid = (left + right) // 2
            if sorted_intervals[mid][1][0] >= end:
                right = mid - 1
            else:
                left = mid + 1
        if left < len(intervals):
            result[i] = sorted_intervals[left][0]
    return result
```

## Сложность

- Время: O(n log n)
- Память: O(n)
