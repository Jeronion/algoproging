## Определение

Если искомая величина монотонна, можно бинарно искать её значение, а для проверки использовать отдельную функцию. Часто применяется в задачах на минимизацию/максимизацию.

## 1. Прямоугольники (solution_1923)

Даны размеры прямоугольника `w` и `h` и количество квадратов `n`. Нужно найти минимальную сторону квадрата, в который можно разместить `n` прямоугольников. Проверка: в квадрате со стороной `x` помещается `(x // w) * (x // h)` прямоугольников.

```python
def minimal_square(w, h, n):
    left = max(w, h)
    right = max(w * n, h * n)
    while left <= right:
        mid = (left + right) // 2
        if (mid // w) * (mid // h) >= n:
            right = mid - 1
        else:
            left = mid + 1
    return left
```

## 2. Агрессивные коровы (solution_1)

Найти максимальное минимальное расстояние между коровами на координатной прямой. Проверка: можно ли разместить `K` коров с расстоянием не менее `x`.

```python
def can_place(arr, dist, k):
    count = 1
    last = arr[0]
    for pos in arr[1:]:
        if pos - last >= dist:
            count += 1
            last = pos
    return count >= k

def max_min_distance(arr, k):
    arr.sort()
    left, right = 1, arr[-1] - arr[0]
    while left <= right:
        mid = (left + right) // 2
        if can_place(arr, mid, k):
            left = mid + 1
        else:
            right = mid - 1
    return right
```

## 3. Монетки (arrangeCoins, LeetCode 441)

Найти максимальное количество полных рядов пирамиды из `n` монет.

```python
def arrangeCoins(n: int) -> int:
    left, right = 0, n
    while left <= right:
        mid = (left + right) // 2
        if (mid + 1) * mid // 2 > n:
            right = mid - 1
        else:
            left = mid + 1
    return right
```
