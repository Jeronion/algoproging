## Определение

Дан массив, который сначала строго возрастает, затем строго убывает. Требуется найти индекс пикового элемента.

---

## Реализация (LeetCode 852)

```python
def peakIndexInMountainArray(arr: List[int]) -> int:
    left, right = 0, len(arr) - 1
    while left < right:
        mid = (left + right) // 2
        if arr[mid] < arr[mid + 1]:
            left = mid + 1
        else:
            right = mid
    return left
```

---

## Сложность

- Время: O(log n)
- Память: O(1)
