## Определение

Для каждого заклинания найти количество зелий, которые в паре дают произведение ≥ success. Зелья можно отсортировать, а для каждого заклинания выполнить бинарный поиск.

---

## Реализация (LeetCode 2300)

```python
def successfulPairs(spells: List[int], potions: List[int], success: int) -> List[int]:
    potions.sort()
    res = []
    for spell in spells:
        left, right = 0, len(potions) - 1
        while left <= right:
            mid = (left + right) // 2
            if potions[mid] * spell >= success:
                right = mid - 1
            else:
                left = mid + 1
        # left — первый индекс, где условие выполняется
        res.append(len(potions) - left)
    return res
```

---

## Сложность

- Время: O(m log n), где m = len(spells), n = len(potions)
- Память: O(1) дополнительно
