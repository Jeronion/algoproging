## Определение

В отсортированном массиве с повторяющимися элементами требуется найти первый и последний индекс вхождения заданного элемента. Классический приём — выполнить два бинарных поиска: один для левой границы (первое вхождение), другой для правой (последнее).

---

## Постановка задачи (LeetCode 34)

Дан отсортированный массив целых чисел `nums` и целое `target`. Вернуть `[left, right]` — индексы первого и последнего вхождения `target`. Если `target` отсутствует, вернуть `[-1, -1]`.

---

## Реализация

```python
def searchRange(nums: List[int], target: int) -> List[int]:
    # ищем левую границу
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] >= target:
            right = mid - 1
        else:
            left = mid + 1
    # left — первый индекс, где nums[left] >= target
    if left == len(nums) or nums[left] != target:
        return [-1, -1]

    # ищем правую границу
    l, r = 0, len(nums) - 1
    while l <= r:
        mid = (l + r) // 2
        if nums[mid] <= target:
            l = mid + 1
        else:
            r = mid - 1
    # l — первый индекс, где nums[l] > target
    return [left, l - 1]
```

---

## Сложность

- Время: O(log n)
- Память: O(1)
