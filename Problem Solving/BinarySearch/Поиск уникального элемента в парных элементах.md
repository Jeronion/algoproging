## Определение

В отсортированном массиве, где каждый элемент встречается дважды, кроме одного, найти этот уникальный элемент. Используется бинарный поиск с проверкой чётности индекса.

---

## Реализация (LeetCode 540)

```python
def singleNonDuplicate(nums: List[int]) -> int:
    left, right = 0, len(nums) - 1
    while left < right:
        mid = (left + right) // 2
        if mid % 2 == 1:
            mid -= 1
        if nums[mid] == nums[mid + 1]:
            left = mid + 2
        else:
            right = mid
    return nums[left]
```

---

## Сложность

- Время: O(log n)
- Память: O(1)
