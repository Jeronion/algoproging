## Определение

Квадратная матрица называется **симметричной**, если элементы, симметричные относительно главной диагонали, равны: `a[i][j] == a[j][i]` для всех i, j.

## Реализация

```python
def is_symmetric_matrix():
    N = int(input())
    grid = [list(map(int, input().split())) for _ in range(N)]
    for i in range(N):
        for j in range(i + 1, N):
            if grid[i][j] != grid[j][i]:
                return "NO"
    return "YES"
```

## Сложность

- **Время:** O(N²) — проход по верхней треугольной части.
- **Память:** O(N²) для хранения матрицы.
