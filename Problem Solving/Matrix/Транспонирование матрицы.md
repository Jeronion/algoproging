## Определение

**Транспонирование матрицы** — это операция, при которой строки исходной матрицы становятся столбцами, а столбцы — строками. Формально, для матрицы `mat` размера `n × m` её транспонированная версия `tMat` имеет размер `m × n`, где `tMat[j][i] = mat[i][j]`.

---

## Два подхода

### 1. Для прямоугольных матриц (с дополнительной памятью)

Создаётся новая матрица размером `m × n`, куда копируются элементы с заменой индексов.

```python
def transpose(mat):
    rows, cols = len(mat), len(mat[0])
    tMat = [[0] * rows for _ in range(cols)]
    for i in range(rows):
        for j in range(cols):
            tMat[j][i] = mat[i][j]
    return tMat
```

**Сложность:**
- Время: O(n·m)
- Память: O(n·m)

### 2. Для квадратных матриц (in-place, без дополнительной памяти)

Обмениваются элементы относительно главной диагонали. Достаточно пройти только по верхней треугольной части (i < j).

```python
def transpose_square(mat):
    n = len(mat)
    for i in range(n):
        for j in range(i + 1, n):
            mat[i][j], mat[j][i] = mat[j][i], mat[i][j]
    return mat
```

**Сложность:**
- Время: O(n²)
- Память: O(1)

---

## Примечания

- Для прямоугольной матрицы in-place транспонирование невозможно без дополнительной памяти, так как размер матрицы меняется.
- Алгоритм для квадратных матриц использует только обмен элементов, что экономит память.
- Транспонирование часто применяется в линейной алгебре, обработке изображений, машинном обучении.
