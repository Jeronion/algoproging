## Определение

Дан двумерный массив (матрица) размера `m × n`. Требуется вывести все его элементы в **спиральном порядке** — начиная с левого верхнего угла, двигаясь вправо по первой строке, затем вниз по последнему столбцу, затем влево по последней строке, затем вверх по первому столбцу, и так далее, постепенно сужая границы.

---

## Идея решения (Boundary Traversal)

Вместо имитации движения с отслеживанием посещённых ячеек можно использовать **границы (boundaries)**. Мы поддерживаем четыре переменные: `top`, `bottom`, `left`, `right`, которые обозначают текущие границы ещё не напечатанной части матрицы.

На каждом шаге:

1. Печатаем верхнюю строку от `left` до `right`, затем увеличиваем `top`.
2. Печатаем правый столбец от `top` до `bottom`, затем уменьшаем `right`.
3. Если `top <= bottom`, печатаем нижнюю строку от `right` до `left`, затем уменьшаем `bottom`.
4. Если `left <= right`, печатаем левый столбец от `bottom` до `top`, затем увеличиваем `left`.

Процесс повторяется, пока `top <= bottom` и `left <= right`.

---

## Алгоритм

1. Инициализировать `top = 0`, `bottom = m-1`, `left = 0`, `right = n-1`.
2. Пока `top <= bottom` и `left <= right`:
   - Пройти по `i` от `left` до `right`: добавить `mat[top][i]`.
   - `top += 1`.
   - Пройти по `i` от `top` до `bottom`: добавить `mat[i][right]`.
   - `right -= 1`.
   - Если `top <= bottom`:
     - Пройти по `i` от `right` до `left` (в обратном порядке): добавить `mat[bottom][i]`.
     - `bottom -= 1`.
   - Если `left <= right`:
     - Пройти по `i` от `bottom` до `top` (в обратном порядке): добавить `mat[i][left]`.
     - `left += 1`.
3. Вернуть собранный список.

## Реализация на Python

```python
def spiral_traversal(mat):
    if not mat or not mat[0]:
        return []
    
    m, n = len(mat), len(mat[0])
    res = []
    top, bottom = 0, m - 1
    left, right = 0, n - 1

    while top <= bottom and left <= right:
        # верхняя строка
        for i in range(left, right + 1):
            res.append(mat[top][i])
        top += 1

        # правый столбец
        for i in range(top, bottom + 1):
            res.append(mat[i][right])
        right -= 1

        # нижняя строка (если есть)
        if top <= bottom:
            for i in range(right, left - 1, -1):
                res.append(mat[bottom][i])
            bottom -= 1

        # левый столбец (если есть)
        if left <= right:
            for i in range(bottom, top - 1, -1):
                res.append(mat[i][left])
            left += 1

    return res
```

---

## Сложность

- **Время:** O(m·n) — каждый элемент посещается ровно один раз.
- **Память:** O(1) дополнительно (не считая памяти для результата).

---

## Примечания

- Алгоритм работает для матриц любого размера, в том числе прямоугольных.
- Важно проверять условия `if top <= bottom` и `if left <= right` после сдвига границ, чтобы не выйти за пределы при печати нижней строки или левого столбца.
- Этот метод считается оптимальным и наиболее часто используется на собеседованиях.
