## Определение

Задача Иосифа (Josephus problem) — классическая задача: в кругу стоят `N` человек (пронумерованных от 1 до N). Начинается отсчёт, и каждый `K`-й удаляется из круга. Требуется вывести порядок удаления.

Задача с сайта informatics.msk.ru (задача №112506 из раздела «Стеки, очереди, деки»).

---

## Постановка задачи

**Вход:** два целых числа `N` и `K` (1 ≤ N, K ≤ 1000).  
**Выход:** последовательность номеров удаляемых людей в порядке их исключения.

Пример:
```
N = 7, K = 3
Вывод: 3 6 2 7 5 1 4
```

---

## Идея решения

Создаём циклический связный список из элементов 1..N. Начинаем с хвоста (предыдущего перед головой). Для каждого удаления делаем `K-1` шагов по кругу, затем удаляем следующий узел и выводим его значение. Повторяем, пока не удалим все.

---

## Реализация

```python
def josephus_problem():
    N, K = map(int, input().split())
    # создаём циклический список
    head = ListNode(1)
    tail = head
    for i in range(2, N + 1):
        tail.next = ListNode(i)
        tail = tail.next
    tail.next = head  # замыкаем

    # теперь tail указывает на голову (1), а head остаётся головой
    # но для удобства будем использовать tail как предыдущий перед удаляемым
    # сделаем tail = узел перед head (это последний элемент, т.к. список зациклен)
    # на самом деле после замыкания tail указывает на последний элемент (N), а head на 1
    # поэтому всё готово

    for _ in range(N):
        for _ in range(K - 1):
            tail = tail.next
        print(tail.next.val, end=" ")
        tail.next = tail.next.next
```

В коде пользователя `tail` изначально указывает на последний узел (после создания списка). После замыкания `tail.next == head`. Внутренний цикл `for _ in range(K-1)` сдвигает `tail` на `K-1` шаг, затем `tail.next` — удаляемый узел.

---

## Сложность

- **Время:** O(N·K) — если N и K до 1000, то достаточно.
- **Память:** O(N) для хранения списка.

---

## Примечания

- Можно решить и математически, но реализация на списке наглядна.
- Циклический список позволяет избежать проверок на конец.
