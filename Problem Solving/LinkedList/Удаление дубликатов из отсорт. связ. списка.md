## Определение

Дан отсортированный по возрастанию связный список. Удалить дубликаты так, чтобы каждый элемент встречался только один раз, и вернуть голову изменённого списка.

Задача известна как [LeetCode 83. Remove Duplicates from Sorted List](https://leetcode.com/problems/remove-duplicates-from-sorted-list/).

---

## Постановка задачи

**Вход:** голова отсортированного списка.  
**Выход:** новая голова без повторяющихся элементов (остаётся по одному).

Пример:
```
head = [1,1,2,3,3]
Результат: [1,2,3]
```

---

## Идея решения

Проходим по списку одним указателем. Если текущий узел и следующий имеют одинаковые значения, пропускаем следующий узел: `head.next = head.next.next`. Иначе сдвигаемся вперёд.

---

## Реализация

```python
def deleteDuplicates(head: Optional[ListNode]) -> Optional[ListNode]:
    current = head
    while current and current.next:
        if current.val == current.next.val:
            current.next = current.next.next
        else:
            current = current.next
    return head
```

---

## Сложность

- **Время:** O(n)
- **Память:** O(1)

---

## Примечания

- В отличие от предыдущей задачи, удаляются только лишние копии, но первый экземпляр остаётся.
