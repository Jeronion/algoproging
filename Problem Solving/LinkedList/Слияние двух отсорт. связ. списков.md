## Определение

Даны два отсортированных по возрастанию связных списка. Требуется слить их в один отсортированный список и вернуть его голову.

Задача известна как [LeetCode 21. Merge Two Sorted Lists](https://leetcode.com/problems/merge-two-sorted-lists/).

---

## Постановка задачи

**Вход:** головы двух отсортированных списков `list1` и `list2`.  
**Выход:** голова объединённого отсортированного списка.

Пример:
```
list1 = [1,2,4]
list2 = [1,3,4]
Результат: [1,1,2,3,4,4]
```

---

## Идея решения

Создаём фиктивную голову `dummy` и указатель `current`. Пока оба списка не пусты, сравниваем значения головных узлов, меньший добавляем в результат и сдвигаем соответствующий указатель. После выхода из цикла прицепляем оставшийся хвост.

---

## Реализация

```python
def mergeTwoLists(list1: Optional[ListNode], list2: Optional[ListNode]) -> Optional[ListNode]:
    dummy = ListNode()
    current = dummy
    while list1 and list2:
        if list1.val < list2.val:
            current.next = list1
            list1 = list1.next
        else:
            current.next = list2
            list2 = list2.next
        current = current.next
    current.next = list1 if list1 else list2
    return dummy.next
```

---

## Сложность

- **Время:** O(m + n)
- **Память:** O(1) (без учёта результата)
