## Определение

Дан односвязный список. Требуется найти значение **среднего узла**.

- Если количество узлов **нечётное** – вернуть значение единственного среднего узла.
- Если **чётное** – вернуть значение **второго** среднего узла (т.е. первый из правой половины).

**Пример:**
```
1 → 2 → 3 → 4 → 5 → None  → 3
2 → 4 → 6 → 7 → 5 → 1 → None → 7
```

---

## Идея решения (быстрый и медленный указатели)

Используем классический метод **tortoise and hare**:
- `slow` каждый шаг сдвигается на один узел.
- `fast` каждый шаг сдвигается на два узла.
- Когда `fast` достигает конца списка, `slow` оказывается ровно на среднем узле (второй средней позиции для чётной длины).

### Алгоритм

1. Инициализировать `slow = head`, `fast = head`.
2. Пока `fast` и `fast.next` не равны `None`:
   - `slow = slow.next`
   - `fast = fast.next.next`
3. Вернуть `slow.data`.

---

## Реализация

```python
class Solution:
    def getMiddle(self, head):
        slow = fast = head
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
        return slow.data
```

---

## Сложность

- **Время:** O(n) – один проход по списку.
- **Память:** O(1) – только два указателя.

---

## Примечания

- Метод работает для списков любой длины (в том числе пустых? В условии гарантируется хотя бы один узел).
- Для чётной длины `slow` останавливается на втором из двух средних узлов, что соответствует условию.
- Альтернативные решения: подсчёт длины с последующим проходом до `n//2` (требует двух проходов).
