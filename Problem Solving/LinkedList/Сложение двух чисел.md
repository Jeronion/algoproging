## Определение

Даны два непустых связных списка, представляющих два неотрицательных целых числа. Цифры хранятся в **обратном порядке** (т.е. первый узел — младший разряд). Требуется сложить эти числа и вернуть результат в виде связного списка (также в обратном порядке). Числа не содержат ведущих нулей, за исключением самого нуля.

Задача известна как [LeetCode 2. Add Two Numbers](https://leetcode.com/problems/add-two-numbers/).

---

## Постановка задачи

**Вход:** два связных списка `l1` и `l2`, каждый узел содержит одну цифру (0–9).  
**Выход:** новый связный список, представляющий сумму чисел.

Пример:
```
l1 = [2,4,3] (число 342)
l2 = [5,6,4] (число 465)
Результат: [7,0,8] (807)
```

---

## Идея решения

Используется моделирование сложения в столбик. Перебираем оба списка одновременно, поддерживая переменную `carry` (перенос). На каждом шаге:
- Берём значения из текущих узлов (если есть) и добавляем к `carry`.
- Создаём новый узел со значением `carry % 10`.
- Обновляем `carry = carry // 10`.
- Переходим к следующим узлам.

Цикл продолжается, пока есть хотя бы один список или ненулевой `carry`.

---

## Реализация

```python
def addTwoNumbers(l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
    dummy = ListNode()
    head = dummy
    carry = 0
    while l1 or l2 or carry:
        if l1:
            carry += l1.val
            l1 = l1.next
        if l2:
            carry += l2.val
            l2 = l2.next
        dummy.next = ListNode(carry % 10)
        dummy = dummy.next
        carry //= 10
    return head.next
```

---

## Сложность

- **Время:** O(max(m, n)), где m, n — длины списков.
- **Память:** O(max(m, n)) для нового списка.

---

## Примечания

- В отличие от наивного преобразования в число, этот метод работает с числами любой длины, не выходя за пределы целочисленных типов.
- Списки передаются уже в обратном порядке, что упрощает сложение.
