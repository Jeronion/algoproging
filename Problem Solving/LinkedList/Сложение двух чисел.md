## Определение

Даны два связных списка, каждый из которых представляет неотрицательное целое число. Цифры хранятся в **обычном порядке** (от старшего разряда к младшему). Требуется вычислить сумму двух чисел и вернуть результат в виде связного списка (также в обычном порядке).

**Пример:**
```
head1: 2 → 4 → 3 (число 243)
head2: 5 → 6 → 4 (число 564)
Сумма: 8 → 0 → 7 (число 807)
```

---

## Идея решения

Классический подход: развернуть оба списка, чтобы работать с младшими разрядами (как при сложении в столбик), затем сложить поразрядно с переносом, после чего развернуть результат. Также необходимо удалить ведущие нули из входных списков (если есть, например, `0→0→1` означает 1, но лучше это сделать, хотя по условию обычно их нет).

### Шаги

1. **Удалить ведущие нули** из обоих списков (если есть) – пропускаем узлы с `0`, пока не встретим ненулевое значение или пока не останется последний узел (число 0).
2. **Развернуть** оба списка (чтобы головы указывали на младшие разряды).
3. **Создать фиктивную голову** для результата.
4. **Пройти по обоим спискам** одновременно, суммируя значения узлов и перенос, создавая новые узлы результата.
5. **Развернуть** полученный список результата (чтобы привести к обычному порядку от старшего разряда).
6. Вернуть голову.

## Реализация (комментированный код)

```python
class Solution:
    def addTwoLists(self, head1, head2):
        # Вспомогательная функция разворота списка
        def reverse(head):
            prev = None
            cur = head
            while cur:
                nxt = cur.next
                cur.next = prev
                prev = cur
                cur = nxt
            return prev
        
        # Удаляем ведущие нули (для корректности, хотя по условию их может не быть)
        while head1 and head1.data == 0 and head1.next:
            head1 = head1.next
        while head2 and head2.data == 0 and head2.next:
            head2 = head2.next
        
        # Разворачиваем списки
        h1 = reverse(head1)
        h2 = reverse(head2)
        
        dummy = Node(0)
        cur = dummy
        carry = 0
        
        # Сложение поразрядно
        while h1 or h2 or carry:
            val = carry
            if h1:
                val += h1.data
                h1 = h1.next
            if h2:
                val += h2.data
                h2 = h2.next
            carry = val // 10
            cur.next = Node(val % 10)
            cur = cur.next
        
        # Разворачиваем результат и возвращаем
        return reverse(dummy.next)
```

## Сложность

- **Время:** O(n + m) — линейные проходы для разворота, сложения и финального разворота.
- **Память:** O(max(n, m)) — для хранения узлов результата (обязательно, так как нужно вернуть новый список).

## Примечания

- В отличие от классической задачи, где цифры уже идут в обратном порядке (LeetCode 2), здесь требуется дополнительный разворот.
- Удаление ведущих нулей не обязательно, если входные данные корректны, но делает код устойчивее.
- Если оба числа равны 0, после разворота и сложения получим один узел со значением 0 (корректно).
