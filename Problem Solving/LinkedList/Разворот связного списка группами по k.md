## Определение

Дан односвязный список и целое число `k`. Требуется развернуть **каждую группу из k узлов** (последовательно, слева направо). Если количество узлов не кратно `k`, то последняя группа (меньше k) также разворачивается. Вернуть голову изменённого списка.

**Пример:**
```
Вход: 1 → 2 → 3 → 4 → 5 → 6 , k = 2
Выход: 2 → 1 → 4 → 3 → 6 → 5
```

---

## Идея решения (итеративный подход с dummy node)

Используется **итеративный метод** с фиктивной головой (`dummy`), чтобы упростить обработку начала списка. Алгоритм:

1. Создать фиктивный узел `node`, который указывает на `head`.
2. Инициализировать `left` и `right` как `node` (оба указывают на фиктивный узел).
3. Пока есть необработанные узлы:
   - Перемещаем `right` на `k` шагов вперёд (если достигаем конца, то останавливаемся, но последняя группа всё равно будет развёрнута).
   - Запоминаем `nxt = right.next` (следующий узел после текущей группы).
   - Отсоединяем группу: `right.next = None`.
   - Запоминаем начало группы: `right = left.next`.
   - Разворачиваем группу: `left.next = reverse(left.next)`.
   - Присоединяем развёрнутую группу к остатку: `right.next = nxt`.
   - Перемещаем `left` на `right` (конец развёрнутой группы).
   - Сбрасываем `right` на `left` (для следующей итерации).
   - Повторяем.

Функция `reverse(head)` разворачивает переданный список и возвращает новую голову.

## Реализация

```python
class Node:
    def __init__(self, val):
        self.data = val
        self.next = None

class Solution:
    def reverseKGroup(self, head, k):
        # Вспомогательная функция разворота списка
        def reverse(head):
            prev = None
            cur = head
            while cur:
                nxt = cur.next
                cur.next = prev
                prev = cur
                cur = nxt
            return prev

        dummy = Node(0)
        dummy.next = head
        left = dummy
        right = dummy
        size = 0

        while right.next:                     # пока есть непроверенные узлы
            while right.next and size < k:    # двигаем правую границу на k шагов
                size += 1
                right = right.next
            nxt = right.next                  # сохраняем следующий узел
            right.next = None                 # отрезаем текущую группу
            right = left.next                 # начало группы (будет развёрнуто)
            left.next = reverse(left.next)    # разворачиваем группу
            right.next = nxt                  # подсоединяем обратно
            left = right                      # перемещаем левую границу в конец развёрнутой группы
            size = 0

        return dummy.next
```

## Сложность

- **Время:** O(n) — каждый узел посещается один раз при развороте и один раз при пробеге.
- **Память:** O(1) — используются только несколько указателей (без учёта стека рекурсии, так как итеративная версия не использует рекурсию).

---

## Примечания

- В коде используется фиктивный узел `node` (или `dummy`), чтобы облегчить обработку головы списка.
- Алгоритм обрабатывает случай, когда последняя группа короче `k` — она тоже разворачивается, как требуется в условии.
- Важно обнулять `right.next` перед разворотом и восстанавливать связь после.
- Итеративный подход предпочтителен рекурсивного из-за ограничений по глубине стека для длинных списков.
