## Определение

Дан односвязный список: $L_0 \to L_1 \to \dots \to L_{n-1} \to L_n$. Требуется переупорядочить его на месте (не меняя значения узлов) так, чтобы он стал: $L_0 \to L_n \to L_1 \to L_{n-1} \to L_2 \to L_{n-2} \to \dots$

**Пример:**
```
Вход: 1 → 2 → 3 → 4 → 5
Выход: 1 → 5 → 2 → 4 → 3
```

---

## Идея решения (O(n) времени, O(1) памяти)

Алгоритм состоит из трёх шагов:

1. **Найти середину** списка с помощью быстрого и медленного указателей (tortoise and hare). В конце медленный указатель (`slow`) указывает на начало второй половины (для чётной длины – на первый элемент правой половины, для нечётной – на средний элемент).
2. **Развернуть вторую половину** списка.
3. **Слить** первую половину с развёрнутой второй, чередуя узлы.

### Детали

- Чтобы найти середину, инициализируем `slow = head`, `fast = head.next`. Двигаем `fast` на два шага, `slow` на один, пока `fast` и `fast.next` не станут `None`. В конце `slow` будет указывать на узел, после которого начинается вторая половина (т.е. `second = slow.next`). Затем отрезаем: `slow.next = None`.
- Разворот второй половины выполняется стандартным способом.
- Слияние: проходим по первой половине (`first`) и развёрнутой второй (`second`), вставляя узлы второй чередующимся образом. Для этого сохраняем `next1` и `next2`, затем обновляем ссылки.

**Важно:** При слиянии последний узел первой половины может соединяться с оставшимися узлами второй, если их длины отличаются на 1 (как в примере с нечётной длиной). Цикл должен продолжаться, пока есть оба списка.

## Реализация на Python

```python
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class Solution:
    def reorderList(self, head):
        if not head or not head.next:
            return head

        # Шаг 1: найти середину
        slow, fast = head, head.next
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next

        # Шаг 2: развернуть вторую половину
        second = slow.next
        slow.next = None   # разрываем список
        prev = None
        while second:
            nxt = second.next
            second.next = prev
            prev = second
            second = nxt
        second = prev      # головa развёрнутой второй половины

        # Шаг 3: слияние двух половин чередованием
        first = head
        while second:
            # сохраняем следующие узлы
            nxt1 = first.next
            nxt2 = second.next

            first.next = second
            second.next = nxt1

            # перемещаем указатели
            first = nxt1
            second = nxt2

        return head
```

## Сложность

- **Время:** O(n) – один проход для поиска середины, один для разворота, один для слияния.
- **Память:** O(1) – только константное количество указателей.

---

## Примечания

- Алгоритм работает на месте, не создавая новых узлов.
- Обработка пустого списка и списка из одного узла выполняется в начале.
- В случае чётной длины первая половина на один узел короче второй, после разворота вторая половина будет иметь ту же длину или на один элемент длиннее? При слиянии цикл `while second` полностью обрабатывает все узлы второй половины, при этом первая половина может содержать лишний узел (как при нечётной длине), который останется на своём месте.
