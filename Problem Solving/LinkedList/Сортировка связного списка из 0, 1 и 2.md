## Определение

Дан односвязный список, состоящий только из узлов со значениями 0, 1 и 2. Требуется переставить узлы так, чтобы сначала шли все 0, затем все 1, затем все 2, сохраняя относительный порядок узлов внутри каждой группы (стабильно). Нужно выполнить преобразование на месте без создания новых узлов.

**Пример:**
```
Вход: 1 → 2 → 2 → 1 → 2 → 0 → 2 → 2
Выход: 0 → 1 → 1 → 2 → 2 → 2 → 2 → 2
```

---

## Идея решения (три фиктивных списка)

Вместо многократного прохода или сортировки обменом данных, используем простое распределение узлов:

1. Создаём три фиктивных узла (`zeros`, `ones`, `twos`) и соответствующие указатели `z`, `o`, `t`, которые будут хвостами каждого из трёх списков.
2. Проходим по исходному списку. Для каждого узла:
   - Отсоединяем его от исходного списка (но так как мы просто переприсваиваем `next`, нам не нужно явно отцеплять).
   - Добавляем в конец соответствующего списка: `z.next = cur`, затем сдвигаем `z`; аналогично для `o` и `t`.
3. После распределения сшиваем списки: 
   - `z.next` указывает на голову списка единиц (если он не пуст, иначе на голову двоек).
   - `o.next` указывает на голову списка двоек.
   - Завершаем список двоек: `t.next = None`.
4. Возвращаем `zeros.next` (первый настоящий узел после фиктивной головы нулей).

Этот метод работает за один проход, без создания новых узлов, и сохраняет порядок внутри каждой группы, так как мы добавляем узлы в конец списков в порядке обхода.

## Реализация

```python
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class Solution:
    def segregate(self, head):
        zeros = Node(0)   # фиктивные головы
        ones = Node(0)
        twos = Node(0)
        z = zeros
        o = ones
        t = twos

        cur = head
        while cur:
            if cur.data == 0:
                z.next = cur
                z = z.next
            elif cur.data == 1:
                o.next = cur
                o = o.next
            else:  # 2
                t.next = cur
                t = t.next
            cur = cur.next

        # Соединяем списки в порядке 0 -> 1 -> 2
        z.next = ones.next if ones.next else twos.next
        o.next = twos.next
        t.next = None

        return zeros.next
```

---

## Сложность

- **Время:** O(n) — один проход по списку.
- **Память:** O(1) — дополнительно только три фиктивных узла (можно было бы обойтись без них, используя проверки на `None`, но так код проще).

---

## Примечания

- Алгоритм работает на месте, не создавая копий узлов.
- Сохраняется относительный порядок следования нулей, единиц и двоек (стабильная сегрегация).
- Если какой‑то из списков пуст, сшивание корректно обрабатывается через проверку `ones.next` и `twos.next`.
