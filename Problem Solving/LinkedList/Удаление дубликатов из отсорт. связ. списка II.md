## Определение

Дан отсортированный по возрастанию связный список. Удалить все узлы, которые имеют дубликаты (т.е. встречаются более одного раза), оставив только уникальные элементы.

Задача известна как [LeetCode 82. Remove Duplicates from Sorted List II](https://leetcode.com/problems/remove-duplicates-from-sorted-list-ii/).

---

## Постановка задачи

**Вход:** голова отсортированного списка.  
**Выход:** новая голова без дублирующихся элементов.

Пример:
```
head = [1,2,3,3,4,4,5]
Результат: [1,2,5]
```

---

## Идея решения

Используем фиктивную голову и два указателя:
- `left` — последний узел, который гарантированно не удаляется (изначально фиктивный).
- `right` — текущий просматриваемый узел.

Проходим по списку. Если `right.val == right.next.val`, пропускаем все узлы с этим значением и соединяем `left.next` с первым узлом, имеющим другое значение. Иначе двигаем `left` вперёд.

---

## Реализация

```python
def deleteDuplicates(head: Optional[ListNode]) -> Optional[ListNode]:
    dummy = ListNode(-1, head)
    left = dummy
    right = head
    while right and right.next:
        if right.val == right.next.val:
            # пропускаем все узлы с этим значением
            while right.next and right.val == right.next.val:
                right = right.next
            left.next = right.next
        else:
            left = left.next
        right = right.next
    return dummy.next
```

---

## Сложность

- **Время:** O(n)
- **Память:** O(1)

---

## Примечания

- В отличие от задачи "удалить дубликаты, оставив один экземпляр", здесь удаляются все узлы с повторяющимися значениями.
