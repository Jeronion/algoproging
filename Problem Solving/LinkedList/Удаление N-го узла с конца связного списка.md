## Определение

Дан односвязный список и целое число `n`. Требуется удалить **n-й узел с конца** списка и вернуть голову изменённого списка.

**Пример:**
```
Вход: 1 → 2 → 3 → 4 → 5, n = 2
Выход: 1 → 2 → 3 → 5
```

---

## Идея решения (два указателя)

Используем два указателя `left` и `right`. Сначала сдвигаем `right` на `n` шагов вперёд. Если `right` стал `None`, значит нужно удалить первый узел (голову). Иначе двигаем оба указателя, пока `right.next` не станет `None`. В этот момент `left` будет указывать на узел **перед** удаляемым. Выполняем `left.next = left.next.next`.

### Алгоритм

1. Установить `left = right = head`.
2. Сдвинуть `right` на `n` узлов вперёд.
3. Если `right is None`, то нужно удалить голову: вернуть `head.next`.
4. Иначе пока `right.next` существует:
   - `left = left.next`
   - `right = right.next`
5. После цикла `left.next` указывает на удаляемый узел. Перенаправить `left.next = left.next.next`.
6. Вернуть `head`.

---

## Реализация

```python
class Solution:
    def removeNfromEnd(self, head, n):
        left = right = head
        for _ in range(n):
            right = right.next
        if right is None:
            return head.next
        while right.next:
            left = left.next
            right = right.next
        left.next = left.next.next
        return head
```

---

## Сложность

- **Время:** O(L), где L — длина списка (один проход + сдвиг правого указателя).
- **Память:** O(1) — только два указателя.

---

## Примечания

- Алгоритм работает за один проход по списку (без предварительного подсчёта длины).
- Обработка удаления головы (когда `n` равно длине списка) выполняется корректно.
- Гарантируется, что `n` находится в допустимых пределах (1 ≤ n ≤ длина списка).
