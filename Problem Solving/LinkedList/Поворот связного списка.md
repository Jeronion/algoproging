## Определение

Дан связный список и целое число `k`. Нужно повернуть список вправо на `k` позиций.

Задача известна как [LeetCode 61. Rotate List](https://leetcode.com/problems/rotate-list/).

---

## Постановка задачи

**Вход:** голова списка и `k` (неотрицательное).  
**Выход:** новая голова после поворота.

Пример:
```
head = [1,2,3,4,5], k = 2
Результат: [4,5,1,2,3]
```

---

## Идея решения

1. Найти длину списка `length` и замкнуть его в цикл (последний узел указывает на голову).
2. Вычислить реальное количество сдвигов: `k %= length`. Если `k == 0`, вернуть исходную голову.
3. Найти узел, который станет новым хвостом: для этого пройти `length - k - 1` шагов от старой головы.
4. Новый хвост разорвать, новый голова — следующий за ним.

---

## Реализация

```python
def rotateRight(head: Optional[ListNode], k: int) -> Optional[ListNode]:
    if not head:
        return head

    # находим длину и последний узел
    length = 1
    tail = head
    while tail.next:
        length += 1
        tail = tail.next

    k %= length
    if k == 0:
        return head

    # замыкаем в кольцо
    tail.next = head

    # ищем новую голову: она будет через length - k шагов
    new_tail = head
    for _ in range(length - k - 1):
        new_tail = new_tail.next
    new_head = new_tail.next
    new_tail.next = None

    return new_head
```

---

## Сложность

- **Время:** O(n)
- **Память:** O(1)

---

## Примечания

- Замыкание в цикл и последующий разрыв — ключевой приём.
- Обработка `k` по модулю длины.
