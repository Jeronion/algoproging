## Определение

Дан односвязный список. Если в нём есть цикл, требуется вернуть **количество узлов** в этом цикле. Если цикла нет, вернуть `0`.

**Пример:**
```
Список: 1 → 2 → 3 → 4 → 5 → 2 (цикл начиная с узла 2)
Длина цикла: 4 (2→3→4→5→2)
```

---

## Идея решения (алгоритм Флойда + подсчёт длины)

1. Сначала обнаруживаем цикл с помощью алгоритма Флойда (быстрый и медленный указатели). Если `fast` и `slow` встретились, цикл существует.
2. Затем, чтобы найти **длину цикла**, фиксируем один указатель (`slow`) на точке встречи, а второй (`fast` или переиспользуем `slow`) начинаем двигаться по циклу, считая шаги, пока не вернёмся в точку встречи.

### Алгоритм

1. Инициализировать `slow = fast = head`.
2. Пока `fast` и `fast.next` не `None`:
   - `slow = slow.next`
   - `fast = fast.next.next`
   - Если `slow == fast`, то цикл найден → переходим к подсчёту.
3. Если цикл не найден, вернуть `0`.
4. Для подсчёта: установить `size = 1`, `slow = slow.next`. Пока `slow != fast`: `slow = slow.next`, `size += 1`. Вернуть `size`.

**Почему это работает:** Указатели встретились внутри цикла. Двигая один из них по кругу, мы сделаем полный оборот и вернёмся к точке встречи, подсчитав количество узлов.

---

## Реализация

```python
class Solution:
    def lengthOfLoop(self, head):
        if head is None:
            return 0
        
        slow = fast = head
        # Поиск цикла
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
            if slow == fast:           # цикл обнаружен
                size = 1
                slow = slow.next
                while slow != fast:
                    slow = slow.next
                    size += 1
                return size
        return 0
```

---

## Сложность

- **Время:** O(n) — сначала обход до точки встречи, затем обход цикла.
- **Память:** O(1) — только указатели.

---

## Примечания

- Гарантируется, что указатель `fast` и `fast.next` существуют при проверке в цикле.
- Если в списке нет цикла, функция возвращает `0`.
- Подсчёт длины цикла начинается с точки встречи, поэтому ни один узел не будет пропущен.
