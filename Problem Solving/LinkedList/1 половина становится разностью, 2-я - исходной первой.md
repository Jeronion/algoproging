## Определение

Дан односвязный список. Требуется модифицировать его следующим образом:

- Для каждой пары узлов, симметричных относительно центра (первый с последним, второй с предпоследним и т.д.), новое значение узла из **первой половины** становится равным `(значение парного узла из второй половины) − (текущее значение узла первой половины)`.
- Затем вторая половина списка заменяется **исходными значениями** узлов первой половины (теми, которые были до их изменения).
- Если длина списка нечётная, средний узел остаётся неизменным.

**Пример:**
```
Вход: 10 → 4 → 5 → 3 → 6
Выход: -4 → -1 → 5 → 4 → 10
Пояснение: 
- Пары: (10,6) → 6-10 = -4, (4,3) → 3-4 = -1; средний 5 не меняется.
- Вторая половина (3,6) заменяется исходными значениями первой половины (4,10) в том же порядке.
```

---

## Идея решения

Вместо того чтобы хранить исходные значения первой половины отдельно, можно использовать сам список:

1. **Найти середину** (первый узел второй половины). Для этого используем два указателя (`slow`, `fast`).
2. **Разделить список** на две половины. Указатель `prev` будет указывать на последний узел первой половины.
3. **Перевернуть вторую половину**, чтобы было удобно идти от конца к началу.
4. **Одновременно пройти** по первой половине и по перевёрнутой второй, обновляя значения:
   - `temp = cur.data` (запоминаем исходное значение узла первой половины).
   - `cur.data = slow.data - cur.data` (новое значение первой половины).
   - `slow.data = temp` (вторая половина получает исходное значение первой половины).
5. **Снова перевернуть вторую половину**, чтобы восстановить исходный порядок узлов (теперь в ней лежат исходные значения первой половины в правильном порядке).
6. Соединить первую и вторую половины.

## Реализация

```python
class Solution:
    def modify_the_list(self, head):
        # Функция для разворота списка
        def reverse(head):
            prev = None
            cur = head
            while cur:
                nxt = cur.next
                cur.next = prev
                prev = cur
                cur = nxt
            return prev

        # Поиск середины (slow указывает на последний узел первой половины)
        slow, fast = head, head
        while fast.next and fast.next.next:
            slow = slow.next
            fast = fast.next.next

        # Разделяем и переворачиваем вторую половину
        second_half = reverse(slow.next)
        slow.next = second_half

        # Проходим по обеим половинам одновременно
        cur = head
        while second_half:
            # Сохраняем исходное значение узла первой половины
            original_first = cur.data
            # Обновляем значение первой половины
            cur.data = second_half.data - cur.data
            # Записываем исходное значение первой половины во вторую
            second_half.data = original_first
            cur = cur.next
            second_half = second_half.next

        # Снова переворачиваем вторую половину, чтобы вернуть порядок
        slow.next = reverse(slow.next)
        return head
```

---

## Сложность

- **Время:** O(n) – один проход для поиска середины, один для разворота, один для обновления и ещё один для обратного разворота.
- **Память:** O(1) – используются только указатели.

---

## Примечания

- Для нечётной длины средний узел остаётся нетронутым: он не попадает ни в первую, ни во вторую половину, так как `slow` останавливается на узле перед серединой, а `slow.next` – это первый узел второй половины.
- В коде используется трюк: после обновления первая половина уже изменена, но мы сохранили исходное значение в переменной и присвоили его узлу второй половины.
- Последний разворот необходим, потому что после первого разворота вторая половина идёт в обратном порядке; после обновления значений мы должны вернуть её в прямой порядок.
