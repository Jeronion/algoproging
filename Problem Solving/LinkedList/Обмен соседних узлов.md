## Определение

Дан связный список. Требуется поменять местами каждые два соседних узла и вернуть голову изменённого списка.

Задача известна как [LeetCode 24. Swap Nodes in Pairs](https://leetcode.com/problems/swap-nodes-in-pairs/).

---

## Постановка задачи

**Вход:** голова списка.  
**Выход:** новая голова после обмена пар.

Пример:
```
head = [1,2,3,4]
Результат: [2,1,4,3]
```

---

## Идея решения

Используется фиктивная голова. Проходим по списку, меняя местами пары узлов. Для каждой пары `(prev, cur, next)`:
- `prev.next = next`
- `cur.next = next.next`
- `next.next = cur`
- Перемещаем `prev` на `cur` и продолжаем.

---

## Реализация

```python
def swapPairs(head: Optional[ListNode]) -> Optional[ListNode]:
    dummy = ListNode(0, head)
    prev = dummy
    cur = head
    while cur and cur.next:
        nxt = cur.next          # второй узел пары
        t = nxt.next            # следующий за парой

        # меняем связи
        nxt.next = cur
        cur.next = t
        prev.next = nxt

        # перемещаемся на следующую пару
        prev = cur
        cur = t
    return dummy.next
```

---

## Сложность

- **Время:** O(n)
- **Память:** O(1)

---

## Примечания

- Если в списке нечётное количество узлов, последний остаётся на месте.
- Решение использует только константную дополнительную память.
