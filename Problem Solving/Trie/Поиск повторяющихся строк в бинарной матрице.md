## Определение

Дана бинарная матрица `mat` размера `n × m`, состоящая из 0 и 1. Требуется найти все строки, которые являются **дубликатами** (полностью совпадают с одной из ранее встреченных строк). Вернуть их индексы в порядке появления.

**Пример:**
```
mat = [[1, 0],
       [1, 0]]
Вывод: [1] (строка 1 повторяет строку 0)
```

---

## Постановка задачи

**Вход:** матрица `mat` размера `n × m` (1 ≤ n, m ≤ 10³), элементы 0 или 1.  
**Выход:** массив индексов строк (0-индексация), которые являются дубликатами (повторяют какую-либо предыдущую строку).

---

## Идея решения

Задача сводится к эффективному поиску повторяющихся последовательностей бит. Один из способов — использовать **префиксное дерево (Trie)**, где каждый узел хранит два возможных перехода (0 и 1). При вставке строки мы проходим по битам, создавая узлы, если их нет. Если при обходе строки мы **не создали ни одного нового узла** (т.е. весь путь уже существовал), это означает, что данная строка уже встречалась ранее — она является дубликатом.

### Почему это работает?

- Первое появление строки создаёт все необходимые узлы в Trie.
- При повторном появлении все узлы уже существуют, и обход не создаёт новых узлов.
- Таким образом, мы можем определить дубликат, не храня сами строки, а только структуру Trie.

---

## Алгоритм

1. Создать корень Trie.
2. Инициализировать пустой список `res` для индексов дубликатов.
3. Для каждой строки `row` (индекс `i`):
   - Установить `curr = root` и флаг `is_new = False` (предполагаем, что строка новая).
   - Для каждого бита `bit` в `row`:
     - Если у `curr` нет ребёнка для `bit`, создать новый узел и установить `is_new = True`.
     - Перейти к дочернему узлу.
   - Если `is_new == False` (т.е. не было создано ни одного нового узла), значит строка уже существовала → добавить `i` в `res`.
4. Вернуть `res`.

---

## Реализация на Python

```python
class TrieNode:
    def __init__(self):
        self.children = [None, None]   # 0 и 1

class Solution:
    def repeatedRows(self, mat):
        root = TrieNode()
        res = []
        n, m = len(mat), len(mat[0])
        
        for i in range(n):
            curr = root
            is_new = False
            for bit in mat[i]:
                if curr.children[bit] is None:
                    curr.children[bit] = TrieNode()
                    is_new = True
                curr = curr.children[bit]
            if not is_new:
                res.append(i)
        
        return res
```

---

## Сложность

- **Время:** O(n·m) — каждая строка обрабатывается за O(m), всего n строк.
- **Память:** O(n·m) в худшем случае (если все строки различны и не имеют общих префиксов). В среднем память меньше благодаря общим префиксам.

---

## Альтернативные подходы

- **Хеширование строк:** преобразовать каждую строку в строку бит и хранить в хеш-таблице. Проще в реализации, но требует преобразования и хранения строк.
- **Использование множества кортежей:** `set(tuple(row) for row in mat)` и проверка повторений. Это O(n·m) времени и O(n·m) памяти, но без использования Trie.
- **Битовые маски:** если m ≤ 64, можно упаковать строку в целое число и использовать множество, что очень эффективно.

Trie-подход особенно полезен, если нужно поддерживать множество строк с возможностью быстрого поиска префиксов, но в данной задаче он просто элегантен.

---

## Примечания

- Строки бинарные, поэтому у каждого узла ровно два ребёнка.
- Индексы дубликатов возвращаются в порядке их появления (как в примерах).
- Если строка встречается несколько раз, все её повторения (кроме первого) будут добавлены в результат.
