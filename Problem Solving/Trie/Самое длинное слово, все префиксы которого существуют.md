## Определение

Задача заключается в том, чтобы среди заданного массива слов найти **самое длинное слово**, для которого все его префиксы (включая само слово) также присутствуют в массиве. Если таких слов несколько, выбрать **лексикографически наименьшее**. Это классическая задача на использование префиксного дерева (Trie), где мы сначала вставляем все слова, а затем проверяем, все ли префиксы каждого слова являются полноценными словами.

**Пример:**
```
words = ["ab", "a", "abc", "abd"]
Здесь "abc" имеет префиксы "a" и "ab", которые присутствуют. "abd" также подходит, но "abc" лексикографически меньше, поэтому ответ "abc".
```

---

## Постановка задачи

**Вход:** массив `words` (строки из строчных латинских букв).  
**Выход:** строка — самое длинное слово, все префиксы которого есть в массиве. В случае равной длины выбирается лексикографически наименьшая строка.

---

## Идея решения

Используется **префиксное дерево (Trie)** для хранения всех слов и отметки окончания каждого слова. Алгоритм состоит из двух этапов:

1. **Вставка всех слов в Trie:** Каждый узел помечается флагом `isEnd`, указывающим, что путь от корня до этого узла образует полное слово.
2. **Проверка каждого слова:** Для каждого слова проходим по Trie от корня до последнего символа. На каждом шаге (включая последний) проверяем, что узел помечен как `isEnd` (т.е. текущий префикс является словом). Если хотя бы один префикс отсутствует, слово не подходит.
3. **Выбор лучшего:** Среди всех подходящих слов выбираем самое длинное; при равенстве длин — лексикографически меньшее.

---

## Алгоритм (по шагам)

1. Создать класс `TrieNode` с массивом из 26 дочерних указателей и булевым флагом `isEnd`.
2. Создать класс `Trie` с методом `insert` для вставки слова и методом `allPrefixesExist`, проверяющим все префиксы.
3. Вставить все слова из входного массива в Trie.
4. Для каждого слова проверить `allPrefixesExist`.
5. Обновлять результат, если текущее слово длиннее или равно длине текущего результата и лексикографически меньше.
6. Вернуть результат.

## Реализация на Python

```python
class TrieNode:
    def __init__(self):
        self.children = [None] * 26
        self.isEnd = False

class Trie:
    def __init__(self):
        self.root = TrieNode()

    def insert(self, word):
        node = self.root
        for ch in word:
            idx = ord(ch) - ord('a')
            if not node.children[idx]:
                node.children[idx] = TrieNode()
            node = node.children[idx]
        node.isEnd = True

    def allPrefixesExist(self, word):
        node = self.root
        for ch in word:
            idx = ord(ch) - ord('a')
            node = node.children[idx]
            if not node or not node.isEnd:
                return False
        return True

def longestWord(words):
    trie = Trie()
    for w in words:
        trie.insert(w)

    result = ""
    for w in words:
        if trie.allPrefixesExist(w):
            if len(w) > len(result) or (len(w) == len(result) and w < result):
                result = w
    return result

# Пример
arr = ["ab", "a", "abc", "abd"]
print(longestWord(arr))  # "abc"
```

## Сложность

- **Время:** O(N·L), где N — количество слов, L — средняя длина слова. Вставка и проверка каждого слова требуют O(L) операций.
- **Память:** O(N·L) в худшем случае для хранения Trie (все узлы). В среднем меньше, если слова имеют общие префиксы.

## Примечания

- Задача известна на LeetCode как [720. Longest Word in Dictionary](https://leetcode.com/problems/longest-word-in-dictionary/).
- Альтернативное решение: можно отсортировать слова по длине и использовать множество для проверки префиксов, но Trie даёт более структурированный и эффективный способ, особенно при большом количестве слов.
- Важно, что проверка префиксов включает **само слово**, поэтому флаг `isEnd` должен быть установлен для последнего символа каждого слова.
- Если массив пуст, результат должен быть пустой строкой (или `""`).
