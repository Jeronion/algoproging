## Определение

Даны целые числа `n` (длина последовательности), `l` и `r` (границы значений). Требуется сгенерировать **битонную последовательность** длины `n`, которая сначала строго возрастает, затем строго убывает. Все элементы должны лежать в диапазоне `[l, r]`. Среди всех возможных таких последовательностей нужно выбрать **лексикографически наибольшую**. Если построить последовательность невозможно, вернуть `[-1]`.

**Пример:**
```
n = 5, l = 3, r = 10 → [9, 10, 9, 8, 7]
n = 7, l = 2, r = 5 → [2, 3, 4, 5, 4, 3, 2]
```

---

## Условие существования

Битонная последовательность максимальной длины, которую можно построить из чисел `[l, r]`, имеет вид: `l, l+1, ..., r-1, r, r-1, ..., l`. Её длина = `(r - l) * 2 + 1`. Если `n` больше этого значения, построить последовательность невозможно.

---

## Идея решения (двусторонняя очередь)

Используем `deque` для сборки последовательности, добавляя элементы как в начало, так и в конец. Лексикографически наибольшая последовательность достигается, если сделать пик как можно больше и расположить элементы в определённом порядке:

1. Начинаем с `r-1` (чтобы получить в итоге наибольший первый элемент).
2. Добавляем в **конец** убывающие числа от `r` до `l`, пока не наберём нужную длину.
3. Если после этого длина ещё меньше `n`, добавляем в **начало** убывающие числа от `r-2` до `l`.

### Алгоритм

1. Проверить, возможно ли построение: если `n > (r - l) * 2 + 1`, вернуть `[-1]`.
2. Создать `deque`, положить в него `r-1`.
3. Добавлять в конец числа от `r` вниз до `l`, пока размер очереди не достигнет `n`.
4. Если размер всё ещё меньше `n`, добавлять в начало числа от `r-2` вниз до `l`, пока размер не станет `n`.
5. Вернуть список из очереди.

---

## Реализация на Python

```python
from collections import deque

def bitonic_array(n, l, r):
    # проверка возможности
    if n > (r - l) * 2 + 1:
        return [-1]

    dq = deque()
    dq.append(r - 1)

    # добавляем убывающую часть в конец
    i = r
    while i >= l and len(dq) < n:
        dq.append(i)
        i -= 1

    # добавляем возрастающую часть в начало (при необходимости)
    i = r - 2
    while i >= l and len(dq) < n:
        dq.appendleft(i)
        i -= 1

    return list(dq)
```

---

## Сложность

- **Время:** O(n)
- **Память:** O(n) для хранения очереди и результата.

---

## Примечания

- Алгоритм использует тот факт, что для получения лексикографически наибольшей последовательности нужно сделать первый элемент максимально возможным (поэтому начинаем с `r-1`, а не с `r`).
