## Условие

Натуральное число называется «красивым», если в его разложение на простые множители входят только числа 2, 3 и 5 (например, 1 не считается, так как начинаем с 2).  
Требуется вывести **первые N** таких чисел в порядке возрастания (N ≤ 1600).

**Пример:**  
N = 10 → 2, 3, 4, 5, 6, 8, 9, 10, 12, 15

---

## Идея

Каждое следующее число получается умножением одного из уже полученных на 2, 3 или 5.  
Поддерживаем три указателя `i2`, `i3`, `i5`, указывающие на минимальные индексы в массиве, такие что умножение чисел на соответствующие множители ещё не давало очередное число.  
На каждом шаге выбираем минимальное из трёх кандидатов:

```
next2 = numbers[i2] * 2
next3 = numbers[i3] * 3
next5 = numbers[i5] * 5
next = min(next2, next3, next5)
```

Затем сдвигаем указатели для тех множителей, которые дали `next` (чтобы избежать дубликатов).

---

## Примечания

- Фиктивная единица в начале упрощает вычисления (умножение на неё даёт сами множители).
- Указатели сдвигаются только при равенстве, что автоматически удаляет дубликаты.
- Этот алгоритм известен как генерация «уродливых чисел» (ugly numbers).--

---

## Реализация

```python
def generate_ugly_numbers():
    N = int(input())
    numbers = [1]  # первый элемент — фиктивная единица
    i2 = i3 = i5 = 0
    for _ in range(N):
        n2 = numbers[i2] * 2
        n3 = numbers[i3] * 3
        n5 = numbers[i5] * 5
        nxt = min(n2, n3, n5)
        numbers.append(nxt)
        if nxt == n2:
            i2 += 1
        if nxt == n3:
            i3 += 1
        if nxt == n5:
            i5 += 1
    print(*numbers[1:])  # пропускаем фиктивную единицу
```

---

## Сложность

- **Время:** O(N) — каждый шаг выполняется за константное время.
- **Память:** O(N) — для хранения всех чисел.
