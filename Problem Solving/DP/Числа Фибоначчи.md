## Определение

**Числа Фибоначчи** — последовательность, в которой каждое следующее число равно сумме двух предыдущих:

$$
F_1 = 1,\; F_2 = 1,\; F_n = F_{n-1} + F_{n-2} \quad (n \ge 3)
$$

В некоторых вариантах используется нулевая индексация: $F_0 = 0,\; F_1 = 1$.

---

## Реализация (динамическое программирование)

```python
def fibonacci(n):
    if n <= 2:
        return 1
    dp = [0] * (n + 1)
    dp[1], dp[2] = 1, 1
    for i in range(3, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]
```

## Сложность

- **Время:** O(n)
- **Память:** O(n) (можно оптимизировать до O(1) храня только два последних значения)
