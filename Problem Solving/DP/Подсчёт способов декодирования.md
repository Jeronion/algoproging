## Определение

Сообщение, состоящее из букв A–Z, кодируется в числа по правилу:
- 'A' → 1, 'B' → 2, ..., 'Z' → 26.

Дана строка `digits`, содержащая только цифры. Требуется подсчитать количество способов, которыми эту строку можно декодировать обратно в буквы. Например, "123" можно декодировать как "ABC" (1,2,3), "LC" (12,3) и "AW" (1,23) → 3 способа.

**Пример:**
```
digits = "123" → 3
digits = "90" → 0 (недопустимо, так как '0' нельзя декодировать)
digits = "05" → 0 (ведущий ноль недопустим)
```

---

## Постановка задачи

**Вход:** строка `digits` длины n (1 ≤ n ≤ 10³).
**Выход:** количество способов декодирования (целое число). Если строка не может быть декодирована ни одним способом, вернуть `0`.

---

## Идея решения (динамическое программирование)

Пусть `dp[i]` — количество способов декодировать префикс `digits[0..i]` (длины i+1). Рассмотрим два варианта для последнего символа:

1. **Одна цифра:** если `digits[i] != '0'`, то её можно декодировать как букву от A до I. Тогда количество способов для префикса длины i+1 увеличивается на `dp[i-1]` (или на 1, если i=0).
2. **Две цифры:** если `digits[i-1] != '0'` и число `10*digits[i-1] + digits[i]` находится в диапазоне [10, 26], то их можно декодировать как одну букву (J–Z). Тогда добавляем `dp[i-2]` (или 1, если i=1).

База: `dp[0] = 1`, если `digits[0] != '0'`, иначе 0.

Таким образом, `dp[i]` вычисляется как сумма этих двух слагаемых.

Этот подход работает за O(n) времени и O(n) памяти (можно оптимизировать до O(1), но в решении используется массив).

---

## Алгоритм (по шагам)

1. Если `digits[0] == '0'`, вернуть `0`.
2. Инициализировать массив `dp` размера `n` нулями; `dp[0] = 1`.
3. Для `i` от 1 до `n-1`:
   - Если `digits[i] != '0'`, то `dp[i] += dp[i-1]`.
   - Если `digits[i-1] != '0'` и `int(digits[i-1:i+1]) <= 26`, то:
     - если `i >= 2`, `dp[i] += dp[i-2]`;
     - иначе `dp[i] += 1`.
4. Вернуть `dp[n-1]`.

## Реализация на Python

```python
class Solution:
    def countWays(self, digits):
        n = len(digits)
        if n == 0 or digits[0] == '0':
            return 0
        dp = [0] * n
        dp[0] = 1
        for i in range(1, n):
            # одна цифра
            if digits[i] != '0':
                dp[i] += dp[i-1]
            # две цифры
            if digits[i-1] != '0' and int(digits[i-1:i+1]) <= 26:
                if i >= 2:
                    dp[i] += dp[i-2]
                else:
                    dp[i] += 1
        return dp[-1]
```

---

## Сложность

- **Время:** O(n) – один проход по строке.
- **Память:** O(n) для массива `dp`. Можно свести к O(1), храня только два последних значения.

---

## Примечания

- Ведущие нули недопустимы: если первый символ '0', ответ 0.
- '0' может встречаться только в составе '10' или '20', иначе декодирование невозможно.
- При двузначном числе, его значение должно быть от 10 до 26 включительно.
- Ответ может быть большим, но в задаче он гарантированно помещается в 32-битное целое.
