## Условие

Дана лестница из N ступенек. На каждой ступеньке указана стоимость. Мальчик начинает перед первой ступенькой и может перейти на следующую ступеньку или перепрыгнуть через одну. Требуется найти минимальную суммарную стоимость достижения последней ступеньки.

**Пример:**  
N=3, стоимости [1, 3, 1] → ответ 2 (1→3).

---

## Примечания

- Для N=1 ответ — стоимость первой ступеньки.
- Для N=2 — минимум из стоимости второй ступеньки (прыжок с земли) и суммы первой и второй.
- Алгоритм работает за счёт того, что после обновления `arr[i]` хранит минимальную стоимость достижения i-й ступеньки.

---
## Идея

Пусть `dp[i]` — минимальная стоимость достижения i-й ступеньки (0-индексация).  
На i-ю ступеньку можно попасть с (i-1)-й или (i-2)-й:

```
dp[i] = cost[i] + min(dp[i-1], dp[i-2])
```

Базовые случаи:  
`dp[0] = cost[0]`, `dp[1] = cost[1] + min(dp[0], 0)` (0 — стоимость старта).

---

## Реализация

```python
def min_cost_path_linear():
    N = int(input())
    arr = list(map(int, input().split()))   # стоимости
    for i in range(2, N):
        arr[i] += min(arr[i-1], arr[i-2])
    return arr[N-1]
```

Массив `arr` используется как `dp` и модифицируется на месте.

---

## Сложность

- **Время:** O(N)
- **Память:** O(1) дополнительно (массив изменяется на месте)
