## Определение

Дана строка `s`, состоящая из строчных латинских букв. Для каждой позиции `i` (от 0 до n‑1) нужно найти **первый неповторяющийся** символ в префиксе `s[0..i]` (т.е. символ, который встречается в этом префиксе ровно один раз, причём из всех таких символов выбирается тот, который появился раньше всех). Если такого символа нет, используется символ `'#'`.

**Пример:**
```
s = "aabc"
Результат: "a#bb"
Пояснение:
- "a" → 'a'
- "aa" → нет неповторяющихся → '#'
- "aab" → первый неповторяющийся: 'b'
- "aabc" → неповторяющиеся: 'b' и 'c'; 'b' появился раньше → 'b'
```

---

## Идея решения (очередь + массив частот)

Используем **очередь** для хранения символов в порядке их появления, а также массив `count` из 26 элементов для подсчёта частот каждого символа.

При обработке каждого нового символа:
1. Увеличиваем его частоту.
2. Если частота символа стала равна 1, добавляем его в конец очереди.
3. Затем, пока очередь не пуста и частота символа в её начале > 1, удаляем его из очереди (этот символ уже повторяется, поэтому он не может быть ответом).
4. После этого, если очередь не пуста, первый символ в очереди — искомый (он встречается ровно один раз и появился раньше всех). Иначе ответ `'#'`.

Этот метод работает за O(n) времени и использует O(1) дополнительной памяти (массив на 26 элементов, очередь может хранить до n символов в худшем случае, но обычно меньше; в рамках ограничений это допустимо).

## Реализация на Python

```python
from collections import deque

def first_non_repeating(s):
    freq = [0] * 26
    q = deque()
    result = []
    for ch in s:
        idx = ord(ch) - ord('a')
        freq[idx] += 1
        if freq[idx] == 1:
            q.append(ch)
        while q and freq[ord(q[0]) - ord('a')] > 1:
            q.popleft()
        if q:
            result.append(q[0])
        else:
            result.append('#')
    return ''.join(result)

# Пример
s = "aabc"
print(first_non_repeating(s))  # a#bb
```

---

## Сложность

- **Время:** O(n) – каждый символ обрабатывается один раз, и каждый символ попадает в очередь и удаляется из неё не более одного раза.
- **Память:** O(26) для массива частот + O(n) для очереди (в худшем случае, если все символы различны, очередь может содержать все n элементов).
