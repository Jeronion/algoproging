## Определение

Дана очередь `q` чётного размера. Требуется переставить её элементы так, чтобы они шли в следующем порядке: первый элемент первой половины, первый элемент второй половины, затем второй элемент первой половины, второй элемент второй половины и так далее. Иными словами, выполнить **чередующееся слияние** двух половин.

**Пример:**
```
Вход: [2, 4, 3, 1] → [2, 3, 4, 1]
Вход: [3, 5] → [3, 5]
```

---

## Подход 1: Использование стека (O(n) времени, O(n) памяти)

### Идея

Стек используется для временного хранения первой половины в обратном порядке, что позволяет затем правильно чередовать элементы.

### Алгоритм

1. Найти половину размера `half = n // 2`.
2. Извлечь из очереди первые `half` элементов и поместить их в стек.
3. Извлечь элементы из стека (теперь в обратном порядке) и добавить их обратно в очередь. Теперь в очереди сначала идёт перевёрнутая первая половина, затем вторая половина в исходном порядке.
4. Извлечь из очереди первые `half` элементов (это снова первая половина, но теперь в исходном порядке) и добавить их в конец. Теперь очередь имеет порядок: вторая половина (исходная), первая половина (исходная).
5. Снова извлечь первые `half` элементов (теперь это исходная вторая половина) и поместить их в стек.
6. Теперь в очереди осталась только исходная первая половина. Поочерёдно извлекать элемент из стека (вторая половина) и элемент из очереди (первая половина), добавляя их в конец очереди.

### Реализация

```python
from collections import deque

def interleave_queue(q):
    n = len(q)
    half = n // 2
    stack = []
    # 1. Первая половина в стек
    for _ in range(half):
        stack.append(q.popleft())
    # 2. Стек -> очередь (обратный порядок)
    while stack:
        q.append(stack.pop())
    # 3. Первая половина в конец
    for _ in range(half):
        q.append(q.popleft())
    # 4. Снова первая половина в стек
    for _ in range(half):
        stack.append(q.popleft())
    # 5. Чередование
    while stack:
        q.append(stack.pop())
        q.append(q.popleft())
    return q
```

## Подход 2: Использование двух вспомогательных очередей (O(n) времени, O(n) памяти)

### Идея

Разделить исходную очередь на две отдельные очереди (первая половина и вторая половина), затем последовательно брать по одному элементу из каждой и помещать в исходную.

### Реализация

```python
def interleave_queue_two_queues(q):
    half = len(q) // 2
    first = deque()
    second = deque()
    for _ in range(half):
        first.append(q.popleft())
    while q:
        second.append(q.popleft())
    while first and second:
        q.append(first.popleft())
        q.append(second.popleft())
    return q
```

## Подход 3: Через массив (O(n) времени, O(n) памяти)

### Идея

Скопировать элементы в массив, а затем заполнить очередь, чередуя элементы из первой и второй половин.

### Реализация

```python
def interleave_queue_array(q):
    arr = list(q)
    q.clear()
    half = len(arr) // 2
    for i in range(half):
        q.append(arr[i])
        q.append(arr[i + half])
    return q
```

---

## Сложность

Все три подхода имеют:

- **Время:** O(n)
- **Память:** O(n) (для стека, вспомогательных очередей или массива)

---

## Примечания

- Исходная очередь модифицируется. Если нужен оригинал, создайте копию.
- Размер очереди должен быть чётным (по условию задачи).
- В Python удобно использовать `collections.deque` для эффективных операций `popleft()` и `append()`.
