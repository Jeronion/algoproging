## Определение

Даны первые `k` членов последовательности (массив `arr`). Каждый следующий член `F_n` для `n > k` вычисляется как произведение `k` предыдущих членов:

$$
F_n = F_{n-1} \cdot F_{n-2} \cdot \dots \cdot F_{n-k}
$$

Требуется найти `F_n` для заданного `n`.

**Пример:**
```
n = 5, arr = [1, 2] (k=2)
Последовательность: F1=1, F2=2
F3 = 2*1 = 2
F4 = 2*2 = 4
F5 = 4*2 = 8
Результат: 8
```

## Идея решения (скользящее окно с деком)

Наивный подход требует O(n·k) времени. Оптимальное решение использует **скользящее окно** для поддержания произведения последних `k` членов.

### Алгоритм

1. Если `n ≤ k`, вернуть `arr[n-1]`.
2. Инициализировать `deque` для хранения последних `k` членов и переменную `product` – произведение этих членов.
3. Заполнить `deque` начальными `k` членами, вычислить `product`.
4. Для `i` от `k` до `n`:
   - Новый член `new_val = product`. Добавить его в `deque`.
   - Если `i == n`, вернуть `new_val`.
   - Обновить `product`: удалить самый старый элемент из окна (разделить произведение на него), затем умножить на новый член.
5. Вернуть последний вычисленный член.

### Реализация

```python
from collections import deque

def findNthTerm(n, arr):
    k = len(arr)
    if n <= k:
        return arr[n-1]
    
    dq = deque(arr)
    product = 1
    for x in arr:
        product *= x
    
    for i in range(k, n):
        new_val = product
        dq.append(new_val)
        product //= dq.popleft()
        product *= new_val
    
    return product
```

- **Время:** O(n)
- **Память:** O(k)
