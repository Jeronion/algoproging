## Определение

Даны две строки `txt` (текст) и `pat` (образец). Требуется найти **первое вхождение** `pat` в `txt` в виде непрерывной подстроки и вернуть его 0-индекс. Если образец не найден, вернуть `-1`. Запрещено использовать встроенные функции поиска подстроки.

Задача известна на GeeksforGeeks как [First Occurence](https://www.geeksforgeeks.org/problems/first-occurrence-of-a-string--141631/1).

**Примеры:**
- `txt = "GeeksForGeeks", pat = "For"` → `5`
- `txt = "GeeksForGeeks", pat = "Fr"` → `-1`

---

## Идея решения

Используется **алгоритм Кнута-Морриса-Пратта (KMP)**, который ищет все вхождения за линейное время. В данной модификации поиск останавливается при первом же совпадении, что ускоряет работу в среднем случае.

Алгоритм:
1. Строим **префикс-функцию (LPS)** для образца `pat`. LPS[i] = длина наибольшего собственного префикса `pat[0..i]`, который также является её суффиксом.
2. Проходим по тексту, используя два указателя `i` (по тексту) и `j` (по образцу):
   - Если символы совпадают → увеличиваем оба указателя.
   - Если `j` достиг длины образца → нашли вхождение, возвращаем `i - j`.
   - При несовпадении:
     - Если `j > 0`, откатываем `j` на `lps[j-1]` (не трогаем `i`).
     - Иначе увеличиваем `i`.
3. Если цикл завершён без нахождения → возвращаем `-1`.

---

## Реализация на Python

```python
class Solution:
    def build_lps(self, pat: str) -> list:
        m = len(pat)
        lps = [0] * m
        length = 0          # длина предыдущего lps
        i = 1
        while i < m:
            if pat[i] == pat[length]:
                length += 1
                lps[i] = length
                i += 1
            else:
                if length == 0:
                    lps[i] = 0
                    i += 1
                else:
                    length = lps[length - 1]
        return lps

    def firstOccurence(self, txt: str, pat: str) -> int:
        n, m = len(txt), len(pat)
        if m == 0:
            return -1
        lps = self.build_lps(pat)
        i = j = 0
        while i < n:
            if txt[i] == pat[j]:
                i += 1
                j += 1
                if j == m:
                    return i - j   # первое вхождение
            else:
                if j != 0:
                    j = lps[j - 1]
                else:
                    i += 1
        return -1
```

---

## Сложность

- **Время:** O(n + m) — построение LPS занимает O(m), проход по тексту — O(n).
- **Память:** O(m) — массив LPS.

---

## Примечания

- В отличие от классического KMP, который ищет все вхождения, здесь алгоритм завершается при первом совпадении.
- Если образец длиннее текста, цикл поиска не начнётся (или быстро завершится) и вернётся -1.
- Альтернативные решения: наивный поиск O(n*m) или встроенный `txt.find(pat)`, но по условию встроенные функции запрещены.
