## Определение

Дан массив целых чисел и целевое значение `target`. Нужно найти сумму трёх элементов, ближайшую к `target`. Вернуть эту сумму.

Задача известна как [LeetCode 16. 3Sum Closest](https://leetcode.com/problems/3sum-closest/).

---

## Идея решения

Сортируем массив. Фиксируем первый элемент `nums[i]`, затем используем два указателя `l = i+1`, `r = n-1`. Двигаем их в зависимости от того, больше или меньше текущая сумма `target`. Обновляем ближайший результат.

---

## Реализация

```python
def threeSumClosest(nums: List[int], target: int) -> int:
    nums.sort()
    n = len(nums)
    closest = nums[0] + nums[1] + nums[2]
    for i in range(n - 2):
        l, r = i + 1, n - 1
        while l < r:
            total = nums[i] + nums[l] + nums[r]
            if total == target:
                return target
            if abs(total - target) < abs(closest - target):
                closest = total
            if total < target:
                l += 1
            else:
                r -= 1
    return closest
```

---

## Сложность

- **Время:** O(n²) — сортировка O(n log n) + двойной цикл O(n²)
- **Память:** O(1)
