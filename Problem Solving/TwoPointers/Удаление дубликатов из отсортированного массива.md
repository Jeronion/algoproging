## Определение

Дан отсортированный по возрастанию массив целых чисел. Требуется удалить дубликаты **на месте**, чтобы каждый уникальный элемент встречался только один раз. Относительный порядок элементов должен сохраняться. Вернуть новую длину массива (количество уникальных элементов), а первые `k` элементов массива должны содержать эти уникальные значения.

Задача известна как [LeetCode 26. Remove Duplicates from Sorted Array](https://leetcode.com/problems/remove-duplicates-from-sorted-array/).

---

## Идея решения

Используется **метод двух указателей**:
- `slow` — указывает на позицию, куда нужно вставить следующий уникальный элемент.
- `fast` — проходит по массиву, ища новые уникальные значения.

Если `nums[fast]` отличается от предыдущего уникального элемента (`nums[slow-1]`), копируем его в `nums[slow]` и сдвигаем `slow`. В конце `slow` — количество уникальных элементов.

---

## Реализация

```python
def removeDuplicates(nums: List[int]) -> int:
    if not nums:
        return 0
    slow = 1
    for fast in range(1, len(nums)):
        if nums[fast] != nums[slow - 1]:
            nums[slow] = nums[fast]
            slow += 1
    return slow
```

**Пояснение:** `slow` начинается с 1, так как первый элемент всегда уникален. `fast` пробегает с 1 до конца, сравнивая с последним записанным уникальным значением.

---

## Сложность

- **Время:** O(n) — один проход по массиву.
- **Память:** O(1) — все изменения происходят на месте.
