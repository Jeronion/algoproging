## Определение

Дан массив, содержащий только три значения: 0, 1, 2. Нужно отсортировать его так, чтобы все 0 шли первыми, затем все 1, затем все 2. Изменения производятся на месте.

Задача известна как [LeetCode 75. Sort Colors](https://leetcode.com/problems/sort-colors/).

---

## Идея решения

Используется алгоритм **Dutch National Flag** (флаг Нидерландов) с тремя указателями:
- `left` — граница для 0 (все элементы слева от `left` — нули)
- `right` — граница для 2 (все элементы справа от `right` — двойки)
- `mid` — текущий указатель.

Если `nums[mid] == 0` — меняем с `nums[left]`, сдвигаем оба.
Если `nums[mid] == 2` — меняем с `nums[right]`, сдвигаем `right`.
Если `nums[mid] == 1` — просто сдвигаем `mid`.

---

## Реализация

```python
def sortColors(nums: List[int]) -> None:
    left, mid, right = 0, 0, len(nums) - 1
    while mid <= right:
        if nums[mid] == 0:
            nums[left], nums[mid] = nums[mid], nums[left]
            left += 1
            mid += 1
        elif nums[mid] == 2:
            nums[right], nums[mid] = nums[mid], nums[right]
            right -= 1
        else:  # nums[mid] == 1
            mid += 1
```
