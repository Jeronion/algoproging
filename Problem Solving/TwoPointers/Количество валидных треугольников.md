## Определение

Дан массив целых чисел. Нужно найти количество троек `(i, j, k)`, из которых можно составить треугольник (сумма двух любых сторон больше третьей). Порядок не важен.

Задача известна как [LeetCode 611. Valid Triangle Number](https://leetcode.com/problems/valid-triangle-number/).

---

## Идея решения

Сортируем массив. Фиксируем наибольшую сторону `nums[i]` (идём справа налево). Используем два указателя `left` и `right` на оставшейся части. Для каждого `i` ищем пары, где `nums[left] + nums[right] > nums[i]`. Если условие выполнено, все элементы между `left` и `right` тоже подходят, прибавляем `right - left` и сдвигаем `right`. Иначе сдвигаем `left`.

---

## Реализация

```python
def triangleNumber(nums: List[int]) -> int:
    nums.sort()
    n = len(nums)
    count = 0
    for i in range(n - 1, 1, -1):
        left, right = 0, i - 1
        while left < right:
            if nums[left] + nums[right] > nums[i]:
                count += right - left
                right -= 1
            else:
                left += 1
    return count
```

## Сложность

- **Время:** O(n²) — сортировка O(n log n) + двойной цикл O(n²)
- **Память:** O(1)
