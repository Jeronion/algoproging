## Определение

Дан **отсортированный** по возрастанию массив целых чисел `arr` и целевое значение `target`. Требуется найти **пару** индексов `(i, j)` (обычно 1‑индексация), таких что `arr[i] + arr[j] = target`. Гарантируется, что решение существует (или нет, в зависимости от задачи).

**Пример:**
```
arr = [2, 7, 11, 15], target = 9
Ответ: (1, 2) (поскольку 2 + 7 = 9)
```

---

## Идея решения (два указателя)

Поскольку массив **отсортирован**, можно использовать **метод двух указателей**:

1. Установить `left = 0` (начало массива), `right = n-1` (конец массива).
2. Пока `left < right`:
   - Вычислить `current_sum = arr[left] + arr[right]`.
   - Если `current_sum == target`, то пара найдена.
   - Если `current_sum < target`, то нужно увеличить сумму → сдвинуть `left` вправо.
   - Если `current_sum > target`, то нужно уменьшить сумму → сдвинуть `right` влево.
3. Если указатели встретились, пара не найдена (можно вернуть `(-1, -1)`).

---

## Алгоритм

1. Инициализировать `left = 0`, `right = n-1`.
2. Пока `left < right`:
   - `sum = arr[left] + arr[right]`
   - Если `sum == target` → вернуть `(left, right)` (или `left+1, right+1` для 1‑индексации)
   - Если `sum < target` → `left += 1`
   - Если `sum > target` → `right -= 1`
3. Вернуть `(-1, -1)`.

---

## Реализация на Python

```python
def two_sum_sorted(arr, target):
    left, right = 0, len(arr) - 1
    while left < right:
        s = arr[left] + arr[right]
        if s == target:
            return [left + 1, right + 1]  # 1‑индексация
        elif s < target:
            left += 1
        else:
            right -= 1
    return [-1, -1]

# Пример
arr = [2, 7, 11, 15]
target = 9
print(two_sum_sorted(arr, target))  # [1, 2]
```

---

## Сложность

- **Время:** O(n) — один проход по массиву.
- **Память:** O(1) — только переменные.

---

## Примечания

- Метод работает **только для отсортированных массивов**.
- Если массив не отсортирован, его нужно предварительно отсортировать (но тогда потеряется исходный порядок индексов).
- Для неотсортированного массива используется хеш-таблица (O(n) времени, O(n) памяти).
