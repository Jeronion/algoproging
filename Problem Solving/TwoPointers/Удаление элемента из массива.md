## Определение

Дан массив целых чисел `nums` и значение `val`. Нужно удалить все вхождения `val` на месте. Относительный порядок остальных элементов может быть изменён. Вернуть новую длину массива (количество элементов, не равных `val`). Первые `k` элементов массива должны содержать все значения, отличные от `val`.

Задача известна как [LeetCode 27. Remove Element](https://leetcode.com/problems/remove-element/).

---

## Идея решения

Два указателя:
- `slow` — место для следующего элемента, не равного `val`.
- `fast` — бежит по массиву.

Если `nums[fast] != val`, копируем его в `nums[slow]` и сдвигаем `slow`. Иначе пропускаем.

---

## Реализация

```python
def removeElement(nums: List[int], val: int) -> int:
    slow = 0
    for fast in range(len(nums)):
        if nums[fast] != val:
            nums[slow] = nums[fast]
            slow += 1
    return slow
```

## Сложность

- **Время:** O(n)
- **Память:** O(1)
