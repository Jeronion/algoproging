## Определение

Дана строка `s`. Нужно проверить, является ли она палиндромом, игнорируя регистр и неалфавитно-цифровые символы.

Задача известна как [LeetCode 125. Valid Palindrome](https://leetcode.com/problems/valid-palindrome/).

---

## Идея решения

Два указателя — `left` в начале, `right` в конце. Пропускаем неалфавитно-цифровые символы, сравниваем символы в нижнем регистре. Если хоть раз не совпали — возвращаем `False`.

---

## Реализация

```python
def isPalindrome(s: str) -> bool:
    left, right = 0, len(s) - 1
    while left < right:
        while left < right and not s[left].isalnum():
            left += 1
        while left < right and not s[right].isalnum():
            right -= 1
        if s[left].lower() != s[right].lower():
            return False
        left += 1
        right -= 1
    return True
```

---

## Сложность

- **Время:** O(n)
- **Память:** O(1)
