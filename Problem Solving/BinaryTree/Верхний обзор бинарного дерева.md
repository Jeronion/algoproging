## Определение

**Верхний обзор** бинарного дерева — это набор узлов, видимых при взгляде на дерево сверху. Узлы упорядочиваются по горизонтальному расстоянию (HD) от корня:

- Корень имеет HD = 0.
- Левый потомок имеет HD = HD родителя - 1.
- Правый потомок имеет HD = HD родителя + 1.

Для каждого горизонтального расстояния в верхний обзор попадает только **самый верхний** узел (ближайший к корню). Если несколько узлов имеют одинаковый HD, выбирается тот, который встретился первым при обходе сверху вниз (т.е. с наименьшей глубиной). Результат выводится в порядке возрастания HD (слева направо).

**Пример:**
```
        1
       / \
      2   3
     / \   \
    4   5   6
```
Верхний обзор: [2, 1, 3] (узел 4 скрыт за 2, узел 6 скрыт за 3).

---

## Постановка задачи

Дан корень бинарного дерева. Требуется вернуть список значений узлов верхнего обзора.

---

## Идея решения (BFS)

Используется **обход в ширину (BFS)** с очередью. Для каждого узла храним его горизонтальное расстояние (HD). При посещении узла проверяем, есть ли уже запись для этого HD в словаре. Если записи нет, значит, этот узел является самым верхним для данной вертикали – добавляем его значение. Если запись уже существует, пропускаем (оставляем первый встреченный, т.е. ближайший к корню).

### Алгоритм

1. Если дерево пустое, вернуть пустой список.
2. Создать `deque` и поместить в него корень с HD = 0.
3. Создать словарь `top_map` для хранения первого встреченного значения для каждого HD.
4. Инициализировать `mn = 0` и `mx = 0` для границ HD.
5. Пока очередь не пуста:
   - Извлечь узел и его HD.
   - Если `hd` ещё нет в `top_map`, добавить `node.data` (это будет самый верхний узел).
   - Обновить `mn` и `mx`.
   - Если есть левый потомок, добавить его с HD - 1.
   - Если есть правый потомок, добавить его с HD + 1.
6. После обхода, для каждого HD от `mn` до `mx` взять значение из `top_map` и добавить в результат.
7. Вернуть результат.

## Реализация

```python
from collections import deque

class Solution:
    def topView(self, root):
        if not root:
            return []
        q = deque([(root, 0)])
        top_map = {}
        mn, mx = 0, 0
        
        while q:
            node, hd = q.popleft()
            if hd not in top_map:
                top_map[hd] = node.data
            mn = min(mn, hd)
            mx = max(mx, hd)
            if node.left:
                q.append((node.left, hd - 1))
            if node.right:
                q.append((node.right, hd + 1))
        
        return [top_map[hd] for hd in range(mn, mx + 1)]
```

## Сложность

- **Время:** O(n), где n — количество узлов. Каждый узел обрабатывается один раз.
- **Память:** O(n) для словаря и очереди.

---

## Примечания

- BFS гарантирует, что мы обрабатываем узлы уровень за уровнем, поэтому первый встреченный узел для данного HD будет самым верхним (ближайшим к корню).
- Если в словаре уже есть запись для HD, мы не перезаписываем её, что соответствует правилам верхнего обзора.
- Альтернативный подход с DFS также возможен, но требует передачи глубины для сравнения; BFS-решение проще и естественнее.
