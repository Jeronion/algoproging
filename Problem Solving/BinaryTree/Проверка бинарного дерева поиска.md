## Определение

**Бинарное дерево поиска (BST)** — это бинарное дерево, в котором для каждого узла выполняются следующие свойства:

- Все значения в **левом поддереве** строго **меньше** значения узла.
- Все значения в **правом поддереве** строго **больше** значения узла.
- Левое и правое поддеревья сами являются BST (рекурсивное свойство).

Требуется проверить, является ли данное бинарное дерево корректным BST.

---

## Постановка задачи

Дан корень бинарного дерева. Вернуть `true`, если дерево является BST, иначе `false`.

**Пример:**
```
        2
       / \
      1   3
           \
            5
```
Это BST, так как все левые меньше, правые больше.

```
        2
         \
          7
         / \
        6   9
```
Не является BST, так как 6 находится в левом поддереве 7, но меньше 7.

---

## Идея решения (inorder-обход)

Ключевое свойство BST: **inorder-обход** (левое → корень → правое) даёт строго возрастающую последовательность. Если при обходе мы встретим нарушение порядка (текущий элемент ≤ предыдущего), то дерево не является BST.

Используется итеративный обход с помощью **стека**, который позволяет обойти дерево без рекурсии, что экономит память и избегает переполнения стека для глубоких деревьев.

### Алгоритм

1. Инициализировать пустой стек `st`, указатель `cur = root` и переменную `prev = -∞` (минимально возможное значение).
2. Пока `cur` не `None` или стек не пуст:
   - Идём по левым потомкам, помещая узлы в стек, пока не дойдём до `None`.
   - Извлекаем узел из стека (`cur = st.pop()`).
   - Если `prev >= cur.data`, то порядок нарушен — возвращаем `False`.
   - Обновляем `prev = cur.data`.
   - Переходим к правому поддереву: `cur = cur.right`.
3. Если обход завершён без нарушений — возвращаем `True`.

---

## Реализация

```python
class Solution:
    def isBST(self, root):
        stack = []
        cur = root
        prev = float('-inf')
        while stack or cur:
            while cur:
                stack.append(cur)
                cur = cur.left
            cur = stack.pop()
            if prev >= cur.data:
                return False
            prev = cur.data
            cur = cur.right
        return True
```

---

## Сложность

- **Время:** O(n), где n — количество узлов. Каждый узел посещается один раз.
- **Память:** O(h), где h — высота дерева (глубина стека). В сбалансированном дереве O(log n), в худшем случае (вырожденном) O(n).

---

## Примечания

- Алгоритм работает для уникальных значений, но также корректен для дубликатов, если по определению BST не допускает равенства (строго меньше/больше). При равенстве возвращается `False`.
- Рекурсивное решение может быть короче, но итеративный подход предпочтительнее для больших деревьев из-за контроля над памятью.
- Альтернативный метод: передача допустимых диапазонов `(min, max)` для каждого узла при обходе (рекурсивно). Он также работает за O(n) и O(h) памяти, но требует проверки границ.
