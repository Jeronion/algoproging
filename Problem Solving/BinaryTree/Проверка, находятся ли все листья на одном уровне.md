## Определение

В бинарном дереве все листовые узлы (узлы, не имеющие детей) должны находиться на **одинаковой глубине** (уровне), чтобы дерево считалось удовлетворяющим условию. Требуется проверить, так ли это.

**Пример:**
```
        12
       /  \
      5    7
     /      \
    3        1
```
Листья: 3 и 1 — оба на глубине 2 → true.

```
        12
       /  \
      5    7
     /
    3
```
Листья: 3 (глубина 2) и 7 (глубина 1) → false.

---

## Постановка задачи

Дан корень бинарного дерева. Необходимо вернуть `true`, если все листья находятся на одной глубине, и `false` в противном случае. Если дерево пустое или содержит только один узел, возвращать `true` (по условию задачи корень может быть листом).

---

## Идея решения (DFS с запоминанием глубины)

Используем **обход в глубину (DFS)**, передавая текущую глубину. При обнаружении первого листа запоминаем его глубину в переменной `depth`. Для каждого следующего листа сравниваем его глубину с запомненной. Если хотя бы один лист имеет другую глубину, устанавливаем флаг `depth = False`.

### Алгоритм

1. Инициализировать переменную `depth = None`.
2. Рекурсивная функция `dfs(node, dist)`:
   - Если `node` равен `None` — возврат.
   - Если `node` является листом (нет левого и правого потомка):
     - Если `depth` ещё не установлена, сохраняем `dist`.
     - Иначе если `depth != dist`, устанавливаем `depth = False`.
   - Рекурсивно вызываем `dfs(node.left, dist + 1)` и `dfs(node.right, dist + 1)`.
3. После обхода, если `depth` равен `False` — возвращаем `False`, иначе `True` (так как `bool(depth)` даёт `True` для любого числа и `False` только для `False` и `None`).

---

## Реализация

```python
class Solution:
    def check(self, root):
        depth = None
        
        def dfs(node, dist):
            nonlocal depth
            if node is None:
                return
            if node.left is None and node.right is None:
                if depth is None:
                    depth = dist
                elif depth != dist:
                    depth = False
            dfs(node.left, dist + 1)
            dfs(node.right, dist + 1)
        
        dfs(root, 1)   # начальная глубина = 1
        return bool(depth)   # если depth == False → False, иначе True
```

---

## Сложность

- **Время:** O(n) — каждый узел посещается ровно один раз.
- **Память:** O(h) — глубина стека рекурсии (h — высота дерева). В худшем случае для вырожденного дерева O(n).

---

## Примечания

- При обнаружении первого листа `depth` сохраняет его глубину. При последующих сравнениях, если глубина отличается, `depth` становится `False`, что сигнализирует о нарушении условия.
- Функция `bool(depth)` возвращает `False`, если `depth` равен `False` (или `None`, но после обхода `depth` никогда не остаётся `None`, так как есть хотя бы один лист). Если `depth` — число, возвращает `True`.
- Начальная глубина установлена в `1`, что корректно для корня на уровне 1. Можно также использовать `0` и сравнивать одинаково.
