## Определение

Два бинарных дерева называются **зеркальными**, если они являются зеркальным отражением друг друга относительно вертикальной оси. Это означает, что для каждого узла в первом дереве его левое поддерево соответствует правому поддереву узла во втором дереве, и наоборот.

Задача: по корневым узлам `a` и `b` определить, являются ли деревья зеркальными.

---

## Идея решения (рекурсивный обход)

Используем рекурсивную функцию, которая одновременно обходит оба дерева в зеркально‑симметричном порядке:

1. Если оба узла `None` → возвращаем `True`.
2. Если один из узлов `None`, а другой нет → возвращаем `False`.
3. Если значения узлов не совпадают → возвращаем `False`.
4. Иначе рекурсивно проверяем:
   - левое поддерево первого с правым поддеревом второго: `dfs(a.left, b.right)`
   - правое поддерево первого с левым поддеревом второго: `dfs(a.right, b.left)`

Если все рекурсивные вызовы возвращают `True`, то деревья зеркальны.

---

## Реализация

```python
class Solution:
    def areMirror(self, a, b):
        def dfs(node1, node2):
            if node1 is None and node2 is None:
                return True
            if node1 is None or node2 is None:
                return False
            if node1.data != node2.data:
                return False
            return dfs(node1.left, node2.right) and dfs(node1.right, node2.left)
        
        return dfs(a, b)
```

---

## Сложность

- **Время:** O(n), где n — количество узлов в меньшем дереве. Каждый узел посещается один раз.
- **Память:** O(h), где h — высота дерева (глубина рекурсии). В худшем случае для вырожденного дерева h = n.

---

## Примечания

- Алгоритм работает для деревьев с любыми значениями (сравнение данных).
- Рекурсивный подход естественен, но для глубоких деревьев можно использовать итеративный обход с явным стеком.
