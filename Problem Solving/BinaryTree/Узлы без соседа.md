## Определение

В бинарном дереве два узла называются соседями (братьями/сёстрами), если они имеют одного родителя. Узел, являющийся единственным ребёнком своего родителя, не имеет соседа. Требуется найти все такие узлы и вернуть их значения в порядке возрастания. Корень не включается, так как у него нет родителя.

**Примеры:**
- Дерево `[37, 20, N, N, 113]` → узлы 20 и 113 не имеют соседей → `[20, 113]`.
- Дерево `[1, 2, 3, 4]` → узел 4 не имеет соседа → `[4]`.
- Дерево `[37, 40, 13]` → все узлы имеют соседа → `[-1]`.

---

## Постановка задачи

Дан корень бинарного дерева. Найти все узлы, которые не имеют соседа. Вернуть их значения в порядке возрастания. Если таких узлов нет, вернуть `[-1]`.

---

## Идея решения (DFS)

Используется обход в глубину (DFS). Для каждого узла проверяем, есть ли у него ровно один ребёнок. Если да, то этот ребёнок не имеет соседа – добавляем его значение в результат. Затем рекурсивно обходим детей.

### Алгоритм

1. Инициализировать пустой список `res`.
2. Рекурсивная функция `dfs(node)`:
   - Если `node` равен `None` – завершить.
   - Если у `node` есть только левый ребёнок (и нет правого), добавить `node.left.data` в `res`.
   - Если у `node` есть только правый ребёнок (и нет левого), добавить `node.right.data` в `res`.
   - Рекурсивно вызвать `dfs(node.left)` и `dfs(node.right)`.
3. Если `res` не пуст, отсортировать и вернуть, иначе вернуть `[-1]`.

---

## Реализация

```python
class Solution:
    def noSibling(self, root):
        res = []
        def dfs(node):
            if node is None:
                return
            # если у узла есть только левый ребёнок
            if node.left and not node.right:
                res.append(node.left.data)
            # если только правый
            if node.right and not node.left:
                res.append(node.right.data)
            dfs(node.left)
            dfs(node.right)
        
        dfs(root)
        return sorted(res) if res else [-1]
```

---

## Сложность

- **Время:** O(n), где n – количество узлов. Каждый узел посещается ровно один раз.
- **Память:** O(h) для стека рекурсии (h – высота дерева) плюс O(k) для результата, где k – количество узлов без соседа. В худшем случае O(n).

---

## Примечания

- Корень не проверяется на наличие соседа (у него нет родителя).
- Дерево может быть пустым – тогда возвращаем `[-1]`.
- Используется обход в глубину, но можно также использовать BFS.
- Узлы без соседа собираются в список, затем сортируются, так как порядок не гарантируется.
