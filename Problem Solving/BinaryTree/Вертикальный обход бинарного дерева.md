## Определение

Вертикальный обход бинарного дерева — это упорядочивание узлов по их **горизонтальному расстоянию** от корня. Корень имеет расстояние `0`. Левый потомок получает `расстояние - 1`, правый — `расстояние + 1`. Узлы с одинаковым расстоянием находятся на одной вертикальной линии. Требуется вывести все узлы, сгруппированные по вертикальным линиям, начиная с самой левой (минимальное расстояние) до самой правой (максимальное расстояние). Внутри одной вертикальной линии узлы выводятся в порядке **level-order** (по уровням сверху вниз).

**Пример:**
```
        1
       / \
      2   3
     / \   \
    4   5   6
```
Вертикальные линии:
- расстояние -2: $[4]$
- расстояние -1: $[2]$
- расстояние 0: $[1, 5]$
- расстояние 1: $[3]$
- расстояние 2: $[6]$
Результат: $[[4], [2], [1, 5], [3], [6]]$

## Идея решения (BFS с хеш-таблицей)

Используем **обход в ширину (BFS)** с очередью, чтобы гарантировать, что узлы на каждой вертикали будут обработаны в порядке уровней (сверху вниз). Для каждого узла храним его горизонтальное расстояние `dist`. При посещении узла добавляем его значение в словарь `hm[dist]`. Также отслеживаем минимальное и максимальное расстояние, чтобы в конце пройти по всем вертикалям в правильном порядке.

### Алгоритм

1. Создать `deque` и поместить в него корень с расстоянием `0`.
2. Создать словарь `hm`, где ключ — расстояние, значение — список узлов.
3. Инициализировать `mn = 0`, `mx = 0` для границ расстояний.
4. Пока очередь не пуста:
   - Извлечь узел и его расстояние `dist`.
   - Добавить `node.data` в `hm[dist]`.
   - Если есть левый потомок, добавить его в очередь с расстоянием `dist - 1`, обновить `mn`.
   - Если есть правый потомок, добавить его в очередь с расстоянием `dist + 1`, обновить `mx`.
5. После обхода сформировать результат: для `i` от `mn` до `mx` включительно добавить `hm[i]` в результирующий список.
6. Вернуть результат.

## Реализация

```python
from collections import deque, defaultdict

class Solution:
    def verticalOrder(self, root):
        if not root:
            return []
        queue = deque([(root, 0)])
        hm = defaultdict(list)
        mn, mx = 0, 0
        
        while queue:
            node, dist = queue.popleft()
            hm[dist].append(node.data)
            
            if node.left:
                mn = min(mn, dist - 1)
                queue.append((node.left, dist - 1))
            if node.right:
                mx = max(mx, dist + 1)
                queue.append((node.right, dist + 1))
        
        res = []
        for i in range(mn, mx + 1):
            res.append(hm[i])
        return res
```

## Сложность

- **Время:** O(n), где n — количество узлов. Каждый узел посещается ровно один раз.
- **Память:** O(n) для хранения словаря и очереди.

---

## Примечания

- BFS гарантирует, что внутри одной вертикали узлы будут расположены в порядке сверху вниз (level-order), что соответствует условию задачи.
- Альтернативный подход с DFS также возможен, но тогда необходимо сортировать узлы внутри каждой вертикали по уровню, что усложняет код.
- Словарь `defaultdict(list)` упрощает добавление элементов, не требуя проверки существования ключа.
