## Определение

**Наименьший общий предок (LCA)** двух узлов `n1` и `n2` в бинарном дереве — это самый глубокий узел, который является предком обоих узлов (т.е. находится на пути от корня к каждому из них). При этом узел может быть предком самого себя (если один из узлов является предком другого).

Задача: по корню `root` и значениям двух узлов `n1` и `n2` (гарантируется, что оба присутствуют в дереве) найти узел LCA.

---

## Постановка задачи

Дан корень бинарного дерева с уникальными значениями и два узла `n1` и `n2`. Вернуть узел, являющийся их наименьшим общим предком.

**Пример:**
```
        1
       / \
      2   3
     / \   \
    4   5   6
```
- LCA(4,5) = 2
- LCA(4,6) = 1
- LCA(2,4) = 2 (так как 2 является предком 4)

---

## Идея решения (рекурсивный DFS)

Используется **обход в глубину (DFS)**. Рекурсивно обходим дерево: для каждого узла проверяем, совпадает ли его значение с `n1` или `n2`. Если да, возвращаем этот узел. Иначе рекурсивно ищем в левом и правом поддеревьях.

- Если оба поддерева возвращают не-`None`, значит текущий узел является LCA (потому что один узел найден в левом поддереве, другой — в правом).
- Если только одно поддерево вернуло не-`None`, то LCA находится в этом поддереве (или это один из узлов, который был найден).
- Если оба вернули `None`, то в этом поддереве нет ни одного из искомых узлов.

Этот алгоритм работает за O(n) времени и O(h) памяти (глубина рекурсии), где n — количество узлов, h — высота дерева.

---

## Реализация

```python
class Node:
    def __init__(self, val):
        self.data = val
        self.left = None
        self.right = None

class Solution:
    def lca(self, root, n1, n2):
        def dfs(node):
            if node is None:
                return None
            # Если текущий узел равен одному из искомых, возвращаем его
            if node.data == n1 or node.data == n2:
                return node
            
            left = dfs(node.left)
            right = dfs(node.right)
            
            # Если оба поддерева вернули не-None, текущий узел — LCA
            if left and right:
                return node
            # Иначе возвращаем непустое поддерево
            return left if left else right
        
        return dfs(root)
```

---

## Сложность

- **Время:** O(n), где n — количество узлов. Каждый узел посещается один раз.
- **Память:** O(h), где h — высота дерева (глубина стека рекурсии). В худшем случае для вырожденного дерева O(n).

---

## Примечания

- Алгоритм предполагает, что оба узла присутствуют в дереве (по условию задачи).
- Если один из узлов является предком другого, то `dfs` вернёт этот узел (предка), так как при достижении его он сразу возвращается, и рекурсия в его поддереве не продолжается. Это корректно, так как LCA в этом случае является предок.
- Значения узлов уникальны, поэтому сравнение по значению корректно.
- Альтернативные подходы: можно найти пути от корня к каждому узлу и затем найти последний общий узел в этих путях (требует O(n) памяти и времени). Рекурсивный метод более эффективен по памяти.
