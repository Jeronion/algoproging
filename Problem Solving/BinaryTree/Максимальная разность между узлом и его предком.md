## Определение

Дано бинарное дерево, где каждый узел содержит целое число. Требуется найти **максимальное значение** разности `value(A) - value(B)`, где `A` — предок `B` (то есть `A` находится на пути от корня к `B`, и `A` строго выше `B`). Если таких пар несколько, нужно вернуть максимальную разность; гарантируется, что дерево содержит как минимум два узла (по условию, число рёбер ≥ 2).

**Пример:**
```
       5
      / \
     2   1
```
Потомки для узла 5: 2 и 1. Разности: 5-2=3, 5-1=4 → максимум = 4.

```
       1
      / \
     2   3
          \
           7
```
Возможные пары: (1,2) → -1, (1,3) → -2, (1,7) → -6, (3,7) → -4. Максимальная = -1.

---

## Постановка задачи

Дан корень бинарного дерева `root`. Найти максимальное значение `node.data - descendant.data`, где `descendant` — любой узел в поддереве `node`, не равный самому `node`. Если таких пар нет (дерево из одного узла), то результат не определён, но по условию узлов ≥ 2.

---

## Идея решения

Используется **постордерный обход** (left → right → root). Для каждого узла мы вычисляем **минимальное значение среди всех узлов в его поддереве** (включая сам узел). Затем для текущего узла (как потенциального предка `A`) мы можем вычислить разность `current.data - min_value_in_subtree`, где `min_value_in_subtree` берётся из поддеревьев потомков (но не включая сам узел). Однако в нашем коде мы вычисляем минимум среди левого, правого поддерева и самого узла, и затем обновляем ответ разностью `root.data - min(l, r)`, где `l` и `r` — минимальные значения в левом и правом поддеревьях. Это эквивалентно нахождению минимума среди всех потомков, так как `min(l, r)` даёт минимум среди всех узлов в левом и правом поддеревьях. Затем для родительских узлов мы возвращаем минимум из `l, r, root.data`, чтобы вышестоящие узлы могли использовать это значение.

### Почему это работает?

- Для каждого узла мы знаем минимальное значение в его поддереве (включая его самого). Это позволяет для любого предка вычислить максимальную разность с минимальным потомком.
- Так как мы хотим максимизировать разность, логично выбирать в качестве `B` узел с наименьшим значением среди всех потомков `A`.
- Обход выполняется снизу вверх, поэтому к моменту обработки узла все поддеревья уже обработаны, и мы знаем их минимумы.

## Реализация

```python
class Solution:
    def maxDiff(self, root):
        mx = float("-inf")
        
        def dfs(node):
            nonlocal mx
            if node is None:
                return float("inf")
            # лист
            if node.left is None and node.right is None:
                return node.data
            
            left_min = dfs(node.left)
            right_min = dfs(node.right)
            
            # обновляем максимум разности
            mx = max(mx, node.data - min(left_min, right_min))
            
            # возвращаем минимальное значение в поддереве
            return min(left_min, right_min, node.data)
        
        dfs(root)
        return mx
```

### Пояснение к коду

- `dfs` возвращает минимальное значение в поддереве текущего узла.
- Для пустого поддерева возвращается `inf`, чтобы оно не влияло на минимум.
- Для листового узла возвращается его значение (минимальное в поддереве — это он сам).
- Для внутреннего узла:
  1. Рекурсивно получаем минимумы левого и правого поддеревьев.
  2. Вычисляем потенциальную разность: `node.data - min(left_min, right_min)`. Это соответствует выбору наименьшего потомка.
  3. Обновляем глобальный максимум.
  4. Возвращаем минимум среди `left_min`, `right_min` и `node.data`.

---

## Сложность

- **Время:** O(n), где n — количество узлов. Каждый узел посещается один раз.
- **Память:** O(h), где h — высота дерева (глубина стека рекурсии). В худшем случае (вырожденное дерево) O(n).

---

## Примечания

- Если все разности отрицательны, ответом будет максимальная (наименьшая по модулю) отрицательная разность. Это обрабатывается корректно, так как `mx` инициализируется `-inf` и обновляется на всех узлах, кроме листьев (но листья не могут быть предками). Для дерева из двух узлов (корень и потомок) ответ будет разность между ними.
- Значения узлов могут быть равны 0, поэтому `float("-inf")` подходит для инициализации.
- В коде используется `nonlocal mx`, чтобы изменять переменную внутри вложенной функции.
- Альтернативный подход: можно передавать текущий минимум сверху вниз, но постордерный метод позволяет избежать передачи параметров.
