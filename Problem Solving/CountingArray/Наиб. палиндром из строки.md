## Условие

Дана последовательность из **N** заглавных латинских букв (N ≤ 100 000). Разрешается переставлять буквы и удалять некоторые. Требуется составить **палиндром наибольшей длины**. Если таких несколько, выбрать первый в **алфавитном порядке**.

**Ограничение:** алгоритм должен работать за O(N), запрещено использовать встроенные сортировки.

---

## Идея решения

1. **Подсчёт частот** — проходим по строке и считаем, сколько раз встречается каждая буква (массив на 26 элементов).
2. **Формирование половины палиндрома** — для каждой буквы берём `count // 2` экземпляров и добавляем их в строку `seq`. Эти символы составят левую половину будущего палиндрома.
   - Например, для буквы `A` с частотой 3 берём `3 // 2 = 1` символ.
3. **Выбор центрального символа** — среди букв, у которых остался неиспользованный символ (нечётное количество), выбираем **первую в алфавитном порядке**. Если такой буквы нет, центр пуст.
4. **Построение ответа** — `seq` + центральный символ + `seq` в обратном порядке.

### Почему это даёт максимальную длину?
- Мы используем все возможные пары букв, так как каждая пара даёт два символа в палиндроме. Количество пар ограничено половиной чётных вхождений. Это обеспечивает максимально возможную длину.
- Выбор наименьшего центрального символа гарантирует первый палиндром в алфавитном порядке (поскольку левая половина уже минимальна из-за того, что `seq` строится в алфавитном порядке).

---

## Сложность
- **Время:** O(N) — один проход для подсчёта, затем константные операции (обход 26 букв).
- **Память:** O(1) — массив фиксированного размера плюс строка для результата.

---

## Реализация на Python

```python
def solve(s: string):
    cnt = [0] * 26
    for ch in s:
        cnt[ord(ch) - ord('A')] += 1

    left = []
    for i in range(26):
        if cnt[i] >= 2:
            left.append(chr(i + ord('A')) * (cnt[i] // 2))
        cnt[i] %= 2            # оставляем только остаток (0 или 1)

    center = ''
    for i in range(26):
        if cnt[i] == 1:
            center = chr(i + ord('A'))
            break

    left_str = ''.join(left)
    print(left_str + center + left_str[::-1])
```

### Пояснения к коду
- `left` — список строк, каждая из которых состоит из повторяющейся буквы. Собираем в алфавитном порядке.
- После использования чётных количеств обновляем `cnt[i]` (остаток 0 или 1).
- Центр — первый встреченный остаток (он же наименьший по алфавиту).
- Финальный вывод — склеивание левой части, центра и перевёрнутой левой.
