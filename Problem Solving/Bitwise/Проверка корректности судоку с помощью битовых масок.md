## Определение

Дана частично заполненная доска судоку размером 9×9, где `0` обозначает пустую клетку. Требуется проверить, является ли текущая конфигурация **валидной**, то есть не нарушает ли она правила судоку:
- в каждой строке цифры от 1 до 9 не повторяются,
- в каждом столбце цифры не повторяются,
- в каждом из 9 подквадратов 3×3 цифры не повторяются.

Пустые клетки (0) игнорируются. Валидность текущей конфигурации **не гарантирует**, что судоку имеет решение — она лишь проверяет отсутствие очевидных конфликтов.

---

## Идея решения (битовая маска)

Вместо использования массивов для отслеживания встреченных цифр (как в наивном подходе), можно применить **битовые маски**: каждому числу от 1 до 9 соответствует один бит в 9-битном числе (например, бит 0 для 1, бит 1 для 2, ..., бит 8 для 9). Для каждой строки, столбца и подквадрата хранится целое число (int), биты которого показывают, какие цифры уже встречались.

- Проверка, встречалась ли цифра `d` (1..9): `(mask & (1 << (d-1))) != 0`
- Отметка цифры как встреченной: `mask |= (1 << (d-1))`

Алгоритм:
1. Создать три массива по 9 элементов для строк, столбцов и подквадратов, заполненные нулями.
2. Пройти по всем клеткам (i, j).
3. Если клетка пуста (`mat[i][j] == 0`), пропустить.
4. Вычислить бит для текущего числа: `bit = 1 << (mat[i][j] - 1)`.
5. Проверить, установлен ли этот бит в маске текущей строки `row[i]`. Если да → вернуть `False`.
6. Установить бит в `row[i]`.
7. Аналогично для столбца `col[j]` и для подквадрата `sub[(i//3)*3 + (j//3)]`.
8. Если все клетки обработаны без конфликта, вернуть `True`.

## Реализация на Python

```python
class Solution:
    def isValid(self, mat):
        # Маски для строк, столбцов и подквадратов (9 бит)
        rows = [0] * 9
        cols = [0] * 9
        subs = [0] * 9

        for i in range(9):
            for j in range(9):
                num = mat[i][j]
                if num == 0:
                    continue
                bit = 1 << (num - 1)
                # Проверка строки
                if rows[i] & bit:
                    return False
                rows[i] |= bit
                # Проверка столбца
                if cols[j] & bit:
                    return False
                cols[j] |= bit
                # Проверка подквадрата
                idx = (i // 3) * 3 + j // 3
                if subs[idx] & bit:
                    return False
                subs[idx] |= bit
        return True
```

## Сложность

- **Время:** O(81) = O(1), так как доска фиксированного размера (9×9). По сути, константа.
- **Память:** O(1) — три массива по 9 целых чисел.

---

## Примечания

- Метод эффективно использует битовые операции, что быстрее и экономичнее по памяти, чем использование множеств или списков.
- Подквадраты нумеруются с 0 до 8: `(i // 3) * 3 + j // 3`.
- Если число `num` равно 0 (пустая клетка), пропускаем.
- Если в какой-то строке/столбце/подквадрате встречается дубликат, возвращаем `False` сразу.
