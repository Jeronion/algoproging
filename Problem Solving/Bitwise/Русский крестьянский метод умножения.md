## Определение

**Русский крестьянский метод** — это алгоритм умножения двух целых чисел, использующий только операции сложения, удвоения и деления на два (сдвиги). Он не использует оператор умножения и основан на следующем свойстве:

- Если $b$ чётное: $a \times b = (a \times 2) \times (b / 2)$
- Если $b$ нечётное: $a \times b = (a \times 2) \times (b / 2) + a$

Алгоритм многократно удваивает $a$ и делит $b$ пополам (целочисленно), накапливая сумму, когда $b$ нечётно.

---

## Алгоритм

1. Инициализировать `result = 0`.
2. Пока `b > 0`:
   - Если `b` нечётное (`b & 1 == 1`), добавить `a` к `result`.
   - Удвоить `a` (`a <<= 1`).
   - Разделить `b` на 2 (`b >>= 1`).
3. Вернуть `result`.

---

## Реализация на Python

```python
def multiply(a, b):
    result = 0
    while b > 0:
        if b & 1:
            result += a
        a <<= 1
        b >>= 1
    return result

# Пример
print(multiply(2, 5))  # 10
print(multiply(6, 9))  # 54
```

---

## Сложность

- **Время:** O(log b) — количество итераций равно количеству битов в `b`.
- **Память:** O(1) — только несколько переменных.

---

## Примечания

- Алгоритм работает для неотрицательных чисел. Для отрицательных можно взять абсолютные значения и скорректировать знак.
- Название связано с тем, что метод был известен в Древнем Египте и использовался русскими крестьянами.
- Эквивалентен умножению через двоичное разложение: $a \times b = a \times \sum (b_i \cdot 2^i) = \sum (a \cdot 2^i) \cdot b_i$.
