## Определение

Дан массив целых чисел `arr` размера `n`. Требуется найти сумму значений XOR **всех возможных непустых подмножеств** (включая одноэлементные). XOR пустого подмножества обычно не учитывается.

**Пример:**
```
arr = [1, 2, 3]
Подмножества:
{1} → 1
{2} → 2
{3} → 3
{1,2} → 3
{1,3} → 2
{2,3} → 1
{1,2,3} → 0
Сумма = 1+2+3+3+2+1+0 = 12
```

---

## Идея решения (битовый вклад)

Каждый бит (позиция `i`) вносит вклад в общую сумму независимо. Для данного бита:

- Пусть `k` — количество элементов массива, у которых этот бит установлен.
- Если `k = 0`, бит никогда не появится в XOR, вклад = 0.
- Если `k > 0`, то бит будет установлен в XOR подмножества тогда и только тогда, когда в подмножество входит **нечётное количество** элементов с установленным битом.

Количество способов выбрать нечётное количество элементов из `k` равно `2^(k-1)`.  
Количество способов выбрать любое количество элементов из `n-k` элементов с нулевым битом равно `2^(n-k)`.  
Таким образом, общее количество подмножеств, в которых бит будет установлен, равно `2^(n-k) * 2^(k-1) = 2^(n-1)`.

**Удивительно, но результат не зависит от `k` (лишь бы `k > 0`)!**  
Следовательно, каждый бит, который присутствует хотя бы в одном элементе массива, вносит вклад `2^(n-1) * 2^i`.

Поэтому общая сумма равна:
```
(побитовое ИЛИ всех элементов) * 2^(n-1)
```

---

## Алгоритм

1. Вычислить `bits = OR` всех элементов массива.
2. Вернуть `bits * (1 << (n-1))`.

---

## Реализация на Python

```python
def subset_xor_sum(arr):
    n = len(arr)
    bits = 0
    for num in arr:
        bits |= num
    return bits * (1 << (n - 1))

# Пример
arr = [1, 2, 3]
print(subset_xor_sum(arr))  # 12
```

---

## Сложность

- **Время:** O(n) — один проход для вычисления OR.
- **Память:** O(1) — несколько переменных.

---

## Примечания

- Формула работает для любых целых чисел, включая отрицательные (в Python битовое представление отрицательных чисел бесконечно длинное, но на практике OR даёт корректный результат).
- В случае пустого массива (`n=0`) формула даёт `0`, но по условию обычно `n >= 1`.
- Этот результат можно доказать через комбинаторные рассуждения или через линейность ожидания (каждый бит входит в половину всех подмножеств, если он есть хотя бы в одном элементе).
