## Определение

**Код Грея** (Gray code) — это упорядоченная последовательность всех $2^n$ двоичных строк длины $n$, в которой соседние строки отличаются ровно в одном бите. Также называется отражённым двоичным кодом.

**Пример для n = 2:** `"00"`, `"01"`, `"11"`, `"10"`.

---

## Постановка задачи

Дано целое число `n` ($1 \le n \le 16$). Требуется сгенерировать все $2^n$ n-битных кодов Грея в правильном порядке. Каждый код должен быть представлен в виде строки из `'0'` и `'1'`.

---

## Идея решения (битовая манипуляция)

Коды Грея можно сгенерировать с помощью простой формулы для **i-го кода Грея** (где $0 \le i < 2^n$):

$$
G(i) = i \oplus (i \gg 1)
$$

Здесь $\oplus$ — побитовое XOR, $\gg$ — сдвиг вправо.

**Почему это работает?**
- При сдвиге `i >> 1` все биты смещаются вправо, младший бит теряется, а слева добавляется 0.
- XOR с исходным `i` приводит к тому, что все последовательные одинаковые биты (которые были подряд в `i` и `i>>1`) взаимно уничтожаются (дают 0), а остаются только те биты, где произошло изменение. Это создаёт последовательность, в которой каждое следующее число отличается от предыдущего ровно в одном бите.

Таким образом, достаточно пройти `i` от `0` до `2^n - 1`, вычислить `g = i ^ (i >> 1)` и преобразовать результат в n-битную двоичную строку (с ведущими нулями).

---

## Алгоритм

1. Инициализировать пустой список `res`.
2. Для `i` от `0` до `(1 << n) - 1`:
   - Вычислить `g = i ^ (i >> 1)`.
   - Преобразовать `g` в двоичную строку длины `n` (дополнив нулями слева).
   - Добавить строку в `res`.
3. Вернуть `res`.

---

## Реализация на Python

```python
class Solution:
    def graycode(self, n):
        res = []
        for i in range(1 << n):
            g = i ^ (i >> 1)
            # Формируем n-битную строку
            s = ""
            for j in range(n - 1, -1, -1):
                s += '1' if (g >> j) & 1 else '0'
            res.append(s)
        return res
```

Более компактный вариант с `format`:

```python
def graycode(n):
    return [format(i ^ (i >> 1), '0{}b'.format(n)) for i in range(1 << n)]
```

## Сложность

- **Время:** O(2^n · n) — каждое из $2^n$ чисел преобразуется в строку длины n.
- **Память:** O(2^n · n) для хранения результата (обязательно, так как нужно вернуть все строки).

---

## Пример

Для `n = 3`:

| i | i >> 1 | i ^ (i>>1) | двоичная (3 бита) |
|---|--------|------------|-------------------|
| 0 | 0      | 0          | 000               |
| 1 | 0      | 1          | 001               |
| 2 | 1      | 3          | 011               |
| 3 | 1      | 2          | 010               |
| 4 | 2      | 6          | 110               |
| 5 | 2      | 7          | 111               |
| 6 | 3      | 5          | 101               |
| 7 | 3      | 4          | 100               |

Результат: `["000","001","011","010","110","111","101","100"]`.

---

## Примечания

- Код Грея широко применяется в цифровой логике (для уменьшения ошибок при переключении состояний), в алгоритмах генерации перестановок, в задачах «Ханойские башни» и т.д.
- Существует также рекурсивный метод построения (отражение), но битовый вариант более эффективен и прост.
- При `n` до 16 результат помещается в память (2^16 * 16 ≈ 1 млн символов).
