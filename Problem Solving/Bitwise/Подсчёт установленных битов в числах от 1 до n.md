## Определение

Дано целое число `n`. Требуется найти **суммарное количество единичных битов** в двоичной записи всех чисел от 1 до `n` включительно.

**Пример:**
- `n = 4` → числа: 1 (001), 2 (010), 3 (011), 4 (100) → битов: 1+1+2+1 = 5.

---

## Идея решения (поразрядный подсчёт)

Подсчёт ведётся **по каждому биту** независимо. Для каждого бита (позиции) выясняем, сколько чисел в диапазоне `[1, n]` имеют этот бит равным 1.

Рассмотрим `i`-й бит (считая с 0, младший бит). Период повторения шаблона для этого бита — `2^(i+1)`. В каждом полном периоде:
- Первые `2^i` чисел имеют бит = 0,
- Следующие `2^i` чисел имеют бит = 1.

Таким образом, количество единиц в одном полном периоде = `2^i`.

Для чисел от 0 до `n` (включительно) формула для количества единиц в `i`-м бите:

```
total = (n+1) // (2^(i+1)) * (2^i) + max(0, (n+1) % (2^(i+1)) - (2^i))
```

Здесь:
- `(n+1)` — количество чисел от 0 до n.
- `(n+1) // (2^(i+1))` — количество полных периодов.
- `(n+1) % (2^(i+1))` — остаток, дающий неполный период.
- Если остаток превышает `2^i`, то добавляем `остаток - 2^i` единиц.

Так как мы считаем от 1 до n, а формула для от 0 до n, результат не меняется, потому что 0 не даёт единиц.

---

## Реализация

```python
class Solution:
    def countSetBits(self, n):
        i = 1          # i = позиция бита + 1 (1 << i — это 2^(i))
        total = 0
        while (1 << (i - 1)) <= n:
            period = 1 << i          # 2^i
            half_period = 1 << (i - 1)  # 2^(i-1)
            full_cycles = (n + 1) // period
            remainder = (n + 1) % period
            total += full_cycles * half_period + max(0, remainder - half_period)
            i += 1
        return total
```

---

## Сложность

- **Время:** O(log n) — количество битов в числе n (не более 31 для 10^9).
- **Память:** O(1).

---

## Примечания

- Алгоритм также известен как **метод подсчёта битов по шаблонам**.
- Вместо цикла по битам можно использовать встроенную функцию `bit_count()` для каждого числа, но это было бы O(n log n).
- Данный метод оптимален для больших n (до 10^8 и выше).
