## Определение

Дан массив целых чисел `arr` длины `n`. Рассмотрим все **непрерывные подмассивы** (subarrays). Для каждого подмассива вычислим побитовый XOR его элементов, а затем вычислим XOR всех этих результатов. Требуется найти итоговое значение.

**Пример:**
```
arr = [1, 2, 3]
Подмассивы:
[1] → 1
[2] → 2
[3] → 3
[1,2] → 3
[2,3] → 1
[1,2,3] → 0
XOR результатов = 1 ^ 2 ^ 3 ^ 3 ^ 1 ^ 0 = 2
```

---

## Идея решения (contribution)

Как и для суммы всех подмассивов, каждый элемент `arr[i]` входит в `(i+1)*(n-i)` подмассивов. В операции XOR элемент учитывается **столько раз**, сколько подмассивов его содержат. Поскольку `x ^ x = 0`, элемент даёт вклад в финальный XOR тогда и только тогда, когда он входит в **нечётное** количество подмассивов.

Следовательно, в итоговый XOR нужно включить `arr[i]`, если `(i+1)*(n-i)` нечётно.

Произведение двух чисел нечётно ⇔ оба множителя нечётны. Значит, условие:

- `(i+1)` нечётно → `i` чётное (0,2,4,...)
- `(n-i)` нечётно → `i` и `n` разной чётности.

Таким образом, для чётного `n` второй множитель всегда чётен (`n-i` чётно при любом `i`), поэтому ни один индекс не подходит → **результат = 0**.

Для нечётного `n` второй множитель нечётен, когда `i` чётное. Поэтому подходят все чётные индексы `i = 0, 2, 4, ..., n-1`.

---

## Упрощённая формула

Если `n` чётное → `result = 0`.

Если `n` нечётное → `result = arr[0] ^ arr[2] ^ arr[4] ^ ... ^ arr[n-1]` (все элементы с чётными индексами).

## Реализации

### Полная версия (проверка каждого индекса)

```python
def subarray_xor_full(arr):
    n = len(arr)
    result = 0
    for i in range(n):
        if ((i + 1) % 2 == 1) and ((n - i) % 2 == 1):
            result ^= arr[i]
    return result
```

### Оптимизированная версия (только чётные индексы при нечётном n)

```python
class Solution:
    def subarrayXor(self, arr):
        n = len(arr)
        if n % 2 == 0:
            return 0
        xor = 0
        for i in range(0, n, 2):
            xor ^= arr[i]
        return xor
```

---

## Сложность

- **Время:** O(n) — один проход по массиву.
- **Память:** O(1).

---

## Пример

`arr = [1, 2, 3, 4, 5]` (n=5, нечётное)

- Чётные индексы: 0,2,4 → элементы 1,3,5.
- `1 ^ 3 = 2`, `2 ^ 5 = 7`.
- Результат = 7.

---

## Примечания

- Метод основан на технике **subarray contribution** и свойстве XOR.
- Решение не требует перебора всех подмассивов и работает за линейное время.
- В приведённой оптимизированной версии достаточно проверить только чётность `n`, что делает код максимально простым.
