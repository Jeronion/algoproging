## Определение

**Расстояние Хэмминга** между двумя целыми числами — это количество позиций, в которых соответствующие биты этих чисел различаются. Иными словами, это число единичных битов в результате побитового XOR (исключающего ИЛИ) двух чисел.

---

## Постановка задачи

Даны два целых числа `n1` и `n2`. Требуется найти расстояние Хэмминга между ними.

**Примеры:**
- `n1 = 9` (двоичное `1001`), `n2 = 14` (`1110`) → различаются в 3 битах → ответ `3`.
- `n1 = 4` (`0100`), `n2 = 8` (`1000`) → различаются в 2 битах → ответ `2`.

---

## Идея решения

1. Вычислить `x = n1 ^ n2` (побитовое XOR). В результате единичные биты будут на тех позициях, где исходные числа различаются.
2. Подсчитать количество единичных битов в `x`. Это и есть расстояние Хэмминга.

Подсчёт битов можно выполнить последовательным сдвигом или с помощью встроенной функции `bit_count()` (Python 3.8+).

---

## Реализация на Python

```python
def hamming_distance(n1, n2):
    x = n1 ^ n2
    set_bits = 0
    while x > 0:
        set_bits += x & 1
        x >>= 1
    return set_bits

# Пример
print(hamming_distance(9, 14))  # 3
```

Более короткая версия с `bit_count()`:

```python
def hamming_distance(n1, n2):
    return (n1 ^ n2).bit_count()
```

---

## Сложность

- **Время:** O(1) — для целых чисел фиксированной разрядности (максимум 32 или 64 бита). В худшем случае — O(log max(n1,n2)) битовых операций.
- **Память:** O(1) — используется только несколько переменных.

---

## Примечания

- Расстояние Хэмминга широко применяется в теории кодирования (обнаружение и исправление ошибок), криптографии, биоинформатике и сравнении строк (например, для метрик схожести).
- В Python 3.8+ можно использовать метод `int.bit_count()` для эффективного подсчёта единичных битов.
- В языках C/C++ есть встроенная функция `__builtin_popcount()`.
