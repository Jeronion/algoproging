## Задача

Проверить, установлен ли бит числа на заданной позиции (равен 1).

## Решение

Применяем **побитовое И** с маской, в которой нужный бит равен 1. Если результат не равен нулю, бит установлен.

```python
def is_bit_set(num, pos):
    return (num & (1 << pos)) != 0

# Примеры
print(is_bit_set(5, 0))  # 5 (101) → бит 0 = 1 → True
print(is_bit_set(5, 1))  # бит 1 = 0 → False
print(is_bit_set(5, 2))  # бит 2 = 1 → True
```

## Объяснение

- Число 5: `101`
- Маска для pos=0: `001` → `101 & 001 = 001` → не 0 → True
- Маска для pos=1: `010` → `101 & 010 = 000` → 0 → False

## Сложность

- Время: O(1)
- Память: O(1)
