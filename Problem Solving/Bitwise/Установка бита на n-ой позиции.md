## Задача

Установить (сделать равным 1) бит числа на заданной позиции (считая от младшего бита, позиция 0).

## Решение

Для установки бита используется операция **побитовое ИЛИ** с маской, в которой только нужный бит равен 1. Маска создаётся сдвигом единицы на нужное количество позиций: `1 << pos`.

```python
def set_bit(num, pos):
    return num | (1 << pos)

# Пример
num = 4      # 100 (двоичное)
pos = 1
result = set_bit(num, pos)
print(result)  # 6 (110)
```

## Объяснение

- Исходное число: `4` → `100`
- Маска: `1 << 1` → `010`
- Результат: `100 | 010` = `110` (6)

Бит на позиции 1 становится 1, остальные не меняются.

## Сложность

- Время: O(1)
- Память: O(1)
