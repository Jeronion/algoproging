## Определение

Дан массив `arr` размера `n-1`, содержащий **различные** целые числа от `1` до `n` (включительно), но одно число пропущено. Требуется найти пропущенное число.

**Пример:**
```
Вход: [1, 2, 3, 5] (n = 5)
Выход: 4
```

---

## Идея решения (XOR)

Используются свойства XOR:
- `x ^ x = 0`
- `x ^ 0 = x`
- Коммутативность и ассоциативность

Вычисляем `xor1` = XOR всех чисел от 1 до `n` (полного диапазона).  
Вычисляем `xor2` = XOR всех элементов массива.  
Тогда `xor1 ^ xor2` даст пропущенное число, потому что все присутствующие числа сократятся (дадут 0), а пропущенное останется.

---

## Оптимизация: вычисление XOR диапазона 1..n без цикла

## Реализация

```python
class Solution:
    def missingNum(self, arr):
        n = len(arr) + 1
        # XOR от 1 до n через формулу
        xor1 = {0: n, 1: 1, 2: n + 1, 3: 0}[n % 4]
        xor2 = 0
        for x in arr:
            xor2 ^= x
        return xor1 ^ xor2
```

---

## Сложность

- **Время:** O(n) — один проход по массиву.
- **Память:** O(1) — несколько переменных.

---

## Примечания

- Метод использует XOR и не требует дополнительной памяти.
- Не страдает от переполнения (в отличие от суммы).
- Работает даже для очень больших `n` (до 10⁶ и выше).
- Альтернативные способы: сумма арифметической прогрессии, хеш-таблица, сортировка.
