## Определение

Дан массив `arr` размера `n`, содержащий числа от `1` до `n`. Известно, что одно число из этого диапазона **пропущено**, а другое **повторяется** (встречается дважды). Все остальные числа встречаются ровно один раз. Требуется найти повторяющееся и пропущенное числа.

**Пример:**
```
Вход: [2, 2] (n=2) → Выход: [2, 1] (повторяется 2, пропущено 1)
Вход: [1, 3, 3] (n=3) → Выход: [3, 2]
```

---

## Идея решения (XOR)

Используются свойства XOR:
- `a ^ a = 0`
- `a ^ 0 = a`
- Коммутативность и ассоциативность

1. Вычислим `xor_all` = XOR всех элементов массива.
2. Вычислим `xor_range` = XOR всех чисел от 1 до `n`.
3. Тогда `xor_all ^ xor_range = (duplicate ^ missing)`, потому что все остальные числа встречаются по два раза (один раз в массиве, один раз в диапазоне) и сокращаются.
4. Найдём **правый установленный бит** в полученном значении: `rightmost = (xor_all ^ xor_range) & -(xor_all ^ xor_range)`.
5. Разделим все числа (и элементы массива, и числа от 1 до n) на две группы: те, у которых этот бит установлен, и те, у которых не установлен. Поскольку `duplicate` и `missing` различаются в этом бите, они попадут в разные группы.
6. Вычислим XOR для каждой группы отдельно по элементам массива и по диапазону, затем объединим. В результате получим `duplicate` и `missing` (в каком-то порядке).
7. Чтобы определить, какое из них повторяется, достаточно один раз пройти по массиву и проверить, какое из двух чисел встречается дважды.

---

## Алгоритм

1. Инициализировать `xor_arr = 0`, `xor_range = 0`.
2. Для `i` от 1 до `n`: `xor_range ^= i`.
3. Для каждого `x` в `arr`: `xor_arr ^= x`.
4. `xor_all = xor_arr ^ xor_range`.
5. `rightmost = xor_all & -xor_all`.
6. Инициализировать `x = 0`, `y = 0`.
7. Для каждого `x_val` в `arr`:
   - Если `x_val & rightmost`: `x ^= x_val`
   - Иначе: `y ^= x_val`
8. Для `i` от 1 до `n`:
   - Если `i & rightmost`: `x ^= i`
   - Иначе: `y ^= i`
9. Теперь `x` и `y` — искомые числа (одно повторяется, другое пропущено).
10. Пройти по массиву, чтобы определить, какое из них встречается дважды:
    - Если `count(x) == 2`, то `dup = x, miss = y`
    - Иначе `dup = y, miss = x`.
11. Вернуть `[dup, miss]`.

## Реализация на Python

```python
class Solution:
    def missingAndRepeating(self, arr):
        n = len(arr)
        xor_arr = 0
        xor_range = 0
        for i in range(1, n + 1):
            xor_range ^= i
        for num in arr:
            xor_arr ^= num
        
        xor_all = xor_arr ^ xor_range
        rightmost = xor_all & -xor_all
        
        x = y = 0
        for num in arr:
            if num & rightmost:
                x ^= num
            else:
                y ^= num
        for i in range(1, n + 1):
            if i & rightmost:
                x ^= i
            else:
                y ^= i
        
        # Определяем, что повторяется
        dup = x if arr.count(x) == 2 else y
        miss = y if dup == x else x
        return [dup, miss]
```

## Сложность

- **Время:** O(n) — три прохода по массиву (XOR, разделение, подсчёт).
- **Память:** O(1) — несколько переменных.

## Пример

`arr = [4, 3, 6, 2, 1, 1]`, `n = 6`.

- `xor_arr = 4^3^6^2^1^1 = (1^1)^2^3^4^6 = 2^3^4^6 = (2^3=1)^4=5^6=3`
- `xor_range = 1^2^3^4^5^6 = (1^2=3)^3=0^4=4^5=1^6=7`
- `xor_all = 3 ^ 7 = 4` (двоичное 100)
- `rightmost = 4 & -4 = 4`
- Разделение:
  - Группа 1 (бит 2 установлен): числа: 4 (100), 6 (110), 4? из массива: 4,6,1? 1(001) нет, 3(011) нет, 2(010) нет. Итого в массиве: 4,6 → XOR = 4^6=2. Из диапазона: числа с битом 2: 4,5(101),6 → XOR = 4^5^6 = (4^5=1)^6=7. В итоге `x = 2^7 = 5`.
  - Группа 0: массив: 3,2,1,1 → XOR = 3^2^1^1 = 3^2=1. Диапазон: 1,2,3 → XOR = 1^2^3=0. `y = 1^0=1`.
- `x=5`, `y=1`. В массиве 1 встречается дважды → `dup=1`, `miss=5`. Ответ: `[1,5]`.

---

## Примечания

- Альтернативное решение: через сумму и сумму квадратов (решение системы уравнений).
- XOR-метод не требует дополнительной памяти и эффективен для больших n.
- Обратите внимание, что `arr.count(x)` делает лишний проход; можно было запомнить одно из чисел и проверить его во втором проходе, но для простоты оставлено так.
