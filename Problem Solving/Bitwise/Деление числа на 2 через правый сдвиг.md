## Задача

Разделить целое число на 2 (целочисленно) с помощью битовой операции.

## Решение

Оператор **правый сдвиг** `>>` сдвигает биты вправо, отбрасывая младшие биты. Для положительных чисел это эквивалентно целочисленному делению на `2^k`.

```python
def divide_by_two(num):
    return num >> 1

# Пример
num = 12
result = divide_by_two(num)
print(result)  # 6

num = 13
result = divide_by_two(num)
print(result)  # 6 (отбрасывается остаток)
```

## Объяснение

- `12` (`1100`) >> 1 → `110` = 6
- `13` (`1101`) >> 1 → `110` = 6 (младший бит теряется)

## Примечание

Для отрицательных чисел поведение правого сдвига зависит от языка (в Python сдвиг вправо для отрицательных чисел сохраняет знак, заполняя единицами слева, но результат всё равно эквивалентен целочисленному делению с округлением вниз).

## Сложность

- Время: O(1)
- Память: O(1)
