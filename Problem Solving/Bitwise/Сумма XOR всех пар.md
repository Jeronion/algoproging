## Определение

Дан массив целых чисел `arr` размера `n`. Требуется найти сумму результатов побитового XOR для **всех** пар `(i, j)` таких, что `0 ≤ i < j < n`. Другими словами, вычислить:

$$
\sum_{0 \le i < j < n} (arr[i] \oplus arr[j])
$$

где $\oplus$ — операция XOR (исключающее ИЛИ).

**Пример:**
```
arr = [7, 3, 5]
Пары: 7⊕3 = 4, 7⊕5 = 2, 3⊕5 = 6
Сумма = 4 + 2 + 6 = 12
```

---

## Идея решения (поразрядный подсчёт)

Прямой перебор всех пар требует O(n²), что неприемлемо для `n ≤ 10⁵`. Используем тот факт, что XOR — побитовая операция, и биты в разных позициях **независимы**.

Для каждого бита `j` (от 0 до 31, так как числа ≤ 10⁵ < 2³¹) подсчитаем:
- `count_ones` — количество чисел в массиве, у которых `j`-й бит установлен в 1.
- `count_zeros = n - count_ones`.

Вклад `j`-го бита в общую сумму равен количеству пар, в которых этот бит в результате XOR равен 1, умноженному на вес бита `2^j`. В паре `(a, b)` бит в XOR будет 1 тогда и только тогда, когда биты в `a` и `b` различны (один 0, другой 1). Количество таких пар = `count_ones * count_zeros`.

Таким образом, общая сумма = сумма по всем битам `(count_ones * count_zeros) * 2^j`.

---

## Алгоритм

1. Инициализировать `total = 0`.
2. Для каждого бита `j` от 0 до 31 (или до максимального бита в максимальном числе):
   - Подсчитать `ones` = количество элементов, у которых `(num >> j) & 1 == 1`.
   - `zeros = n - ones`.
   - Добавить к `total` значение `ones * zeros * (1 << j)`.
3. Вернуть `total`.

---

## Реализация на Python

```python
class Solution:
    def sumXOR(self, arr, n):
        total = 0
        for j in range(32):          # 32 бита (для чисел до 10^9 достаточно)
            ones = 0
            for num in arr:
                if num & (1 << j):
                    ones += 1
            zeros = n - ones
            total += ones * zeros * (1 << j)
        return total
```

---

## Сложность

- **Время:** O(32·n) = O(n) — константа 32 не зависит от n.
- **Память:** O(1) — несколько переменных.

---

## Пример работы

`arr = [7, 3, 5]` (n=3)

- Бит 0 (1): числа: 7 (1), 3 (1), 5 (1) → ones=3, zeros=0 → вклад = 0.
- Бит 1 (2): 7 (1), 3 (1), 5 (0) → ones=2, zeros=1 → вклад = 2*1*2 = 4.
- Бит 2 (4): 7 (1), 3 (0), 5 (1) → ones=2, zeros=1 → вклад = 2*1*4 = 8.
- Остальные биты — нули.
Сумма = 4 + 8 = 12.

---

## Примечания

- Метод применим для любых целых чисел, не только положительных (XOR работает одинаково).
- Для больших значений (до 10⁹) достаточно 31 бита, но можно взять 32 для безопасности.
- Альтернативный подход: предварительно вычислить `count_ones` для каждого бита за O(32·n), затем вычислить сумму.
