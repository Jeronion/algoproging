## Определение

Дана строка `s` из заглавных латинских букв и целое число `k`. Разрешено выполнить **не более `k` операций** замены любого символа на любой другой заглавный символ. Требуется найти **максимальную длину** непрерывной подстроки, которую можно превратить в строку из **одинаковых символов** (все символы в подстроке станут одинаковыми) с помощью не более `k` замен.

**Пример:**
```
s = "ABBA", k = 2 → ответ 4 (весь строку можно превратить в "BBBB" за 2 замены)
s = "ADBD", k = 1 → ответ 3 (из "ADBD" можно получить "ADDD" → подстрока "DDD")
```

---

## Идея решения (скользящее окно)

Используем два указателя `l` и `r` для поддержания текущего окна. Основное наблюдение: в окне длины `len` мы можем оставить наиболее частый символ, а остальные `len - max_freq` заменить на него. Если `len - max_freq ≤ k`, то окно **валидно** (можно сделать все символы одинаковыми). Нам нужно найти максимальную длину валидного окна.

При движении `r` вправо:

- Увеличиваем частоту `s[r]` в массиве `freq`.
- Обновляем `max_freq` – максимальную частоту любого символа в текущем окне.
- Если `(r - l + 1) - max_freq > k`, то окно невалидно: нужно сдвинуть `l` вправо (уменьшить частоту `s[l]`).
- В противном случае обновляем ответ `res = max(res, r - l + 1)`.

Поскольку мы не знаем точное значение `max_freq` при сдвиге `l`, но знаем, что максимум может уменьшиться, однако этого достаточно: он никогда не станет больше, чем было, поэтому условие остаётся корректным.

---

## Алгоритм

1. Инициализировать `freq = [0] * 26` (частоты символов в окне), `l = 0`, `max_freq = 0`, `res = 0`.
2. Для `r` от `0` до `n-1`:
   - Индекс `idx = ord(s[r]) - ord('A')`.
   - `freq[idx] += 1`.
   - `max_freq = max(max_freq, freq[idx])`.
   - Пока `(r - l + 1) - max_freq > k`:
     - `freq[ord(s[l]) - ord('A')] -= 1`.
     - `l += 1`.
   - `res = max(res, r - l + 1)`.
3. Вернуть `res`.

---

## Реализация на Python

```python
class Solution:
    def longestSubstr(self, s: str, k: int) -> int:
        freq = [0] * 26
        l = 0
        max_freq = 0
        res = 0
        for r in range(len(s)):
            idx = ord(s[r]) - ord('A')
            freq[idx] += 1
            max_freq = max(max_freq, freq[idx])
            # если количество символов, которые нужно заменить, превышает k, сдвигаем окно
            while (r - l + 1) - max_freq > k:
                freq[ord(s[l]) - ord('A')] -= 1
                l += 1
            res = max(res, r - l + 1)
        return res
```

## Сложность

- **Время:** O(n) — каждый символ обрабатывается не более двух раз.
- **Память:** O(1) — массив из 26 элементов.

## Примечания

- Классическая задача, известная как **LeetCode 424. Longest Repeating Character Replacement**.
- Алгоритм использует тот факт, что `max_freq` может не уменьшаться при сдвиге `l`, но это не влияет на корректность, так как мы используем его для проверки условия. Если `max_freq` окажется завышенным, условие может сработать позже, но мы всё равно сдвигаем окно до тех пор, пока окно не станет валидным. В итоге максимальная длина будет найдена.
- Важно, что буквы только заглавные, поэтому массив из 26 элементов.
