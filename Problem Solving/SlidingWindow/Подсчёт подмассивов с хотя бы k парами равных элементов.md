## Определение

Дан массив целых чисел `arr` и положительное целое `k`. Пара равных элементов в подмассиве определяется как два индекса `(i, j)` с `i < j` и `arr[i] = arr[j]`. Требуется подсчитать количество **непрерывных подмассивов**, которые содержат **хотя бы `k` таких пар**.

**Пример:**
```
arr = [2, 1, 4, 5, 2, 1], k = 1
Валидные подмассивы (содержат хотя бы одну пару):
[2,1,4,5,2], [2,1,4,5,2,1], [1,4,5,2,1] → итого 3.
```

---

## Идея решения (скользящее окно)

Используем метод **двух указателей** (sliding window) с поддержкой частотного словаря и счётчика пар. Основное наблюдение: при добавлении нового элемента `x` в конец окна, он образует `freq[x]` новых пар (с каждым предыдущим вхождением `x`). Общее количество пар в окне `cur` увеличивается на `freq[x]`, после чего `freq[x]` увеличивается на 1.

Когда `cur` становится ≥ `k`, **все подмассивы, заканчивающиеся на текущей позиции `i` (и далее до конца массива), будут содержать эти пары** (так как расширение вправо только добавляет элементы, но не уменьшает количество уже имеющихся пар). Поэтому можно сразу добавить к ответу количество таких подмассивов: `(n - i)`. После этого нужно сдвинуть левый указатель `j` вправо, чтобы попытаться найти другие окна. При удалении элемента `y` из левой части окна, он уменьшает `freq[y]`, и количество пар уменьшается на `freq[y]` (новое значение частоты после удаления, так как пары с этим элементом пропадают). Затем `cur -= freq[y]` и `freq[y]--`? В коде порядок: сначала `freq[arr[j]] -= 1`, затем `cur -= freq[arr[j]]` – это корректно, так как после уменьшения частоты, `freq[arr[j]]` равно числу оставшихся вхождений этого элемента, и именно столько пар было с этим элементом до удаления? Разберём: до удаления было `f` вхождений `y`. Пары с `y`: количество пар вида `(y, другой y)` равно `f*(f-1)/2`, но в `cur` хранится общее количество пар, а не коэффициент. Удобнее использовать подход: при добавлении `y` мы увеличиваем `cur` на `freq[y]`, после чего увеличиваем `freq[y]`. При удалении `y` мы сначала уменьшаем `freq[y]`, а затем вычитаем из `cur` новое значение `freq[y]` (которое равно количеству оставшихся вхождений). Это эквивалентно удалению пар, образованных с удаляемым элементом.

Таким образом, алгоритм линейный.

---

## Алгоритм

1. Инициализировать `freq` – словарь частот элементов в текущем окне.
2. Инициализировать `j = 0` (левый указатель), `cur = 0` (текущее количество пар), `result = 0`.
3. Для `i` от `0` до `n-1` (правый указатель):
   - `cur += freq[arr[i]]` (новые пары с текущим элементом).
   - `freq[arr[i]] += 1`.
   - Пока `cur >= k`:
     - `result += n - i` (все подмассивы, заканчивающиеся на `i` или правее, валидны).
     - Удаляем левый элемент `arr[j]`:
       - `freq[arr[j]] -= 1`.
       - `cur -= freq[arr[j]]` (после уменьшения).
       - `j += 1`.
4. Вернуть `result`.

## Реализация на Python

```python
from collections import defaultdict

def countKPairsSubarray(arr, k):
    n = len(arr)
    freq = defaultdict(int)
    j = 0
    cur = 0
    result = 0
    for i in range(n):
        cur += freq[arr[i]]
        freq[arr[i]] += 1
        while cur >= k:
            result += n - i
            # удаляем левый элемент
            freq[arr[j]] -= 1
            cur -= freq[arr[j]]
            j += 1
    return result

# Пример
arr = [2, 1, 4, 5, 2, 1]
k = 1
print(countKPairsSubarray(arr, k))  # 3
```

## Сложность

- **Время:** O(n) — каждый элемент добавляется и удаляется из окна не более одного раза.
- **Память:** O(n) — в худшем случае словарь может хранить все различные элементы массива.

## Примечания

- Алгоритм работает для любых целых чисел (не только неотрицательных).
- При `k = 0` все подмассивы валидны (включая пустые? по условию, вероятно, подмассив длины <2 тоже? В задаче пары определяются как два разных индекса, поэтому для подмассива длины 1 нет пар. Но если `k = 0`, то условие `cur >= 0` всегда истинно, и алгоритм добавит `n - i` для каждого `i`. Это соответствует подсчёту всех подмассивов (включая подмассивы длины 1). Проверьте, нужно ли это в вашей задаче.
- В примере `arr = [1,2,3,4,5], k=2` ответ 0.
- Метод является классическим примером использования скользящего окна для подсчёта подмассивов с условием на количество пар.
