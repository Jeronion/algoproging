## Определение

Даны две строки `s` и `p`. Требуется найти наименьшую **непрерывную подстроку** (окно) в `s`, которая содержит **все символы** строки `p` (включая повторяющиеся), т.е. частоты символов в окне должны быть не меньше, чем в `p`. Если таких окон несколько, вернуть то, у которого наименьший начальный индекс. Если окна нет, вернуть `""`.

**Пример:**
```
s = "timetopractice", p = "toc" → "toprac" (содержит t, o, c)
s = "zoomlazapzo", p = "oza" → "apzo" (содержит o, z, a)
```

---

## Идея решения (скользящее окно с двумя указателями)

Используем два указателя `left` и `right` для поддержания текущего окна. Словарь `countP` хранит частоты символов в `p`. Словарь `countS` хранит частоты символов в текущем окне `s[left:right+1]`. Также поддерживаем переменную `formed` — количество символов, частоты которых в окне достигли требуемого уровня из `p`.

1. **Расширяем** окно вправо (`right++`), добавляя символ `s[right]` в `countS`. Если после добавления `countS[ch] == countP[ch]`, увеличиваем `formed`.
2. Как только `formed == len(p)` (все символы `p` присутствуют в окне), пытаемся **сжать** окно слева (`left++`), уменьшая `countS[s[left]]`. Если при этом `countS[s[left]]` становится меньше, чем нужно в `countP`, уменьшаем `formed`.
3. На каждом шаге, когда окно валидно, запоминаем минимальную длину и начальный индекс.
4. Возвращаем подстроку минимальной длины (или `""`).

## Реализация

```python
def minWindow(s: str, p: str) -> str:
    if len(s) < len(p):
        return ""

    countP = [0] * 256
    for ch in p:
        countP[ord(ch)] += 1

    countS = [0] * 256
    left = 0
    right = 0
    formed = 0
    min_len = float('inf')
    start_idx = -1

    while right < len(s):
        ch = s[right]
        countS[ord(ch)] += 1
        if countP[ord(ch)] > 0 and countS[ord(ch)] == countP[ord(ch)]:
            formed += 1

        # сжимаем окно, пока оно валидно
        while left <= right and formed == len(p):
            ch_left = s[left]
            # обновляем минимальную длину
            if right - left + 1 < min_len:
                min_len = right - left + 1
                start_idx = left
            # убираем левый символ
            countS[ord(ch_left)] -= 1
            if countP[ord(ch_left)] > 0 and countS[ord(ch_left)] < countP[ord(ch_left)]:
                formed -= 1
            left += 1

        right += 1

    return "" if start_idx == -1 else s[start_idx:start_idx + min_len]
```

---

## Сложность

- **Время:** O(|s| + |p|) — каждый символ обрабатывается не более двух раз.
- **Память:** O(1) — массивы фиксированного размера 256.

---

## Примечания

- Используются массивы длины 256 (для ASCII-символов). Для Unicode можно использовать словарь.
- Алгоритм учитывает повторяющиеся символы (например, `p = "aa"` требует двух `a` в окне).
- Если требуется вернуть `"-1"` вместо пустой строки, измените возвращаемое значение.

## Мой код

```python
class Solution:
    def minWindow(self, s, p):
        def idx(x):
            return ord(x) - ord("a")
        
        p_arr = [0] * 26
        for i in p:
            p_arr[idx(i)] += 1
            
        s_arr = [0] * 26
        c, l = 0, 0
        res = [float("-inf"), float("inf")]
        
        for r in range(len(s)):
            
            s_arr[idx(s[r])] += 1
            if s_arr[idx(s[r])] <= p_arr[idx(s[r])]:
                c += 1
                
            if c == len(p):
                while c == len(p):
                    s_arr[idx(s[l])] -= 1
                    if s_arr[idx(s[l])] < p_arr[idx(s[l])]:
                        c -= 1
                    l += 1
                
                if res[1] - res[0] > r - l:
                    res = [l, r]
        
        if res[0] == float("-inf"):
            return ""
        return s[res[0] - 1:res[1] + 1]
```
