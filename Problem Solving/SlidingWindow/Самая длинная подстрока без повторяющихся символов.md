## Определение

Дана строка `s`, состоящая из строчных латинских букв (в общем случае любых символов). Требуется найти длину самой длинной **непрерывной подстроки**, в которой **все символы различны**.

**Пример:**
```
s = "geeksforgeeks" → ответ 7 ("eksforg")
s = "aaa" → ответ 1
s = "abcdefabcbb" → ответ 6 ("abcdef")
```

---

## Идея решения (скользящее окно)

Используем два указателя `l` (левый) и `r` (правый), чтобы поддерживать окно, в котором все символы уникальны. По мере увеличения `r`:

- Если символ `s[r]` уже присутствует в окне (его частота > 0), сдвигаем `l` вправо, удаляя символы до тех пор, пока дубликат не исчезнет.
- После этого (или если символа не было) добавляем `s[r]` в окно (увеличиваем частоту).
- Длина текущего окна = `r - l + 1`. Обновляем максимальную длину.

Для отслеживания частот используем массив из 26 элементов (для строчных букв) или словарь (для общего случая).

---

## Алгоритм

1. Инициализировать `freq` – массив частот (26 нулей), `l = 0`, `max_len = 0`.
2. Для `r` от `0` до `n-1`:
   - Пока `freq[ord(s[r]) - ord('a')] > 0` (символ уже есть в окне):
     - Уменьшить `freq[ord(s[l]) - ord('a')]`.
     - `l += 1`.
   - Увеличить частоту для `s[r]`.
   - `max_len = max(max_len, r - l + 1)`.
3. Вернуть `max_len`.

---

## Реализация на Python

```python
class Solution:
    def longestUniqueSubstr(self, s: str) -> int:
        freq = [0] * 26
        l = 0
        max_len = 0
        for r in range(len(s)):
            while freq[ord(s[r]) - ord('a')] > 0:
                freq[ord(s[l]) - ord('a')] -= 1
                l += 1
            freq[ord(s[r]) - ord('a')] += 1
            max_len = max(max_len, r - l + 1)
        return max_len
```

**Примечание:** Для строк, содержащих любые символы (Unicode), следует использовать `defaultdict(int)` для частот.

---

## Сложность

- **Время:** O(n) – каждый символ обрабатывается не более двух раз.
- **Память:** O(1) – массив фиксированного размера (26) или O(алфавит) для обобщённого случая.

---

## Примечания

- Классическая задача, известная как "Longest Substring Without Repeating Characters" (LeetCode 3).
- Метод скользящего окна является оптимальным.
- Вместо массива частот можно использовать множество (`set`), но при удалении левого символа нужно знать, какой именно убрать, что сложнее. Массив частот позволяет легко отслеживать количество, а не только факт присутствия.
