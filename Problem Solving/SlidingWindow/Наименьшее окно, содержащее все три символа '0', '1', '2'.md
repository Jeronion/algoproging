## Определение

Дана строка `s`, состоящая только из символов `'0'`, `'1'`, `'2'`. Требуется найти **минимальную длину** непрерывной подстроки, которая содержит **каждый из трёх символов хотя бы один раз**. Если такой подстроки нет, вернуть `-1`.

**Пример:**
```
s = "10212" → ответ 3 ("102" содержит 1,0,2)
s = "12121" → ответ -1 (нет символа '0')
```

---

## Идея решения (скользящее окно)

Используем два указателя `l` (левый) и `r` (правый), чтобы поддерживать окно, в котором храним частоты символов. При движении `r` вправо добавляем символ `s[r]` в счётчик. Если какой-то символ встречается в окне больше одного раза, сдвигаем `l` вправо, чтобы удалить лишние. Как только окно содержит все три символа (т.е. частоты каждого > 0), мы запоминаем минимальную длину `r - l + 1`. Если находим длину 3 — сразу возвращаем её (это минимально возможная). В конце возвращаем найденный минимум или -1.

### Алгоритм

1. Создать массив `freq` размера 3 (для символов '0','1','2'), инициализированный нулями.
2. Установить `left = 0`, `min_len = ∞`.
3. Пройти `right` от `0` до `n-1`:
   - Увеличить `freq[int(s[right])]` на 1.
   - Пока `freq[int(s[left])] > 1` (левый символ встречается в окне более одного раза), уменьшить его частоту и сдвинуть `left` вправо.
   - Если `freq[0] > 0` и `freq[1] > 0` и `freq[2] > 0`, обновить `min_len = min(min_len, right - left + 1)`.
   - Если `min_len == 3`, сразу вернуть 3.
4. Вернуть `min_len`, если он не бесконечность, иначе -1.

---

## Реализация

```python
class Solution:
    def smallestSubstring(self, s: str) -> int:
        freq = [0, 0, 0]
        left = 0
        min_len = float('inf')
        for right in range(len(s)):
            freq[int(s[right])] += 1
            # сжимаем окно, если левый символ лишний
            while freq[int(s[left])] > 1:
                freq[int(s[left])] -= 1
                left += 1
            # если есть все три символа
            if freq[0] > 0 and freq[1] > 0 and freq[2] > 0:
                min_len = min(min_len, right - left + 1)
                if min_len == 3:
                    return 3
        return min_len if min_len != float('inf') else -1
```

## Сложность

- **Время:** O(n) — каждый символ обрабатывается не более двух раз.
- **Память:** O(1) — массив фиксированного размера.

---

## Примечания

- Алгоритм учитывает, что окно может содержать лишние повторяющиеся символы, которые не мешают наличию всех трёх, но мешают минимальности. Мы сжимаем окно только когда левый символ встречается **более одного раза** (не важно, входит ли он в тройку). Это правильная стратегия для поддержания наименьшего окна.
- Если `min_len` достигает 3, дальше искать не имеет смысла, так как это теоретический минимум.
- Если строка не содержит одного из символов, `freq` для него останется 0 и условие никогда не выполнится → вернётся -1.
