## Определение

Дана строка `s`, состоящая из строчных латинских букв, и целое число `k`. Требуется найти длину самой длинной **непрерывной подстроки**, которая содержит **ровно `k` различных символов**. Если такой подстроки нет, вернуть `-1`.

**Пример:**
```
s = "aabacbebebe", k = 3 → ответ 7 ("cbebebe")
s = "aaaa", k = 2 → ответ -1
s = "aabaaab", k = 2 → ответ 7 (вся строка)
```

---

## Идея решения (скользящее окно)

Используем два указателя `l` и `r` для поддержания окна. Также ведём счётчик `c` – количество **различных** символов в текущем окне. Для подсчёта частот используем массив `hm` размера 26.

- При расширении окна вправо (`r`):
  - Если частота `s[r]` была 0, то увеличиваем `c` (появился новый уникальный символ).
  - Увеличиваем частоту этого символа.
- Если `c > k`, то окно содержит более `k` различных символов – его нужно сжимать слева:
  - Пока `c > k` и `l < r`:
    - Уменьшаем частоту символа `s[l]`.
    - Если его частота стала 0, уменьшаем `c` (символ исчез из окна).
    - `l++`.
- Если `c == k`, то текущее окно валидно. Обновляем максимальную длину `res = max(res, r - l + 1)`.

После обработки всей строки возвращаем `res`, если она была обновлена, иначе `-1`.

---

## Реализация

```python
class Solution:
    def longestKSubstr(self, s: str, k: int) -> int:
        n = len(s)
        freq = [0] * 26
        left = 0
        unique = 0          # количество различных символов в окне
        max_len = 0
        
        for right in range(n):
            idx = ord(s[right]) - ord('a')
            if freq[idx] == 0:
                unique += 1
            freq[idx] += 1
            
            # сжимаем окно, если уникальных > k
            while unique > k:
                left_idx = ord(s[left]) - ord('a')
                freq[left_idx] -= 1
                if freq[left_idx] == 0:
                    unique -= 1
                left += 1
            
            # если ровно k, обновляем ответ
            if unique == k:
                max_len = max(max_len, right - left + 1)
        
        return max_len if max_len > 0 else -1
```

## Сложность

- **Время:** O(n) – каждый символ обрабатывается не более двух раз.
- **Память:** O(1) – массив фиксированной длины 26.

---

## Примечания

- Алгоритм использует классический метод **скользящего окна**.
- Условие `while unique > k` гарантирует, что окно всегда содержит `≤ k` различных символов.
- Если после всей строки `unique` никогда не было равно `k`, возвращается `-1`.
- Входные ограничения: длина строки до 10⁵, `k ≤ 26`.
- Альтернативный подход: можно использовать `defaultdict` для учёта частот, но массив эффективнее для строчных букв.

## Мой код

```python
class Solution:
    def longestKSubstr(self, s, k):
        res, l, c = 0, 0, 0
        hm = [0] * 26
        for r in range(len(s)):
            c += not hm[ord(s[r]) - ord("a")]
            hm[ord(s[r]) - ord("a")] += 1
            
            while c > k:
                c -= (hm[ord(s[l]) - ord("a")] == 1)
                hm[ord(s[l]) - ord("a")] -= 1
                l += 1
                
            if c == k:
                res = max(res, r - l + 1)
        
        return res if res else -1
```
