## Определение

Дан массив целых чисел `arr` и размер окна `k`. Требуется для каждого непрерывного подмассива (окна) длины `k` вывести количество **уникальных** (различных) элементов в этом окне.

**Пример:**
```
arr = [1, 2, 1, 3, 4, 2, 3], k = 4
Окна:
[1,2,1,3] → 3 уникальных
[2,1,3,4] → 4
[1,3,4,2] → 4
[3,4,2,3] → 3
Результат: [3,4,4,3]
```

---

## Идея решения (скользящее окно + хеш-таблица)

Используем **два указателя** для поддержания окна, но не сумму, а **частотный словарь** (hash map), хранящий количество вхождений каждого элемента в текущем окне. При сдвиге окна:

- Удаляем левый элемент: уменьшаем его счётчик; если он стал 0 — удаляем ключ или просто считаем, что уникальных стало на 1 меньше.
- Добавляем правый элемент: увеличиваем счётчик; если до этого его не было (счётчик был 0), то уникальных стало на 1 больше.

Таким образом, на каждом шаге количество уникальных элементов поддерживается за O(1) (после O(1) операций со словарём).

### Алгоритм

1. Создать словарь `freq` для подсчёта частот.
2. Заполнить первое окно (первые `k` элементов): для каждого элемента увеличить счётчик в `freq`. Количество уникальных элементов в первом окне = размер `freq`.
3. Записать это число в результат.
4. Для `i` от `k` до `n-1`:
   - Удалить `arr[i-k]`:
     - Уменьшить его счётчик.
     - Если счётчик стал 0, удалить ключ (или просто учитывать нулевые счётчики как отсутствие).
   - Добавить `arr[i]`:
     - Увеличить счётчик.
     - Если раньше его не было (счётчик стал 1), увеличить счётчик уникальных.
   - Записать текущее количество уникальных в результат.
5. Вернуть результат.

## Реализация на Python

```python
class Solution:
    def countDistinct(self, arr, k):
        n = len(arr)
        res = [0] * (n - k + 1)
        freq = {}
        # первое окно
        for i in range(k):
            freq[arr[i]] = freq.get(arr[i], 0) + 1
        res[0] = len(freq)
        
        # остальные окна
        for i in range(k, n):
            # удаляем arr[i-k]
            freq[arr[i-k]] -= 1
            if freq[arr[i-k]] == 0:
                del freq[arr[i-k]]
            # добавляем arr[i]
            freq[arr[i]] = freq.get(arr[i], 0) + 1
            res[i - k + 1] = len(freq)
        return res
```

### Примечание к приведённому коду

В исходном коде использовалась переменная `res[i+1] = res[i]` с последующими корректировками, что эквивалентно, но менее читаемо. Приведённая выше версия проще.

## Сложность

- **Время:** O(n) — каждый элемент обрабатывается дважды (один раз добавляется, один раз удаляется).
- **Память:** O(k) — в худшем случае в окне могут быть все различные элементы.

---

## Примечания

- Если значения элементов могут быть очень большими, словарь всё равно эффективен.
- Можно также использовать `collections.Counter`, но ручной словарь даёт больше контроля.
- Важно правильно обрабатывать удаление: только когда счётчик становится 0, нужно уменьшать количество уникальных.
