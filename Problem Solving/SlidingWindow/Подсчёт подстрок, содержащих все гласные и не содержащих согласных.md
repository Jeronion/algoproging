## Определение

Дана строка `str`, состоящая из строчных латинских букв. Требуется подсчитать количество **непрерывных подстрок**, которые:

- содержат **каждую из гласных букв** (`a`, `e`, `i`, `o`, `u`) **хотя бы один раз**;
- **не содержат согласных** (т.е. состоят **только из гласных**).

**Пример:**
```
str = "aeoibsddaaeiouudb"
Подстроки, подходящие под условие: "aaeiouu", "aeiouu", "aeiou", "aaeiou" → ответ 4.
```

---

## Идея решения (скользящее окно)

Эффективный алгоритм использует **скользящее окно** и работает за O(n). Поддерживаем окно, которое содержит только гласные буквы. При расширении окна вправо:

- Если встречаем согласную – сбрасываем всё окно и начинаем заново со следующей позиции.
- Если встречаем гласную – обновляем её частоту в словаре.
- Чтобы окно было минимальным, но содержало все гласные, мы сдвигаем левую границу, пока не уберём лишние повторяющиеся гласные (но не нарушая наличие всех пяти).
- Как только в окне присутствуют все пять гласных, каждая позиция от текущей левой границы до первой возможной (с учётом того, что можно начинать подстроку с любого символа, который не нарушает наличие всех гласных) даёт валидные подстроки, заканчивающиеся на текущем `i`. В коде это количество равно `1 + pref`, где `pref` — количество «лишних» левых символов, которые можно отбросить, сохранив все гласные.

### Алгоритм

1. Инициализировать `freq` – словарь для частот гласных в текущем окне.
2. Установить `cnt = 0` – количество различных гласных в окне.
3. Установить `j = 0` – левая граница окна.
4. Установить `pref = 0` – количество дополнительных позиций слева, которые можно удалить, не потеряв ни одной гласной.
5. Для каждого `i` от `0` до `n-1`:
   - Если `s[i]` – гласная:
     - Увеличить `freq[s[i]]`; если `freq[s[i]] == 1`, увеличить `cnt`.
     - Пока `freq[s[j]] > 1` (т.е. левый символ встречается лишний раз), уменьшить `freq[s[j]]`, сдвинуть `j` и увеличить `pref`.
     - Если `cnt == 5` (все гласные присутствуют), то добавить к ответу `1 + pref`.
   - Если `s[i]` – согласная:
     - Сбросить всё: `cnt = 0`, `pref = 0`, `freq.clear()`, `j = i + 1`.
6. Вернуть ответ.

## Реализация на Python

```python
def countVowelSubstrings(s: str) -> int:
    vowels = set('aeiou')
    freq = {}
    ans = 0
    n = len(s)
    j = 0
    cnt = 0      # количество различных гласных в окне
    pref = 0     # количество лишних символов слева (можно удалить)
    
    for i in range(n):
        ch = s[i]
        if ch in vowels:
            freq[ch] = freq.get(ch, 0) + 1
            if freq[ch] == 1:
                cnt += 1
            # Сжимаем окно слева, если какой-то гласной больше 1
            while freq.get(s[j], 0) > 1:
                freq[s[j]] -= 1
                pref += 1
                j += 1
            if cnt == 5:
                ans += 1 + pref
        else:
            # Согласная — сброс
            freq.clear()
            cnt = 0
            pref = 0
            j = i + 1
    return ans
```

## Сложность

- **Время:** O(n) — каждый символ обрабатывается ограниченное число раз.
- **Память:** O(1) — словарь содержит не более 5 ключей.

---

## Примечания

- Наивное решение (перебор всех подстрок) имеет сложность O(n²) и не подходит для длинных строк.
- Алгоритм учитывает, что подстроки должны состоять **только из гласных** и содержать **все пять гласных**. При встрече согласной окно полностью сбрасывается.
- Переменная `pref` накапливает количество позиций, на которые можно сдвинуть левую границу, не потеряв ни одной гласной. Это позволяет подсчитать все подстроки, заканчивающиеся в текущей позиции, одним действием.
