## Определение

Дана строка `s`, состоящая из строчных латинских букв. Найдите **длину наименьшей непрерывной подстроки**, которая содержит **все символы**, встречающиеся в строке `s` (каждый хотя бы один раз).

**Пример:**
```
s = "aabcbcdbca" → уникальные символы: a,b,c,d → наименьшая подстрока "dbca" длины 4.
s = "aaab" → уникальные: a,b → "ab" длины 2.
s = "geeksforgeeks" → ответ 7 ("eksforg" или "ksforge").
```

---

## Идея решения (скользящее окно)

1. Подсчитываем **количество уникальных символов** в строке: `k = len(set(s))`.
2. Используем два указателя `l` (левый) и `r` (правый) для поддержания окна.
3. Массив `freq[26]` хранит частоты символов в текущем окне.
4. Переменная `unique` – количество различных символов в окне.
5. Двигаем `r` вправо, добавляя символы. Если частота символа стала 1, увеличиваем `unique`.
6. Когда `unique == k` (окно содержит все символы):
   - Обновляем минимальную длину `min_len`.
   - Пытаемся сжать окно слева: удаляем `s[l]`, уменьшаем частоту. Если частота стала 0, уменьшаем `unique`.
   - Сдвигаем `l`.
7. Возвращаем `min_len`.

## Реализация (ваш код, слегка приведённый к единообразию)

```python
class Solution:
    def findSubString(self, s: str) -> int:
        k = len(set(s))                     # всего уникальных символов
        freq = [0] * 26
        left = 0
        unique = 0
        min_len = float('inf')

        for right in range(len(s)):
            idx = ord(s[right]) - ord('a')
            freq[idx] += 1
            if freq[idx] == 1:
                unique += 1

            while unique == k:
                min_len = min(min_len, right - left + 1)
                left_idx = ord(s[left]) - ord('a')
                freq[left_idx] -= 1
                if freq[left_idx] == 0:
                    unique -= 1
                left += 1

        return min_len
```

## Сложность

- **Время:** O(n) – каждый символ обрабатывается дважды (при расширении и при сжатии).
- **Память:** O(1) – массив из 26 элементов.

## Примечания

- Задача аналогична **LeetCode 76. Minimum Window Substring**, но там требуется содержать все символы заданной строки-паттерна, а здесь – все символы исходной строки.
- Если строка состоит из одного повторяющегося символа, ответ `1`.
- Если строка уже содержит все символы в минимальной подстроке, алгоритм найдёт её.
