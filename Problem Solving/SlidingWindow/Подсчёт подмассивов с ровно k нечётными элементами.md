## Определение

Дан массив целых положительных чисел `arr` и целое число `k`. Требуется подсчитать количество **непрерывных подмассивов**, которые содержат **ровно k нечётных чисел**.

**Пример:**
```
arr = [2, 5, 6, 9], k = 2 → 2 подмассива: [2,5,6,9] и [5,6,9]
arr = [2, 2, 5, 6, 9, 2, 11], k = 2 → 8 подмассивов
```

---

## Идея решения (скользящее окно + разность)

Прямой перебор всех подмассивов (O(n²)) слишком медленный для n ≤ 10⁵. Используем метод **скользящего окна** для подсчёта подмассивов с **не более чем k** нечётными, а затем вычитаем подмассивы с **не более чем k-1** нечётными. Результат = количество подмассивов с ровно k нечётными.

### Подсчёт подмассивов с ≤ x нечётными (atMostX)

Поддерживаем окно `[start, i]` (скользящее окно), в котором количество нечётных элементов не превышает `x`. При расширении `i` вправо:
- Если встретили нечётный элемент, увеличиваем счётчик `odd`.
- Пока `odd > x`, сдвигаем `start` вправо, уменьшая счётчик, если выходит нечётный элемент.
- На каждой итерации все подмассивы, заканчивающиеся на `i` и начинающиеся в интервале `[start, i]`, удовлетворяют условию (их количество = `i - start + 1`). Суммируем.

Таким образом, `atMostX(arr, x)` возвращает количество подмассивов с не более чем `x` нечётными.

### Итоговая формула

```
count_subarrays_with_exactly_k_odds = atMostX(arr, k) - atMostX(arr, k - 1)
```

Если `k == 0`, то `atMostX(arr, -1)` следует обработать как 0 (или просто не вызывать). В реализации `atMostX` предполагается, что `x >= 0`. Для `k = 0` можно просто вернуть `atMostX(arr, 0)`.

---

## Реализация

```python
def atMostX(arr, x):
    if x < 0:
        return 0
    n = len(arr)
    left = 0
    odd = 0
    count = 0
    for right in range(n):
        if arr[right] % 2 == 1:
            odd += 1
        while odd > x:
            if arr[left] % 2 == 1:
                odd -= 1
            left += 1
        count += right - left + 1
    return count

def countSubarrays(arr, k):
    return atMostX(arr, k) - atMostX(arr, k - 1)
```

---

## Сложность

- **Время:** O(n) — один проход для вычисления `atMostX`, константное количество вызовов.
- **Память:** O(1) — только указатели и счётчики.

---

## Примечания

- Алгоритм работает для любых целых чисел (достаточно проверять чётность).
- Подход основан на стандартной технике «скользящее окно + разность» для подсчёта подмассивов с точным значением параметра.
- Краевой случай: `k = 0` — ищем подмассивы без нечётных чисел. Тогда `atMostX(arr, -1)` даёт 0, и ответ = `atMostX(arr, 0)`.

## Мой код

```python
def countSubarrays(self, arr, k):
    arr.append(1)          # добавляем 1 в конец
    n = len(arr)
    l, c, m = 0, 0, 0
    prev_r = None
    for r in range(n):
        m += arr[r] % 2
        if m > k:
            prev_l = l
            while l < n and arr[l] % 2 == 0:
                l += 1
            l += 1
            m -= 1
            c += (r - prev_r) * (l - prev_l)
        if arr[r] % 2:
            prev_r = r
    return c
```
