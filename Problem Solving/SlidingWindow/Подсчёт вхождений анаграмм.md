## Определение

Даны две строки: `txt` (текст) и `pat` (паттерн). Требуется подсчитать количество вхождений **анаграмм** `pat` в `txt`. Анаграмма — это перестановка символов, то есть подстрока `txt` длины `|pat|`, содержащая те же символы в тех же количествах, что и `pat`.

**Пример:**
```
txt = "forxxorfxdofr", pat = "for"
Анаграммы: "for", "orf", "ofr" → ответ 3.
txt = "aabaabaa", pat = "aaba" → ответ 4.
```

---

## Идея решения (скользящее окно + частотный массив)

Используем метод **скользящего окна** фиксированной длины `k = len(pat)`. Для сравнения анаграмм нам нужно проверить, совпадают ли частоты символов в текущем окне с частотами в `pat`. Поскольку алфавит состоит из 26 строчных букв, можно использовать массивы частот `hmw` (для `pat`) и `hm` (для текущего окна).

**Алгоритм:**

1. Построить частотный массив `hmw` для `pat`.
2. Инициализировать окно первыми `k` символами `txt`, заполняя `hm`. При этом вести счётчик `c` — количество символов, частота которых в окне **не превышает** требуемую (т.е. они «покрыты»).
   - При добавлении символа: если его частота в окне `≤` частоте в `hmw`, увеличиваем `c`.
3. Если `c == k` (все символы окна покрыты), то это анаграмма → увеличиваем ответ.
4. Затем сдвигаем окно: удаляем левый символ, добавляем правый новый.
   - При удалении: если до удаления частота левого символа `≤` частоте в `hmw`, уменьшаем `c`.
   - После удаления уменьшаем его частоту.
   - При добавлении нового символа: если после добавления его частота `≤` частоте в `hmw`, увеличиваем `c`.
5. На каждом шаге после обновления окна проверяем `c == k`.

Этот подход работает за O(n) времени и O(1) памяти (массивы на 26).

## Реализация

```python
class Solution:
    def search(self, pat, txt):
        k = len(pat)
        # частоты паттерна
        freq_pat = [0] * 26
        for ch in pat:
            freq_pat[ord(ch) - ord('a')] += 1
        
        freq_win = [0] * 26
        count = 0          # количество совпадающих частот в окне
        ans = 0
        
        # первое окно
        for i in range(k):
            idx = ord(txt[i]) - ord('a')
            freq_win[idx] += 1
            if freq_win[idx] <= freq_pat[idx]:
                count += 1
        if count == k:
            ans += 1
        
        # скольжение
        for i in range(k, len(txt)):
            # удаляем левый символ
            left_idx = ord(txt[i - k]) - ord('a')
            if freq_win[left_idx] <= freq_pat[left_idx]:
                count -= 1
            freq_win[left_idx] -= 1
            
            # добавляем правый символ
            right_idx = ord(txt[i]) - ord('a')
            freq_win[right_idx] += 1
            if freq_win[right_idx] <= freq_pat[right_idx]:
                count += 1
            
            if count == k:
                ans += 1
        
        return ans
```

## Сложность

- **Время:** O(|txt| + |pat|) — один проход по тексту.
- **Память:** O(1) — два массива из 26 элементов.

---

## Примечания

- Вместо сравнения всех 26 частот на каждом шаге, мы поддерживаем счётчик `count`, что ускоряет проверку.
- Условие `freq_win[idx] <= freq_pat[idx]` при добавлении гарантирует, что мы учитываем только символы, которые ещё не превысили нужное количество. При удалении аналогично.
- Если окно содержит избыточные символы (больше, чем нужно), они не влияют на `count`, и условие `count == k` не выполнится, что правильно.
- Задача известна на GeeksforGeeks как "Count Occurences of Anagrams".

## Мой код

```python
class Solution:
    def search(self, w, s):
        def idx(x):
            return ord(x) - ord("a")
        
        res = 0
        
        k = len(w)
        hmw = [0] * 26
        for i in w:
            hmw[idx(i)] += 1
        
        c = 0
        hm = [0] * 26
        for i in range(k):
            hm[idx(s[i])] += 1
            if hm[idx(s[i])] <= hmw[idx(s[i])]:
                c += 1
            res += (c == k)
        
        for l in range(len(s) - k):
            r = l + k - 1
            
            if hm[idx(s[l])] <= hmw[idx(s[l])]:
                c -= 1
            hm[idx(s[l])] -= 1
            
            hm[idx(s[r + 1])] += 1
            if hm[idx(s[r + 1])] <= hmw[idx(s[r + 1])]:
                c += 1
            
            res += (c == k)
        
        return res
```
