## Определение

Две строки `s1` и `s2` называются **анаграммами**, если они состоят из одних и тех же символов, причём каждый символ встречается одинаковое количество раз. Порядок символов не важен.

**Пример:**
```
s1 = "geeks", s2 = "kseeg" → true
s1 = "listen", s2 = "silent" → true
s1 = "allergy", s2 = "allergyy" → false (лишняя 'y')
```

---

## Идея решения

Поскольку строки состоят только из строчных английских букв (`a`–`z`), можно использовать **массив частот** из 26 элементов.

1. Если длины строк не равны → сразу `False`.
2. Пройти по обеим строкам одновременно:
   - Для символа из `s1` увеличить счётчик на 1.
   - Для символа из `s2` уменьшить счётчик на 1.
3. Если после прохода все счётчики равны нулю → `True`, иначе `False`.

Этот метод требует O(n) времени и O(1) дополнительной памяти.

---

## Реализация на Python

```python
class Solution:
    def areAnagrams(self, s1, s2):
        if len(s1) != len(s2):
            return False

        freq = [0] * 26
        for i in range(len(s1)):
            freq[ord(s1[i]) - ord('a')] += 1
            freq[ord(s2[i]) - ord('a')] -= 1

        return not any(freq)   # all zero?
```

---

## Сложность

- **Время:** O(n), где n — длина строк.
- **Память:** O(1) — массив фиксированного размера.

---

## Примечания

- Алгоритм использует тот факт, что алфавит ограничен 26 буквами. Для Unicode-строк пришлось бы использовать словарь.
- Если строки могут содержать не только строчные буквы, нужно расширить диапазон или применять `Counter`.
