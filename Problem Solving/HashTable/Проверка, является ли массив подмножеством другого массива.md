## Определение

Даны два массива `a[]` и `b[]`. Требуется определить, является ли массив `b[]` **подмножеством** массива `a[]` с учётом кратности элементов. То есть каждый элемент из `b[]` должен встречаться в `a[]` не меньшее число раз, чем в `b[]`.

**Пример:**
```
a = [11, 7, 1, 13, 21, 3, 7, 3]
b = [11, 3, 7, 1, 7]
Вывод: true (в a есть 11, три 7? нет, две 7 и 3, 1 — всё есть)
```

---

## Идея решения

Используем **хеш-таблицу** (словарь) для подсчёта частот элементов в массиве `a`. Затем проходим по массиву `b` и для каждого элемента:
- Если элемент отсутствует в словаре или его счётчик стал нулевым — возвращаем `False`.
- Иначе уменьшаем счётчик на 1.

Если все элементы `b` успешно «израсходованы», возвращаем `True`.

---

## Реализация на Python

```python
from collections import Counter

class Solution:
    def isSubset(self, a, b):
        freq = Counter(a)           # подсчёт частот в a
        for val in b:
            if freq.get(val, 0) == 0:
                return False
            freq[val] -= 1
        return True
```

---

## Сложность

- **Время:** O(len(a) + len(b)) — один проход для подсчёта частот, один проход для проверки.
- **Память:** O(len(a)) — для хранения словаря частот.

---

## Примечания

- Метод `Counter` из модуля `collections` создаёт словарь, где ключи — уникальные элементы, значения — их количество.
- Для элементов, отсутствующих в `freq`, вызов `freq.get(val, 0)` возвращает `0`, что корректно обрабатывается.
- Альтернативное решение: отсортировать оба массива и использовать два указателя, но хеширование проще и быстрее на практике.
