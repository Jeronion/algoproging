## Определение

Дан массив длины `n`. Для каждого элемента с индексом `i` (0-индексация) требуется определить, **сколько существует непрерывных подмассивов**, в которые этот элемент входит.

Эта величина является базовой для техники **«вклад элемента»** (contribution technique), используемой в задачах на сумму, XOR, произведение и другие агрегатные функции по всем подмассивам.

---

## Постановка задачи

Дан массив `arr` длины `n`. Требуется для каждого `i` (0 ≤ i < n) найти количество подмассивов `[L, R]` (L ≤ i ≤ R), где `L` и `R` — индексы начала и конца подмассива.

**Пример:**
```
arr = [a, b, c] (n=3)
Подмассивы: [a], [b], [c], [a,b], [b,c], [a,b,c]
Подсчёт вхождений:
- a входит в подмассивы: [a], [a,b], [a,b,c] → 3
- b входит в [b], [a,b], [b,c], [a,b,c] → 4
- c входит в [c], [b,c], [a,b,c] → 3
```

---

## Формула

Элемент с индексом `i` входит в любой подмассив, у которого:

- левая граница `L` может быть любой от `0` до `i` (включительно) → `(i+1)` вариантов.
- правая граница `R` может быть любой от `i` до `n-1` (включительно) → `(n-i)` вариантов.

Таким образом, общее количество подмассивов, содержащих `arr[i]`, равно произведению этих двух чисел:

```
count(i) = (i + 1) * (n - i)
```

---

## Вывод

Рассмотрим все возможные начала подмассива, которые не правее `i`: это индексы `0, 1, ..., i` — всего `i+1` вариант.

Для каждого фиксированного начала `L` подмассив должен закончиться где-то на или после `i`. Индексы конца: `i, i+1, ..., n-1` — всего `n-i` вариантов.

По правилу произведения получаем `(i+1)*(n-i)`.

---

## Реализация на Python (вычисление для всех элементов)

```python
def subarray_contain_counts(n):
    """Возвращает список количества подмассивов, содержащих элемент с индексом i."""
    counts = []
    for i in range(n):
        counts.append((i + 1) * (n - i))
    return counts

# Пример
n = 3
print(subarray_contain_counts(n))  # [3, 4, 3]
```

---

## Сложность

- **Время:** O(n) для вычисления всех, либо O(1) для одного индекса.
- **Память:** O(1) дополнительно (не считая выходной массив).

---

## Примечания

- Важно: формула справедлива для **непрерывных подмассивов** (contiguous subarrays).
- Для подмассивов с дополнительными условиями (например, только чётной длины) формула меняется, но подход остаётся.
