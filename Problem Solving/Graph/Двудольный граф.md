## Определение

**Двудольный граф** — это граф, вершины которого можно разбить на два непересекающихся множества (доли) так, что каждое ребро соединяет вершины из разных множеств. Эквивалентное определение: граф можно раскрасить в два цвета так, что смежные вершины всегда имеют разные цвета.

**Пример:**
- Граф с рёбрами `(0-1, 1-2)` является двудольным (цвета: 0-красный, 1-синий, 2-красный).
- Цикл нечётной длины (например, треугольник) не является двудольным.

---

## Постановка задачи

Дан неориентированный граф с `V` вершинами и `E` рёбрами. Определить, является ли он двудольным.

**Вход:**
- `V` — количество вершин.
- `edges` — список рёбер `[u, v]`.

**Выход:** `True`, если граф двудольный, иначе `False`.

---

## Идея решения (BFS с раскраской)

Используем **поиск в ширину (BFS)** для раскраски вершин в два цвета (0 и 1). Начинаем с произвольной вершины, окрашиваем её в цвет 0, затем обходим всех её соседей и окрашиваем их в противоположный цвет (1). Если при обходе встречается сосед, уже имеющий тот же цвет, что и текущая вершина, то граф не является двудольным.

Так как граф может быть несвязным, запускаем BFS для каждой ещё неокрашенной вершины.

### Алгоритм

1. Построить список смежности `graph`.
2. Создать массив `colors` размера `V`, изначально заполненный `None` (цвет не назначен).
3. Для каждой вершины `v` от 0 до `V-1`:
   - Если `colors[v]` уже задан, пропустить.
   - Иначе начать BFS: добавить `v` в очередь, установить `colors[v] = 0`.
   - Пока очередь не пуста:
     - Извлечь вершину `u`.
     - Для каждого соседа `w`:
       - Если `colors[w] == colors[u]`, то конфликт → вернуть `False`.
       - Если `colors[w] is None`, то установить `colors[w] = 1 - colors[u]` и добавить `w` в очередь.
4. Если все компоненты успешно окрашены, вернуть `True`.

---

## Реализация на Python

```python
from collections import deque, defaultdict

class Solution:
    def isBipartite(self, V, edges):
        graph = defaultdict(list)
        for u, v in edges:
            graph[u].append(v)
            graph[v].append(u)
        
        colors = [None] * V
        
        for start in range(V):
            if colors[start] is not None:
                continue
            q = deque([start])
            colors[start] = 0
            while q:
                u = q.popleft()
                for v in graph[u]:
                    if colors[v] == colors[u]:
                        return False
                    if colors[v] is None:
                        colors[v] = 1 - colors[u]
                        q.append(v)
        return True
```

## Сложность

- **Время:** O(V + E) — обход всех вершин и рёбер.
- **Память:** O(V + E) для хранения графа и O(V) для массива цветов и очереди.

---

## Примечания

- Граф без рёбер (все вершины изолированы) является двудольным (можно разбить на два любых множества).
- Наличие цикла нечётной длины — достаточное условие недвудольности. Например, треугольник требует 3 цветов, поэтому не подходит.
- Альтернативно можно использовать DFS с передачей цвета, но BFS здесь также удобен.
- Массив `colors` может содержать значения `0` и `1`; `None` означает неокрашенность.
