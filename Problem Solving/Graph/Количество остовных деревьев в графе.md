## Определение

**Остовное дерево (spanning tree)** неориентированного связного графа — это подграф, который содержит все вершины графа, не имеет циклов и содержит ровно `V-1` рёбер. Количество таких деревьев может быть вычислено с помощью **теоремы Кирхгофа (Matrix-Tree Theorem)**.

**Пример:**
- Граф, являющийся деревом (V-1 ребро), имеет ровно 1 остовное дерево.
- Полный граф `K_4` имеет `4^(4-2) = 16` остовных деревьев.

---

## Постановка задачи

Дан связный неориентированный граф с `V` вершинами, заданный матрицей смежности `graph[V][V]` (1 — ребро, 0 — отсутствие). Требуется найти общее число остовных деревьев графа.

---

## Идея решения (теорема Кирхгофа)

Теорема Кирхгофа утверждает, что количество остовных деревьев равно **любому главному минору** матрицы Кирхгофа (Лапласиана) размера `(V-1)×(V-1)`.

### Шаги

1. **Построить матрицу Лапласа `L` размера `V × V`**:
   - `L[i][i]` = степень вершины `i` (количество рёбер, инцидентных `i`).
   - `L[i][j]` = `-1`, если есть ребро между `i` и `j` (i ≠ j).
   - `L[i][j]` = 0 в остальных случаях.

2. **Удалить последнюю строку и последний столбец** (или любую одну строку и один столбец) — получить матрицу `minor` размера `(V-1) × (V-1)`.

3. **Вычислить определитель** матрицы `minor`. Абсолютное значение определителя равно числу остовных деревьев.

### Почему это работает?

Матрица Лапласа представляет все возможные подграфы. Её определитель после удаления одной строки/столбца равен сумме по всем остовным деревьям (каждое даёт вклад 1). Это следствие теоремы Кирхгофа.

## Реализация на Python

```python
def determinant(mat, n):
    """Вычисляет определитель матрицы n×n методом Гаусса (с плавающей точкой)."""
    det = 1.0
    for i in range(n):
        # Поиск ведущего элемента
        pivot = i
        for j in range(i + 1, n):
            if abs(mat[j][i]) > abs(mat[pivot][i]):
                pivot = j
        if abs(mat[pivot][i]) < 1e-9:
            return 0
        if i != pivot:
            mat[i], mat[pivot] = mat[pivot], mat[i]
            det *= -1
        det *= mat[i][i]
        # Обнуление элементов ниже
        for j in range(i + 1, n):
            factor = mat[j][i] / mat[i][i]
            for k in range(i, n):
                mat[j][k] -= factor * mat[i][k]
    return det

def count_spanning_trees(graph, V):
    # Построение матрицы Лапласа
    L = [[0.0] * V for _ in range(V)]
    for i in range(V):
        degree = 0
        for j in range(V):
            if graph[i][j] == 1:
                degree += 1
                if i != j:
                    L[i][j] = -1.0
        L[i][i] = float(degree)
    
    # Удаляем последнюю строку и столбец
    minor = [[L[i][j] for j in range(V - 1)] for i in range(V - 1)]
    # Определитель (результат округляем до целого)
    return round(determinant(minor, V - 1))

# Пример использования
V = 4
graph = [
    [0, 1, 1, 1],
    [1, 0, 1, 1],
    [1, 1, 0, 1],
    [1, 1, 1, 0]
]
print(count_spanning_trees(graph, V))  # 16
```

---

## Сложность

- **Время:** O(V³) — из-за вычисления определителя методом Гаусса.
- **Память:** O(V²) для хранения матрицы Лапласа и минора.

---

## Примечания

- Для несвязных графов количество остовных деревьев равно 0 (определитель будет равен 0).
- Для полного графа `K_n` существует формула Кэли: `n^(n-2)`.
- Теорема Кирхгофа работает для мультиграфов (если учитывать кратность рёбер), но в данной задаче граф простой.
