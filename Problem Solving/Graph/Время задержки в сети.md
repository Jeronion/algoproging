## Определение

Дан ориентированный взвешенный граф, представляющий сеть узлов. Каждое ребро `(u, v, w)` означает, что сигнал, отправленный из узла `u`, достигает узла `v` через `w` единиц времени. Задан источник `src`. Требуется найти **минимальное время**, за которое сигнал достигнет всех узлов сети. Если хотя бы один узел недостижим, вернуть `-1`.

**Пример:**
```
V = 3, edges = [[0,2,1], [2,1,2], [0,1,4]], src = 0
Вывод: 3 (сигнал идёт 0→2 (1), затем 2→1 (2), итого 3)
```

---

## Идея решения

Задача сводится к поиску кратчайших расстояний от источника до всех остальных узлов. Поскольку веса рёбер неотрицательные, можно использовать **алгоритм Дейкстры** с приоритетной очередью (min-heap). После вычисления расстояний до всех узлов, ответом будет **максимальное** из этих расстояний (время, необходимое для покрытия самого удалённого узла). Если какая-то вершина осталась недостижимой (расстояние равно бесконечности), возвращаем `-1`.

### Алгоритм

1. Построить список смежности из рёбер.
2. Инициализировать массив `dist` размера `V` со значением `∞`; `dist[src] = 0`.
3. Поместить `(0, src)` в min-heap.
4. Пока куча не пуста:
   - Извлечь `(d, u)`.
   - Если `d > dist[u]`, пропустить (устаревшая запись).
   - Для каждого соседа `(w, v)` из `graph[u]`:
     - Если `dist[u] + w < dist[v]`, обновить `dist[v]` и добавить `(dist[v], v)` в кучу.
5. После завершения Дейкстры найти максимальное значение среди `dist`.
6. Если максимум равен `∞`, вернуть `-1`, иначе вернуть максимум.

## Реализация (из вашего кода)

```python
from collections import defaultdict
import heapq

class Solution:
    def findDelayTime(self, V, edges, src):
        graph = defaultdict(list)
        for u, v, w in edges:
            graph[u].append((w, v))
        
        dist = [float("inf")] * V
        dist[src] = 0
        pq = [(0, src)]
        
        while pq:
            d, u = heapq.heappop(pq)
            if d > dist[u]:
                continue
            for w, v in graph[u]:
                if dist[u] + w < dist[v]:
                    dist[v] = dist[u] + w
                    heapq.heappush(pq, (dist[v], v))
        
        res = max(dist)
        return -1 if res == float("inf") else res
```

## Сложность

- **Время:** O(E log V), где E — число рёбер, V — число вершин (стандартная сложность Дейкстры с кучей).
- **Память:** O(V + E) для хранения графа и массивов.

---

## Примечания

- Алгоритм Дейкстры работает только с **неотрицательными** весами (в условии задачи веса ≥ 0).
- Если граф содержит отрицательные веса, необходимо использовать Беллмана-Форда.
- Ответ — максимальное расстояние, так как сигнал распространяется параллельно; все узлы получат его к моменту, когда будет достигнут самый дальний.
