## Определение

**Цикл в неориентированном графе** — это путь, начинающийся и заканчивающийся в одной вершине, без повторения рёбер и вершин (кроме стартовой/конечной). Задача состоит в том, чтобы проверить, содержит ли граф хотя бы один цикл. Для решения этой задачи можно использовать **систему непересекающихся множеств (Disjoint Set Union, DSU)**.

---

## Постановка задачи

Дан неориентированный граф без петель с `V` вершинами (пронумерованными от 0 до V-1) и `E` рёбрами. Требуется определить, есть ли в графе цикл.

**Вход:**
- `V` — количество вершин.
- `edges` — список рёбер `[u, v]`.

**Выход:** `True`, если цикл существует, иначе `False`.

---

## Идея решения (DSU)

В неориентированном графе при добавлении рёбер в DSU цикл возникает, когда мы пытаемся соединить две вершины, которые уже принадлежат одному множеству. Если вершины находятся в разных множествах, их можно объединить без образования цикла. Таким образом, алгоритм:

1. Инициализировать DSU для всех вершин.
2. Последовательно обрабатывать каждое ребро.
3. Для ребра `(u, v)`:
   - Найти представителей множеств, содержащих `u` и `v`.
   - Если представители совпадают — ребро создаёт цикл, вернуть `True`.
   - Иначе объединить множества (union).
4. Если после обработки всех рёбер цикл не обнаружен, вернуть `False`.

Этот метод эффективно использует операции `find` и `union` с оптимизациями (сжатие путей и объединение по рангу), что обеспечивает почти константное время.

---

## Алгоритм

1. Создать объект DSU с `V` элементами.
2. Для каждого ребра `(u, v)` в списке `edges`:
   - Если `find(u) == find(v)`, то цикл найден → вернуть `True`.
   - Иначе выполнить `union(u, v)`.
3. Если цикл не найден, вернуть `False`.

---

## Реализация на Python

```python
class DSU:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [1] * n

    def find(self, x):
        # Path compression
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])
        return self.parent[x]

    def union(self, a, b):
        # Union by rank
        ra = self.find(a)
        rb = self.find(b)
        if ra == rb:
            return False  # уже в одном множестве (цикл)
        if self.rank[ra] < self.rank[rb]:
            self.parent[ra] = rb
        elif self.rank[rb] < self.rank[ra]:
            self.parent[rb] = ra
        else:
            self.parent[rb] = ra
            self.rank[ra] += 1
        return True

class Solution:
    def detectCycle(self, V, edges):
        dsu = DSU(V)
        for u, v in edges:
            if not dsu.union(u, v):
                return True
        return False
```

## Сложность

- **Время:** O(E · α(V)), где α(V) — обратная функция Аккермана (практически константа). Это достигается за счёт сжатия путей и объединения по рангу.
- **Память:** O(V) для хранения массивов `parent` и `rank`.

---

## Примечания

- Алгоритм работает только для **неориентированных** графов без петель.
- Если граф несвязный, то цикл может находиться в любой компоненте связности; DSU будет обрабатывать каждое ребро независимо.
- В отличие от DFS, DSU не требует обхода всего графа, а проверяет циклы в процессе добавления рёбер. Это особенно удобно, если рёбра поступают динамически.
- Для ориентированных графов DSU не подходит (там используется другой подход, например, DFS с тремя состояниями или алгоритм Кана).
