## Определение

Дан ориентированный граф с `V` вершинами и `E` рёбрами. Требуется определить, содержит ли граф **цикл** — путь, начинающийся и заканчивающийся в одной вершине, с соблюдением направления рёбер. В ориентированном графе цикл может быть обнаружен с помощью обхода в глубину (DFS) с отслеживанием вершин, находящихся в текущем стеке рекурсии.

**Пример:**
- `edges = [[0,1], [1,2], [2,0]]` → `True` (цикл 0→1→2→0)
- `edges = [[0,1], [0,2], [1,2], [2,3]]` → `False`

---

## Постановка задачи

**Вход:**
- `V` — количество вершин (от 0 до V-1).
- `edges` — список направленных рёбер `[u, v]`.

**Выход:** `True`, если в графе есть хотя бы один цикл, иначе `False`.

---

## Идея решения (DFS с рекурсивным стеком)

В ориентированном графе наличие цикла можно проверить с помощью DFS, отслеживая вершины, которые находятся в **текущем пути** рекурсии (стек вызовов). Если при обходе мы встречаем вершину, которая уже есть в этом стеке, значит, мы нашли цикл. Вершины, которые уже обработаны полностью (посещены и удалены из стека), не приводят к циклу.

### Алгоритм

1. Построить список смежности `graph` из рёбер.
2. Использовать два множества:
   - `visited` — вершины, которые были полностью обработаны (все соседи посещены).
   - `recStack` — вершины, которые находятся в текущем стеке рекурсии (на пути от корня).
3. Для каждой вершины `v` от `0` до `V-1`:
   - Если `v` ещё не посещена, запустить DFS.
4. Рекурсивная функция `dfs(v)`:
   - Если `v` уже в `recStack` → найден цикл, вернуть `True`.
   - Если `v` уже в `visited` → пропустить (возврат `False`).
   - Добавить `v` в `visited` и `recStack`.
   - Для каждого соседа `u` из `graph[v]`:
     - Если `dfs(u)` возвращает `True` → цикл найден, передать дальше.
   - Удалить `v` из `recStack` (возврат назад).
   - Вернуть `False`.
5. Если ни один вызов не вернул `True`, циклов нет.

---

## Реализация на Python

```python
from collections import defaultdict

class Solution:
    def isCyclic(self, V, edges):
        graph = defaultdict(list)
        for u, v in edges:
            graph[u].append(v)
        
        visited = set()
        recStack = set()
        
        def dfs(v):
            if v in recStack:
                return True
            if v in visited:
                return False
            
            visited.add(v)
            recStack.add(v)
            
            for u in graph[v]:
                if dfs(u):
                    return True
            
            recStack.remove(v)
            return False
        
        for v in range(V):
            if v not in visited:
                if dfs(v):
                    return True
        return False
```

---

## Сложность

- **Время:** O(V + E) — обход всех вершин и рёбер.
- **Память:** O(V) для множеств `visited`, `recStack` и стека рекурсии (в худшем случае O(V)).

---

## Примечания

- Множество `recStack` отслеживает вершины на текущем пути, что позволяет отличать циклы от повторного посещения уже обработанных вершин.
- Альтернативный подход: использовать массив `state` (0 — не посещена, 1 — в процессе, 2 — обработана) вместо двух множеств, что улучшает производительность.
- Если граф не является ациклическим, алгоритм вернёт `True` при обнаружении первого же цикла.
