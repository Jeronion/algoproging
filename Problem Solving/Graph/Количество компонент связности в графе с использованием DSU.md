## Определение

**Компонента связности** неориентированного графа — это максимальное множество вершин, в котором между любой парой вершин существует путь. Две вершины принадлежат одной компоненте, если они достижимы друг из друга. Количество компонент связности — это число таких непересекающихся множеств, покрывающих все вершины графа.

**Пример:**
```
Вход: V = 5, edges = [[0,1], [1,2], [3,4]]
Компоненты: {0,1,2} и {3,4} → ответ 2.
```

---

## Постановка задачи

Дан неориентированный граф с `V` вершинами (пронумерованными от 0 до V-1) и списком рёбер `edges`. Требуется найти количество связных компонент графа.

**Вход:**
- `V` — количество вершин.
- `edges` — список пар `[u, v]` (рёбра).

**Выход:** число компонент связности.

---

## Идея решения (DSU)

Используем **систему непересекающихся множеств (Disjoint Set Union, DSU)**: каждая вершина изначально образует своё множество. При добавлении ребра `(u, v)` объединяем множества, содержащие `u` и `v`. После обработки всех рёбер количество различных корней (представителей) в DSU равно числу компонент связности.

### Оптимизации

- **Сжатие путей** (Path Compression) в `find` ускоряет последующие операции.
- **Объединение по рангу** (Union by Rank) минимизирует высоту деревьев.

После всех объединений проходим по всем вершинам, находим их корни через `find` и сохраняем уникальные корни в множество. Размер множества — ответ.

---

## Алгоритм

1. Инициализировать DSU для `V` вершин: `parent[i] = i`, `rank[i] = 1`.
2. Для каждого ребра `(u, v)`:
   - Выполнить `union(u, v)`.
3. Создать пустое множество `roots`.
4. Для каждой вершины `i` от 0 до V-1:
   - `roots.add(find(i))`.
5. Вернуть `len(roots)`.

---

## Реализация на Python

```python
class DSU:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [1] * n
    
    def find(self, x):
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])
        return self.parent[x]
    
    def union(self, a, b):
        ra = self.find(a)
        rb = self.find(b)
        if ra == rb:
            return
        if self.rank[ra] < self.rank[rb]:
            self.parent[ra] = rb
        elif self.rank[rb] < self.rank[ra]:
            self.parent[rb] = ra
        else:
            self.parent[rb] = ra
            self.rank[ra] += 1

class Solution:
    def countComponents(self, V, edges):
        dsu = DSU(V)
        for u, v in edges:
            dsu.union(u, v)
        roots = set()
        for v in range(V):
            roots.add(dsu.find(v))
        return len(roots)
```

---

## Сложность

- **Время:** O((V+E) · α(V)), где α — обратная функция Аккермана (практически константа). Фактически O(V+E) с учётом сжатия путей.
- **Память:** O(V) для массивов `parent` и `rank`.

---

## Примечания

- Граф может быть несвязным — алгоритм корректно обрабатывает все компоненты.
- Если граф пуст (нет рёбер), ответ будет равен V (каждая вершина — отдельная компонента).
- DSU также может использоваться для динамического добавления рёбер и запросов о связности.
