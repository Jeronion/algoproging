## Определение

Задача **Alien Dictionary** заключается в определении порядка символов в неизвестном языке по заданному словарю слов, которые отсортированы лексикографически согласно этому порядку. Требуется восстановить возможный порядок букв (или вернуть пустую строку, если данные противоречивы). Это классическая задача на **топологическую сортировку** ориентированного графа, где вершины — буквы, а направленные рёбра задают отношение «предшествует»: если в двух соседних словах на первой различающейся позиции стоят буквы `a` и `b`, то `a` должна идти перед `b`.

---

## Постановка задачи

**Вход:** массив слов `words`, отсортированных по правилам инопланетного языка.
**Выход:** строка, содержащая уникальные буквы в порядке, согласующемся со словарём (любой допустимый порядок). Если порядок не может быть определён или данные противоречивы, вернуть `""`.

**Пример:**
```
words = ["baa", "abcd", "abca", "cab", "cad"]
Вывод: "bdac" (или другой допустимый порядок)
```
**Ограничения:** количество слов ≤ 500, длина слова ≤ 100.

---

## Идея решения

1. Из попарного сравнения соседних слов извлекаются правила упорядочения букв. Если в паре `(w1, w2)` на некоторой позиции `i` буквы различаются, то `w1[i]` должна идти раньше `w2[i]` — добавляем ребро в граф.
2. Если одно слово является **префиксом** другого, и первое длиннее второго, то это противоречит лексикографическому порядку (например, `"ab"` и `"a"` — `"a"` должна быть раньше, но `"ab"` длиннее, что невозможно). В этом случае возвращаем `""`.
3. Построенный граф должен быть **ациклическим**; порядок букв находится с помощью топологической сортировки (алгоритм Кана).
4. Если в графе есть цикл или не все буквы удалось упорядочить, возвращаем `""`.

---

## Алгоритм (пошагово)

1. Собрать все уникальные буквы из слов в множество `letters`.
2. Для каждой пары соседних слов `(w1, w2)`:
   - Найти первую позицию `i`, где `w1[i] != w2[i]`.
   - Если такая позиция найдена, добавить ребро `w1[i] → w2[i]` в граф.
   - Если `w1` длиннее `w2` и `w2` является префиксом `w1` (т.е. все символы до `len(w2)` совпадают), то это противоречие → вернуть `""`.
3. Вычислить входящие степени (`indegree`) для всех букв.
4. Инициализировать очередь всеми буквами с `indegree == 0`.
5. Пока очередь не пуста:
   - Извлечь букву `u`, добавить в результат.
   - Для каждого соседа `v` из `graph[u]` уменьшить `indegree[v]`; если `indegree[v] == 0`, добавить в очередь.
6. Если длина результата равна числу уникальных букв, вернуть строку; иначе (есть цикл) вернуть `""`.

## Реализация на Python

```python
from collections import defaultdict, deque

class Solution:
    def findOrder(self, words):
        n = len(words)
        
        # собираем все уникальные буквы
        letters = set()
        for word in words:
            for ch in word:
                letters.add(ch)
        
        graph = defaultdict(list)
        # строим рёбра из сравнения соседних слов
        for i in range(1, n):
            w1, w2 = words[i-1], words[i]
            j = 0
            while j < min(len(w1), len(w2)):
                if w1[j] != w2[j]:
                    graph[w1[j]].append(w2[j])
                    break
                j += 1
            else:
                # если одно слово — префикс другого
                if len(w1) > len(w2):
                    return ""
        
        # вычисляем in-degree
        indegree = defaultdict(int)
        for u in letters:
            for v in graph[u]:
                indegree[v] += 1
        
        # алгоритм Кана
        q = deque([v for v in letters if indegree[v] == 0])
        res = []
        while q:
            u = q.popleft()
            res.append(u)
            for v in graph[u]:
                indegree[v] -= 1
                if indegree[v] == 0:
                    q.append(v)
        
        return "".join(res) if len(res) == len(letters) else ""
```

## Сложность

- **Время:** O(Σ длины слов + количество рёбер) ≈ O(N·L), где N — число слов, L — средняя длина. Поиск всех букв и сравнение слов линейны по суммарной длине.
- **Память:** O(V + E), где V — количество уникальных букв (≤ 26), E — число рёбер (≤ V²).

---

## Примечания

- Порядок букв в ответе не единственный; алгоритм возвращает любой из возможных.
- Проверка на префикс обязательна: если `w1` длиннее `w2` и `w2` является префиксом `w1`, то нарушается условие лексикографической упорядоченности (в алфавитном порядке более короткое слово должно идти раньше).
- Граф может содержать цикл (например, `"ab", "ba"`), тогда очередь не охватит все вершины, и ответ будет пустым.
