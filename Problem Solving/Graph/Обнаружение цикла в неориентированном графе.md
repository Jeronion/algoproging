## Определение

Дан неориентированный граф с `V` вершинами и `E` рёбрами, заданный списком рёбер `edges`. Требуется определить, содержит ли граф **цикл** (т.е. существует ли путь, начинающийся и заканчивающийся в одной вершине, с рёбрами, не повторяющимися). Граф может состоять из нескольких компонент связности.

**Пример:**
- `edges = [[0,1], [0,2], [1,2], [2,3]]` → `True` (цикл 0-1-2-0)
- `edges = [[0,1], [1,2], [2,3]]` → `False`

---

## Постановка задачи

**Вход:**
- `V` — количество вершин.
- `edges` — список рёбер `[u, v]`.

**Выход:** `True`, если в графе есть хотя бы один цикл, иначе `False`.

---

## Идея решения (DFS с отслеживанием родителя)

Используем **обход в глубину (DFS)** для каждой непосещённой вершины. В процессе обхода передаём родительскую вершину (`parent`), чтобы не учитывать обратное ребро к родителю как цикл. Если при обходе встречается уже посещённая вершина, которая **не является родителем** текущей, то найден цикл.

### Алгоритм

1. Построить список смежности `graph` из списка рёбер.
2. Создать множество `visited` для отслеживания посещённых вершин.
3. Для каждой вершины `v` от `0` до `V-1`:
   - Если `v` ещё не посещена, запустить DFS.
4. DFS:
   - Пометить вершину как посещённую.
   - Для каждого соседа `neighbor`:
     - Если сосед не посещён, рекурсивно вызвать DFS от него с родителем `v`. Если рекурсия вернула `True` — цикл найден.
     - Если сосед уже посещён и не является родителем — это цикл, вернуть `True`.
5. Если ни один вызов не вернул `True`, циклов нет.

## Реализация

```python
from collections import defaultdict

class Solution:
    def isCycle(self, V, edges):
        graph = defaultdict(list)
        for u, v in edges:
            graph[u].append(v)
            graph[v].append(u)
        
        visited = set()
        
        def dfs(node, parent):
            visited.add(node)
            for neighbor in graph[node]:
                if neighbor not in visited:
                    if dfs(neighbor, node):
                        return True
                elif neighbor != parent:
                    return True
            return False
        
        for v in range(V):
            if v not in visited:
                if dfs(v, -1):
                    return True
        return False
```

---

## Сложность

- **Время:** O(V + E) — обход всех вершин и рёбер.
- **Память:** O(V + E) для хранения графа и множества `visited` (плюс стек рекурсии).

---

## Примечания

- Для неориентированного графа важно передавать родителя, чтобы не принимать ребро назад за цикл.
- Граф может содержать несколько компонент связности, поэтому запускаем DFS для каждой непосещённой вершины.
- Альтернативно можно использовать BFS с сохранением родителя для каждой вершины.
