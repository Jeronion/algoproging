## Определение

Дан массив `arr`, представляющий высоты столбцов гистограммы (ширина каждого столбца = 1). Требуется найти площадь **самого большого прямоугольника**, который можно вписать в гистограмму (прямоугольник должен состоять из непрерывных столбцов, а его высота ограничена минимальной высотой выбранных столбцов).

**Пример:**
```
arr = [60, 20, 50, 40, 10, 50, 60] → 100 (прямоугольник высотой 50 и шириной 2)
```

---

## Идея решения (монотонный стек)

Используем стек для хранения **индексов** столбцов в порядке возрастания их высот. Алгоритм:

1. Проходим по массиву слева направо.
2. Для каждого столбца `i`:
   - Пока стек не пуст и высота текущего столбца **меньше** высоты столбца на вершине стека, это означает, что для столбца на вершине стека текущий столбец является **правой границей** (first smaller to the right).
   - Извлекаем индекс `tp` из стека, вычисляем площадь прямоугольника с высотой `arr[tp]`:
     - Ширина = `i - (индекс предыдущего элемента в стеке после извлечения) - 1` (если стек пуст, то ширина = `i`).
     - Обновляем максимум.
   - После этого помещаем текущий индекс `i` в стек.
3. После обработки всех столбцов, для оставшихся в стеке индексов (у них нет правой границы) считаем ширину = `n - (предыдущий индекс) - 1` (или `n`, если стек пуст).

Этот метод вычисляет для каждого столбца максимальную площадь, где он является наименьшим (ограничивающим) в прямоугольнике.

---

## Алгоритм

1. Инициализировать пустой стек `stack` и `max_area = 0`.
2. Для `i` от 0 до `n-1`:
   - Пока `stack` не пуст и `arr[stack[-1]] >= arr[i]`:
     - `top = stack.pop()`
     - `width = i` если `stack` пуст, иначе `i - stack[-1] - 1`
     - `max_area = max(max_area, arr[top] * width)`
   - `stack.append(i)`
3. После цикла: пока `stack` не пуст:
   - `top = stack.pop()`
   - `width = n` если `stack` пуст, иначе `n - stack[-1] - 1`
   - `max_area = max(max_area, arr[top] * width)`
4. Вернуть `max_area`.

---

## Реализация

```python
def largest_rectangle_area(heights):
    stack = []
    max_area = 0
    n = len(heights)
    for i in range(n):
        while stack and heights[stack[-1]] >= heights[i]:
            top = stack.pop()
            width = i if not stack else i - stack[-1] - 1
            max_area = max(max_area, heights[top] * width)
        stack.append(i)
    while stack:
        top = stack.pop()
        width = n if not stack else n - stack[-1] - 1
        max_area = max(max_area, heights[top] * width)
    return max_area

# Пример
arr = [60, 20, 50, 40, 10, 50, 60]
print(largest_rectangle_area(arr))  # 100
```

---

## Сложность

- **Время:** O(n) — каждый индекс помещается в стек и извлекается не более одного раза.
- **Память:** O(n) — для стека.

---

## Примечания

- Ключевое наблюдение: для каждого столбца прямоугольник, где он является минимальным, ограничен ближайшими меньшими столбцами слева и справа.
- Алгоритм не требует отдельных массивов для `prev_smaller` и `next_smaller`, вычисляя всё за один проход.
- Работает для массивов с любыми целыми неотрицательными числами.
