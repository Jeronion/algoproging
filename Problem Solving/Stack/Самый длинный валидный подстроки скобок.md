## Определение

Дана строка `s` из символов `'('` и `')'`. Найти длину самой длинной непрерывной подстроки, являющейся правильной скобочной последовательностью.

**Примеры:**
```
s = "(()())" → 6
s = "())" → 2
s = "()(()" → 2
```

---

## Подход 1: Использование стека

Сложность: O(n) времени, O(n) памяти.

### Алгоритм

1. Создать стек, положить в него `-1`.
2. `max_len = 0`.
3. Для `i` от `0` до `len(s)-1`:
   - Если `s[i] == '('`, push(i).
   - Если `s[i] == ')'`:
     - pop()
     - Если стек пуст, push(i)
     - Иначе `max_len = max(max_len, i - stack[-1])`
4. Вернуть `max_len`.

### Реализация

```python
def max_length_stack(s):
    stack = [-1]
    max_len = 0
    for i, ch in enumerate(s):
        if ch == '(':
            stack.append(i)
        else:
            stack.pop()
            if not stack:
                stack.append(i)
            else:
                max_len = max(max_len, i - stack[-1])
    return max_len
```

## Подход 2: Два прохода (без дополнительной памяти)

Сложность: O(n) времени, O(1) памяти.

### Алгоритм

1. `max_len = 0`
2. **Левый проход**:
   - `open = close = 0`
   - Для каждого `ch` в `s`:
     - Если `ch == '('`, `open += 1`
     - Если `ch == ')'`, `close += 1`
     - Если `open == close`: `max_len = max(max_len, 2 * close)`
     - Если `close > open`: `open = close = 0`
3. **Правый проход**:
   - `open = close = 0`
   - Для каждого `ch` в `s` в обратном порядке:
     - Если `ch == ')'`, `close += 1`
     - Если `ch == '('`, `open += 1`
     - Если `open == close`: `max_len = max(max_len, 2 * open)`
     - Если `open > close`: `open = close = 0`
4. Вернуть `max_len`.

### Реализация

```python
def max_length_two_pass(s):
    max_len = 0
    open_cnt = close_cnt = 0
    for ch in s:
        if ch == '(':
            open_cnt += 1
        else:
            close_cnt += 1
        if open_cnt == close_cnt:
            max_len = max(max_len, 2 * close_cnt)
        elif close_cnt > open_cnt:
            open_cnt = close_cnt = 0
    
    open_cnt = close_cnt = 0
    for ch in reversed(s):
        if ch == ')':
            close_cnt += 1
        else:
            open_cnt += 1
        if open_cnt == close_cnt:
            max_len = max(max_len, 2 * open_cnt)
        elif open_cnt > close_cnt:
            open_cnt = close_cnt = 0
    
    return max_len
```
