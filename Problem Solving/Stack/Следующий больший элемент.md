## Определение

Дан массив целых чисел `arr`. Для каждого элемента нужно найти **ближайший справа** элемент, который **больше** текущего. Если такого нет, результатом будет `-1`.

**Пример:**
```
arr = [1, 3, 2, 4] → [3, 4, 4, -1]
arr = [6, 8, 0, 1, 3] → [8, -1, 1, 3, -1]
```

---

## Идея решения (монотонный стек)

Используем стек для хранения индексов элементов, для которых ещё не найден следующий больший элемент. Проходим по массиву слева направо:

- Пока стек не пуст и текущий элемент `arr[i]` больше элемента, соответствующего вершине стека (`arr[stack[-1]]`), это значит, что для элемента в вершине стека текущий элемент является **следующим большим**.
- Записываем результат для этого индекса: `res[stack.pop()] = arr[i]`.
- После того как все такие элементы из стека обработаны, помещаем текущий индекс `i` в стек.
- После завершения цикла для всех оставшихся в стеке индексов следующий больший элемент отсутствует, ставим `-1`.

### Алгоритм

1. Инициализировать `res = [-1] * n` и пустой стек.
2. Для `i` от `0` до `n-1`:
   - Пока стек не пуст и `arr[stack[-1]] < arr[i]`:
     - `idx = stack.pop()`
     - `res[idx] = arr[i]`
   - `stack.append(i)`
3. Вернуть `res`.

---

## Реализация

```python
class Solution:
    def nextGreaterElement(self, arr):
        n = len(arr)
        res = [-1] * n
        stack = []
        for i in range(n):
            while stack and arr[stack[-1]] < arr[i]:
                idx = stack.pop()
                res[idx] = arr[i]
            stack.append(i)
        return res
```

---

## Сложность

- **Время:** O(n) — каждый элемент попадает в стек и выходит из него не более одного раза.
- **Память:** O(n) в худшем случае (стек может содержать все индексы, например, для убывающей последовательности).

---

## Примечания

- Алгоритм работает для любых целых чисел (включая отрицательные).
- Для случая, когда требуется найти следующий **больший или равный**, условие меняется на `<=`.
- Можно также решить задачу с помощью двух вложенных циклов за O(n²), но это неэффективно для больших массивов.
