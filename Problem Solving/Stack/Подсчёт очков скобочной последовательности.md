## Определение

Дана сбалансированная строка `s`, состоящая только из символов `'('` и `')'`. Оценка (score) вычисляется по правилам:
- `"()"` имеет оценку 1
- `AB` (конкатенация) имеет оценку `score(A) + score(B)`
- `(A)` имеет оценку `2 * score(A)`

Требуется вернуть общую оценку строки.

**Примеры:**
```
s = "()" → 1
s = "(())" → 2
s = "()()" → 2
s = "(()(()))" → 6
```

---

## Идея решения (стек)

Используем стек, в котором хранятся оценки для текущих уровней вложенности. Изначально кладём `0`.

Проходим по строке:
- Если встречаем `'('`, кладём в стек `0` (начало нового уровня).
- Если встречаем `')'`, извлекаем верхнее значение `tmp`:
  - Если `tmp == 0`, значит внутри скобок ничего не было → оценка 1.
  - Иначе внутри была подстрока с оценкой `tmp` → оценка `2 * tmp`.
  - Добавляем полученную оценку к новому верхнему элементу стека (родительскому уровню).

В конце в стеке остаётся одно число — итоговая оценка.

---

## Реализация

```python
def score_of_parentheses(s):
    stack = [0]
    for c in s:
        if c == '(':
            stack.append(0)
        else:
            tmp = stack.pop()
            val = 2 * tmp if tmp > 0 else 1
            stack[-1] += val
    return stack[-1]
```

---

## Сложность

- **Время:** O(n) — один проход по строке.
- **Память:** O(n) — в худшем случае стек содержит все открывающие скобки.

---

## Примечания

- В стеке всегда хранятся оценки для незакрытых уровней.
- Конкатенация `AB` обрабатывается автоматически: после закрытия одного уровня его оценка добавляется к родительскому, а затем следующий уровень (соседний) будет добавлен к тому же родительскому.
