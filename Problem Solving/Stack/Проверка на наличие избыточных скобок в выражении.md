## Определение

Дано строковое выражение `s`, содержащее операторы `+`, `-`, `*`, `/`, цифры (или буквы) и круглые скобки, образующие корректную скобочную последовательность. Требуется определить, есть ли в выражении **избыточные** (redundant) скобки – то есть такие скобки, внутри которых нет оператора (т.е. всё выражение внутри скобок состоит только из одного операнда или уже сгруппировано так, что скобки не нужны). Например, `(a+b)` – не избыточны (есть оператор), а `((a+b))` – внешние скобки избыточны, так как внутри них только подвыражение `(a+b)` без оператора относительно внешних скобок. Также `(a)` – избыточны.

**Примеры:**
```
"((a+b))" → true (внешние скобки избыточны)
"(a+(b)/c)" → true (b окружено скобками без оператора внутри)
"(a+b+(c+d))" → false (все скобки содержат операторы)
```

---

## Идея решения (стек)

Используем стек для отслеживания символов. Проходим по строке:

- Если текущий символ **не** `')'`, помещаем его в стек (включая `'('`, операнды и операторы).
- Если встречаем `')'`, то начинаем выталкивать из стека элементы до тех пор, пока не встретим соответствующую открывающую скобку `'('`. Во время выталкивания проверяем, есть ли среди извлечённых символов **хотя бы один оператор** (`+`, `-`, `*`, `/`). Если операторов нет – значит, внутри скобок нет операции, и скобки избыточны – возвращаем `True`.
- После обработки всей строки, если ни разу не было избыточных скобок, возвращаем `False`.

### Алгоритм

1. Создать пустой стек `st`.
2. Для каждого символа `ch` в строке:
   - Если `ch == ')'`:
     - Установить `flag = True` (предполагаем, что скобки избыточны).
     - Извлекать из стека элементы, пока не встретится `'('`:
       - Если извлечённый символ – оператор, установить `flag = False`.
     - Если `flag` осталось `True`, вернуть `True`.
   - Иначе (все остальные символы, включая `'('`):
     - Поместить `ch` в стек.
3. Вернуть `False`.

---

## Реализация

```python
def check_redundancy(s):
    stack = []
    for ch in s:
        if ch == ')':
            redundant = True
            while stack and stack[-1] != '(':
                top = stack.pop()
                if top in {'+', '-', '*', '/'}:
                    redundant = False
            stack.pop()   # удаляем '('
            if redundant:
                return True
        else:
            stack.append(ch)
    return False
```

## Сложность

- **Время:** O(n) – каждый символ обрабатывается один раз.
- **Память:** O(n) – в худшем случае стек может содержать все открывающие скобки и операторы.

---

## Примечания

- Выражение гарантированно корректно (скобки правильно расставлены), поэтому можно не проверять на пустой стек при встрече `')'`.
- Операторы могут быть как бинарными, так и унарными? По условию подразумеваются бинарные операции, потому что унарные не создают избыточности в обычном понимании.
