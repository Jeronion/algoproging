## Условие задачи

Дан массив целых чисел, размер которого заранее неизвестен (но указан в первой строке входного файла). Требуется вывести элементы массива в обратном порядке, сохранив их в одной строке через пробел.

**Входные данные (input.txt):**
- Первая строка: размер массива `N`.
- Вторая строка: `N` целых чисел через пробел.

**Выходные данные (output.txt):**
- Одна строка: элементы исходного массива в обратном порядке, разделённые пробелами.

**Пример:**

| input.txt     | output.txt |
|---------------|------------|
| 5<br>5 1 4 2 3 | 3 2 4 1 5 |

---

## Идея решения

1. Прочитать размер массива `N` (фактически не обязателен для алгоритма, если мы будем читать элементы до конца строки).
2. Создать пустой стек (в Python удобно использовать `list` или `collections.deque`).
3. Для каждого числа из второй строки:
   - Поместить число в стек (операция `push`).
4. После того как все числа прочитаны, последовательно извлекать числа из стека (операция `pop`) и записывать их в выходной файл.

---

## Реализация на Python

```python
def reverse_array():
    with open("input.txt", "r") as fin, open("output.txt", "w") as fout:
        # Читаем размер (можно проигнорировать)
        fin.readline()
        # Читаем строку с числами
        numbers = map(int, fin.readline().strip().split())
        stack = []
        for num in numbers:
            stack.append(num)      # push
        while stack:
            fout.write(str(stack.pop()) + " ")  # pop и вывод
```

**Альтернативная реализация с использованием `deque` как стека:**

```python
from collections import deque

def reverse_array():
    with open("input.txt", "r") as fin, open("output.txt", "w") as fout:
        fin.readline()
        numbers = map(int, fin.readline().strip().split())
        stack = deque()
        for num in numbers:
            stack.append(num)      # push в конец
        while stack:
            fout.write(str(stack.pop()) + " ")   # pop с конца
```

---

## Сложность

- **Время:** O(N), где N — количество элементов. Каждый элемент добавляется в стек и извлекается из него один раз.
- **Память:** O(N) — для хранения стека.
