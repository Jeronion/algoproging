## Определение

Дана строка `s`, представляющая неотрицательное целое число, и целое число `k`. Требуется удалить ровно `k` цифр так, чтобы оставшееся число (в том же порядке) было **минимально возможным**. Ведущие нули в результате не допускаются; если результат пуст, вернуть `"0"`.

**Пример:**
```
s = "4325043", k = 3 → "2043"
s = "765028321", k = 5 → "221"
```

---

## Идея решения (жадный алгоритм со стеком)

Чтобы получить минимальное число, нужно на каждой позиции по возможности удалять **большие** цифры, стоящие перед **меньшими**. Используем **стек**:

1. Проходим по символам строки.
2. Пока стек не пуст, верхний элемент стека больше текущего символа и `k > 0`, удаляем (выталкиваем) верхний элемент (уменьшаем `k`).
3. Затем помещаем текущий символ в стек.
4. После обработки всей строки, если `k` ещё не исчерпан (`k > 0`), удаляем последние `k` символов (так как они уже идут в порядке неубывания и для минимизации нужно удалить с конца).
5. Формируем результирующую строку, отбрасывая ведущие нули. Если строка пуста, возвращаем `"0"`.

### Алгоритм

1. Инициализировать пустой стек `st`.
2. Для каждого символа `ch` в `s`:
   - Пока `st` не пуст и `k > 0` и `st[-1] > ch`, выполнить `pop()` и `k -= 1`.
   - Добавить `ch` в стек.
3. Если `k > 0`, удалить последние `k` символов из стека (обрезать).
4. Пропустить ведущие нули, начиная с начала стека.
5. Если оставшаяся часть пуста, вернуть `"0"`, иначе вернуть её как строку.

---

## Реализация

```python
class Solution:
    def removeKdig(self, s: str, k: int) -> str:
        stack = []
        for ch in s:
            while stack and k > 0 and stack[-1] > ch:
                stack.pop()
                k -= 1
            stack.append(ch)
        
        # Если остались удаления, убираем с конца
        if k > 0:
            stack = stack[:-k]
        
        # Удаляем ведущие нули
        result = ''.join(stack).lstrip('0')
        return result if result else '0'
```

---

## Сложность

- **Время:** O(n) — каждый символ добавляется и удаляется из стека не более одного раза.
- **Память:** O(n) — стек может хранить до n символов.

---

## Примечания

- Алгоритм гарантирует минимальность результата за счёт жадного удаления «плохих» цифр.
- Обработка ведущих нулей важна: `lstrip('0')` удаляет их, а если строка становится пустой, возвращается `"0"`.
- Удаление из конца после цикла необходимо, если `k` не использовано полностью (например, `s = "12345"`).
