

Дан массив `arr[]`. Найдите префиксную сумму массива.
Префиксный массив сумм — это другой массив `prefixSum[]` того же размера, такой что:

### Примеры

**Ввод:**
`arr[] = [10, 20, 10, 5, 15]`

**Вывод:**
`[10, 30, 40, 45, 60]`

**Пояснение:**
Для каждого индекса `i` складываем все элементы от `0` до `i`:

* `prefixSum[0] = 10`
* `prefixSum[1] = 10 + 20 = 30`
* `prefixSum[2] = 10 + 20 + 10 = 40`
  и так далее.

---

## Реализация префиксных сумм

Идея заключается в создании массива `prefixSum[]` размером `n`.
Для каждого индекса `i` в диапазоне от `1` до `n - 1` выполняется:

```
prefixSum[i] = prefixSum[i - 1] + arr[i]
```

### Шаги решения задачи

1. Объявить новый массив `prefixSum[]` того же размера, что и входной массив
2. Запустить цикл `for` для обхода входного массива
3. Для каждого индекса прибавлять текущий элемент массива к предыдущему значению префиксной суммы

### Реализация на Python

```python
# функция для нахождения префиксного массива сумм
def prefSum(arr):
    n = len(arr)

    # массив для хранения префиксных сумм
    prefixSum = [0] * n

    # инициализация первого элемента
    prefixSum[0] = arr[0]

    # прибавляем текущий элемент к предыдущей сумме
    for i in range(1, n):
        prefixSum[i] = prefixSum[i - 1] + arr[i]

    return prefixSum
```

### Сложность алгоритма

* **Временная сложность:** `O(n)`
* **Дополнительная память:** `O(n)`
