
# Введение в массивы

Последнее обновление: 10 сентября 2025 г.

Массив — это коллекция элементов одного типа, которые хранятся в смежных областях памяти. Это одна из самых популярных и простых структур данных, используемых в программировании.

## Основные термины массива

- **Элемент массива:** Элементы — это объекты, хранящиеся в массиве.
    
- **Индекс массива:** Элементы массива доступны по их индексам. В большинстве языков программирования индексация начинается с 0.
    

## Представление массива в памяти

В массиве все элементы или ссылки на них хранятся в смежных областях памяти. Это позволяет эффективно получать доступ к элементам и выполнять операции с ними.

## Объявление массива

Массивы можно объявлять по-разному в разных языках программирования. Для наглядности приведены примеры объявления массива на некоторых языках:

**Python:** Все типы списков создаются одинаково

```python
arr = []
```

## Инициализация массива

Массивы можно инициализировать различными способами в зависимости от языка. Примеры на Python:

```python
# Список целых чисел
arr = [1, 2, 3, 4, 5]

# Список символов (строк)
arr = ['a', 'b', 'c', 'd', 'e']

# Список чисел с плавающей точкой
arr = [1.4, 2.0, 24.0, 5.0, 0.0]
```

## Зачем нужны массивы?

Предположим, что в классе пять студентов, и нам нужно хранить их оценки. Мы могли бы создать пять отдельных переменных, но если студентов станет очень много, управлять данными будет сложно. В этом случае удобнее использовать массив.

## Важность массивов

### Типы массивов

Массивы классифицируются по двум признакам:

1. По размеру
    
2. По количеству измерений
    

#### Типы массивов по размеру

1. **Массив фиксированного размера**
    
    - Размер массива изменить нельзя. Память выделяется только под указанное количество элементов.
        
    - Если размер неизвестен и выделена большая память, часть памяти будет простаивать. Если выделено меньше памяти, чем нужно, элементы не поместятся.
        

Пример в Python:

```python
# Создание массива фиксированного размера 5, заполненного нулями
arr = [0] * 5
print(arr)
```

2. **Динамический массив**
    
    - Размер массива меняется во время выполнения программы. Можно добавлять и удалять элементы по мере необходимости. Память выделяется и освобождается динамически.
        

Пример в Python:

```python
# Динамический массив
arr = []
```

#### Типы массивов по измерениям

1. **Одномерный массив (1-D Array)**
    
    - Элементы хранятся в строку, один за другим.
        
2. **Многомерный массив**
    
    - Массив с более чем одним измерением. Позволяет хранить сложные данные в виде таблиц и матриц.
        
    - **Двумерный массив (2-D Array или матрица):** Массив массивов, представляющий таблицу с строками и столбцами.
        
    - **Трёхмерный массив (3-D Array):** Массив, состоящий из двумерных массивов.
