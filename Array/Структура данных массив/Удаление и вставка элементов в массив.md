
Массив — это структура данных, которая хранит элементы одного типа в смежных областях памяти. Одними из самых часто используемых операций над массивами являются **удаление** и **вставка** элементов. Эти операции зависят от позиции элемента в массиве.

---

## 1. Вставка элемента по позиции

**Вставка** элемента означает добавление нового элемента в определённое место массива. При этом все элементы, находящиеся после позиции вставки, сдвигаются на одну позицию вправо.

### Шаги вставки элемента:

1. Определяем позицию, на которую нужно вставить элемент.
2. Сдвигаем все элементы начиная с этой позиции на одну ячейку вправо.
3. Вставляем новый элемент на указанную позицию.

### Пример на Python:

```python
arr = [10, 20, 30, 40, 50]
pos = 2   # Вставка на 3-ю позицию (индекс 2)
value = 25

# Вставка элемента
arr.insert(pos, value)

print("Массив после вставки:", arr)
```

**Вывод:**

```
Массив после вставки: [10, 20, 25, 30, 40, 50]
```

**Сложность операции:**

* Время: O(n) (поскольку необходимо сдвинуть элементы)
* Доп. память: O(1)

---

## 2. Удаление элемента по позиции

**Удаление** элемента означает удаление элемента с определённой позиции массива. После удаления все элементы, находящиеся после этой позиции, сдвигаются на одну ячейку влево, чтобы заполнить «пустое» место.

### Шаги удаления элемента:

1. Определяем индекс элемента, который нужно удалить.
2. Сдвигаем все элементы после этого индекса на одну ячейку влево.
3. Уменьшаем размер массива (если требуется).

### Пример на Python:

```python
arr = [10, 20, 30, 40, 50]
pos = 3   # Удаляем 4-й элемент (индекс 3)

# Удаление элемента
arr.pop(pos)

print("Массив после удаления:", arr)
```

**Вывод:**

```
Массив после удаления: [10, 20, 30, 50]
```

**Сложность операции:**

* Время: O(n) (поскольку необходимо сдвинуть элементы)
* Доп. память: O(1)

---

## 3. Особенности операций по позиции

| Операция                  | Время выполнения | Замечания                                        |
| ------------------------- | ---------------- | ------------------------------------------------ |
| Вставка в начало массива  | O(n)             | Все элементы сдвигаются вправо                   |
| Вставка в конец массива   | O(1)             | Если массив динамический (например, Python list) |
| Вставка в середину        | O(n)             | Сдвиг половины элементов                         |
| Удаление в начале массива | O(n)             | Все элементы сдвигаются влево                    |
| Удаление в конце массива  | O(1)             | Просто уменьшаем размер массива                  |
| Удаление в середине       | O(n)             | Сдвиг элементов после позиции                    |

> **Примечание:** В языках с **фиксированным размером массива** (например, C/C++) при вставке или удалении нужно вручную управлять памятью и сдвигать элементы. В **динамических массивах** (Python list, Java ArrayList) память управляется автоматически.

---

## 4. Применения

* **Вставка по позиции:** сортировка, добавление новых данных в список, обработка очередей.
* **Удаление по позиции:** фильтрация данных, удаление ошибок, динамическое управление массивом.
