
Сортировка пузырьком — самый простой алгоритм сортировки, который работает путём многократного обмена соседних элементов, если они находятся в неправильном порядке. Этот алгоритм неэффективен для больших наборов данных, так как его средняя и худшая временная сложность довольно высока.

**Принцип работы:**

1. Массив сортируется за несколько проходов. После первого прохода максимальный элемент оказывается в конце (на своей правильной позиции). Аналогично, после второго прохода второй по величине элемент оказывается на предпоследней позиции, и так далее.
    
2. На каждом проходе обрабатываются только элементы, которые ещё не заняли свои позиции. После k проходов k наибольших элементов перемещаются на последние k позиций.
    
3. В проходе сравниваются оставшиеся элементы и при необходимости меняются местами, если больший элемент находится перед меньшим. После этого наибольший элемент среди оставшихся оказывается на правильной позиции.
    

---

## Оптимизированная реализация на Python

```python
def bubbleSort(arr):
    n = len(arr)
    
    # Проходим по всем элементам массива
    for i in range(n):
        swapped = False

        # Последние i элементов уже на своих местах
        for j in range(0, n-i-1):
            # Сравниваем соседние элементы и меняем их местами, если нужно
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True
        
        # Если за проход не произошло ни одного обмена, останавливаем алгоритм
        if not swapped:
            break
```

---

## Анализ сложности сортировки пузырьком

- **Временная сложность:** `O(n²)`
    
- **Дополнительная память:** `O(1)`
    

---

## Преимущества Bubble Sort

- Легко понять и реализовать.
    
- Не требует дополнительной памяти.
    
- Стабильный алгоритм сортировки — элементы с одинаковым ключом сохраняют свой относительный порядок.
    

---

## Недостатки Bubble Sort

- Временная сложность `O(n²)` делает его очень медленным для больших массивов.
    
- Практически не применяется в реальных задачах. Используется в основном в образовательных целях для изучения различных способов сортировки.
