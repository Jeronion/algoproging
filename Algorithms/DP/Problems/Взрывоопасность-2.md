## Условие

Имеются три типа контейнеров: **A** (особо опасные), **B** (неопасные), **C** (совсем неопасные). Контейнеры укладываются в вертикальную стопку. Стопка **безопасна**, если в ней нет двух контейнеров **A** подряд. Требуется посчитать количество безопасных стопок длины **N** $(1 \le N \le 20$).

**Пример:**  
При $N = 2$ безопасные стопки: AB, AC, BA, BB, BC, CA, CB, CC (все, кроме AA). Ответ = 8.

---

## Решение с помощью одной рекуррентности

Приведённый код использует компактную формулу:

```python
def count_domino_tilings_variant():
    N = int(input())
    dp = [1] * (N + 1)
    dp[1] = 3
    for i in range(2, N + 1):
        dp[i] = (dp[i - 1] + dp[i - 2]) * 2
    return dp[-1]
```

### Как это работает

Обозначим $F(n)$ — количество безопасных стопок длины $n$.  
Рассмотрим последний контейнер:

- Если последний — **B** или **C** (2 варианта), то перед ним может быть **любая** безопасная стопка длины $n-1$ (все $F(n-1)$ вариантов).
- Если последний — **A**, то перед ним **не может быть A**, значит предпоследний контейнер — B или C (2 варианта), а перед ними — любая безопасная стопка длины $n-2$ (все $F(n-2)$ вариантов).

Таким образом:

$$
F(n) = 2 \cdot F(n-1) + 2 \cdot F(n-2)
$$

Базовые случаи:
- $F(1) = 3$ (одна стопка: A, B, C — все безопасны).
- $F(2) = 8$ (как в примере).

Вычисление идёт по формуле $F(n) = 2 \cdot (F(n-1) + F(n-2))$.

### Почему код выглядит именно так

- `dp[0] = 1` — это вспомогательное значение, необходимое для работы формулы при $n=2$ (чтобы `dp[2] = (dp[1] + dp[0]) * 2` дало 8).
- `dp[1] = 3` — базовое значение для длины 1.
- Цикл с `range(2, N+1)` вычисляет все последующие значения.

### Сложность

- **Время:** $O(N)$ — один проход.
- **Память:** $O(N)$ для хранения массива (можно сократить до $O(1)$, храня только два предыдущих значения).
