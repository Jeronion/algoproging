## Определение

**Трибоначчи** — последовательность, в которой каждый следующий член равен сумме трёх предыдущих. В задаче «Мячик на лесенке» количество способов спуститься с N-й ступеньки на землю (прыжками на 1, 2 или 3 ступени) подчиняется правилу:

$$
T_1 = 1,\; T_2 = 2,\; T_3 = 4,\; T_n = T_{n-1} + T_{n-2} + T_{n-3} \quad (n \ge 4)
$$

---

## Реализация (динамическое программирование)

```python
def tribonacci(n):
    if n == 0:
        return 0
    dp = [0] * (n + 2)
    dp[1], dp[2], dp[3] = 1, 2, 4
    for i in range(4, n + 1):
        dp[i] = dp[i-1] + dp[i-2] + dp[i-3]
    return dp[n]
```

## Сложность

- **Время:** O(n)
- **Память:** O(n) (можно оптимизировать до O(1))

---

## Применение

- Количество способов перемещения по лестнице с шагами 1, 2, 3
- Обобщение чисел Фибоначчи для задач с большим количеством шагов
- Алгоритмические задачи на динамическое программирование
