## Условие

Калькулятор выполняет три операции:
- прибавить 1
- умножить на 2
- умножить на 3

Из числа 1 получить число N (1 ≤ N ≤ 10⁶), минимизируя количество операций. Требуется вывести последовательность операций (цифрами 1,2,3).

**Пример:** N=5 → 3 операции (например: 1→2→4→5? нет, 1*2=2, 2*2=4, +1=5, но оптимальнее 1*3=3, +1=4, +1=5? 3 операции. На самом деле минимальное количество операций — 3, одна из возможных последовательностей: 1*2=2, 2*3=6? 5 не получится. Классический пример: 5 требует 3 операции: 1→2→4→5 (операции: *2, *2, +1). Код выводит последовательность в обратном порядке, затем разворачивает.)

---

## Идея

Динамика: `dp[x]` — минимальное количество операций для получения x.  
Для x>1:  
- `dp[x] = dp[x-1] + 1` (операция 1)  
- если x%20: `dp[x] = min(dp[x], dp[x//2]+1)` (операция 2)  
- если x%30: `dp[x] = min(dp[x], dp[x//3]+1)` (операция 3)

Запоминаем предка и тип операции для восстановления.

---

## Примечания

- `bytearray` для dp и op экономят память.
- `array('i')` для prev (индексы до 10⁶ помещаются в 4 байта).
- Вывод последовательности операций — `''.join(reversed(res))` даёт строку без разделителей.
- Для N=1 программа ничего не выводит (0 операций).

## Реализация (с восстановлением пути)

```python
def min_operations_to_one():
    N = int(input())
    if N == 1:
        print('')
        return

    dp = bytearray(N + 1)        # количество операций
    prev = array('i', [0]) * (N + 1)  # предыдущее число
    op = bytearray(N + 1)        # код операции (1,2,3)

    for x in range(2, N + 1):
        dp[x] = dp[x - 1] + 1
        prev[x] = x - 1
        op[x] = 1

        if x % 2 == 0 and dp[x // 2] + 1 < dp[x]:
            dp[x] = dp[x // 2] + 1
            prev[x] = x // 2
            op[x] = 2

        if x % 3 == 0 and dp[x // 3] + 1 < dp[x]:
            dp[x] = dp[x // 3] + 1
            prev[x] = x // 3
            op[x] = 3

    # Восстановление последовательности операций
    res = []
    cur = N
    while cur != 1:
        res.append(str(op[cur]))
        cur = prev[cur]

    print(''.join(reversed(res)))
```

---

## Сложность

- **Время:** O(N)
- **Память:** O(N) для массивов dp, prev, op (можно оптимизировать, но для N≤10⁶ допустимо).
