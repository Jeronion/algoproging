
## Определение

**Теорема Мастера** — это инструмент для решения рекуррентных соотношений вида:

$$ T(n) = aT\left(\frac{n}{b}\right) + f(n) $$

где:
- $n$ — размер задачи,
- $a \ge 1$ — количество подзадач,
- $b > 1$ — коэффициент уменьшения размера,
- $f(n)$ — затраты на разделение и объединение (вне рекурсивных вызовов).

Теорема позволяет быстро определить **асимптотическую оценку** $T(n)$ для алгоритмов, относящихся к принципу [[Разделяй и властвуй]], без построения дерева рекурсии.

---

## Ключевые моменты

- Теорема Мастера применима **только** к рекуррентным соотношениям вида $T(n) = aT(n/b) + f(n)$ с постоянными $a \ge 1$, $b > 1$.
- Она даёт **асимптотическую оценку**, не учитывая константы и малые значения $n$.
- Если теорема не подходит, используют метод подстановки, метод дерева рекурсии или метод итераций.
- Продвинутая версия охватывает более широкий класс функций $f(n)$, включая логарифмические множители.

---

## Три случая теоремы Мастера

### Случай 1: Работа на листьях доминирует

Если $f(n) = O\big(n^{\log_b a - \epsilon}\big)$ для некоторой константы $\epsilon > 0$, то

$$ T(n) = \Theta\big(n^{\log_b a}\big) $$

### Случай 2: Работа распределена равномерно

Если $f(n) = \Theta\big(n^{\log_b a} \log^k n\big)$ для некоторой константы $k \ge 0$, то

$$ T(n) = \Theta\big(n^{\log_b a} \log^{k+1} n\big) $$

### Случай 3: Работа на корне доминирует

Если $f(n) = \Omega\big(n^{\log_b a + \epsilon}\big)$ для некоторой константы $\epsilon > 0$, и выполняется условие регулярности

$$ a f\!\left(\frac{n}{b}\right) \le c f(n) \quad \text{для некоторой } c < 1 \text{ и всех достаточно больших } n, $$

то

$$ T(n) = \Theta\big(f(n)\big) $$

---

## Интуитивное объяснение через дерево рекурсии

- **Случай 1**: рост числа листьев $n^{\log_b a}$ перевешивает стоимость разделения/объединения $f(n)$. Ответ — $n^{\log_b a}$.
- **Случай 2**: стоимость листьев и корня сопоставимы; появляется дополнительный логарифмический множитель.
- **Случай 3**: стоимость разделения/объединения доминирует; ответ — $f(n)$.

## Структура рекурсии «разделяй и властвуй»

```python
def solve(problem):
    if is_trivial(problem):
        return direct_solution(problem)
    else:
        subproblems = divide(problem)          # a подзадач размера n/b
        solutions = [solve(sub) for sub in subproblems]
        return combine(solutions)              # f(n) работа
```

Время работы описывается именно приведённым рекуррентным соотношением.

---

## Когда теорема **не применима**

- $T(n)$ не монотонна (например, $T(n) = \sin n$).
- $f(n)$ не является полиномом или не подходит под форму (например, $2^n$).
- Размер подзадачи не кратен $n/b$ (например, $T(n) = T(\lfloor n/2 \rfloor) + \dots$ — обычно работает, но нужно аккуратно).
- $a$ и $b$ не постоянны.

## Примеры

### Бинарный поиск

$$ T(n) = T(n/2) + O(1) $$

- $a = 1$, $b = 2$, $f(n) = O(1) = \Theta(n^{\log_2 1}) = \Theta(1)$
- Случай 2 с $k = 0$ → $T(n) = \Theta(\log n)$

### Сортировка слиянием

$$ T(n) = 2T(n/2) + O(n) $$

- $a = 2$, $b = 2$, $\log_b a = 1$, $f(n) = \Theta(n) = \Theta(n^{\log_2 2})$
- Случай 2 с $k = 0$ → $T(n) = \Theta(n \log n)$

### Умножение матриц по Штрассену (упрощённо)

$$ T(n) = 7T(n/2) + O(n^2) $$

- $a = 7$, $b = 2$, $\log_2 7 \approx 2.8$, $f(n) = O(n^2) = O(n^{2.8 - 0.8})$
- Случай 1 → $T(n) = \Theta(n^{\log_2 7})$

### Другой пример

$$ T(n) = 3T(n/2) + n^2 $$

- $a = 3$, $b = 2$, $\log_2 3 \approx 1.585$, $f(n) = n^2 = \Omega(n^{1.585 + \epsilon})$
- Проверка условия регулярности: $3 \cdot (n/2)^2 = 3n^2/4 \le c n^2$ при $c = 3/4 < 1$ — выполняется.
- Случай 3 → $T(n) = \Theta(n^2)$

## Продвинутая версия (с логарифмическими множителями)

Если рекуррентность имеет вид

$$ T(n) = aT(n/b) + \Theta(n^k \log^p n), $$

где $k \ge 0$, $p \in \mathbb{R}$, то:

- **Если $a > b^k$:**  
  $$ T(n) = \Theta(n^{\log_b a}) $$

- **Если $a = b^k$:**  
  - $p > -1$: $T(n) = \Theta(n^{\log_b a} \log^{p+1} n)$  
  - $p = -1$: $T(n) = \Theta(n^{\log_b a} \log \log n)$  
  - $p < -1$: $T(n) = \Theta(n^{\log_b a})$

- **Если $a < b^k$:**  
  - $p \ge 0$: $T(n) = \Theta(n^k \log^p n)$  
  - $p < 0$: $T(n) = \Theta(n^k)$

### Примеры продвинутой версии

$$ T(n) = 3T(n/2) + \log^2 n $$

- $a=3$, $b=2$, $k=0$, $p=2$
- $b^k = 1 < a$ → $T(n) = \Theta(n^{\log_2 3})$

$$ T(n) = 2T(n/2) + n \log^2 n $$

- $a=2$, $b=2$, $k=1$, $p=2$
- $a = b^k = 2$ → $p > -1$ → $T(n) = \Theta(n \log^3 n)$

$$ T(n) = 2T(n/2) + n / \log n $$

- Здесь $f(n) = n / \log n = n^1 \log^{-1} n$ → $k=1$, $p=-1$.
- $a = b^k$ → $p = -1$ → $T(n) = \Theta(n \log \log n)$
