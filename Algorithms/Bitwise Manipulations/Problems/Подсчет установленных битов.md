## Задача

Подсчитать количество битов, равных 1, в двоичном представлении целого числа (также называется вес Хэмминга или population count).

## Решение

Простой итеративный метод: пока число не станет нулём, проверяем младший бит и сдвигаем вправо.

```python
def count_set_bits(n):
    count = 0
    while n:
        count += n & 1
        n >>= 1
    return count

# Пример
print(count_set_bits(5))  # 5 (101) → 2
print(count_set_bits(13)) # 13 (1101) → 3
```

## Оптимизация

Более быстрый метод — использовать `n & (n-1)` для обнуления младшего установленного бита на каждой итерации. Количество итераций равно числу единиц.

```python
def count_set_bits_fast(n):
    count = 0
    while n:
        n &= n - 1
        count += 1
    return count
```

## Сложность

- Время: O(количество битов) или O(количество единиц) для второго метода.
- Память: O(1)
