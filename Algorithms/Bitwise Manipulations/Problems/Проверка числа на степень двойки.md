## Задача

Определить, является ли целое положительное число степенью двойки (например, 1, 2, 4, 8, 16...).

## Решение

Число, являющееся степенью двойки, в двоичном представлении имеет ровно один установленный бит. Свойство: `x & (x - 1)` обнуляет самый младший установленный бит. Если после этого получился 0, то бит был один, значит число — степень двойки. Также нужно учесть, что 0 не является степенью двойки.

```python
def is_power_of_two(x):
    return x > 0 and (x & (x - 1)) == 0

# Примеры
print(is_power_of_two(1))   # True (2^0)
print(is_power_of_two(2))   # True
print(is_power_of_two(3))   # False
print(is_power_of_two(16))  # True
```

## Объяснение

- Для `x = 16` (`10000`), `x-1 = 15` (`01111`), `10000 & 01111 = 0` → True.
- Для `x = 12` (`1100`), `x-1 = 11` (`1011`), `1100 & 1011 = 1000` (8) ≠ 0 → False.

## Сложность

- Время: O(1)
- Память: O(1)
