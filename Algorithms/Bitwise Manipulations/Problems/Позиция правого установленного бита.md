## Задача

Найти позицию самого младшего (правого) бита, равного 1, в двоичном представлении числа. Позиции считаются от 0 (младший бит). Если число равно 0, вернуть 0 или специальное значение.

## Решение

Один из способов: выделить самый младший единичный бит с помощью `n & -n` (в дополнительном коде это даёт число, содержащее только этот бит). Затем найти его позицию, сдвигая вправо до нуля.

```python
def rightmost_set_bit_pos(n):
    if n == 0:
        return 0
    # Выделяем самый правый установленный бит
    rightmost = n & -n
    pos = 0
    while rightmost > 1:
        rightmost >>= 1
        pos += 1
    return pos

# Пример
print(rightmost_set_bit_pos(18))  # 18 = 10010 → правый единичный бит на позиции 1
```

Альтернативно можно использовать встроенные методы, например `(n & -n).bit_length() - 1`.

```python
def rightmost_set_bit_pos_fast(n):
    if n == 0:
        return 0
    return (n & -n).bit_length() - 1
```

## Объяснение

- Для `n = 18` (`10010`), `-n` в дополнительном коде = `01110`, `n & -n = 00010` (2), позиция = 1.

## Сложность

- Время: O(1) при использовании `bit_length()` (или O(log n) при цикле, но на практике быстро)
- Память: O(1)
