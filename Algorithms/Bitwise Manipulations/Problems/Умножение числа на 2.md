## Задача

Умножить целое число на 2 с помощью битовой операции.

## Решение

Оператор **левый сдвиг** `<<` сдвигает все биты числа влево на указанное количество позиций, что эквивалентно умножению на `2^k`.

```python
def multiply_by_two(num):
    return num << 1

# Пример
num = 12
result = multiply_by_two(num)
print(result)  # 24
```

## Объяснение

- `12` в двоичной: `1100`
- Сдвиг влево на 1: `11000` → 24
- Каждый сдвиг влево на 1 бит умножает на 2.

## Примечание

Операция работает быстрее обычного умножения на низком уровне, но современные компиляторы и интерпретаторы часто оптимизируют умножение на степени двойки в сдвиги. Полезно для встроенных систем и оптимизаций.

## Сложность

- Время: O(1)
- Память: O(1)
