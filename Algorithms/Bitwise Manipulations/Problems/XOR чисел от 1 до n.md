## Задача

Вычислить побитовое исключающее ИЛИ (XOR) всех целых чисел от 1 до n включительно.

## Решение

Существует математическая закономерность, позволяющая вычислить результат за O(1) без цикла:

- Если `n % 4 == 0` → результат `n`
- Если `n % 4 == 1` → результат `1`
- Если `n % 4 == 2` → результат `n + 1`
- Если `n % 4 == 3` → результат `0`

```python
def xor_1_to_n(n):
    remainder = n % 4
    if remainder == 0:
        return n
    elif remainder == 1:
        return 1
    elif remainder == 2:
        return n + 1
    else:  # remainder == 3
        return 0

# Примеры
print(xor_1_to_n(5))  # 1 ^ 2 ^ 3 ^ 4 ^ 5 = 1
print(xor_1_to_n(6))  # 7
print(xor_1_to_n(7))  # 0
print(xor_1_to_n(8))  # 8
```

## Объяснение

Закономерность основана на свойствах XOR и циклическом повторении паттерна каждые 4 числа. Полезно для задач, где требуется быстрое вычисление без перебора.

## Сложность

- Время: O(1)
- Память: O(1)
