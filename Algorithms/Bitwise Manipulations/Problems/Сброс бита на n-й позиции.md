## Задача

Сбросить (сделать равным 0) бит числа на заданной позиции.

## Решение

Используется операция **побитовое И** с маской, в которой все биты, кроме нужного, равны 1, а нужный — 0. Такая маска получается инверсией маски установки: `~(1 << pos)`.

```python
def clear_bit(num, pos):
    return num & ~(1 << pos)

# Пример
num = 7      # 111 (двоичное)
pos = 1
result = clear_bit(num, pos)
print(result)  # 5 (101)
```

## Объяснение

- Исходное число: `7` → `111`
- Маска: `~(1 << 1)` = `~(010)` = `...111101` (в бесконечной разрядности, но в Python работает с целыми)
- Результат: `111 & 101` = `101` (5)

Бит на позиции 1 становится 0, остальные сохраняются.

## Сложность

- Время: O(1)
- Память: O(1)
