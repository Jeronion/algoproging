## Задача

Переключить (инвертировать) бит числа на заданной позиции: 0 → 1, 1 → 0.

## Решение

Используется операция **побитовое исключающее ИЛИ (XOR)** с маской, в которой нужный бит равен 1. XOR с 1 инвертирует бит.

```python
def toggle_bit(num, pos):
    return num ^ (1 << pos)

# Пример
num = 4      # 100
pos = 1
result = toggle_bit(num, pos)
print(result)  # 6 (110)

num = 7      # 111
pos = 1
result = toggle_bit(num, pos)
print(result)  # 5 (101)
```

## Объяснение

- Для числа 4 (100): маска 010 → 100 ^ 010 = 110 (6)
- Для числа 7 (111): 111 ^ 010 = 101 (5)

Бит на позиции pos инвертируется, остальные биты не меняются.

## Сложность

- Время: O(1)
- Память: O(1)
