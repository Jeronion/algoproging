
## Определение

**Битовые операторы** — это операторы, которые работают непосредственно с двоичными битами чисел. Они выполняются на уровне аппаратуры (АЛУ процессора) и позволяют эффективно манипулировать данными, устанавливать, сбрасывать и проверять отдельные биты, а также выполнять быстрые арифметические операции (умножение и деление на степени двойки).

---

## Основные операторы

### AND (`&`)

Возвращает 1 только если оба соответствующих бита равны 1.

```python
a = 7   # 111 (двоичное)
b = 4   # 100
print(a & b)  # 4 (100)
```

### OR (`|`)

Возвращает 1 если хотя бы один из битов равен 1.

```python
a = 12  # 1100
b = 25  # 11001 (приводится к общей длине)
print(a | b)  # 29 (11101)
```

### XOR (`^`) — исключающее ИЛИ

Возвращает 1 если биты различны, 0 если одинаковы.

```python
a = 12
b = 25
print(a ^ b)  # 21 (10101)
```

### NOT (`~`) — побитовое отрицание

Инвертирует все биты числа (включая знаковый). Для положительных чисел даёт `-(x+1)`.

```python
a = 0
print(~a)      # -1
print(int(not a))  # 1 (логическое НЕ, не побитовое)
```

### Левый сдвиг (`<<`)

Сдвигает биты влево на указанное количество позиций, справа добавляются нули. Эквивалентно умножению на `2^k`.

```python
num = 5   # 101
print(num << 1)  # 10 (1010) = 10
print(num << 2)  # 20 (10100) = 20
```

### Правый сдвиг (`>>`)

Сдвигает биты вправо. Для положительных чисел слева добавляются нули, для отрицательных — единицы (зависит от языка). Эквивалентно целочисленному делению на `2^k`.

```python
num = 5
print(num >> 1)  # 2 (10) = 2
print(num >> 2)  # 1 (1)  = 1
print(num >> 3)  # 0
```
