
## Определение

**Алгоритм Дейкстры** — это жадный алгоритм для нахождения кратчайших путей от одной вершины (источника) до всех остальных вершин во **взвешенном графе** с **неотрицательными весами рёбер**. Алгоритм широко применяется в навигационных системах, сетевой маршрутизации, планировании и других областях.

---

## Описание работы

1. **Инициализация**:  
   - Задаём расстояние до стартовой вершины равным 0, до всех остальных — бесконечность.
   - Создаём множество непосещённых вершин (или приоритетную очередь).

2. **Основной цикл**:  
   - Извлекаем из непосещённых вершину с минимальным текущим расстоянием (на первом шаге это старт).
   - Для каждого соседа этой вершины вычисляем новое расстояние как `расстояние[текущая] + вес ребра`.
   - Если новое расстояние меньше текущего расстояния до соседа, обновляем его.
   - Помечаем текущую вершину как посещённую (удаляем из множества непосещённых).

3. **Завершение**:  
   - Когда все вершины обработаны или оставшиеся недостижимы, алгоритм завершается. Результат — массив кратчайших расстояний от старта до каждой вершины.

---

## Ограничения

- Рёбра должны иметь **неотрицательные веса**. При наличии отрицательных весов алгоритм может работать некорректно (для таких случаев используется алгоритм Беллмана-Форда).
- Не находит сам путь, только расстояния (можно модифицировать для сохранения предков).

## Реализации

### 1. Базовая реализация (без приоритетной очереди)

Для небольших графов можно просто искать минимум линейным проходом.

```python
def dijkstra_simple(graph, start):
    vertices = list(graph.keys())
    dist = {v: float('inf') for v in vertices}
    dist[start] = 0
    visited = set()

    while len(visited) < len(vertices):
        # Выбираем непосещённую вершину с минимальным dist
        u = None
        min_dist = float('inf')
        for v in vertices:
            if v not in visited and dist[v] < min_dist:
                min_dist = dist[v]
                u = v
        if u is None:
            break
        visited.add(u)
        for neighbor, weight in graph[u].items():
            if neighbor not in visited:
                new_dist = dist[u] + weight
                if new_dist < dist[neighbor]:
                    dist[neighbor] = new_dist
    return dist
```

**Сложность:** O(V²) — подходит для плотных графов.

### 2. Реализация с приоритетной очередью (heapq)

Это наиболее эффективный и распространённый способ.

```python
import heapq

def dijkstra(graph, start):
    dist = {vertex: float('inf') for vertex in graph}
    dist[start] = 0
    pq = [(0, start)]  # очередь с приоритетом: (расстояние, вершина)

    while pq:
        current_dist, u = heapq.heappop(pq)
        # Если расстояние в очереди больше уже найденного, пропускаем
        if current_dist > dist[u]:
            continue
        for v, weight in graph[u].items():
            new_dist = dist[u] + weight
            if new_dist < dist[v]:
                dist[v] = new_dist
                heapq.heappush(pq, (new_dist, v))
    return dist
```

---

## Сложность
- **Время:** O((V + E) log V) при использовании двоичной кучи.  
- **Память:** O(V) для хранения расстояний и очереди.

---

## Восстановление пути

Чтобы не только узнать расстояние, но и сам путь, нужно хранить для каждой вершины предка (`parent`), через которого было получено минимальное расстояние.

```python
def dijkstra_path(graph, start, target):
    dist = {v: float('inf') for v in graph}
    parent = {v: None for v in graph}
    dist[start] = 0
    pq = [(0, start)]

    while pq:
        current_dist, u = heapq.heappop(pq)
        if u == target:
            break
        if current_dist > dist[u]:
            continue
        for v, w in graph[u].items():
            new_dist = dist[u] + w
            if new_dist < dist[v]:
                dist[v] = new_dist
                parent[v] = u
                heapq.heappush(pq, (new_dist, v))

    # Восстановление пути
    if parent[target] is None and target != start:
        return None  # путь не найден
    path = []
    v = target
    while v is not None:
        path.append(v)
        v = parent[v]
    return list(reversed(path))
```
