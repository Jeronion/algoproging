## Определение

**BFS (Breadth-First Search, поиск в ширину)** — алгоритм обхода графа или дерева, который исследует вершины **по уровням** (слоям), начиная с выбранной стартовой вершины. Сначала посещаются все непосредственные соседи стартовой вершины, затем соседи этих соседей и так далее. BFS использует **очередь** для хранения вершин, которые нужно посетить.

---

## Описание работы

1. Помечаем стартовую вершину как посещённую и помещаем её в очередь.
2. Пока очередь не пуста:
   - Извлекаем вершину из начала очереди.
   - Обрабатываем её (например, выводим, сохраняем расстояние).
   - Для каждого непосещённого соседа:
     - Помечаем его как посещённый.
     - Записываем расстояние (или родителя) и добавляем в конец очереди.

Таким образом, вершины обрабатываются в порядке увеличения расстояния от старта.

---

## Реализация

Это классический способ реализации BFS (рекурсивная - бесполезное говно).

```python
from collections import deque

def bfs(graph, start):
    visited = set()
    queue = deque([start])
    visited.add(start)
    
    while queue:
        vertex = queue.popleft()
        print(vertex, end=' ')  # обработка вершины
        
        for neighbor in graph[vertex]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)
    
    return visited

# Пример графа (список смежности)
graph = {
    'A': ['B', 'C'],
    'B': ['A', 'D', 'E'],
    'C': ['A', 'F'],
    'D': ['B'],
    'E': ['B', 'F'],
    'F': ['C', 'E']
}

bfs(graph, 'A')  # A B C D E F (или A B C D E F, зависит от порядка соседей)
```

---

## Сложность

- **Время:** O(V + E), где V — число вершин, E — число рёбер. Каждая вершина и каждое ребро обрабатываются один раз.
- **Память:** O(V) для хранения очереди и множества посещённых вершин (в худшем случае, например, для полного графа очередь может содержать почти все вершины).

## Сравнение BFS и DFS

| Характеристика      | BFS                                   | DFS                                   |
|---------------------|---------------------------------------|---------------------------------------|
| Порядок обхода      | В ширину (по уровням)                 | В глубину (до конца пути)             |
| Используемая структура | Очередь (queue)                     | Стек (рекурсия или явный стек)        |
| Применение          | Кратчайший путь, обход по слоям       | Поиск циклов, топологическая сортировка, комбинаторные задачи |
| Память в худшем случае | O(V) (очередь)                      | O(V) (стек вызовов/явный стек)        |
| Свойства            | Находит минимальное расстояние в невзвешенном графе | Не гарантирует минимальное расстояние |
