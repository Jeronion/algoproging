## Определение

**Дек (Double-Ended Queue)** — линейная структура данных, позволяющая добавлять и удалять элементы **с обеих сторон**: начало и конец.

- Стек → LIFO`
- Очередь → FIFO
- Дек → сочетает возможности стека и очереди

> Метафора: поезд, к которому можно прицеплять вагоны спереди и сзади, и отцеплять их с обеих сторон.

---

## Основные понятия

| Понятие  | Описание                                                         |
| -------- | ---------------------------------------------------------------- |
| Front    | Первый элемент дека                                              |
| Rear     | Последний элемент дека                                           |
| Size     | Текущее количество элементов                                     |
| Capacity | Максимальное количество элементов (для фиксированных реализаций) |

Все операции выполняются за **O(1)**.

---

## Основные операции

|Операция|Описание|
|---|---|
|`push_front(x)`|Добавление в начало|
|`push_back(x)`|Добавление в конец|
|`pop_front()`|Удаление с начала|
|`pop_back()`|Удаление с конца|
|`peek_front()`|Первый элемент без удаления|
|`peek_back()`|Последний элемент без удаления|
|`isEmpty()`|Проверка пустоты|
|`size()`|Количество элементов|

## Дек как обобщение стека и очереди

| Структура | Добавление     | Удаление       |
| --------- | -------------- | -------------- |
| Стек      | С конца        | С конца        |
| Очередь   | С конца        | С начала       |
| Дек       | С обеих сторон | С обеих сторон |

> Примеры:
> 
> - `push_back` + `pop_back` → стек
> - `push_back` + `pop_front` → очередь

---

## Применение дека

1. **Реализация стека и очереди** — заменяет обе структуры
2. **Скользящее окно (1Sliding Window)** — поиск максимума/минимума за O(n)
3. **LRU Cache** — вытеснение наименее недавно использованных элементов
4. **Палиндромы** — сравнение элементов с начала и конца
5. **Планирование задач** — BFS и алгоритмы графов

|Свойство|Стек|Очередь|Дек|
|---|---|---|---|
|Добавление|С одной стороны|С одной стороны|С двух сторон|
|Удаление|С одной стороны|С одной стороны|С двух сторон|
|Гибкость|Низкая|Средняя|Высокая|
