## Определение

**Обход матрицы** — последовательное посещение всех её элементов. Обычно выполняется по строкам (row-major) или по столбцам (column-major). Также существуют специальные обходы: по спирали, змейкой, диагоналями и т.д.

---

## Типы обхода

### Базовый обход по строкам

Это самый распространённый способ: внешний цикл по строкам, внутренний по столбцам.

```python
def traverse_row_major(matrix):
    for i in range(len(matrix)):
        for j in range(len(matrix[0])):
            print(matrix[i][j], end=' ')
        print()  # новая строка для красоты

matrix = [[1, 2, 3], [4, 5, 6]]
traverse_row_major(matrix)
# Вывод:
# 1 2 3 
# 4 5 6
```

### Обход по столбцам

Внешний цикл по столбцам, внутренний по строкам.

```python
def traverse_column_major(matrix):
    rows = len(matrix)
    cols = len(matrix[0])
    for j in range(cols):
        for i in range(rows):
            print(matrix[i][j], end=' ')
        print()

# Для той же матрицы:
# 1 4 
# 2 5 
# 3 6
```

## Сложность
- **Время:** O(m·n)
- **Память:** O(1)

## Другие виды обхода

- **Спиральный обход** — движение по спирали от внешнего края к центру.
- **Змейкой** — чётные строки слева направо, нечётные справа налево.
- **По диагоналям** — обход всех диагоналей.

Они полезны в олимпиадном программировании и обработке изображений.
