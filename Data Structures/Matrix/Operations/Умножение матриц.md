
## Определение

**Умножение матриц** — операция, при которой вычисляется матрица **C = A · B**, где элемент **cᵢⱼ** равен сумме произведений элементов i-й строки **A** на соответствующие элементы j-го столбца **B**.

## Примечания

- Число столбцов **A** должно равняться числу строк **B**.
- Если **A** размера **m × n**, **B** размера **n × k**, то результат **C** — размера **m × k**.
- Умножение матриц **не коммутативно** (в общем случае **A·B ≠ B·A**).

## Формула

$$c_{ij} = \sum_{t=1}^{n} a_{it} b_{tj}$$

## Реализация на Python

### Чистый Python (наивный алгоритм)

```python
def multiply_matrices(A, B):
    m = len(A)
    n = len(A[0])
    k = len(B[0])
    if n != len(B):
        raise ValueError("Incompatible dimensions")
    
    # Инициализация результата нулями
    result = [[0 for _ in range(k)] for _ in range(m)]
    
    for i in range(m):
        for j in range(k):
            for t in range(n):
                result[i][j] += A[i][t] * B[t][j]
    return result

# Пример
A = [[1, 2], [3, 4]]
B = [[5, 6], [7, 8]]
print(multiply_matrices(A, B))  # [[19, 22], [43, 50]]
```

### С NumPy

```python
import numpy as np

A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])
print(A @ B)  # или np.dot(A, B)
# [[19 22]
#  [43 50]]
```

## Сложность

- **Время:** O(m·n·k) для наивного алгоритма. Существуют более быстрые алгоритмы (например, Штрассена), но на практике для небольших матриц используется наивный.
- **Память:** O(m·k) для результата.
