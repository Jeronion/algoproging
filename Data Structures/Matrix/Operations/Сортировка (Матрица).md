## Определение

**Сортировка матрицы** может означать:
- Сортировку каждой строки независимо.
- Сортировку всей матрицы как одномерного массива (превращение в список, сортировка, обратное заполнение).
- Сортировку строк матрицы по какому-либо критерию (например, по сумме элементов в строке).

## Сортировка строк по отдельности

Каждая строка сортируется независимо.

```python
def sort_rows(matrix):
    for row in matrix:
        row.sort()
    return matrix

matrix = [[3, 1, 2], [9, 7, 8]]
print(sort_rows(matrix))  # [[1, 2, 3], [7, 8, 9]]
```

## Глобальная сортировка (все элементы)

```python
def sort_global(matrix):
    rows = len(matrix)
    cols = len(matrix[0])
    # Превращаем в плоский список
    flat = [matrix[i][j] for i in range(rows) for j in range(cols)]
    flat.sort()
    # Обратно заполняем матрицу (по строкам)
    for i in range(rows):
        for j in range(cols):
            matrix[i][j] = flat[i * cols + j]
    return matrix

matrix = [[3, 1, 2], [9, 7, 8]]
print(sort_global(matrix))  # [[1, 2, 3], [7, 8, 9]]
```

## Сортировка строк по сумме элементов

```python
def sort_rows_by_sum(matrix):
    # Сортируем строки по убыванию суммы (для примера)
    matrix.sort(key=lambda row: sum(row), reverse=True)
    return matrix

matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
print(sort_rows_by_sum(matrix))  # Строки с большей суммой будут первыми
```

## Сложность

- Сортировка каждой строки отдельно: O(m·n log n), если использовать TimSort (стандартную сортировку Python).
- Глобальная сортировка: O(m·n log(m·n)).
- Сортировка строк по ключу: O(m log m) на сравнение строк плюс затраты на вычисление ключа.
