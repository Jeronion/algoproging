## Определение

**Поиск в матрице** — нахождение заданного элемента (или его позиции) в матрице. В зависимости от структуры матрицы применяются разные алгоритмы.

## Линейный поиск (неотсортированная матрица)

Проходим по всем элементам используя [[Обход (Матрица)]], пока не найдём нужный.

```python
def linear_search(matrix, target):
    for i in range(len(matrix)):
        for j in range(len(matrix[0])):
            if matrix[i][j] == target:
                return (i, j)  # возвращаем координаты
    return None

matrix = [[1, 2, 3], [4, 5, 6]]
print(linear_search(matrix, 5))  # (1, 1)
```

### Сложность: O(m·n)

## Поиск в отсортированной матрице (по строкам и столбцам)

Если матрица отсортирована так, что каждая строка и каждый столбец упорядочены по возрастанию (например, матрица Юнга), можно применить алгоритм за O(m+n).

**Идея:** начинаем с правого верхнего угла (или левого нижнего). Сравниваем текущий элемент с искомым:
- Если равен — нашли.
- Если больше — идём влево (столбец уменьшается).
- Если меньше — идём вниз (строка увеличивается).

```python
def search_sorted_matrix(matrix, target):
    if not matrix:
        return None
    rows = len(matrix)
    cols = len(matrix[0])
    i = 0
    j = cols - 1
    while i < rows and j >= 0:
        if matrix[i][j] == target:
            return (i, j)
        elif matrix[i][j] > target:
            j -= 1
        else:
            i += 1
    return None

# Пример: матрица, где каждая строка и столбец отсортированы
matrix = [
    [1, 4, 7, 11],
    [2, 5, 8, 12],
    [3, 6, 9, 16]
]
print(search_sorted_matrix(matrix, 5))  # (1, 1)
print(search_sorted_matrix(matrix, 10)) # None
```

### Сложность: O(m + n)

## Бинарный поиск в каждой строке

Если строки отсортированы независимо, можно применить бинарный поиск по каждой строке.

```python
def search_binary_in_rows(matrix, target):
    for i, row in enumerate(matrix):
        # Бинарный поиск в строке (предполагаем, что строки отсортированы)
        lo, hi = 0, len(row) - 1
        while lo <= hi:
            mid = (lo + hi) // 2
            if row[mid] == target:
                return (i, mid)
            elif row[mid] < target:
                lo = mid + 1
            else:
                hi = mid - 1
    return None
```

### Сложность: O(m log n)

Выбор метода зависит от свойств матрицы.
