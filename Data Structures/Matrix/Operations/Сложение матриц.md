
## Определение

**Сложение матриц** — поэлементная операция, при которой соответствующие элементы двух матриц складываются. Результатом является матрица того же размера.

## Примечания

- Матрицы должны быть **одинакового размера** (одинаковое количество строк и столбцов).
- Сложение коммутативно: **A + B = B + A**.
- Сложение ассоциативно: **(A + B) + C = A + (B + C)**.

## Формула

Если **A = [aᵢⱼ]** и **B = [bᵢⱼ]**, то **(A + B)ᵢⱼ = aᵢⱼ + bᵢⱼ**

## Реализация на Python

### Чистый Python (списки списков)

```python
def add_matrices(A, B):
    rows = len(A)
    cols = len(A[0])
    # Проверка размеров
    if rows != len(B) or cols != len(B[0]):
        raise ValueError("Matrices must have the same dimensions")
    
    result = [[A[i][j] + B[i][j] for j in range(cols)] for i in range(rows)]
    return result

# Пример
A = [[1, 2], [3, 4]]
B = [[5, 6], [7, 8]]
print(add_matrices(A, B))  # [[6, 8], [10, 12]]
```

### С NumPy

```python
import numpy as np

A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])
print(A + B)  # [[ 6  8] [10 12]]
```

## Сложность

- **Время:** O(m·n), где m — число строк, n — число столбцов.
- **Память:** O(m·n) для результата (если не считать входные матрицы).
