
## Определение

Дан двумерный массив (матрица) размера `n × n`. Требуется вывести все элементы матрицы в порядке, соответствующем **диагоналям**, идущим из левого верхнего угла в правый нижний (т.е. диагонали с направлением «вниз-влево»). Порядок обхода: сначала все диагонали, начинающиеся с первой строки (слева направо), затем все диагонали, начинающиеся с последнего столбца (сверху вниз, исключая самую первую диагональ, уже пройденную).

Пример для матрицы 3×3:

```
1 2 3
4 5 6
7 8 9
```

Обход даёт: `1 2 4 3 5 7 6 8 9`.

---

## Идея решения

Каждая диагональ в матрице (направленная вниз-влево) характеризуется тем, что сумма индексов `(i + j)` постоянна. Однако удобнее генерировать диагонали, начиная с каждой ячейки первой строки (элементы `(0, col)` для `col = 0..n-1`), а затем с каждой ячейки последнего столбца (кроме `(0, n-1)`, чтобы не дублировать). Для каждой начальной точки движемся по диагонали: увеличиваем `i` на 1, уменьшаем `j` на 1, пока не выйдем за пределы матрицы.

---

## Алгоритм

1. Пройти по всем столбцам первой строки (`col` от 0 до `n-1`):
   - Установить `i = 0`, `j = col`.
   - Пока `i < n` и `j >= 0`, вывести `mat[i][j]`, затем `i++`, `j--`.
2. Пройти по всем строкам последнего столбца, начиная со второй строки (`row` от 1 до `n-1`):
   - Установить `i = row`, `j = n-1`.
   - Пока `i < n` и `j >= 0`, вывести `mat[i][j]`, затем `i++`, `j--`.

---

## Реализация на Python

```python
def diagonal_traversal(mat):
    n = len(mat)

    # Диагонали, начинающиеся с первой строки
    for col in range(n):
        i, j = 0, col
        while i < n and j >= 0:
            print(mat[i][j], end=" ")
            i += 1
            j -= 1

    # Диагонали, начинающиеся с последнего столбца (кроме первого элемента)
    for row in range(1, n):
        i, j = row, n - 1
        while i < n and j >= 0:
            print(mat[i][j], end=" ")
            i += 1
            j -= 1

# Пример
mat = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
diagonal_traversal(mat)  # 1 2 4 3 5 7 6 8 9
```

---

## Сложность

- **Время:** O(n²) — каждый элемент посещается ровно один раз.
- **Память:** O(1) — дополнительная память не требуется (не считая входных данных).

---

## Примечания

- Алгоритм работает для квадратной матрицы. Для прямоугольной матрицы нужно учитывать размеры `m × n`, но общий принцип сохраняется (начальные точки — первая строка и последний столбец, с учётом границ).
- Существуют и другие порядки обхода диагоналей (например, зигзагом), но этот — один из наиболее естественных.
