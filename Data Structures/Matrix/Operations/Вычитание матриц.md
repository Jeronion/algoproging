
## Определение

**Вычитание матриц** — поэлементная операция, при которой из элементов первой матрицы вычитаются соответствующие элементы второй. Результат — матрица того же размера.

## Примечания

- Матрицы должны быть **одинакового размера**.
- Вычитание **не коммутативно**: **A - B ≠ B - A** (кроме тривиальных случаев).

## Формула

**(A - B)ᵢⱼ = aᵢⱼ - bᵢⱼ**

## Реализация

### Чистый Python

```python
def subtract_matrices(A, B):
    rows = len(A)
    cols = len(A[0])
    if rows != len(B) or cols != len(B[0]):
        raise ValueError("Matrices must have the same dimensions")
    
    result = [[A[i][j] - B[i][j] for j in range(cols)] for i in range(rows)]
    return result

# Пример
A = [[5, 6], [7, 8]]
B = [[1, 2], [3, 4]]
print(subtract_matrices(A, B))  # [[4, 4], [4, 4]]
```

### С NumPy

```python
import numpy as np

A = np.array([[5, 6], [7, 8]])
B = np.array([[1, 2], [3, 4]])
print(A - B)  # [[4 4] [4 4]]
```

## Сложность
- **Время:** O(m·n)
- **Память:** O(m·n)
