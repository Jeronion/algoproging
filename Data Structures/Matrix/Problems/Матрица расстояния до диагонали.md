## Определение

Заполнить матрицу размера N×M значениями `abs(i - j)`, т.е. расстоянием от элемента до главной диагонали.

## Реализация

```python
def generate_distance_matrix():
    N, M = map(int, input().split())
    grid = [[abs(i - j) for j in range(M)] for i in range(N)]
    for row in grid:
        print(*row)
```

## Сложность

- **Время:** O(N·M)
- **Память:** O(N·M) для хранения матрицы.
