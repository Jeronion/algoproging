## Определение

Создать матрицу размера N×M, где элемент `grid[i][j] = i + j` (0-индексация).

## Реализация

```python
def generate_sum_index_matrix():
    N, M = map(int, input().split())
    grid = [[i + j for j in range(M)] for i in range(N)]
    for row in grid:
        print(*row)
```

## Сложность

- **Время:** O(N·M)
- **Память:** O(N·M) для хранения матрицы.
