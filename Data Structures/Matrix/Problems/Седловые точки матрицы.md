## Определение

**Седловая точка** — элемент матрицы, который одновременно является минимальным в своей строке и максимальным в своём столбце.

## Реализация

```python
def find_saddle_points():
    N, M = map(int, input().split())
    matrix = [list(map(int, input().split())) for _ in range(N)]

    row_mins = [min(row) for row in matrix]
    col_maxs = [max(matrix[i][j] for i in range(N)) for j in range(M)]

    saddle_points = []
    for i in range(N):
        for j in range(M):
            if matrix[i][j] == row_mins[i] and matrix[i][j] == col_maxs[j]:
                saddle_points.append((i+1, j+1))   # 1-индексация

    if saddle_points:
        for point in saddle_points:
            print(point[0], point[1])
    else:
        print(0)
```

## Сложность

- **Время:** O(N·M) — нахождение минимумов строк, максимумов столбцов и проверка.
- **Память:** O(N·M) для матрицы и O(N+M) для вспомогательных массивов.
