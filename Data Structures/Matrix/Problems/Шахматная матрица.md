## Определение

Заполнить матрицу размера N×M значениями 0 и 1 в шахматном порядке (как на шахматной доске). Обычно верхний левый угол — 0 или 1.

## Реализация

```python
def generate_checkerboard_matrix():
    N, M = map(int, input().split())
    grid = [[int(i % 2 != j % 2) for j in range(M)] for i in range(N)]
    for row in grid:
        print(*row)
```

## Сложность

- **Время:** O(N·M)
- **Память:** O(N·M) для хранения матрицы.
