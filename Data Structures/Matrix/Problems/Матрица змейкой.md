## Определение

Заполнить матрицу размера N×M натуральными числами «змейкой»: первая строка слева направо, вторая — справа налево, третья — снова слева направо и т.д.

## Реализация

```python
def generate_snake_matrix():
    N, M = map(int, input().split())
    grid = [[(M * i) + (j + 1 if i % 2 == 0 else M - j) for j in range(M)] for i in range(N)]
    for row in grid:
        print(*row, sep='\t')
```

## Сложность

- **Время:** O(N·M)
- **Память:** O(N·M) для хранения матрицы.
