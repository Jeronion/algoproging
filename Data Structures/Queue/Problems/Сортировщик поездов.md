## Условие задачи

На сортировочную станцию прибывают поезда с вагонами **N** различных типов. В полночь все вагоны, прибывшие за сутки, нужно выстроить в один состав в следующем порядке: сначала все вагоны типа 1 (в порядке их прибытия), затем все вагоны типа 2 (в порядке прибытия), …, наконец, все вагоны типа N.

Входные данные:
- Первая строка: число N (количество типов вагонов).
- Далее идут строки, каждая в формате: `<тип вагона> <номер вагона>` (номер — четырёхзначное число). Строки заданы в порядке прибытия.

Требуется вывести в выходной файл список вагонов в отсортированном составе в том же формате (тип и номер).

**Пример:**

Вход:
```
2
1 2345
2 1234
2 3876
1 2330
2 1236
1 0087
```

Выход:
```
1 2345
1 2330
1 0087
2 1234
2 3876
2 1236
```

---

## Идея решения

1. Создаём массив из `N` очередей (одна очередь для каждого типа вагонов).
2. Читаем входные данные построчно. Для каждой строки:
   - Определяем тип `t` и номер `num`.
   - Добавляем `num` в очередь с индексом `t-1` (так как типы нумеруются с 1).
3. После чтения всех вагонов проходим по всем типам от 1 до N и для каждого типа:
   - Пока очередь не пуста, извлекаем из неё номер и выводим строку в формате `тип номер`.

Для эффективной работы используем `collections.deque` — двусторонняя очередь, которая позволяет добавлять в конец и удалять из начала за O(1).

## Реализация на Python

```python
from collections import deque

def distribute_tasks_by_type():
    with open("input.txt", "r") as fin, open("output.txt", "w") as fout:
        N = int(fin.readline().strip())
        # Создаём список из N пустых деков
        queues = [deque() for _ in range(N)]

        # Читаем оставшиеся строки
        for line in fin:
            t, num = line.strip().split()
            t = int(t)
            # Добавляем номер в очередь соответствующего типа
            queues[t - 1].append(num)

        # Выводим вагоны по порядку типов
        for i in range(N):
            while queues[i]:
                fout.write(f"{i + 1} {queues[i].popleft()}\n")
```

---

## Сложность

- **Время:** O(M), где M — общее количество вагонов. Каждый вагон один раз добавляется в очередь и один раз извлекается.
- **Память:** O(M) — для хранения всех номеров в очередях.
