## Определение

**Queue (очередь)** — линейная структура данных, работающая по принципу **[[FIFO]]**, где первый добавленный элемент извлекается первым.

---

## Основные термины

- **Front (голова):** позиция первого элемента в очереди (отсюда происходит удаление).
- **Rear (хвост):** позиция последнего добавленного элемента (сюда добавляются новые элементы).
- **Size (размер):** текущее количество элементов в очереди.

---

## Основные операции

| Операция     | Описание                               |
|--------------|----------------------------------------|
| `enqueue(x)` | добавить элемент в конец очереди       |
| `dequeue()`  | удалить элемент из начала очереди      |
| `front()`    | получить первый элемент (без удаления) |
| `isEmpty()`  | проверить, пуста ли очередь            |
| `size()`     | получить количество элементов          |

Все основные операции при правильной реализации выполняются за время **O(1)**.

### Array Queue vs Linked List Queue

| Свойство            | Очередь на циклическом массиве                                  | Очередь на связном списке                                                    |
| ------------------- | --------------------------------------------------------------- | ---------------------------------------------------------------------------- |
| Размер              | Фиксированный (задаётся при создании)                           | Динамический                                                                 |
| Память              | Массив фиксированной длины, нет накладных расходов на указатели | Дополнительная память под указатели next (и prev, если двусвязный)           |
| Переполнение        | Может возникнуть при заполнении                                 | Только при нехватке памяти                                                   |
| Кэш-локальность     | Высокая (элементы в неприрывной памяти)                         | Низкая (узлы разбросаны)                                                     |
| Простота реализации | Требуется управление индексами и учёт цикличности               | Проще в реализации, особенно если не нужно поддерживать фиксированный размер |
