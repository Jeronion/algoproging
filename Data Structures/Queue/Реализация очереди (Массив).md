
## Определение

**Circular Queue (кольцевая очередь)** — это структура данных, которая устраняет ограничения обычной очереди при реализации через массив.

В обычной очереди после нескольких `dequeue()` освобождённое место **не используется повторно**.  
Circular Queue решает эту проблему, соединяя конец массива с его началом.

Таким образом элементы образуют **кольцо**.

---

# Структура реализации

Для реализации используются:

- `arr` — массив для хранения элементов
- `capacity` — максимальный размер очереди
- `front` — индекс первого элемента
- `size` — текущее количество элементов

Начальное состояние:

```
front = 0
size = 0
```

---

# Основные операции

|Операция|Описание|
|---|---|
|`enqueue(x)`|добавление элемента|
|`dequeue()`|удаление элемента|
|`getFront()`|получить первый элемент|
|`getRear()`|получить последний элемент|

---

# Enqueue (добавление)

Добавляет элемент в конец очереди.

### Шаги

1. Проверить переполнение (`size == capacity`)
2. Найти индекс заднего элемента

```
rear = (front + size) % capacity
```

3. Добавить элемент
4. Увеличить `size`

```python
def enqueue(self, x):

    if self.size == self.capacity:
        print("Queue is full!")
        return

    rear = (self.front + self.size) % self.capacity

    self.arr[rear] = x
    self.size += 1
```

---

# Dequeue (удаление)

Удаляет элемент из начала очереди.

### Алгоритм

1. Проверить пустоту
2. Сохранить `arr[front]`
3. Передвинуть `front`

```
front = (front + 1) % capacity
```

4. Уменьшить `size`
5. Вернуть значение

```python
def dequeue(self):

    if self.size == 0:
        print("Queue is empty!")
        return -1

    res = self.arr[self.front]

    self.front = (self.front + 1) % self.capacity
    self.size -= 1

    return res
```

---

# Получить первый элемент

```python
def getFront(self):

    if self.size == 0:
        return -1

    return self.arr[self.front]
```

---

# Получить последний элемент

Формула:

```
rear = (front + size - 1) % capacity
```

```python
def getRear(self):

    if self.size == 0:
        return -1

    rear = (self.front + self.size - 1) % self.capacity

    return self.arr[rear]
```

---

# Полная реализация

```python
class CircularQueue:

    def __init__(self, cap):

        self.arr = [0] * cap
        self.front = 0
        self.size = 0
        self.capacity = cap


    def enqueue(self, x):

        if self.size == self.capacity:
            print("Queue is full!")
            return

        rear = (self.front + self.size) % self.capacity
        self.arr[rear] = x
        self.size += 1


    def dequeue(self):

        if self.size == 0:
            print("Queue is empty!")
            return -1

        res = self.arr[self.front]

        self.front = (self.front + 1) % self.capacity
        self.size -= 1

        return res


    def getFront(self):

        if self.size == 0:
            return -1

        return self.arr[self.front]


    def getRear(self):

        if self.size == 0:
            return -1

        rear = (self.front + self.size - 1) % self.capacity
        return self.arr[rear]
```

# Сложность операций

|Операция|Сложность|

|Enqueue|O(1)|
|Dequeue|O(1)|
|getFront|O(1)|

# Почему Circular Queue лучше обычной

| Свойство                      | Simple Queue  | Circular Queue |
| ----------------------------- | ------------- | -------------- |
| Использование памяти          | неэффективное | эффективное    |
| Повторное использование ячеек | нет           | да             |
| Enqueue                       | O(1)          | O(1)           |
| Dequeue                       | O(1)          | O(1)           |
| Размер                        | динамический  | фиксированный  |
