
## Определение

**Вставка** элемента означает добавление нового элемента в определённое место массива. Все элементы, находящиеся после позиции вставки, сдвигаются на одну позицию вправо.

### Шаги вставки:

1. Определяем позицию для вставки.
2. Сдвигаем все элементы начиная с этой позиции на одну ячейку вправо.
3. Вставляем новый элемент.

### Пример на Python

```python
arr = [10, 20, 30, 40, 50]
pos = 2   # Вставка на 3-ю позицию (индекс 2)
value = 25

arr.insert(pos, value)

# Вывод: [10, 20, 25, 30, 40, 50]
print("Массив после вставки:", arr)
```

**Сложность:**
- Время: O(n)
- Память: O(1)

---

## Особенности вставки

|Место вставки|Время|
|---|---|
|Начало массива|O(n)|
|Конец массива|O(1)|
|Середина массива|O(n)|

> Для динамических массивов (Python list, Java ArrayList) память управляется автоматически. В статических массивах (C/C++) нужно сдвигать элементы вручную.
