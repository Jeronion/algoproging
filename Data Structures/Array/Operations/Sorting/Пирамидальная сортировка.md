## Определение

**Heap Sort** — алгоритм сортировки на основе **бинарной кучи (Binary Heap)**.

### Примечания:

- Улучшенная версия сортировки выбором.
- Многократно находит **максимальный элемент** за `O(log n)`.
- Выполняется на месте (`in-place`).
- **Нестабильный** алгоритм: порядок равных элементов может изменяться.
- Общая сложность: `O(n log n)`.

---

## Принцип работы

1. **Представление массива как полного бинарного дерева:**
    
    - Корень: `arr[0]`
    - Левый ребёнок `i` → `2i + 1`
    - Правый ребёнок `i` → `2i + 2`

2. **Построение Max Heap:**
    
    - Вызываем `heapify` для всех внутренних узлов
    - После построения максимальный элемент оказывается в корне

3. **Сортировка:**
    
    - Меняем корень и последний элемент
    - Уменьшаем размер кучи на 1
    - Восстанавливаем свойство кучи (`heapify`)
    - Повторяем до сортировки всего массива

## Реализация на Python

```python
# Восстановление свойства Max Heap
def heapify(arr, n, i):
    largest = i
    l = 2 * i + 1
    r = 2 * i + 2

    if l < n and arr[l] > arr[largest]:
        largest = l
    if r < n and arr[r] > arr[largest]:
        largest = r

    if largest != i:
        arr[i], arr[largest] = arr[largest], arr[i]
        heapify(arr, n, largest)


# Основная функция сортировки
def heapSort(arr):
    n = len(arr)

    # Построение Max Heap
    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)

    # Извлечение элементов из кучи
    for i in range(n - 1, 0, -1):
        arr[0], arr[i] = arr[i], arr[0]
        heapify(arr, i, 0)


# Пример использования
arr = [9, 4, 3, 8, 10, 2, 5]
heapSort(arr)
print(arr)
```

## Сложность

|Показатель|Значение|
|---|---|
|Время|O(n log n)|
|Построение кучи|O(n)|
|Извлечение элементов|O(n log n)|
|Доп. память (рекурсия)|O(log n)|
|Итеративная версия|O(1)|

---

## Преимущества

- Гарантированная сложность `O(n log n)`
- Минимальное использование памяти
- Работает на месте (`in-place`)

---

## Недостатки

- Нестабильный алгоритм
- Константы выше, чем у Merge Sort
- Часто медленнее QuickSort на практике
