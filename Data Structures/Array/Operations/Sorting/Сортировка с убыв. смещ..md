
## Определение

**Shell Sort** — алгоритм сортировки, который улучшает сортировку вставками, позволяя элементам массива перемещаться на большие расстояния.

### Примечания:

- Алгоритм предложен **Дональдом Шеллом (1959)**
- **Неустойчивый** алгоритм: равные элементы могут менять порядок
- Работает **на месте**, память O(1)
- Эффективнее, чем Bubble Sort и Insertion Sort для средних массивов

---

## Принцип работы

1. Выбирается шаг `gap` — расстояние между сравниваемыми элементами
2. Массив разбивается на несколько подпоследовательностей
3. Каждая подпоследовательность сортируется методом вставок
4. Шаг `gap` уменьшается
5. При `gap = 1` выполняется обычная сортировка вставками

**Пример классической последовательности шагов:**

```
gap = n // 2
gap = gap // 2
...
gap = 1
```

---

## Реализация на Python

```python
def shell_sort(arr: list[int]) -> None:
    n = len(arr)
    gap = n // 2

    while gap > 0:
        for i in range(gap, n):
            temp = arr[i]
            j = i

            # сортировка вставками с шагом gap
            while j >= gap and arr[j - gap] > temp:
                arr[j] = arr[j - gap]
                j -= gap

            arr[j] = temp

        gap //= 2
```

## Сложность

|Показатель|Значение|
|---|---|
|Временная (лучший)|O(n log n)|
|Временная (средняя)|зависит от выбора шагов|
|Временная (худшая)|O(n²)|
|Память|O(1)|

> Выбор последовательности шагов (`gap`) сильно влияет на производительность.

---

## Заключение

- Shell Sort сочетает **простоту реализации** и **экономию памяти**
- Быстрее Bubble Sort и Insertion Sort на средних массивах
- Неустойчивый алгоритм, сложность зависит от последовательности шагов
- Хорош для **частично упорядоченных массивов** и обучения
- Для больших массивов **лучше использовать Quick Sort или Merge Sort**
