## Определение

**Quick Sort (Быстрая сортировка)** — алгоритм сортировки на основе парадигмы **«Разделяй и властвуй»**.

### Примечания:

- Алгоритм **не стабильный** (относительный порядок равных элементов может изменяться)
- Использует рекурсию

---

## Принцип работы

1. **Выбор опорного элемента (Pivot)**
    
    - Первый, последний, случайный или медианный элемент
    - Рандомизация снижает шанс худшего случая

2. **Разбиение массива (Partition)**
    
    - Массив перестраивается вокруг pivot
    - Все элементы меньше pivot слева, больше — справа

3. **Рекурсивная сортировка подмассивов**
    
    - Применяется к левому и правому подмассивам

4. **Базовый случай рекурсии**
    
    - Подмассив содержит 0 или 1 элемент


---

## Разбиение Ломуто (Lomuto Partition)

- Отслеживает индекс меньших элементов `i`
- Проход по массиву: элементы < pivot меняются местами с `arr[i]`
- После прохода pivot помещается на правильную позицию

---

## Реализация на Python

```python
# Разбиение Ломуто
def partition(arr, low, high):
    pivot = arr[high]
    i = low - 1
    for j in range(low, high):
        if arr[j] < pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]
    arr[i + 1], arr[high] = arr[high], arr[i + 1]
    return i + 1

# Рекурсивный QuickSort
def quickSort(arr, low, high):
    if low < high:
        pi = partition(arr, low, high)
        quickSort(arr, low, pi - 1)
        quickSort(arr, pi + 1, high)
```

## Итеративная версия (без рекурсии)

```python
import random

def partition(arr, low, high):
    pivot_index = random.randint(low, high)
    arr[pivot_index], arr[high] = arr[high], arr[pivot_index]
    pivot = arr[high]
    i = low - 1
    for j in range(low, high):
        if arr[j] < pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]
    arr[i + 1], arr[high] = arr[high], arr[i + 1]
    return i + 1

def quick_sort_iterative(arr):
    stack = [(0, len(arr) - 1)]
    while stack:
        low, high = stack.pop()
        if low < high:
            pi = partition(arr, low, high)
            if pi - 1 - low > high - (pi + 1):
                stack.append((low, pi - 1))
                stack.append((pi + 1, high))
            else:
                stack.append((pi + 1, high))
                stack.append((low, pi - 1))
```

## Сложность

|Показатель|Значение|
|---|---|
|Временная (лучший случай)|Ω(n log n)|
|Временная (средний случай)|Θ(n log n)|
|Временная (худший случай)|O(n²)|
|Память (худший случай)|O(n)|
|Память (лучший случай)|O(log n)|

## Преимущества Quick Sort

- Быстрый для больших массивов
- Работает на месте, минимальные накладные расходы по памяти
- Хорошо кэшируется
- Оптимизируемая хвостовая рекурсия
- Подходит для Quickselect и других задач

---

## Недостатки

- Неустойчивый алгоритм
- Худший случай O(n²) при плохом выборе pivot
- Неэффективен для маленьких массивов

---

## Применения

- Сортировка больших массивов в оперативной памяти
- Стандартные библиотеки: `std::sort` (C++), `Arrays.sort` (Java)
- Подготовка данных для алгоритмов с отсортированным вводом
- Поиск k-го наименьшего/наибольшего элемента
- Сжатие данных и вычислительная геометрия
