
## Определение

**Блочная сортировка (Bucket Sort)** — это алгоритм сортировки, который распределяет элементы массива по нескольким блокам (корзинам), затем сортирует каждый блок отдельно (обычно с помощью другой сортировки, например, вставками) и объединяет отсортированные блоки в итоговый массив.

Алгоритм хорошо работает, когда входные элементы равномерно распределены по диапазону. Он является устойчивым, если для сортировки блоков используется устойчивый алгоритм (например, сортировка вставками).

---

## Примечания

- Блочная сортировка предполагает, что элементы распределены равномерно. Если распределение неравномерно, эффективность падает.
- Индекс блока вычисляется исходя из диапазона значений. В примере использовались числа от 0 до 1, поэтому формула `int(n * num)` даёт корректный индекс. Для других диапазонов нужно адаптировать.
- Сортировка вставками внутри блоков выбрана из-за эффективности на малых массивах.
- Блочная сортировка устойчива, если внутри блоков используется устойчивый алгоритм.

---

## Алгоритм

1. Создать `n` пустых блоков (списков), где `n` — размер массива.
2. Для каждого элемента `arr[i]`:
   - Вычислить индекс блока, обычно как `int(n * arr[i])` (если элементы в диапазоне (0,1)).
   - Поместить элемент в соответствующий блок.
3. Отсортировать каждый блок по отдельности (обычно сортировкой вставками).
4. Объединить все блоки в исходный массив, проходя блоки по порядку и добавляя их элементы.

## Реализация на Python

```python
def insertion_sort(bucket):
    for i in range(1, len(bucket)):
        key = bucket[i]
        j = i - 1
        while j >= 0 and bucket[j] > key:
            bucket[j + 1] = bucket[j]
            j -= 1
        bucket[j + 1] = key

def bucket_sort(arr):
    n = len(arr)
    buckets = [[] for _ in range(n)]

    # Распределение элементов по блокам
    for num in arr:
        bi = int(n * num)   # индекс блока
        buckets[bi].append(num)

    # Сортировка каждого блока
    for bucket in buckets:
        insertion_sort(bucket)

    # Объединение блоков
    index = 0
    for bucket in buckets:
        for num in bucket:
            arr[index] = num
            index += 1

# Пример использования
arr = [0.897, 0.565, 0.656, 0.1234, 0.665, 0.3434]
bucket_sort(arr)
print("Sorted array is:")
print(" ".join(map(str, arr)))
```

**Вывод:**
```
Sorted array is:
0.1234 0.3434 0.565 0.656 0.665 0.897
```

## Сложность

- **Время в худшем случае:** O(n²) — если все элементы попадают в один блок, то сортировка этого блока займёт O(n²) при использовании вставками. Можно улучшить до O(n log n), применяя внутри блоков более эффективную сортировку (например, слиянием или пирамидальную), но для маленьких блоков вставки эффективнее.
- **Время в лучшем случае:** O(n + k) — при равномерном распределении элементов по блокам (k — количество блоков, обычно k = n). Каждый блок содержит примерно константное число элементов, и сортировка вставками работает за O(1) на блок.
- **Дополнительная память:** O(n + k) — для хранения блоков.
