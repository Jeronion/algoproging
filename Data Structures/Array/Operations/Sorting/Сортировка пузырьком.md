
## Определение

**Сортировка пузырьком** — простой алгоритм сортировки, который **многократно обменивает соседние элементы**, если они находятся в неправильном порядке.

### Примечания:

- **Стабильный** алгоритм: сохраняет относительный порядок равных элементов.
- Подходит для **малых массивов** или образовательных целей.
- Неэффективен для больших наборов данных: средняя и худшая сложность O(n²).

---

## Принцип работы

1. Массив проходит несколько раз. После каждого прохода **максимальный элемент "всплывает" на своё место**.
2. На каждом проходе сравниваются только элементы, которые ещё не отсортированы.
3. Обмен соседних элементов выполняется, если **левый элемент больше правого**.
4. После k проходов **k наибольших элементов находятся на последних k позициях**.

---

## Реализация на Python

```python
def bubbleSort(arr):
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        if not swapped:
            break
```

- Флаг `swapped` позволяет остановить сортировку раньше, если массив уже отсортирован.

---

## Сложность

| Показатель          | Значение |
| ------------------- | -------- |
| Временная (лучший)  | O(n)     |
| Временная (средняя) | O(n²)    |
| Временная (худшая)  | O(n²)    |
| Память              | O(1)     |

---

## Заключение

- Bubble Sort **легко понять и реализовать**
- Сохраняет стабильность сортировки
- **Неэффективен для больших массивов**, используется в основном для обучения
- Хорошо демонстрирует **принципы обмена и оптимизации проходов**
