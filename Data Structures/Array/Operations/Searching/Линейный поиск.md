## Определение

Линейный поиск — это алгоритм поиска элемента в массиве или списке, который **последовательно проверяет каждый элемент**, пока не найдет искомый или не закончится массив.

- Возвращает индекс найденного элемента или `-1`, если элемент отсутствует.

---

## Пример на Python

```python
def linear_search(arr, x):
    for i, val in enumerate(arr):
        if val == x:
            return i
    return -1
```

---

## Сложность

|Показатель|Значение|
|---|---|
|Лучший случай|O(1)|
|Худший случай|O(N)|
|Средний случай|O(N)|
|Память|O(1)|

---

## Применения

- Неотсортированные массивы
- Малые наборы данных
- Linked List (поиск элементов по узлам)
- Поиск минимума и максимума
- Простая реализация для базовых задач

---

## Преимущества

- Работает с любым массивом, отсортированным или нет
- Не требует дополнительной памяти
- Легко реализуется и понимается

---

## Недостатки

- O(N) делает поиск медленным на больших массивах
- Неэффективен для больших наборов данных

---

## Когда использовать

- Малые массивы
- Неотсортированные данные
- Простые алгоритмические задачи без необходимости оптимизации
