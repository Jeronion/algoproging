## Определение

**Метод Ньютона** (также известный как метод касательных) — это итерационный численный метод для приближённого нахождения корня уравнения. Для вычисления квадратного корня из числа $N$ используется рекуррентная формула:

$$
x_{k+1} = \frac{1}{2} \left( x_k + \frac{N}{x_k} \right)
$$

где $x_0$ — начальное приближение (обычно берут $N$ или $N/2$). Процесс повторяется до тех пор, пока разница между последовательными приближениями не станет меньше заданной погрешности (толерантности).

---

## Идея

Формула получена из метода Ньютона для уравнения $f(x) = x^2 - N = 0$. Производная $f'(x) = 2x$, и итерационная формула принимает указанный вид. Каждое следующее приближение оказывается ближе к истинному значению $\sqrt{N}$.

---

## Алгоритм

1. Выбрать начальное приближение $x_0$ (например, $x_0 = N$ или $x_0 = N/2$).
2. На каждой итерации вычислить новое приближение:
   $$
   x_{\text{new}} = \frac{1}{2} \left( x + \frac{N}{x} \right)
   $$
3. Если $|x_{\text{new}} - x| < \text{tolerance}$, остановиться. Иначе присвоить $x = x_{\text{new}}$ и повторить шаг 2.

---

## Реализации

### Итеративная версия

```python
def square_root_iterative(N, tolerance=1e-5):
    x = N  # начальное приближение
    while True:
        root = 0.5 * (x + N / x)
        if abs(root - x) < tolerance:
            return root
        x = root
```

### Рекурсивная версия

```python
def square_root_recursive(N, guess=None, tolerance=1e-5):
    if guess is None:
        guess = N / 2
    next_guess = 0.5 * (guess + N / guess)
    if abs(next_guess - guess) <= tolerance:
        return next_guess
    return square_root_recursive(N, next_guess, tolerance)
```

---

## Пример

```python
N = 327
tolerance = 0.00001
sqrt = square_root_iterative(N, tolerance)
print(round(sqrt, 4))  # 18.0831
```

---

## Сложность

- **Время:** $O(\log N)$ итераций (квадратичная сходимость). Фактически количество итераций пропорционально $\log \log (1/\text{tolerance})$ — очень быстро.
- **Память:** $O(1)$ для итеративной версии, $O(\log N)$ для рекурсивной (из-за стека вызовов).
