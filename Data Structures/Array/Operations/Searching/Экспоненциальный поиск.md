## Определение

**Экспоненциальный поиск** — алгоритм поиска элемента в **упорядоченном массиве**, который сначала **находит диапазон**, в котором может находиться элемент, а затем применяет бинарный поиск внутри этого диапазона.

* Полезен для **неизвестных или больших массивов**, где прямое сканирование неэффективно.
* Эффективнее обычного бинарного поиска, если элемент находится **близко к началу массива**.

---

## 1. Идея алгоритма

1. Начать с подмассива размера (`i = 1`)
2. Удваивать размер подмассива (`i *= 2`), пока `arr[i] < x`
3. После нахождения индекса `i` выполнить **бинарный поиск** в диапазоне `[i/2, min(i, n-1)]`

> Почему `[i/2, i]`?
> На предыдущей итерации элемент был меньше `x`, поэтому диапазон начинается с `i/2`.

---

## 2. Реализация (рекурсивный бинарный поиск)

```python id="expsearch-recursive"
def binarySearch(arr, l, r, x):
    if r >= l:
        mid = l + (r - l) // 2
        if arr[mid] == x:
            return mid
        if arr[mid] > x:
            return binarySearch(arr, l, mid - 1, x)
        return binarySearch(arr, mid + 1, r, x)
    return -1

def exponentialSearch(arr, n, x):
    if arr[0] == x:
        return 0
    i = 1
    while i < n and arr[i] <= x:
        i *= 2
    return binarySearch(arr, i // 2, min(i, n - 1), x)
```

**Сложность:**

* Время: O(log n)
* Память: O(log n) (рекурсивный стек)

## 3. Реализация (итеративный бинарный поиск)

```python id="expsearch-iterative"
def exponential_search(arr, x):
    n = len(arr)
    if n == 0:
        return -1

    i = 1
    while i < n and arr[i] < x:
        i *= 2

    left = i // 2
    right = min(i, n - 1)

    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == x:
            return mid
        elif arr[mid] < x:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

**Сложность:**
* Время: O(log n)
* Память: O(1)

---

## 4. Применения

* Массивы неизвестного размера или бесконечные массивы
* Быстрый поиск, если элемент находится **близко к началу массива**
* Основа для гибридных алгоритмов поиска с бинарным подходом
