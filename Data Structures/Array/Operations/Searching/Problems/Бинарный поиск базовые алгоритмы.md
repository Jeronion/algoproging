## Классический бинарный поиск

Ищет точное вхождение элемента в отсортированном массиве. Возвращает индекс или -1, если элемент отсутствует.

```python
def binary_search(nums: list[int], target: int) -> int:
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
```

## Поиск левой границы (lower_bound)

Возвращает индекс первого элемента, не меньшего `target`. Если все элементы меньше `target`, возвращает `len(nums)`. Используется для поиска позиции вставки.

```python
def lower_bound(nums: list[int], target: int) -> int:
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] >= target:
            right = mid - 1
        else:
            left = mid + 1
    return left
```

Аналогично, **поиск правой границы** (upper_bound) — индекс первого элемента, большего `target`.

## Поиск ближайшего элемента

Ищет элемент, ближайший по значению к `target`. Задача `solution_2`.

```python
def closest_element(nums: list[int], target: int) -> int:
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    # left — индекс первого >= target
    if left == 0:
        return nums[0]
    if left == len(nums):
        return nums[-1]
    # выбираем ближайший из nums[left-1] и nums[left]
    l_val, r_val = nums[left-1], nums[left]
    return l_val if abs(l_val - target) <= abs(r_val - target) else r_val
```

## Поиск с внешним API (firstBadVersion, guessNumber)

Когда функция сравнения не требует доступа к массиву, а вызывается через API.

```python
def firstBadVersion(n: int) -> int:
    left, right = 1, n
    while left <= right:
        mid = (left + right) // 2
        if isBadVersion(mid):
            right = mid - 1
        else:
            left = mid + 1
    return left

def guessNumber(n: int) -> int:
    left, right = 1, n
    while left <= right:
        mid = (left + right) // 2
        res = guess(mid)
        if res == 0:
            return mid
        elif res == -1:
            right = mid - 1
        else:
            left = mid + 1
```
