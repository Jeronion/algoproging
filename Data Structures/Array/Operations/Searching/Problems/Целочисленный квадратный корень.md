## Определение

Найти целую часть квадратного корня из `x` или проверить, является ли число полным квадратом.

---

## 1. mySqrt (LeetCode 69)

Используем бинарный поиск или метод Ньютона.

```python
# Бинарный поиск
def mySqrt(x: int) -> int:
    left, right = 0, x
    while left <= right:
        mid = (left + right) // 2
        if mid * mid <= x:
            left = mid + 1
        else:
            right = mid - 1
    return right

# Метод Ньютона
def mySqrt_newton(x: int) -> int:
    r = x
    while r * r > x:
        r = (r + x // r) // 2
    return r
```

## 2. Проверка на полный квадрат (LeetCode 367)

```python
# Бинарный поиск
def isPerfectSquare(num: int) -> bool:
    left, right = 0, num
    while left <= right:
        mid = (left + right) // 2
        sq = mid * mid
        if sq == num:
            return True
        elif sq < num:
            left = mid + 1
        else:
            right = mid - 1
    return False

# Метод Ньютона
def isPerfectSquare_newton(num: int) -> bool:
    x = num
    while x * x > num:
        x = (x + num // x) // 2
    return x * x == num
```

---

## Сложность

- Бинарный поиск: O(log x)
- Метод Ньютона: O(log x) с меньшей константой
