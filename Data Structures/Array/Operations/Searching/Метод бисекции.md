
## Определение

**Метод бисекции** (bisection method) — простейший численный метод решения нелинейных уравнений вида $f(x)=0$. Он основан на теореме Больцано — Коши (о промежуточных значениях) и требует только непрерывности функции на заданном отрезке. Является частным случаем [[Бинарный поиск]].

---

## Обоснование

Если непрерывная функция $f(x)$ на концах отрезка $[a, b]$ принимает значения разных знаков ($f(a) \cdot f(b) < 0$), то внутри отрезка существует хотя бы один корень. Метод последовательно делит отрезок пополам и выбирает ту половину, на которой знаки функции на концах различны, сужая интервал локализации корня.

---

## Алгоритм

1. Задаём начальный отрезок $[x_L, x_R]$, на котором $f(x_L) \cdot f(x_R) < 0$ (или знаки различны).
2. Вычисляем середину $x_M = (x_L + x_R)/2$.
3. Если $|f(x_M)|$ меньше заданной точности $\varepsilon_f$ (по оси $y$), то $x_M$ — корень.
4. Иначе проверяем знак $f(x_M)$:
   - Если $f(x_L) \cdot f(x_M) < 0$, корень в левой половине → $x_R = x_M$.
   - Иначе корень в правой половине → $x_L = x_M$.
5. Повторяем шаги 2–4, пока ширина отрезка $(x_R - x_L)$ не станет меньше $\varepsilon_x$ (точность по $x$) или пока $|f(x_M)| \le \varepsilon_f$.

---

## Реализация на Python

```python
def bisection(f, a, b, eps=1e-6):
    if f(a) == 0:
        return a
    if f(b) == 0:
        return b
    while (b - a) > eps:
        c = (a + b) / 2
        if abs(f(c)) < eps:
            return c
        if f(a) * f(c) < 0:
            b = c
        else:
            a = c
    return (a + b) / 2
```

## Пример

Найти корень $x^2 - 6x + 8 = 0$ на отрезке $[0, 10]$ с точностью $10^{-3}$.

- Начальный отрезок: [0, 10], $f(0)=8$, $f(10)=48$ — знаки одинаковы? Нет, здесь нужно другое начальное приближение. Для этого уравнения корни 2 и 4. Возьмём отрезок [0, 3], где $f(0)=8>0$, $f(3)=-1<0$. 
- Итерации:
  1. $c=1.5$, $f(1.5)=1.25>0$ → [1.5,3]
  2. $c=2.25$, $f(2.25)=0.0625>0$ → [2.25,3]
  3. $c=2.625$, $f(2.625)=-0.359<0$ → [2.25,2.625]
  4. $c=2.4375$, $f(2.4375)=-0.152<0$ → [2.25,2.4375]
  5. $c=2.34375$, $f(2.34375)=-0.044<0$ → [2.25,2.34375]
  6. $c=2.296875$, $f(2.296875)=0.009>0$ → [2.296875,2.34375]
  7. $c=2.3203125$, $f(2.3203125)=-0.017<0$ → [2.296875,2.3203125]
  8. $c=2.30859375$, $f(2.30859375)=-0.004<0$ → [2.296875,2.30859375]
  9. $c=2.302734375$, $f(2.302734375)=0.002>0$ → [2.302734375,2.30859375]
- Через 9 итераций длина интервала ≈ 0.0058, середина ≈ 2.30566.

---

## Сложность

- **Скорость сходимости:** линейная. За одну итерацию интервал сокращается в два раза.
- **Число итераций:** $\lceil \log_2\left(\frac{b-a}{\varepsilon}\right) \rceil$.
- **Память:** $O(1)$.

---

## Особенности

- Метод всегда сходится (при выполнении условия смены знака), но медленно.
- Требует изолированного корня и начального отрезка, где функция меняет знак.
- Не требует вычисления производной, устойчив к погрешностям вычислений.
- Для поиска нескольких корней необходимо последовательно применять к разным отрезкам.
