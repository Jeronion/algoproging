
## Определение

**Удаление** элемента означает удаление элемента с определённой позиции массива. После удаления все элементы после этой позиции сдвигаются на одну ячейку влево.

### Шаги удаления:

1. Определяем индекс элемента для удаления.
2. Сдвигаем все элементы после него на одну ячейку влево.
3. Уменьшаем размер массива (если требуется).

### Пример на Python

```python
arr = [10, 20, 30, 40, 50]
pos = 3   # Удаляем 4-й элемент (индекс 3)

arr.pop(pos)

# Вывод: [10, 20, 30, 50]
print("Массив после удаления:", arr)
```

**Сложность:**
- Время: O(n)
- Память: O(1)

---

## Особенности удаления

| Место удаления   | Время |
| ---------------- | ----- |
| Начало массива   | O(n)  |
| Конец массива    | O(1)  |
| Середина массива | O(n)  |

> Динамические массивы управляют памятью автоматически. В статических массивах (C/C++) нужно вручную сдвигать элементы.
