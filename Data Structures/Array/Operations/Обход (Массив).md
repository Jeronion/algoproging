## Виды обхода

### 1. Линейный

```python
arr = [1, 2, 3, 4, 5]

for i in arr:
    print(i, end=" ")
```

**Сложность:**
- Время: O(n)
- Память: O(1)

---

### 2. Обратный

```python
arr = [1, 2, 3, 4, 5]

for i in range(len(arr)-1, -1, -1):
    print(arr[i], end=" ")
```

**Сложность:** 
- Время: O(n)
- Память: O(1)

---

## Методы обхода

### Цикл `for`

```python
arr = [10, 20, 30, 40, 50]

for i in arr:
    print(i, end=' ')
```

### Цикл `while`

```python
arr = [10, 20, 30, 40, 50]
i = 0
n = len(arr)

while i < n:
    print(arr[i], end=" ")
    i += 1
```

**Сложность:**
- Время: O(n)
- Память: O(1)

## Применения

### 1. Поиск элемента

```python
arr = [10, 20, 30, 40, 50]
target = 30

found = False
for i in arr:
    if i == target:
        found = True
        break

print("Найдено" if found else "Не найдено")
```

**Сложность:**
- Время: O(n)
- Память: O(1)

---

### 2. Изменение элементов

```python
arr = [10, 20, 30, 40, 50]

for i in range(len(arr)):
    arr[i] += 5

# Вывод: [15, 25, 35, 45, 55]  
print(arr)
```

**Сложность:**
- Время: O(n)
- Память: O(1)
