
## Определение

Префиксные суммы — это массив накопленных сумм элементов исходного массива:

```
prefixSum[i] = arr[0] + arr[1] + ... + arr[i]
```

Вариант с дополнительным нулём в начале:

```
prefixSum[0] = 0
prefixSum[i] = arr[0] + ... + arr[i - 1]
```

- Удобно для задач с суммами диапазонов
- Позволяет быстро получать суммы подмассивов за O(1)

---

## Основная формула

Сумма подмассива `[l, r]`:

```
sum(l, r) = prefixSum[r + 1] - prefixSum[l]
```

- Нужно заранее построить `prefixSum` за O(n)
- Каждый запрос суммы → O(1)

---

## Применение

- Нужно вычислять **суммы на диапазонах**
- Много **запросов** к массиву
- Баланс слева/справа
- Подмассивы и накопления
- Основа для **Sliding Window**, **Hash + Prefix**, **DP**
