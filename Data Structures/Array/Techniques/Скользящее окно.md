
## Определение

**Скользящее окно** — метод решения задач с подмассивами, подстроками или диапазонами элементов.

- Вместо многократного перебора одних и тех же элементов, поддерживается «окно» фиксированного или переменного размера, которое перемещается по данным.
- Результаты предыдущего окна используются для вычисления результата нового окна.

**Применение:**

- Подмасcивы с заданной суммой
- Самая длинная подстрока с уникальными символами
- Подмассивы/строки фиксированного размера с условиями

---

## Наивный подход — O(n × k) по времени, O(1) по памяти

```python
def maxSum(arr, n, k):
    max_sum = float("-inf")
    for i in range(n - k + 1):
        current_sum = 0
        for j in range(k):
            current_sum += arr[i + j]
        max_sum = max(current_sum, max_sum)
    return max_sum

arr = [5, 2, -1, 0, 3]
k = 3
print(maxSum(arr, len(arr), k))  # 6
```

---

## Техника скользящего окна — O(n) по времени, O(1) по памяти

### Алгоритм

1. Вычисляем сумму первых `k` элементов → `window_sum`
2. Инициализируем `max_sum = window_sum`
3. Линейно сдвигаем окно по массиву:
    
    - вычитаем первый элемент предыдущего окна
    - добавляем последний элемент нового окна
    - обновляем `max_sum`

### Визуальное объяснение

```
arr = [5, 2, -1, 0, 3], k = 3

Шаг 1: [5, 2, -1], сумма = 6 -> max_sum = 6
Шаг 2: [2, -1, 0], сумма = 1 -> max_sum = 6
Шаг 3: [-1, 0, 3], сумма = 2 -> max_sum = 6
```

### Реализация

```python
def maxSum(arr, k):
    n = len(arr)
    if n < k:
        return -1  # Невозможно создать окно
    window_sum = sum(arr[:k])
    max_sum = window_sum
    for i in range(n - k):
        window_sum = window_sum - arr[i] + arr[i + k]
        max_sum = max(max_sum, window_sum)
    return max_sum

arr = [5, 2, -1, 0, 3]
print(maxSum(arr, 3))  # 6
```

## Типы скользящего окна

1. **Фиксированного размера**
    
    - Размер окна известен заранее (`k`)
    - Алгоритм:
        
        1. Считаем первый блок `k` элементов
        2. Сдвигаем окно на 1, обновляем результат

2. **Переменного размера**
    
    - Увеличиваем правый указатель `right`, пока условие выполняется
    - Если условие нарушается → двигаем левый указатель `left`
    - Продолжаем до конца массива

---

## Как распознать задачи на скользящее окно

- Нужно найти **максимальный/минимальный подмассив** или **подстроку** с условием
- Размер подмассива или подстроки может быть задан (`k`)
- Наивное решение с вложенными циклами — O(n²)
- Оптимальное решение с использованием окна — O(n) или O(n log n)
- Ограничения часто: `n ≤ 10^6`

---

## Заключение

- Позволяет уменьшить сложность задач с O(n²) до O(n)
- Эффективно для линейной обработки диапазонов
- Особенно полезно при работе с подмассивами, подстроками и последовательными данными
- Важный навык для алгоритмических задач и собеседований
