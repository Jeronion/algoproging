## Определение

**Counting array** (массив подсчёта) — это простая и эффективная техника для подсчёта частоты встречаемости элементов, когда множество возможных значений невелико и известно заранее. Идея заключается в использовании массива фиксированного размера, где индекс соответствует значению элемента, а значение ячейки — количество его вхождений.

---

## Основная идея

1. Определяем диапазон возможных значений (например, от `min` до `max` или конкретное множество).
2. Создаём массив `count` размером, равным количеству возможных значений (или `max - min + 1`).
3. Проходим по исходным данным и для каждого элемента увеличиваем соответствующую ячейку: `count[element - min] += 1`.
4. Полученный массив содержит частоты элементов.

Этот подход часто используется как первый шаг в алгоритмах, требующих частотного анализа, сортировки подсчётом или построения структур на основе частот.

---

## Пример реализации

```python
def build_counting_array(data, min_val, max_val):
    size = max_val - min_val + 1
    count = [0] * size
    for x in data:
        count[x - min_val] += 1
    return count

# Пример: числа от 0 до 9
data = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]
count = build_counting_array(data, 0, 9)
print(count)  # [0, 2, 1, 2, 1, 3, 1, 0, 0, 1]
```

---
## Когда применять

- Элементы принадлежат **ограниченному и известному диапазону** (например, цифры 0–9, буквы, небольшие целые числа).
- Требуется быстрый подсчёт частот за O(n).
- Необходимо впоследствии обращаться к частотам по значению элемента за O(1).
- Как часть более сложных алгоритмов: сортировка подсчётом, построение палиндрома, поиск моды, частотный анализ.

---

## Сложность

- **Время:** O(n) — один проход по данным.
- **Память:** O(k), где k — размер диапазона значений.
