## Определение

**Индекс равновесия** массива — это такой индекс `i`, что сумма элементов слева от него равна сумме элементов справа. Элемент с индексом `i` не учитывается ни в левой, ни в правой части.

Формально: для массива `arr` длины `n` индекс `i` (0 ≤ i < n) является равновесным, если:

```
arr[0] + arr[1] + ... + arr[i-1] = arr[i+1] + arr[i+2] + ... + arr[n-1]
```

Если `i = 0`, левая сумма считается равной 0. Если `i = n-1`, правая сумма равна 0.

---

## Условие задачи

Дан массив целых чисел. Требуется найти любой индекс равновесия (обычно первый встреченный). Если такого индекса нет, вернуть, например, -1.

---

## Решение с префиксными суммами

### Идея

1. Вычислить общую сумму всех элементов массива `total_sum`.
2. Пройти по массиву слева направо, поддерживая текущую сумму элементов слева от индекса `i` (обозначим `left_sum`).
3. Для каждого индекса `i` правая сумма вычисляется как `total_sum - left_sum - arr[i]`.
4. Если `left_sum == total_sum - left_sum - arr[i]`, то `i` — индекс равновесия.
5. Обновить `left_sum` добавлением `arr[i]` перед переходом к следующему элементу.

Этот способ требует O(1) дополнительной памяти и O(n) времени.

### Реализация на Python

```python
def equilibrium_index(arr):
    total_sum = sum(arr)
    left_sum = 0
    for i, val in enumerate(arr):
        # Правая сумма = total_sum - left_sum - val
        if left_sum == total_sum - left_sum - val:
            return i
        left_sum += val
    return -1
```

### Пример

Для массива `[-7, 1, 5, 2, -4, 3, 0]`:
- `total_sum = 0`
- Проход:
  - i=0: left_sum=0, правая = 0-0-(-7)=7 → не равны.
  - i=1: left_sum=-7, правая = 0-(-7)-1=6 → не равны.
  - i=2: left_sum=-6, правая = 0-(-6)-5=1 → не равны.
  - i=3: left_sum=-1, правая = 0-(-1)-2 = -1 → равны! Возвращаем 3.

---

## Сложность

- **Время:** O(n) — один проход для вычисления префиксных сумм и один проход для проверки.
- **Память:** O(1) для первого метода (без префиксного массива) или O(n) для метода с префиксным массивом.
