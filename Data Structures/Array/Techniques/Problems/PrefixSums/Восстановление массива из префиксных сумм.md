## Постановка задачи

Дан массив `presum`, представляющий префиксные суммы некоторого исходного массива `arr` (той же длины). Требуется восстановить исходный массив.

**Вход:** массив `presum` длины `n`.
**Выход:** массив `arr` длины `n`, для которого `presum[i] = arr[0] + arr[1] + ... + arr[i]`.

---

## Решение

### Идея

Если префиксные суммы определены как накопленные суммы с начала массива, то восстановление тривиально:

- Первый элемент исходного массива равен первой префиксной сумме.
- Каждый следующий элемент равен разности текущей и предыдущей префиксных сумм.

### Реализация на Python

```python
def decode_array(presum):
    n = len(presum)
    arr = [0] * n
    arr[0] = presum[0]                      # первый элемент
    for i in range(1, n):
        arr[i] = presum[i] - presum[i - 1]   # каждый следующий
    return arr
```

### Пример

Пусть `presum = [2, 5, 9, 14]`. Тогда:
- `arr[0] = 2`
- `arr[1] = 5 - 2 = 3`
- `arr[2] = 9 - 5 = 4`
- `arr[3] = 14 - 9 = 5`

Исходный массив: `[2, 3, 4, 5]`. Проверка: префиксные суммы от него действительно дают `[2, 5, 9, 14]`.

---

## Сложность

- **Время:** O(n) — один проход по массиву.
- **Память:** O(n) для хранения результата (не считая входного массива).
