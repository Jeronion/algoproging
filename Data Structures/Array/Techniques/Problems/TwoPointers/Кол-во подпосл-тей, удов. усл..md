## Определение

Дан массив целых чисел и целевое значение `target`. Нужно найти количество подпоследовательностей (не обязательно непрерывных), в которых сумма минимального и максимального элементов ≤ `target`. Вернуть результат по модулю 10⁹+7.

Задача известна как [LeetCode 1498. Number of Subsequences That Satisfy the Given Sum Condition](https://leetcode.com/problems/number-of-subsequences-that-satisfy-the-given-sum-condition/).

---

## Идея решения

Сортируем массив. Для каждого `left` находим наибольший `right` такой, что `nums[left] + nums[right] <= target`. Тогда все подпоследовательности, включающие `nums[left]` и любые элементы из диапазона `[left+1, right]`, удовлетворяют условию. Количество таких подпоследовательностей: `2^(right-left)`. Используем предвычисленные степени двойки по модулю.

---

## Реализация

```python
def numSubseq(nums: List[int], target: int) -> int:
    MOD = 10**9 + 7
    nums.sort()
    n = len(nums)
    power = [1] * n
    for i in range(1, n):
        power[i] = (power[i-1] * 2) % MOD
    
    left, right = 0, n - 1
    result = 0
    while left <= right:
        if nums[left] + nums[right] <= target:
            result = (result + power[right - left]) % MOD
            left += 1
        else:
            right -= 1
    return result
```

---

## Сложность

- **Время:** O(n log n) — сортировка + линейный проход.
- **Память:** O(n) для хранения степеней двойки.
