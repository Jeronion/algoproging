## Определение

Даны два отсортированных по возрастанию массива: `nums1` и `nums2`. Массив `nums1` имеет дополнительную длину `n` в конце, заполненную нулями. Нужно слить `nums2` в `nums1` так, чтобы результат остался отсортированным. Изменения производятся на месте в `nums1`.

Задача известна как [LeetCode 88. Merge Sorted Array](https://leetcode.com/problems/merge-sorted-array/).

---

## Идея решения

Используем два указателя `l` и `r`, указывающие на последние реальные элементы в `nums1` и `nums2`. Заполняем `nums1` с конца, сравнивая текущие максимальные элементы и помещая больший в конец.

---

## Реализация

```python
def merge(nums1: List[int], m: int, nums2: List[int], n: int) -> None:
    l, r = m - 1, n - 1
    while l >= 0 and r >= 0:
        if nums1[l] >= nums2[r]:
            nums1[l + r + 1] = nums1[l]
            l -= 1
        else:
            nums1[l + r + 1] = nums2[r]
            r -= 1
    while r >= 0:
        nums1[l + r + 1] = nums2[r]
        r -= 1
```

---

## Сложность

- **Время:** O(m + n)
- **Память:** O(1)
