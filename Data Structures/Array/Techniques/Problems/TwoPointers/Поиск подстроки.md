## Определение

Даны две строки `haystack` и `needle`. Вернуть индекс первого вхождения `needle` в `haystack`, или `-1`, если `needle` не входит. `needle` может быть пустой строкой (тогда возвращается 0).

Задача известна как [LeetCode 28. Find the Index of the First Occurrence in a String](https://leetcode.com/problems/find-the-index-of-the-first-occurrence-in-a-string/).

---

## Идея решения

Простейший подход — скользящее окно: перебираем все возможные начальные позиции подстроки длины `len(needle)` в `haystack` и сравниваем с `needle`. Для небольших строк этого достаточно.

Более эффективные алгоритмы — КМП или Бойер-Мур, но в данном случае используется наивный.

---

## Реализация

```python
def strStr(haystack: str, needle: str) -> int:
    n = len(needle)
    if n == 0:
        return 0
    for i in range(len(haystack) - n + 1):
        if haystack[i:i + n] == needle:
            return i
    return -1
```

---

## Сложность

- **Время:** O((m-n+1)·n) в худшем случае, где m = len(haystack), n = len(needle).
- **Память:** O(1).
