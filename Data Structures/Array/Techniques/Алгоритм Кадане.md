
## Определение

**Алгоритм Кадане** — это алгоритм динамического программирования для нахождения **максимальной суммы непрерывного подмассива** (подмассива с максимальной суммой) в массиве целых чисел (могут быть и отрицательные). Работает за линейное время O(n) и использует O(1) дополнительной памяти.

---

## Идея

Идея заключается в том, чтобы проходить по массиву слева направо и для каждой позиции `i` вычислять максимальную сумму подмассива, **заканчивающегося в `i`**. Обозначим её `dp[i]`. Тогда:

- `dp[0] = nums[0]`
- `dp[i] = max(nums[i], dp[i-1] + nums[i])`

То есть либо начинаем новый подмассив с `nums[i]`, либо продолжаем предыдущий. Ответом будет максимальное из всех `dp[i]`.

На практике не нужно хранить весь массив `dp`, достаточно одной переменной.

---

## Реализация

```python
def kadane(nums):
    max_current = max_global = nums[0]
    for i in range(1, len(nums)):
        max_current = max(nums[i], max_current + nums[i])
        max_global = max(max_global, max_current)
    return max_global
```

---

## Пример

`nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]`

Проход:
- i=0: current = -2, best = -2
- i=1: current = max(1, -2+1= -1) = 1, best = max(-2,1)=1
- i=2: current = max(-3, 1-3=-2) = -2, best = max(1,-2)=1
- i=3: current = max(4, -2+4=2) = 4, best = max(1,4)=4
- i=4: current = max(-1, 4-1=3) = 3, best = max(4,3)=4
- i=5: current = max(2, 3+2=5) = 5, best = max(4,5)=5
- i=6: current = max(1, 5+1=6) = 6, best = max(5,6)=6
- i=7: current = max(-5, 6-5=1) = 1, best = max(6,1)=6
- i=8: current = max(4, 1+4=5) = 5, best = max(6,5)=6

Ответ: 6 (подмассив [4, -1, 2, 1]).

- **Время:** O(n)
- **Память:** O(1)
