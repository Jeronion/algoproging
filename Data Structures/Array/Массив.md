
## Определение

Массив — это **коллекция элементов одного типа**, которые хранятся в **смежных областях памяти**.

Массивы — одна из самых популярных и простых структур данных в программировании.

---

## Основные термины

- **Элемент массива:** Объекты, хранящиеся в массиве.
- **Индекс массива:** Позиция элемента в массиве, обычно индексация начинается с **0**.

---

## Представление массива в памяти

- Все элементы или ссылки на них хранятся **в смежных областях памяти**.
- Позволяет **быстрый доступ к элементам** и эффективные операции вставки/удаления.

---

## Объявление массива

Примеры на Python:

```python
# Пустой массив
arr = []
# Список целых чисел
arr = [1, 2, 3, 4, 5]
# Список символов (строк)
arr = ['a', 'b', 'c', 'd', 'e']
# Список чисел с плавающей точкой
arr = [1.4, 2.0, 24.0, 5.0, 0.0]
# Список из одинаковых элементов
arr = [0 for _ in range(10)]
```

---

## Применение

Пример:

- Нужно хранить оценки студентов: `[90, 85, 78, 92, 88]`.
- Если студентов станет очень много, **использовать отдельные переменные неудобно**.
- Массив позволяет **структурировать и быстро обрабатывать данные**.

---

## Типы массивов

Массивы классифицируются по двум признакам:

1. По размеру
2. По количеству измерений

### 1. Типы массивов по размеру

1. **Фиксированный размер**
    
    - Размер задаётся при создании.
    - Память выделяется под конкретное количество элементов.

2. **Динамический массив**
    
    - Размер может меняться во время выполнения.
    - Элементы можно добавлять и удалять.

### 2. Типы массивов по измерениям

1. **Одномерный массив (1-D Array)**
    
    - Линейное хранение элементов `[a1, a2, a3, ...]`.

2. **Многомерный массив**
    
    - **2D массив ([[Матрица]]):** массив массивов, таблица `rows x cols`.
    - **3D массив:** массив двумерных массивов.
