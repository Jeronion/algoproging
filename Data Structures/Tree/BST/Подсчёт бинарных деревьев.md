
## Определение

**[[Бинарное дерево поиска (BST)]]** — структура данных, где каждый узел имеет не более двух потомков: левого и правого. Различают **помеченные** (узлы имеют уникальные метки) и **непомеченные** (важна только структура) деревья. Количество различных бинарных деревьев с `n` узлами выражается через **числа Каталана**.

---

## Непомеченные бинарные деревья

Обозначим через `T(n)` количество непомеченных бинарных деревьев с `n` узлами.

### Базовые случаи
- `T(0) = 1` (пустое дерево)
- `T(1) = 1`
- `T(2) = 2`

### Рекуррентная формула
Если один узел является корнем, то оставшиеся `n-1` узлов распределяются между левым и правым поддеревьями. Пусть в левом поддереве `i` узлов, тогда в правом `n-1-i`. Количество комбинаций:

$$
T(n) = \sum_{i=0}^{n-1} T(i) \cdot T(n-1-i)
$$

### Примеры
- `T(3) = T(0)T(2) + T(1)T(1) + T(2)T(0) = 1·2 + 1·1 + 2·1 = 5`
- `T(4) = 1·5 + 1·2 + 2·1 + 5·1 = 14`

Последовательность: 1, 1, 2, 5, 14, 42, 132, … — это **числа Каталана** (`C_n`).

---

## Связь с числами Каталана

Количество непомеченных бинарных деревьев с `n` узлами равно `n`-му числу Каталана:

$$
T(n) = C_n = \frac{1}{n+1} \binom{2n}{n} = \frac{(2n)!}{(n+1)! \, n!}
$$

### Применение к бинарным деревьям поиска (BST)

Количество различных BST с `n` различными ключами также равно $C_n$. При фиксированном порядке ключей выбор корня определяет разбиение, что приводит к той же рекуррентности.

---

## Помеченные бинарные деревья

Для каждого непомеченного дерева можно расставить `n` различных меток `n!` способами. Таким образом, количество помеченных бинарных деревьев:

$$
\text{Помеченных} = C_n \cdot n!
$$

### Пример для n = 3
- Непомеченных: `5`
- Перестановок меток: `3! = 6`
- Всего помеченных: `5 × 6 = 30`

---

## Итоговые формулы

| Тип деревьев                  | Количество для `n` узлов            |
| ----------------------------- | ----------------------------------- |
| Непомеченные бинарные         | $C_n = \frac{1}{n+1} \binom{2n}{n}$ |
| Бинарные деревья поиска (BST) | $C_n$                               |
| Помеченные бинарные           | $C_n \cdot n!$                      |
