
## Определение

**Система непересекающихся множеств (Disjoint Set Union, DSU)** — структура данных, которая хранит коллекцию непересекающихся множеств и поддерживает две основные операции:

- **Find(x)** — найти представителя (корень) множества, содержащего элемент `x`.
- **Union(x, y)** — объединить два множества, содержащие `x` и `y`, в одно.

Используется для эффективного решения задач, связанных с динамическим разбиением на компоненты связности, например, в графах (алгоритм Крускала), проверке принадлежности к одной группе, обнаружении циклов и т.д.

---

## Основные операции

### Инициализация

Каждый элемент изначально является отдельным множеством. Хранится массив `parent`, где `parent[i]` — родитель элемента `i`. Корень множества имеет `parent[root] = root`.

### Find

Возвращает корень множества, к которому принадлежит элемент. Для этого рекурсивно поднимаемся по родителям, пока не достигнем корня.

```python
def find(x):
    if parent[x] != x:
        return find(parent[x])
    return x
```

### Union

Объединяет два множества, устанавливая корень одного как родителя корня другого.

```python
def union(x, y):
    rx = find(x)
    ry = find(y)
    if rx != ry:
        parent[rx] = ry
```

---

## Оптимизации

Наивная реализация может привести к вырожденному дереву (например, при последовательных объединениях), что даёт сложность O(n). Для ускорения применяют две техники:

### 1. Path Compression (сжатие путей)

При выполнении `find` все узлы на пути к корню переподвешиваются непосредственно к корню, что ускоряет последующие вызовы.

```python
def find(x):
    if parent[x] != x:
        parent[x] = find(parent[x])
    return parent[x]
```

### 2. Union by Rank / Size (объединение по рангу/размеру)

При объединении выбирается корень с большим рангом (высотой дерева) или размером, чтобы минимизировать высоту.

```python
def union(x, y):
    rx = find(x)
    ry = find(y)
    if rx == ry:
        return
    if rank[rx] < rank[ry]:
        parent[rx] = ry
    elif rank[rx] > rank[ry]:
        parent[ry] = rx
    else:
        parent[ry] = rx
        rank[rx] += 1
```

Или по размеру:

```python
if size[rx] < size[ry]:
    parent[rx] = ry
    size[ry] += size[rx]
else:
    parent[ry] = rx
    size[rx] += size[ry]
```

---

## Сложность

С обеими оптимизациями амортизированное время выполнения операций составляет **O(α(n))**, где α(n) — обратная функция Аккермана, которая практически не превышает 4 для любых реальных значений n. Это считается почти константным.

- **Память:** O(n) для массивов parent, rank/size.

---

## Полная реализация на Python (с Union by Size и Path Compression)

```python
class DSU:
    def __init__(self, n):
        self.parent = list(range(n))
        self.size = [1] * n

    def find(self, x):
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])
        return self.parent[x]

    def union(self, a, b):
        ra = self.find(a)
        rb = self.find(b)
        if ra == rb:
            return
        if self.size[ra] < self.size[rb]:
            ra, rb = rb, ra
        self.parent[rb] = ra
        self.size[ra] += self.size[rb]
```

---

## Пример использования

```python
dsu = DSU(5)
dsu.union(0, 1)
dsu.union(2, 3)
dsu.union(0, 4)

print(dsu.find(1) == dsu.find(4))  # True (1 и 4 в одном множестве)
print(dsu.find(2) == dsu.find(4))  # False
```
