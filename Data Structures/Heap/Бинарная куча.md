## Определение

**Binary Heap (Бинарная куча)** — структура данных на основе **полного бинарного дерева (Complete Binary Tree)**, которая удовлетворяет **свойству кучи (Heap Property)**. Она является реализацией абстрактного типа данных **Приоритетная очередь**.

### Примечания

- дерево **полностью заполнено по уровням**
- последний уровень заполняется **слева направо**
- корень содержит **минимальный или максимальный элемент**

---

# Представление кучи

Так как куча является **полным бинарным деревом**, её удобно хранить **в [[Массив]]**.

Пусть индекс узла равен `i`, тогда:

| Элемент     | Формула      |
| ----------- | ------------ |
| Parent      | (i - 1) // 2 |
| Left child  | 2 * i + 1    |
| Right child | 2 * i + 2    |
| Root        | 0            |

Это соответствует **обходу дерева в ширину (Level Order)**.

---

## Операции

| Операция                    | Описание                     |
| --------------------------- | ---------------------------- |
| `insert(x)`                 | вставка элемента             |
| `extract_min / extract_max` | удаление корня               |
| `decrease_key`              | уменьшение значения          |
| `delete_key`                | удаление элемента            |
| `heapify`                   | восстановление свойства кучи |

---

## Сложность операций

|Операция|Сложность|
|---|---|
|Insert|O(log n)|
|Extract Min / Max|O(log n)|
|Decrease Key|O(log n)|
|Delete|O(log n)|
|Peek|O(1)|

# Реализация Min Heap (Python)

```python
class MinHeap:

    def __init__(self):
        self.arr = []

    def parent(self, i):
        return (i - 1) // 2

    def left(self, i):
        return 2 * i + 1

    def right(self, i):
        return 2 * i + 2

    def get_min(self):
        return self.arr[0] if self.arr else None

    def insert(self, x):
        self.arr.append(x)
        i = len(self.arr) - 1

        while i > 0 and self.arr[self.parent(i)] > self.arr[i]:
            p = self.parent(i)
            self.arr[i], self.arr[p] = self.arr[p], self.arr[i]
            i = p

    def extract_min(self):
        if len(self.arr) == 0:
            return None

        if len(self.arr) == 1:
            return self.arr.pop()

        root = self.arr[0]
        self.arr[0] = self.arr.pop()

        self.heapify(0)

        return root

    def heapify(self, i):

        smallest = i
        l = self.left(i)
        r = self.right(i)

        if l < len(self.arr) and self.arr[l] < self.arr[smallest]:
            smallest = l

        if r < len(self.arr) and self.arr[r] < self.arr[smallest]:
            smallest = r

        if smallest != i:
            self.arr[i], self.arr[smallest] = self.arr[smallest], self.arr[i]
            self.heapify(smallest)

    def decrease_key(self, i, new_val):
        self.arr[i] = new_val
        
        while i != 0 and self.arr[self.parent(i)] > self.arr[i]:
            p = self.parent(i)
            self.arr[i], self.arr[p] = self.arr[p], self.arr[i]
            i = p
    
    def delete_key(self, i):
        self.decrease_key(i, -float('inf'))
        self.extract_min()
```
