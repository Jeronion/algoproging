
## Определение

**Строка** — это последовательность символов. В большинстве современных языков программирования (Python, Java, JavaScript, C#) строки являются **неизменяемыми** (immutable), то есть после создания их нельзя изменить. В C и C++ строки могут быть изменяемыми (массивы символов) или неизменяемыми (литералы).

---

## Представление в памяти

- **C/C++**: строки — массивы символов, заканчивающиеся нулевым символом `'\0'`.  
  В C++ также есть класс `std::string` (изменяемый).
- **Java/Python/JavaScript/C#**: строки — объекты специальных классов (`String`, `str`), которые неизменяемы.  
- В Python и JavaScript нет отдельного типа char; одиночный символ — это строка длины 1.

## Создание строк в Python

```python
# Одинарные кавычки
s1 = 'Hello'

# Двойные кавычки (удобно, если внутри есть одинарные)
s2 = "I'm a student"

# Тройные кавычки (многострочные)
s3 = '''First line
Second line'''

# Тройные двойные кавычки (аналогично)
s4 = """Another multiline
string"""
```

## Неизменяемость строк

В Python строки нельзя изменить после создания. Попытка присвоить новое значение по индексу вызовет ошибку:

```python
s = "Geeks"
# s[0] = 'g'  # TypeError: 'str' object does not support item assignment
```

Чтобы получить изменённую строку, нужно создать новую.

---

## Основные операции

| Операция               | Описание                                        | Пример (Python)                         |
|------------------------|-------------------------------------------------|------------------------------------------|
| Длина                  | Количество символов                             | `len(s)`                                 |
| Поиск символа          | Найти позицию первого вхождения                 | `s.find('a')`                            |
| Проверка подстроки     | Входит ли подстрока                             | `sub in s`                               |
| Вставка символа        | Создать новую строку с добавленным символом     | `s[:i] + ch + s[i:]`                     |
| Удаление символа       | Создать строку без символа на позиции           | `s[:i] + s[i+1:]`                        |
| Сравнение              | Лексикографическое сравнение                    | `s1 == s2`, `s1 < s2`                    |
| Конкатенация           | Объединение строк                               | `s1 + s2`                                |
| Обращение              | Переворот строки                                | `s[::-1]`                                |
| Поворот                | Циклический сдвиг влево/вправо                  | `s[d:] + s[:d]`                          |
| Проверка палиндрома    | Читается одинаково слева и справа               | `s == s[::-1]`                           |

---

## Примеры кода

```python
# Длина
print(len("Python"))  # 6

# Поиск подстроки
print("world" in "Hello world")  # True

# Конкатенация
print("Hello" + " " + "World")  # Hello World

# Обращение
print("abc"[::-1])  # cba

# Палиндром
def is_palindrome(s):
    return s == s[::-1]

print(is_palindrome("radar"))  # True
```

## Сложность основных операций

| Операция         | Сложность (в Python) |
|------------------|----------------------|
| Доступ по индексу | O(1)                 |
| Длина            | O(1)                 |
| Поиск подстроки  | O(n) (в худшем)      |
| Конкатенация     | O(n + m)             |
| Срез             | O(k) (k — длина среза)|
| Проверка на палиндром | O(n)           |
