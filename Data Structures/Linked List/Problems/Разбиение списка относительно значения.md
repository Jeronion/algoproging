## Определение

Дан связный список и значение `x`. Нужно переставить узлы так, чтобы все узлы со значениями меньше `x` оказались перед узлами со значениями больше или равными `x`. Относительный порядок в каждой группе должен сохраниться.

Задача известна как [LeetCode 86. Partition List](https://leetcode.com/problems/partition-list/).

---

## Постановка задачи

**Вход:** голова списка и `x`.  
**Выход:** новая голова после разбиения.

Пример:
```
head = [1,4,3,2,5,2], x = 3
Результат: [1,2,2,4,3,5]
```

---

## Идея решения

Создаём два фиктивных списка: `left` для узлов `< x`, `right` для узлов `>= x`. Проходим по исходному списку и распределяем узлы по соответствующим спискам (не копируя, а перемещая существующие узлы). В конце соединяем `left` с `right` и обнуляем хвост `right`.

---

## Реализация

```python
def partition(head: Optional[ListNode], x: int) -> Optional[ListNode]:
    left_dummy = ListNode(0)
    right_dummy = ListNode(0)
    left = left_dummy
    right = right_dummy

    while head:
        if head.val < x:
            left.next = head
            left = left.next
        else:
            right.next = head
            right = right.next
        head = head.next

    right.next = None
    left.next = right_dummy.next
    return left_dummy.next
```

---

## Сложность

- **Время:** O(n)
- **Память:** O(1) дополнительно (не считая выходных узлов)

---

## Примечания

- Используется техника «два фиктивных узла» для удобства.
- Необходимо обнулить `right.next`, чтобы избежать цикла.
