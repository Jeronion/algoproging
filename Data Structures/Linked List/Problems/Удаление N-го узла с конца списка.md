## Определение

Дан связный список и целое число `n`. Требуется удалить `n`-й узел с конца списка и вернуть голову изменённого списка.

Задача известна как [LeetCode 19. Remove Nth Node From End of List](https://leetcode.com/problems/remove-nth-node-from-end-of-list/).

---

## Постановка задачи

**Вход:** голова списка и `n` (1 ≤ n ≤ длина списка).  
**Выход:** новая голова после удаления узла.

Пример:
```
head = [1,2,3,4,5], n = 2
Результат: [1,2,3,5]
```

---

## Идея решения

Используется техника двух указателей. Создаём фиктивный узел перед головой, чтобы упростить удаление первого узла. Первый указатель `fast` сдвигаем на `n` шагов вперёд. Затем двигаем оба указателя (`fast` и `slow`), пока `fast` не дойдёт до конца. В этот момент `slow` будет указывать на узел **перед** удаляемым. Выполняем `slow.next = slow.next.next`.

---

## Реализация

```python
def removeNthFromEnd(head: Optional[ListNode], n: int) -> Optional[ListNode]:
    dummy = ListNode(0, head)
    slow = dummy
    fast = head

    # сдвигаем fast на n шагов
    for _ in range(n):
        fast = fast.next

    # двигаем оба, пока fast не станет None
    while fast:
        fast = fast.next
        slow = slow.next

    # удаляем узел
    slow.next = slow.next.next
    return dummy.next
```

---

## Сложность

- **Время:** O(L), где L — длина списка.
- **Память:** O(1)

---

## Примечания

- Фиктивная голова позволяет единообразно обрабатывать удаление первого узла.
- Можно решить за один проход, как показано.
