
## Определение

**Linked List (Связный список)** — линейная структура данных, состоящая из **узлов (nodes)**, где каждый узел содержит:

- данные
- ссылку на следующий узел

Элементы списка **не хранятся в непрерывной области памяти**.

### Примечания

- доступ к элементам осуществляется **последовательно**
- структура легко поддерживает **вставку и удаление**
- используется для реализации других структур данных

Связные списки применяются в:

- **[[Стек]]**
- **[[Очередь]]**
- **[[Дек]]**

## Сравнение: массив vs связный список

|Свойство|Linked List|Array|
|---|---|---|
|Память|непрерывность отсутствует|непрерывная|
|Доступ|O(n)|O(1)|
|Вставка|O(1)|O(n)|
|Удаление|O(1)|O(n)|
|Кэш-эффективность|низкая|высокая|
