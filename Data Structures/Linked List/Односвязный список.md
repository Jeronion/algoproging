
## Определение

**Singly Linked List** — список, где каждый узел содержит **только ссылку на следующий узел**.

Позволяет двигаться только в одну сторону.

---

## Структура узла

```python
class Node:
    def __init__(self, data):
	    __slots__ = ["data", "next"]
        self.data = data
        self.next = None

```
