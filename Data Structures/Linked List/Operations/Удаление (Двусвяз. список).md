
## Определение

**Удаление** из двусвязного списка — операция исключения узла с заданными характеристиками (значение, позиция). При этом необходимо корректно перенаправить ссылки `prev` и `next` соседних узлов, чтобы не нарушить связность списка.

Основные случаи:
- Удаление первого узла (головы)
- Удаление последнего узла
- Удаление узла по значению
- Удаление узла по индексу (позиции)

---

## Структура узла

```python
class Node:
    def __init__(self, data):
        self.data = data
        self.prev = None
        self.next = None
```

---

## 1. Удаление первого узла

Голова перемещается на следующий узел. Если список не пуст, у нового головного узла обнуляем `prev`.

```python
def delete_head(head):
    if head is None:
        return None
    new_head = head.next
    if new_head is not None:
        new_head.prev = None
    return new_head
```

### Сложность:
- **Время:** O(1)

---

## 2. Удаление последнего узла

Находим предпоследний узел (достигаем конца, затем идём на один шаг назад). Обнуляем его `next`. Если в списке один узел, удаляем его, возвращая `None`.

```python
def delete_tail(head):
    if head is None:
        return None
    if head.next is None:
        return None
    current = head
    while current.next:
        current = current.next
    # current — последний узел
    current.prev.next = None
    return head
```

### Сложность:
- **Время:** O(n)

---

## 3. Удаление узла по значению (первое вхождение)

Проходим по списку, отслеживая текущий узел. При совпадении значения корректируем указатели соседей.

```python
def delete_by_value(head, key):
    if head is None:
        return None
    if head.data == key:
        return delete_head(head)
    current = head
    while current and current.data != key:
        current = current.next
    if current is None:
        return head   # элемент не найден
    # Обновляем связи соседей
    if current.prev:
        current.prev.next = current.next
    if current.next:
        current.next.prev = current.prev
    return head
```

### Сложность:
- **Время:** O(n)

---

## 4. Удаление узла по индексу (позиции)

Индексация с 0. Находим узел по индексу, затем корректируем указатели соседей.

```python
def delete_at_position(head, position):
    if head is None:
        return None
    if position == 0:
        return delete_head(head)
    current = head
    for _ in range(position):
        if current is None:
            raise IndexError("Позиция вне диапазона")
        current = current.next
    if current is None:
        raise IndexError("Позиция вне диапазона")
    # current — удаляемый узел
    if current.prev:
        current.prev.next = current.next
    if current.next:
        current.next.prev = current.prev
    return head
```

### Сложность:
- **Время:** O(n)

---

## Сравнение сложности удаления

| Операция           | Сложность |
|--------------------|-----------|
| Первый узел        | O(1)      |
| Последний узел     | O(n)      |
| По значению        | O(n)      |
| По индексу         | O(n)      |

---

## Примечание

В двусвязном списке удаление узла проще, чем в односвязном, так как не требуется запоминать предыдущий узел — доступ к нему есть через `prev`. Однако для поиска узла по значению или индексу всё равно требуется обход.
