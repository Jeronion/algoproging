
## Определение

**Переворот (реверс) связного списка** — операция, изменяющая порядок узлов на противоположный. Первый узел становится последним, а последний — первым.

Для односвязного списка это требует аккуратного перенаправления указателей next. Существуют итеративный и рекурсивный подходы.

---

## Итеративный способ

Используются три указателя:
- `prev` — предыдущий узел (изначально None)
- `current` — текущий узел (начинаем с головы)
- `next_node` — следующий узел, чтобы не потерять связь

На каждом шаге:
1. Сохраняем `current.next` в `next_node`.
2. Перенаправляем `current.next` на `prev`.
3. Сдвигаем `prev` на `current`, `current` на `next_node`.
4. Когда `current` станет `None`, `prev` указывает на новую голову.

```python
def reverse_iterative(head):
    prev = None
    current = head
    while current:
        next_node = current.next
        current.next = prev
        prev = current
        current = next_node
    return prev   # новая голова
```

### Пример

Список: 1 -> 2 -> 3 -> None  
После реверса: 3 -> 2 -> 1 -> None

---

## Рекурсивный способ

Рекурсивно разворачиваем хвост, затем меняем ссылки. Базовый случай: пустой список или один узел.

```python
def reverse_recursive(head):
    if head is None or head.next is None:
        return head
    new_head = reverse_recursive(head.next)
    head.next.next = head
    head.next = None
    return new_head
```

### Объяснение
- Предположим, мы уже развернули список начиная со второго узла.
- Теперь нужно, чтобы второй узел (теперь последний в развёрнутой части) указывал на текущий (head).
- Устанавливаем `head.next.next = head` (это замыкает ссылку с конца).
- Затем `head.next = None`, так как текущий узел становится новым хвостом.

---

## Сложность:
- **Время:** O(n) — каждый узел посещается один раз.
- **Память:**
  - Итеративный: O(1) дополнительной памяти.
  - Рекурсивный: O(n) из-за стека вызовов (глубина рекурсии равна длине списка).
