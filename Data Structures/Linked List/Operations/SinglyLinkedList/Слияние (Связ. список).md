
## Определение

Даны два **отсортированных** по возрастанию односвязных списка. Требуется слить их в один отсортированный список и вернуть его голову. Объединение должно быть выполнено **на месте** (без создания новых узлов), путём перестановки указателей `next`. Оба входных списка могут быть любой длины, включая пустые.

---

## Идея решения

Используем итеративный подход с фиктивной головой (dummy node), чтобы упростить обработку начала результирующего списка. Сравниваем текущие узлы из обоих списков, меньший добавляем в результат, затем сдвигаем указатель в этом списке. Когда один из списков заканчивается, присоединяем оставшийся хвост другого списка.

---

### Алгоритм (итеративный)

1. Создать фиктивный узел `dummy` и указатель `tail = dummy`.
2. Пока оба списка не пусты:
   - Если `l1.val <= l2.val`, то `tail.next = l1`, `l1 = l1.next`
   - Иначе `tail.next = l2`, `l2 = l2.next`
   - `tail = tail.next`
3. Если остались узлы в `l1`, присоединить их: `tail.next = l1`
4. Если остались узлы в `l2`, присоединить их: `tail.next = l2`
5. Вернуть `dummy.next`.

---

## Реализация

```python
class Node:
    def __init__(self, val):
        self.data = val
        self.next = None

def merge_two_lists(l1, l2):
    dummy = Node(0)
    tail = dummy
    while l1 and l2:
        if l1.data <= l2.data:
            tail.next = l1
            l1 = l1.next
        else:
            tail.next = l2
            l2 = l2.next
        tail = tail.next
    tail.next = l1 if l1 else l2
    return dummy.next
```

---

## Сложность

- **Время:** O(n + m), где n, m – длины списков.
- **Память:** O(1)
