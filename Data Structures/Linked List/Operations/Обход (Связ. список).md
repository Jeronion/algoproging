
## Определение

**Обход связного списка** — это процесс посещения каждого узла списка ровно один раз для выполнения некоторой операции (например, вывода значения, поиска элемента, подсчёта количества узлов).

В отличие от массивов, где доступ по индексу происходит за O(1), в связном списке доступ к элементам возможен только последовательно, начиная с головы (head) и следуя по ссылкам next.

---
### Примечания

- **Начало обхода:** всегда с головы списка.
- **Конец обхода:** когда достигнут `None` (последний узел указывает на `None`).
- Обход не изменяет структуру списка.

---

## Реализация обхода

Для обхода односвязного списка используется вспомогательный указатель `current`, который изначально указывает на голову списка. Затем в цикле, пока `current` не станет `None`:
- Выполняется нужное действие с данными текущего узла.
- Указатель перемещается на следующий узел: `current = current.next`.

### Базовый код

```python
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

def traverse(head):
    current = head
    while current:
        print(current.data, end=" -> ")
        current = current.next
    print("None")
```

### Пример использования

```python
# Создадим список: 10 -> 20 -> 30 -> None
head = Node(10)
head.next = Node(20)
head.next.next = Node(30)

traverse(head)  # Вывод: 10 -> 20 -> 30 -> None
```

---

## Сложность:
- **Время:** O(n)
- **Память:** O(1)
