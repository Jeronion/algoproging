
## Определение

**Обход двусвязного списка** — это процесс посещения каждого узла списка ровно один раз. Благодаря наличию двух указателей (`prev` и `next`), обход возможен в двух направлениях: от головы к хвосту (прямой) и от хвоста к голове (обратный).

---

## Структура узла

```python
class Node:
    def __init__(self, data):
        self.data = data
        self.prev = None
        self.next = None
```

---

## Прямой обход (от головы к хвосту)

Начинаем с головы и двигаемся по `next`, пока не достигнем `None`.

```python
def traverse_forward(head):
    current = head
    while current:
        print(current.data, end=" <-> " if current.next else "")
        current = current.next
    print(" -> None")
```

### Пример использования

```python
# Создадим список: 10 <-> 20 <-> 30
head = Node(10)
head.next = Node(20)
head.next.prev = head
head.next.next = Node(30)
head.next.next.prev = head.next

traverse_forward(head)  # 10 <-> 20 <-> 30 -> None
```

### Сложность:
- **Время:** O(n)
- **Память:** O(1)

---

## Обратный обход (от хвоста к голове)

Если у нас есть указатель на хвост (последний узел), можно пройти по `prev` до головы.

```python
def traverse_backward(tail):
    current = tail
    while current:
        print(current.data, end=" <-> " if current.prev else "")
        current = current.prev
    print(" -> None")
```

### Пример использования

```python
# Предположим, у нас есть tail после создания списка
tail = head.next.next  # последний узел
traverse_backward(tail)  # 30 <-> 20 <-> 10 -> None
```

### Сложность:
- **Время:** O(n)
- **Память:** O(1)

---

## Примечания

- **Начало обхода:** в прямом направлении — с головы, в обратном — с хвоста.
- **Конец обхода:** когда достигнут `None` (после последнего узла в любую сторону).
- Обход не изменяет структуру списка.
- Для обратного обхода необходимо иметь ссылку на хвост; если её нет, можно сначала найти хвост прямым обходом, но это увеличит сложность до O(n) дополнительно.
