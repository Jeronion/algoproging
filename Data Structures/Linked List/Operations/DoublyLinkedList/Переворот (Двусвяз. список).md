
## Определение

Дан двусвязный список. Требуется **обратить его** (развернуть) так, чтобы первый узел стал последним, второй — предпоследним и т.д., и вернуть указатель на новую голову. Преобразование должно быть выполнено **на месте** (in‑place).

**Пример:**
```
Исходный: 1 <-> 2 <-> 3
Обращённый: 3 <-> 2 <-> 1
```

---

## Идея решения

В двусвязном списке у каждого узла есть указатели `prev` и `next`. Чтобы обратить список, нужно для каждого узла поменять местами `prev` и `next`, а затем перенаправить голову на последний узел исходного списка (который после обмена станет первым).

Существует два основных подхода:

- **Рекурсивный** — изменяет указатели, продвигаясь вперёд, и возвращает новый головной узел.
- **Итеративный (с двумя указателями)** — обходит список, меняя указатели на месте, без дополнительной памяти.

---

## Рекурсивный подход (O(n) времени, O(n) памяти)

### Алгоритм

1. Базовый случай: если текущий узел `None`, вернуть `None`.
2. Для текущего узла поменять местами `prev` и `next`.
3. Если после обмена `prev` (бывший `next`) стал `None`, значит, текущий узел теперь последний → возвращаем его как новую голову.
4. Иначе рекурсивно вызываем функцию для `curr.prev` (который указывает на следующий узел исходного списка).

### Реализация

```python
class Node:
    def __init__(self, val):
        self.data = val
        self.next = None
        self.prev = None

def reverse_recursive(curr):
    if curr is None:
        return None
    # меняем местами prev и next
    temp = curr.prev
    curr.prev = curr.next
    curr.next = temp
    # если prev стал None, то это новая голова
    if curr.prev is None:
        return curr
    # идём дальше по бывшему next (теперь это prev)
    return reverse_recursive(curr.prev)
```

### Сложность

- **Время:** O(n) — каждый узел обрабатывается один раз.
- **Память:** O(n) — из-за глубины рекурсии (для списка из n узлов).

## Итеративный подход (O(n) времени, O(1) памяти)

### Алгоритм

1. Инициализировать `prevNode = None`, `currNode = head`.
2. Пока `currNode is not None`:
   - Сохранить `prevNode` (текущий `prev`), поменять местами `prev` и `next` у `currNode`.
   - Перейти к следующему узлу исходного списка: `currNode = currNode.prev` (так как после обмена `prev` указывает на бывший `next`).
3. После завершения цикла `prevNode` будет указывать на бывший последний узел (теперь голову). Вернуть `prevNode.prev`, потому что в последней итерации мы обменяли указатели, и новый `prev` стал `None`, а нужная голова — это предыдущий узел.

### Реализация

```python
def reverse_iterative(head):
    if head is None or head.next is None:
        return head
    prevNode = None
    currNode = head
    while currNode is not None:
        prevNode = currNode.prev
        currNode.prev = currNode.next
        currNode.next = prevNode
        currNode = currNode.prev   # переход к следующему
    return prevNode.prev
```

### Сложность

- **Время:** O(n) — один проход по списку.
- **Память:** O(1) — несколько дополнительных указателей.
