
## Определение

Даны два **отсортированных** по возрастанию двусвязных списка. Требуется слить их в один отсортированный двусвязный список и вернуть его голову. Объединение должно быть выполнено **на месте** (без создания новых узлов), путём перестановки указателей `next` и `prev`. Оба входных списка могут быть любой длины, включая пустые.

**Пример:**
```
list1: 1 ↔ 3 ↔ 5 ↔ None
list2: 2 ↔ 4 ↔ 6 ↔ None
Результат: 1 ↔ 2 ↔ 3 ↔ 4 ↔ 5 ↔ 6 ↔ None
```

---

## Отличия от односвязного списка

- При слиянии необходимо также поддерживать корректные ссылки `prev` (на предыдущий узел).
- После объединения результирующий список должен быть корректным двусвязным списком (каждый узел имеет `next` и `prev`).
- Фиктивная голова упрощает код, но не должна быть частью конечного результата.

---

## Идея решения

Используем итеративный подход с фиктивной головой (dummy node). Указатель `tail` будет указывать на последний добавленный узел. При добавлении нового узла:

- Устанавливаем `tail.next = new_node`
- Устанавливаем `new_node.prev = tail`
- Перемещаем `tail = new_node`

После того как один из списков закончится, присоединяем оставшийся хвост, не забывая обновить `prev` для первого присоединяемого узла.

---

### Алгоритм (итеративный)

1. Создать фиктивный узел `dummy` (значение любое, например 0).
2. Установить `tail = dummy`.
3. Пока оба списка не пусты:
   - Если `l1.data <= l2.data`, то выбрать `l1`, иначе `l2`.
   - Выбранный узел `curr` отключаем от его списка (перемещаем указатель соответствующей головы на `curr.next`).
   - Присоединяем `curr` к хвосту: `tail.next = curr`, `curr.prev = tail`, `tail = curr`.
4. Присоединить оставшиеся узлы из `l1` или `l2` (если есть):
   - `tail.next = l_remaining`
   - Если `l_remaining` не `None`, установить `l_remaining.prev = tail`.
5. Обойти присоединённый хвост до конца, чтобы `tail` указывал на последний узел (необязательно, но может пригодиться).
6. Отсоединить фиктивную голову: `result = dummy.next`, и если `result` не `None`, установить `result.prev = None`.
7. Вернуть `result`.

---

## Реализация на Python

```python
class Node:
    def __init__(self, val):
        self.data = val
        self.prev = None
        self.next = None

def merge_two_doubly_lists(l1, l2):
    dummy = Node(0)
    tail = dummy

    while l1 and l2:
        if l1.data <= l2.data:
            curr = l1
            l1 = l1.next
        else:
            curr = l2
            l2 = l2.next

        # Подсоединяем curr к хвосту
        tail.next = curr
        curr.prev = tail
        tail = curr

    # Присоединяем оставшийся хвост (l1 или l2)
    remaining = l1 if l1 else l2
    if remaining:
        tail.next = remaining
        remaining.prev = tail

    # Отсоединяем фиктивный узел
    result = dummy.next
    if result:
        result.prev = None
    return result
```

---

## Сложность

- **Время:** O(n + m), где n и m – длины списков. Каждый узел обрабатывается один раз.
- **Память:** O(1) дополнительно (без учёта рекурсии, так как решение итеративное).
