
## Определение

**Вставка** в двусвязный список — операция добавления нового узла с заданными данными в определённую позицию. Благодаря наличию указателей `prev` и `next`, вставка в двусвязном списке во многих случаях выполняется так же эффективно, как и в односвязном, но требует обновления дополнительных ссылок.

---

## Базовая структура узла

```python
class Node:
    def __init__(self, data):
        self.data = data
        self.prev = None
        self.next = None
```

---

## 1. Вставка в начало (Insert at beginning)

Новый узел становится головой списка. Его `next` указывает на старую голову, а `prev` — на `None`. Если список не пуст, у старой головы обновляем `prev` на новый узел.

```python
def insert_at_beginning(head, data):
    new_node = Node(data)
    new_node.next = head
    if head is not None:
        head.prev = new_node
    return new_node   # новая голова
```

### Сложность:
- **Время:** O(1)

---

## 2. Вставка в конец (Insert at end)

Если список пуст, новый узел становится головой. Иначе нужно пройти до последнего узла, затем присоединить новый узел, обновив `next` текущего хвоста и `prev` нового узла.

```python
def insert_at_end(head, data):
    new_node = Node(data)
    if head is None:
        return new_node
    current = head
    while current.next:
        current = current.next
    current.next = new_node
    new_node.prev = current
    return head
```

### Сложность:
- **Время:** O(n)

---

## 3. Вставка после заданного узла (Insert after a node)

При наличии ссылки на узел `prev_node` (после которого вставляем), новый узел вставляется между `prev_node` и его следующим узлом. Обновляются указатели `next` и `prev` у трёх узлов.

```python
def insert_after(prev_node, data):
    if prev_node is None:
        print("Предыдущий узел не может быть None")
        return
    new_node = Node(data)
    new_node.prev = prev_node
    new_node.next = prev_node.next
    if prev_node.next is not None:
        prev_node.next.prev = new_node
    prev_node.next = new_node
```

### Сложность:
- **Время:** O(1)

---

## 4. Вставка перед заданным узлом (Insert before a node)

Аналогично вставке после, но новый узел вставляется перед `next_node`. Если `next_node` — голова, то новый узел становится новой головой.

```python
def insert_before(next_node, data):
    if next_node is None:
        print("Следующий узел не может быть None")
        return
    new_node = Node(data)
    new_node.prev = next_node.prev
    new_node.next = next_node
    if next_node.prev is not None:
        next_node.prev.next = new_node
    else:
        # new_node становится новой головой
        pass
    next_node.prev = new_node
    return new_node   # может быть новая голова
```

### Сложность:
- **Время:** O(1) (при наличии ссылки на `next_node`)

---

## 5. Вставка на заданную позицию (по индексу)

Индексация с 0 (голова). Если позиция 0 — вставка в начало. Иначе проходим до узла с индексом `pos-1`, после чего выполняем вставку после него.

```python
def insert_at_position(head, data, position):

        nxt.prev = node
```

- **Время:** O(n)

---

## Общая сложность операций вставки

| Операция                 | Сложность |
|--------------------------|-----------|
| В начало                 | O(1)      |
| В конец                  | O(n)      |
| После заданного узла     | O(1)      |
| Перед заданным узлом     | O(1)      |
| По индексу               | O(n)      |

---

## Примечания

- В двусвязном списке вставка после/перед заданным узлом выполняется за O(1) при наличии ссылки на этот узел.
- Вставка по индексу требует обхода списка, что даёт O(n).
- При вставке в начало или перед головой не забудьте обновить указатель `head`.
