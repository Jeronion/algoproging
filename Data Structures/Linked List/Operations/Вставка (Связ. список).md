## Определение

**Вставка** в связный список — операция добавления нового узла с заданными данными в определённую позицию списка. В зависимости от позиции различают:
- Вставку в начало (перед первым узлом)
- Вставку в конец (после последнего узла)
- Вставку в середину (после узла с заданным значением или на заданную позицию)

---

## Базовая структура узла

```python
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None
```

---

## 1. Вставка в начало

Новый узел становится головой списка, его next указывает на старую голову.

```python
def insert_at_beginning(head, data):
    new_node = Node(data)
    new_node.next = head
    return new_node   # новая голова

# Использование:
head = insert_at_beginning(head, 5)
```

### Сложность:
- Время: O(1)

---

## 2. Вставка в конец

Если список пуст, новый узел становится головой. Иначе нужно пройти до последнего узла и присоединить новый узел.

```python
def insert_at_end(head, data):
    new_node = Node(data)
    if head is None:
        return new_node
    current = head
    while current.next:
        current = current.next
    current.next = new_node
    return head
```

### Сложность:
- Время: O(n)

**Требуется обход до конца**

---

## 3. Вставка после заданного узла

Предположим, у нас есть ссылка на предыдущий узел (или его значение). Вставим новый узел сразу после него.

```python
def insert_after(prev_node, data):
    if prev_node is None:
        print("Предыдущий узел не может быть None")
        return
    new_node = Node(data)
    new_node.next = prev_node.next
    prev_node.next = new_node
```

### Сложность:
- Время: O(1) 

**При наличии ссылки на узел.**

---

## 4. Вставка на заданную позицию (по индексу)

Индексация начинается с 0 (голова). Если позиция равна 0 — вставка в начало. Иначе проходим до (pos-1)-го узла и вставляем после него.

```python
def insert_at_position(head, data, position):
    new_node = Node(data)
    if position == 0:
        new_node.next = head
        return new_node
    current = head
    for _ in range(position - 1):
        if current is None:
            raise IndexError("Позиция вне диапазона")
        current = current.next
    new_node.next = current.next
    current.next = new_node
    return head
```

### Сложность:
- Время: O(n)

---

## Общая сложность операций вставки

| Операция           | Сложность |
|--------------------|-----------|
| В начало           | O(1)      |
| В конец            | O(n)      |
| После заданного узла | O(1)   |
| По индексу         | O(n)      |
