
## Определение

**Удаление** из связного списка — операция исключения узла с заданными характеристиками (значение, позиция). При этом необходимо корректно перенаправить ссылки, чтобы не потерять связность списка.

Основные случаи:
- Удаление первого узла (головы)
- Удаление последнего узла
- Удаление узла по значению
- Удаление узла по индексу

---

## Структура узла

```python
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None
```

---

## 1. Удаление первого узла

Голова перемещается на следующий узел. Если список пуст — ничего не делаем.

```python
def delete_head(head):
    if head is None:
        return None
    return head.next   # новая голова
```

### Сложность:
- Время: O(1)

---

## 2. Удаление последнего узла

Требуется найти предпоследний узел и обнулить его next.

```python
def delete_tail(head):
    if head is None or head.next is None:
        return None   # список пуст или из одного узла
    current = head
    while current.next.next:
        current = current.next
    current.next = None
    return head
```

### Сложность:
- Время: O(n)

---

## 3. Удаление узла по значению (первое вхождение)

Проходим по списку, отслеживая предыдущий узел. Если нашли узел с искомым значением, перенаправляем ссылку предыдущего на следующий за удаляемым.

```python
def delete_by_value(head, key):
    if head is None:
        return None
    if head.data == key:
        return head.next   # удаляем голову
    current = head
    while current.next and current.next.data != key:
        current = current.next
    if current.next:   # нашли
        current.next = current.next.next
    return head
```

### Сложность:
- Время: O(n)

---

## 4. Удаление узла по индексу

Индексация с 0. Для удаления узла на позиции i нужно дойти до узла (i-1) и перенаправить его next.

```python
def delete_at_position(head, position):
    if head is None:
        return None
    if position == 0:
        return head.next
    current = head
    for _ in range(position - 1):
        if current is None:
            raise IndexError("Позиция вне диапазона")
        current = current.next
    if current.next is None:
        raise IndexError("Позиция вне диапазона")
    current.next = current.next.next
    return head
```

### Сложность:
- Время: O(n)

---

## Сравнение сложности удаления

| Операция           | Сложность |
|--------------------|-----------|
| Первый узел        | O(1)      |
| Последний узел     | O(n)      |
| По значению        | O(n)      |
| По индексу         | O(n)      |

---

## Примечание для двусвязного списка

В двусвязном списке удаление проще, так как есть ссылка на предыдущий узел. Достаточно скорректировать указатели prev и next у соседей.
