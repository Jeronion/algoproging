
## Определение

**Doubly Linked List** — список, где каждый узел содержит **две ссылки**:

- `prev` — на предыдущий узел
- `next` — на следующий узел

Это позволяет **двигаться в обе стороны**.

---

## Структура узла

```python
class Node:
    def __init__(self, data):
	    __slots__ = ["data", "prev", "next"]
        self.data = data
        self.prev = None
        self.next = None
```
