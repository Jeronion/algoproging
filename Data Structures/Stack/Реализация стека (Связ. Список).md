# Структура реализации

Для реализации используются:

- `Node` — узел списка
- `top` — указатель на вершину стека
- `count` — текущий размер стека

Каждый узел содержит:

- `data` — значение
- `next` — ссылка на следующий узел

Начальное состояние:

```
top = None
```

Это означает **пустой стек**.

---

# Node Structure

```python
class Node:

    def __init__(self, data):
        self.data = data
        self.next = None
```

# Push (добавление)

Добавляет элемент в стек.

### Шаги

1. Создать новый узел
2. Сделать `new_node.next = top`
3. Переместить `top` на новый узел
4. Увеличить размер стека

```python
def push(self, x):

    new_node = Node(x)

    new_node.next = self.top
    self.top = new_node

    self.count += 1
```

---

# Pop (удаление)

Удаляет верхний элемент стека.

### Алгоритм

1. Проверить пуст ли стек
2. Сохранить текущий `top`
3. Переместить `top` на следующий узел
4. Уменьшить размер
5. Вернуть значение

```python
def pop(self):

    if self.top is None:
        print("Stack Underflow")
        return -1

    temp = self.top
    self.top = self.top.next

    self.count -= 1

    return temp.data
```

---

# Peek (верхний элемент)

Возвращает верхний элемент **без удаления**.

```python
def peek(self):

    if self.top is None:
        print("Stack is Empty")
        return -1

    return self.top.data
```

# Проверка пустоты

```python
def isEmpty(self):

```

# Размер стека

```python
def size(self):
```

# Полная реализация

```python
class Node:

    def __init__(self, data):
        self.data = data
        self.next = None


class Stack:

    def __init__(self):
        self.top = None
        self.count = 0


    def push(self, x):

        new_node = Node(x)

        new_node.next = self.top
        self.top = new_node

        self.count += 1


    def pop(self):

        if self.top is None:
            print("Stack Underflow")
            return -1

        temp = self.top
        self.top = self.top.next

        self.count -= 1

        return temp.data


    def peek(self):

        if self.top is None:
            print("Stack is Empty")
            return -1

        return self.top.data


    def isEmpty(self):
        return self.top is None


    def size(self):
        return self.count
```
