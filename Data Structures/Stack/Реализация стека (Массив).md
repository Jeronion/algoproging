
## Структура реализации (Fixed-size stack)

Для реализации стека используется:

- `arr` — массив для хранения элементов
- `capacity` — максимальный размер стека
- `top` — индекс верхнего элемента

Начальное состояние:

```
top = -1
```

Это означает **пустой стек**.

---

# Основные операции

|Операция|Описание|
|---|---|
|`push(x)`|добавление элемента|
|`pop()`|удаление верхнего элемента|
|`peek()`|просмотр верхнего элемента|
|`isEmpty()`|проверка пустоты|
|`isFull()`|проверка переполнения|

---

# Push (добавление)

**Шаги:**

1. Проверить переполнение
2. Увеличить `top`
3. Поместить элемент в `arr[top]`

```python
def push(self, x):

    if self.top == self.capacity - 1:
        print("Stack Overflow")
        return

    self.top += 1
    self.arr[self.top] = x
```

---

# Pop (удаление)

Алгоритм:

1. Проверить пустоту
2. Сохранить `arr[top]`
3. Уменьшить `top`
4. Вернуть значение

```python
def pop(self):

    if self.top == -1:
        print("Stack Underflow")
        return -1

    value = self.arr[self.top]
    self.top -= 1

    return value
```

---

# Peek/Top (верхний элемент)

Возвращает элемент на вершине стека.

```python
def peek(self):

    if self.top == -1:
        print("Stack is Empty")
        return -1

    return self.arr[self.top]
```

---
# Проверка пустоты

```python
def isEmpty(self):
    return self.top == -1
```

---
# Проверка переполнения

```python
def isFull(self):
    return self.top == self.capacity - 1
```

---

# Полная реализация

```python
class MyStack:

    def __init__(self, cap):
        self.capacity = cap
        self.arr = [0] * cap
        self.top = -1


    def push(self, x):

        if self.top == self.capacity - 1:
            print("Stack Overflow")
            return

        self.top += 1
        self.arr[self.top] = x


    def pop(self):

        if self.top == -1:
            print("Stack Underflow")
            return -1

        val = self.arr[self.top]
        self.top -= 1

        return val


    def peek(self):

        if self.top == -1:
            print("Stack Empty")
            return -1

        return self.arr[self.top]


    def isEmpty(self):
        return self.top == -1


    def isFull(self):
        return self.top == self.capacity - 1
```

# Динамический стек (Dynamic Stack)

Фиксированный стек имеет ограничение размера.  
Поэтому на практике используют **динамический массив**.

В Python это обычный `list`.

---

## Реализация

```python
class Stack:

    def __init__(self):
        self.arr = []

    def push(self, x):
        self.arr.append(x)

    def pop(self):
        if not self.arr:
            print("Stack Underflow")
            return -1
        return self.arr.pop()

    def peek(self):
        if not self.arr:
            print("Stack Empty")
            return -1
        return self.arr[-1]

    def isEmpty(self):
        return len(self.arr) == 0

    def size(self):
        return len(self.arr)
```

# Сложность операций

| Операция | Сложность |
| -------- | --------- |
| Push     | O(1)      |
| Pop      | O(1)      |
| Peek     | O(1)      |
| isEmpty  | O(1)      |

# Fixed vs Dynamic Stack

|Свойство|Fixed Stack|Dynamic Stack|
|---|---|---|
|Размер|фиксированный|динамический|
|Overflow|возможен|нет|
|Гибкость|низкая|высокая|
|Реализация|массив|dynamic array|
